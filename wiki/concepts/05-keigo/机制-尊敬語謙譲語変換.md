# 尊敬語謙譲語変換

> 将普通动词变换为尊敬语・谦让语的规则体系。规则变换（お〜になる/お〜する）与特殊动词（いらっしゃる/参る等）两个系统并行，掌握动词对应表和变换判断流程是实用的关键。

## 一、尊敬语的规则变换

**用于没有特殊尊敬形的动词，或代替特殊形**。

### 1.1 お + 連用形 + になる（最礼貌的规则形）

| 普通形 | 尊敬形 | 例句 |
|--------|--------|------|
| 書く | お書き**になる** | 先生がお書きになった |
| 読む | お読み**になる** | 先生がお読みになった |
| 使う | お使い**になる** | 部長がお使いになる |

**注意**：「する」的连用形「し」→「おしになる」不自然，用「なさる」代替。  
「来る」→「いらっしゃる」（特殊形）。

### 1.2 〜れる/られる（通用，礼貌度稍低）

| 普通形 | 尊敬形 |
|--------|--------|
| 書く | 書か**れる** |
| 来る | 来**られる** |
| する | さ**れる** |

---

## 二、尊敬语的特殊动词

| 普通形 | 特殊尊敬語 | 含义 |
|--------|----------|------|
| いる/行く/来る | **いらっしゃる** | 存在/去/来 |
| 言う | **おっしゃる** | 说 |
| 食べる/飲む | **召し上がる** | 吃/喝 |
| 見る | **ご覧になる** | 看 |
| する | **なさる** | 做 |
| くれる | **くださる** | 给（自己） |
| 知っている | **ご存知だ** | 知道 |
| もらう | **いただく**（謙譲）→尊敬=くださる | — |

**いらっしゃる 的活用**（不规则）：

| 活用形 | 形式 | 例 |
|--------|------|-----|
| ます形 | いらっしゃ**います** | 先生がいらっしゃいます |
| テ形 | いらっしゃっ**て** | — |
| 否定 | いらっしゃら**ない** | — |

## 三、谦让语的规则变换

**自己的行为向对方表达谦让**。

### 3.1 お + 連用形 + する

| 普通形 | 谦让形 | 例句 |
|--------|--------|------|
| 書く | お書き**する** | 私がお書きします |
| 持つ | お持ち**する** | 荷物をお持ちします |
| 送る | お送り**する** | 駅までお送りします |

### 3.2 お + 連用形 + いたす（より丁寧）

| 例 |
|-----|
| お書き**いたします** |
| お持ち**いたします** |

**ご + 漢語 + する/いたす**（汉语动词的谦让）：

| 普通形 | 谦让形 |
|--------|--------|
| 説明する | ご説明する/ご説明いたす |
| 連絡する | ご連絡する/ご連絡いたす |
| 案内する | ご案内する/ご案内いたす |

---

## 四、谦让语的特殊动词

| 普通形 | 特殊謙譲語 | 含义 |
|--------|----------|------|
| 行く/来る | **参る（まいる）** | 去/来（自己） |
| 言う | **申す（もうす）** | 说（自己） |
| いる | **おる** | 存在（自己） |
| 食べる/飲む | **いただく** | 吃/喝（接受时） |
| もらう | **いただく** | 接受（来自对方） |
| 見る | **拝見する（はいけんする）** | 看（自己） |
| する | **いたす** | 做（自己） |
| 知る/知っている | **存じる（ぞんじる）** | 知道（自己） |
| 会う | **お目にかかる** | 见面（自己） |
| もらう/食べる（許可を受けて）| **いただく/させていただく** | 承蒙对方允许 |

## 五、普通→尊敬→谦让 三段对应表

| 普通形 | 尊敬語（対方の動作） | 謙譲語（自分の動作） |
|--------|---------------|-------------|
| いる | いらっしゃる | おる |
| 行く | いらっしゃる/おいでになる | 参る |
| 来る | いらっしゃる/おいでになる | 参る/伺う |
| する | なさる | いたす |
| 言う | おっしゃる | 申す |
| 食べる/飲む | 召し上がる | いただく |
| 見る | ご覧になる | 拝見する |
| 知っている | ご存知だ | 存じる |
| もらう | — | いただく |
| くれる | くださる | — |
| あげる | — | 差し上げる |
| 聞く | — | 伺う/拝聴する |
| 訪問する | — | 伺う |
| もらう（許可） | — | させていただく |

## 六、变换判断流程

```
敬語変換の判断フロー
  └─ 誰の動作か？
       └─ 話し相手・目上の人の動作 → 尊敬語
       │   └─ 特殊尊敬語あり？ → 使う（いらっしゃる/おっしゃる等）
       │   └─ 特殊形なし → お〜になる（または〜れる/られる）
       └─ 自分・自分側の動作で相手に敬意を示す → 謙譲語I
           └─ 特殊謙譲語あり？ → 使う（参る/申す/いただく等）
           └─ 特殊形なし → お〜する/ご〜する（または〜いたす）
```

> **実践例**：「部長が来る（来週の会議に）」  
> → 部長の動作 → 尊敬語 → 「来る」の特殊尊敬語＝いらっしゃる  
> →「部長がいらっしゃいます」
