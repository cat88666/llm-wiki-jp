## 一、三种基本句型

| 类型 | 句型 | 例句 |
| --- | --- | --- |
| 名词句 | A は B です | 私(わたし)は学生(がくせい)です。（我是学生。） |
| 形容词句 | A は 形容词です | この部屋(へや)は静(しず)かです。（这个房间很安静。） |
| 动词句 | A は B を Vます | 私(わたし)は本(ほん)を読(よ)みます。（我读书。） |

**为什么分三类？** 因为句末谓语决定了整句的变形方式——名词和ナ形容词靠 `です` 变化，イ形容词靠词尾 `い` 变化，动词靠 `ます` 变化。三套规则互不通用。

### 1.1 基本助词

中文靠语序区分"我吃饭"和"饭吃我"，日语靠助词给每个词贴角色标签，所以助词不能直译成某个中文字。

→ 完整体系详见 [[格助词体系]]

#### は — 主题标记

写作「は」但读 wa（历史遗留，只有做助词时如此）。

| 用法 | 例 |
| --- | --- |
| 提出话题："至于 A" | 私(わたし)は学生(がくせい)です。（我是学生。） |
| 对比：两个事物分别怎样 | 肉(にく)は好(す)きですが、魚(さかな)は嫌(きら)いです。（肉喜欢，但鱼讨厌。） |
| 强调否定：特意指出"这个不" | お酒(さけ)は飲(の)みません。（酒（我）不喝。） |

**は vs が 的核心区别**：`は` 标记"已知话题，下面要说的重点在后半句"；`が` 标记"新信息，重点在 が 前面那个词"。

> **誰(だれ)が**来(き)ましたか。（**谁**来了？）— 不知道谁来了，问的就是 `が` 前面的人  
> 田中(たなか)さん**が**来(き)ました。（田中来了。）— "田中"是新信息，用 `が`  
> 田中(たなか)さん**は**もう帰(かえ)りました。（田中已经回去了。）— 田中已经是话题，重点在"回去了"

#### が — 主语/新信息/对象

| 用法 | 例 |
| --- | --- |
| 存在句主语 | 机(つくえ)の上(うえ)に本(ほん)があります。（桌上有书。） |
| 新信息（回答"谁/什么"） | 田中(たなか)さんが来(き)ました。（田中来了。） |
| 好恶・能力・需要的对象 | 日本語(にほんご)が好(す)きです。（喜欢日语。） |
|  | 英語(えいご)が分(わ)かります。（懂英语。） |
|  | 水(みず)が欲(ほ)しいです。（想要水。） |
| 从属节里的主语 | 雨(あめ)が降(ふ)っているので、傘(かさ)を持(も)っていきます。（因为在下雨，带伞去。） |

**为什么好き・分かる・欲しい用 `が` 不用 `を`？** 因为这些词表达的不是"主动去做"的动作，而是"自然感受到"的状态——喜欢、理解、想要都是感受，不是你对对象施加的动作。日语里感受的对象用 `が`，动作的对象用 `を`。

#### を — 动作对象/经过・离开

写作「を」但读 o（历史遗留，古代读 wo）。

| 用法 | 例 |
| --- | --- |
| 动作对象 | 本(ほん)を読(よ)みます。（读书。） |
| 经过的场所（通过/走过） | 公園(こうえん)を歩(ある)きます。（在公园走。） |
| 离开的场所（出发/离开） | 家(いえ)を出(で)ます。（出门。） |
|  | 大学(だいがく)を卒業(そつぎょう)します。（从大学毕业。） |

**为什么"走公园""出家门"也用 `を`？** `を` 的本质是"动作经过/作用的对象"。走路时公园是你"经过"的空间，出门时家是你"离开"的起点——都是动作作用的对象，所以用 `を`。

#### に — 时间/目的地/存在地/对象/目的

`に` 的核心是"指向一个点"——时间轴上的一个刻度、空间中的一个位置、动作指向的一个对象。

| 用法 | 例 |
| --- | --- |
| 时间点 | 7時(しちじ)に起(お)きます。（7点起床。） |
| 移动目的地 | 学校(がっこう)に行(い)きます。（去学校。） |
| 存在地点 | 教室(きょうしつ)に学生(がくせい)がいます。（教室有学生。） |
| 动作对象（给予/告知的方向） | 友達(ともだち)に本(ほん)をあげます。（给朋友书。） |
|  | 先生(せんせい)に聞(き)きます。（问老师。） |
| 目的（V ます形去掉ます + に行く） | 買(か)い物(もの)に行(い)きます。（去买东西。） |
| 频率基准 | 一日(いちにち)に三回(さんかい)飲(の)みます。（一天喝三次。） |

**今日(きょう)・明日(あした)・毎日(まいにち) 不加 `に`**：`に` 标记的是"坐标刻度"（几点、星期几、几号），而"今天、明天、每天"本身已经定位了时间，不需要再标。

#### で — 动作地点/手段/材料/原因/范围

`で` 的核心是"动作展开的条件"——在哪里做、用什么做、因为什么做。

| 用法 | 例 |
| --- | --- |
| 动作地点 | 図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。） |
| 手段/工具 | バスで行(い)きます。（坐公交去。） |
|  | 日本語(にほんご)で話(はな)します。（用日语说。） |
| 材料 | 木(き)で作(つく)ります。（用木头做。） |
| 原因 | 病気(びょうき)で休(やす)みます。（因为生病休息。） |
| 范围限定 | 世界(せかい)で一番(いちばん)大(おお)きい。（世界上最大。） |
| 数量/人数限定 | 三人(さんにん)で行(い)きます。（三个人去。） |

**に vs で 判断法**：`に` = 静态的"点"（在哪里存在、往哪里去）；`で` = 动态的"场/条件"（在哪里做事、用什么做事）。如果地点后面跟的是动作动词 → `で`；跟的是 いる/ある 或移动终点 → `に`。

#### 其他常用助词（速查）

| 助词 | 功能 | 例 |
| --- | --- | --- |
| も | 也 | 私(わたし)も学生(がくせい)です。（我也是学生。） |
| と | 和（并列）/ 一起 / 引用 | 友達(ともだち)と行(い)きます。（和朋友去。） |
| の | 的（所属/修饰） | 私(わたし)の本(ほん)（我的书） |
| から | 从（起点）/ 因为 | 9時(くじ)から始(はじ)まります。（从9点开始。） |
| まで | 到（终点） | 5時(ごじ)まで働(はたら)きます。（工作到5点。） |
| へ | 朝某方向（读 e） | 東京(とうきょう)へ行(い)きます。（往东京去。） |

**へ vs に（目的地）**：`に` 强调到达的具体地点，`へ` 强调朝某个方向移动。日常口语中大多数情况可互换，`に` 更常用。

**句末决定一切**：日语谓语放最后，句末的 `です / ます / ない / た` 决定礼貌度、时态、肯否定。中文"我**不**去"第二个字就出结论，日语 `行(い)きません` 要听到末尾 `ません` 才知道是否定。

## 二、名词句：A は B です

### 2.1 肯定：A は B です

> 私(わたし)は学生(がくせい)です。（我是学生。）

| 词 | 作用 |
| --- | --- |
| 私(わたし) | 我 |
| は | 主题助词。写作「は」但读 wa——历史遗留：古代助词 は 确实读 ha，后来发音演变为 wa，写法没改。同理助词 `へ` 读 e。只有做助词时如此，普通假名仍读 ha |
| 学生(がくせい) | 学生 |
| です | 礼貌判断助动词，相当于"是" |

`A は B です` 整体理解为"至于 A，是 B"，不要逐字对应中文。

### 2.2 否定：A は B ではありません

> 田中(たなか)さんは先生(せんせい)ではありません。（田中不是老师。）

| 表达 | 说明 |
| --- | --- |
| ではありません | `です` 的礼貌否定 |
| じゃありません | 口语缩略，常见 |
| じゃない | 普通体，随意场合 |

**为什么否定形这么长？** `です` 的底层结构是 `で＋ある`（ある = 存在/是），礼貌否定就是 `で（は）あり＋ません`。口语嫌长，把 `では` 缩成 `じゃ` → `じゃありません`，再简化 → `じゃない`。

### 2.3 疑问：A は B ですか

> これは本(ほん)ですか。（这是书吗？）

| 词 | 作用 |
| --- | --- |
| これ | 这个 |
| は | 主题 |
| 本(ほん) | 书 |
| ですか | 是……吗？ |

**为什么加 `か` 就变问句？** 日语礼貌体不靠语调升降来表示疑问，而是用 `か` 做显式的疑问标记，贴在句末即可，语序不变。

## 三、形容词句

日语形容词分 **イ形容词** 和 **ナ形容词** 两类，变形规则完全不同。

**为什么分两类？** イ形容词是日语固有词，词尾 `い` 本身能活用（くない、かった……）；ナ形容词多来自汉语或外来语（静(しず)か、きれい），词干不能自己变，必须借 `だ / です / な` 连接。来源不同导致变形机制不同。

### 3.1 イ形容词

> この本(ほん)はおもしろいです。（这本书很有趣。）

| 词 | 作用 |
| --- | --- |
| この | 这个（后接名词） |
| 本(ほん) | 书 |
| は | 主题 |
| おもしろい | 有趣（イ形容词） |
| です | 礼貌结尾 |

**变形规则**：去掉词尾 `い`，换上对应后缀。`い` 是イ形容词专用的活用词尾，所有变形都在这个位置发生。

| 时态 | 规则 | 例 | 中文 |
| --- | --- | --- | --- |
| 现在肯定 | 原形 + です | おもしろいです | 有趣 |
| 现在否定 | い → くないです | おもしろくないです | 不有趣 |
| 过去肯定 | い → かったです | おもしろかったです | 以前有趣 |
| 过去否定 | い → くなかったです | おもしろくなかったです | 以前不有趣 |

### 3.2 ナ形容词

> この部屋(へや)は静(しず)かです。（这个房间很安静。）

| 词 | 作用 |
| --- | --- |
| この | 这个 |
| 部屋(へや) | 房间 |
| は | 主题 |
| 静(しず)か | 安静（ナ形容词） |
| です | 礼貌结尾 |

**变形规则**：ナ形容词词干不变，全靠 `です` 部分变化——和名词句变形完全一样。掌握名词句变形，ナ形容词自动通用。

| 时态 | 例 | 中文 |
| --- | --- | --- |
| 现在肯定 | 静(しず)かです | 安静 |
| 现在否定 | 静(しず)かではありません / 静(しず)かじゃないです | 不安静 |
| 过去肯定 | 静(しず)かでした | 以前安静 |
| 过去否定 | 静(しず)かではありませんでした / 静(しず)かじゃなかったです | 以前不安静 |

### 3.3 イ形容词 vs ナ形容词 对比

| 对比 | イ形容词 | ナ形容词 |
| --- | --- | --- |
| 结尾特征 | 以 い 结尾 | 不靠 い 判断，多为汉语词 |
| 修饰名词 | おもしろい本(ほん)（有趣的书） | 静(しず)か**な**部屋(へや)（安静的房间） |
| 否定 | おもしろくない（不有趣） | 静(しず)かじゃない（不安静） |
| 过去 | おもしろかった（以前有趣） | 静(しず)かだった（以前安静） |

**为什么修饰名词时ナ形容词要加 `な`？** イ形容词的 `い` 本身能连接名词（おもしろい→おもしろい本），ナ形容词词干不能直接接名词，需要 `な` 做桥梁（静か→静か**な**部屋）。`な` 就是ナ形容词名字的由来。

**反直觉**：`きれい` 以 `い` 结尾，但它是**ナ形容词**。因为 `きれい` 的 `い` 是词干的一部分（漢字：綺麗(きれい)），不能拆成 `きれ + い` 来活用。  
正确：きれい**な**人(ひと)（漂亮的人）、きれいじゃない（不漂亮）  
错误：~~きれい人~~、~~きれくない~~  
同类词：嫌(きら)い（讨厌）也是ナ形容词。判断法：去掉 `い` 后剩余部分不成词根 → ナ形容词。

## 四、动词句

### 4.1 基本句型：A は B を Vます

> 私(わたし)は本(ほん)を読(よ)みます。（我读书。）

| 词 | 作用 |
| --- | --- |
| 私(わたし) | 我 |
| は | 主题 |
| 本(ほん) | 书 |
| を | 宾语助词。写作「を」读 o——和 `は` 读 wa 一样是历史遗留，古代读 wo，现代已变为 o。`を` 现在只做宾语助词，看到它就知道前面的词是动作对象 |
| 読(よ)みます | 读（礼貌体） |

```text
私は   本を   読みます。
主题    对象    动词
```

### 4.2 动作地点：場所(ばしょ)で V ます

> 図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。）

**に vs で**：都可以接地点，但含义不同。

| 助词 | 用于 | 例 |
| --- | --- | --- |
| に | 静止存在的地点 / 移动的目的地 | 学校(がっこう)に行(い)きます。（去学校。） |
| で | 动作发生的场所 / 工具手段 | 学校(がっこう)で勉強(べんきょう)します。（在学校学习。） |

**为什么同样是"地点"却用不同助词？** `に` 标记"存在位置"或"到达终点"，强调静态的点；`で` 标记"动作展开的舞台"，强调有事在发生。简记：`に` = 在哪里待着 / 往哪里去，`で` = 在哪里干活。

### 4.3 时间点：時間(じかん) + に

> 7時(しちじ)に起(お)きます。（7点起床。）

具体刻度（几点、星期几、几号）加 `に`：7時(しちじ)に、月曜日(げつようび)に。

今日(きょう)、明日(あした)、昨日(きのう)、毎日(まいにち) **不加** `に`。

**为什么？** `に` 标记的是"时间坐标轴上的一个刻度"。而"今天、明天、每天"这些词本身已经定位了时间，不需要再标记。类似的还有 `今(いま)`（现在）也不加 `に`。

## 五、时态变化总表

日语礼貌体只有两个时态（现在/将来、过去）× 两个极性（肯定、否定）= 四种变化。

| 谓语类型 | 现在肯定 | 现在否定 | 过去肯定 | 过去否定 |
| --- | --- | --- | --- | --- |
| 名词 | 学生(がくせい)です（是学生） | 学生(がくせい)ではありません（不是学生） | 学生(がくせい)でした（以前是学生） | 学生(がくせい)ではありませんでした（以前不是学生） |
| イ形容词 | 高(たか)いです（贵） | 高(たか)くないです（不贵） | 高(たか)かったです（以前贵） | 高(たか)くなかったです（以前不贵） |
| ナ形容词 | 静(しず)かです（安静） | 静(しず)かではありません（不安静） | 静(しず)かでした（以前安静） | 静(しず)かではありませんでした（以前不安静） |
| 动词 | 行(い)きます（去） | 行(い)きません（不去） | 行(い)きました（去了） | 行(い)きませんでした（没去） |

**动词礼貌体变形最规律**：`ます`（肯定）→ `ません`（否定）→ `ました`（过去）→ `ませんでした`（过去否定）。`ます` 前面的部分（連用形）始终不变，所有变化集中在 `ます` 这个词尾上。

### 5.1 疑问句

任何陈述句末尾加 `か` 即变为疑问句。

> 明日(あした)、学校(がっこう)に行(い)きますか。（明天去学校吗？）

回答：

| 回答 | 中文 |
| --- | --- |
| はい、行(い)きます。 | 是的，去。 |
| いいえ、行(い)きません。 | 不，不去。 |

## 六、存在句

→ 详见 [[存在表达]]

### 6.1 あります（无生命物体、植物）

> 机(つくえ)の上(うえ)に本(ほん)があります。（桌上有书。）

| 词 | 作用 |
| --- | --- |
| 机(つくえ) | 桌子 |
| の | 的 |
| 上(うえ) | 上面 |
| に | 存在地点 |
| 本(ほん) | 书 |
| が | 标出存在物 |
| あります | 有/在（无生命物） |

### 6.2 います（人、动物）

> 教室(きょうしつ)に学生(がくせい)がいます。（教室里有学生。）

**あります vs います**：

| 动词 | 对象 | 例 |
| --- | --- | --- |
| あります | 无生命物：书、桌子、钱、时间、植物 | 本(ほん)があります。（有书。） |
| います | 有生命且能自主移动：人、猫、狗 | 猫(ねこ)がいます。（有猫。） |

**为什么"有"分两个词？** 日语按"能否自主移动"区分。人和动物会自己动 → `います`；书、桌子不会自己动 → `あります`。

**植物用 `あります`**：虽然植物有生命，但不会自主移动。日语的 `います` 看重的是"自主移动能力"，不是生物学上的"生命"。

## 七、语序与助词

### 7.1 助词标角色，语序可调

```text
私(わたし)は ご飯(はん)を 食(た)べます。（我吃饭。）
  主题         对象        动词
```

因为助词标记了每个词的角色，`ご飯(はん)を 私(わたし)は 食(た)べます` 语序变了但意思不变——`を` 标着"饭是对象"，`は` 标着"我是主题"。

**铁律：动词永远放最后。**

### 7.2 基本语序

```text
[时间] [主题] [地点] [对象] [动词]
```

> 私(わたし)は毎日(まいにち)、家(いえ)で日本語(にほんご)を勉強(べんきょう)します。（我每天在家学习日语。）

| 词 | 作用 |
| --- | --- |
| 私(わたし)は | 主题 |
| 毎日(まいにち) | 时间 |
| 家(いえ)で | 动作地点 |
| 日本語(にほんご)を | 学习对象 |
| 勉強(べんきょう)します | 动词 |

### 7.3 常见错误

| 错误 | 原因 | 正确 |
| --- | --- | --- |
| 私(わたし)は学生(がくせい)。 | 缺 `です`，礼貌体必须有判断词 | 私(わたし)は学生(がくせい)です。（我是学生。） |
| 図書館(としょかん)**に**勉強(べんきょう)します。 | 动作地点用 `で`，不用 `に` | 図書館(としょかん)**で**勉強(べんきょう)します。（在图书馆学习。） |
| 本(ほん)読(よ)む。 | 缺宾语助词 `を` | 本(ほん)**を**読(よ)みます。（读书。） |

## 八、终助词

终助词放句末，标记语气。

| 助词 | 功能 | 例 |
| --- | --- | --- |
| か | 疑问 | これは本(ほん)ですか。（这是书吗？） |
| ね | 确认/共鸣 | 今日(きょう)は暑(あつ)いですね。（今天很热呢。） |
| よ | 告知/提醒 | これは大切(たいせつ)ですよ。（这个很重要哦。） |

**为什么需要终助词？** 中文靠语调和语气词（"吧""啊""呢"）传递态度，日语也类似，但更依赖句末显式标记：`ね` = 我觉得你也同意；`よ` = 你可能不知道，我来告诉你；`か` = 我在提问。

### 8.1 ね

> 今日(きょう)は寒(さむ)いですね。（今天很冷呢。）

`ね` 表示"你应该也有同感吧？"，用来建立共鸣。不加 `ね` 的 `今日(きょう)は寒(さむ)いです` 只是在陈述事实。

### 8.2 よ

> 明日(あした)は休(やす)みですよ。（明天休息哦。）

`よ` 表示"这件事你可能不知道，我告诉你"。**注意**：对上司或不熟的人频繁用 `よ` 会显得强硬，因为底层含义是"我知道你不知道"，有居高临下的感觉。对朋友自然，对长辈慎用。
