# は vs が

> 「は」は主題（何について話すか）を示し、「が」は主語（誰/何が動作するか）を示す。この区別は日本語文法の最難関の一つであり、中国語・英語にはない概念体系。

## 一、主題と主語：根本的に違う概念

```
は = 主題マーカー（Topic）
  「〜については」「〜の話をすると」
  → 旧情報（聞き手が既知のもの）を取り上げる

が = 主語マーカー（Subject）
  「〜が（こそが）」
  → 新情報（聞き手にとって未知）を提示する
```

> **核心区別**：
> - 「田中さん**は**学生です」= 田中さんの話をすると → 学生です（田中さんは既知）
> - 「田中さん**が**学生です」= （誰が学生？）→ 田中さんです（田中さんが新情報）

## 二、五つの判別規則

| # | 規則 | は | が | 例文 |
|---|------|---|---|------|
| 1 | **旧情報 vs 新情報** | 既に話題の人/物 | 初めて導入する人/物 | 昔々、おじいさん**が**いました。おじいさん**は**山へ行きました。 |
| 2 | **対比** | 対比の「は」 | — | コーヒー**は**飲みますが、紅茶**は**飲みません。 |
| 3 | **排他（総記）** | — | 「他でもなく、これが」 | 誰**が**来ましたか？ — 田中さん**が**来ました。 |
| 4 | **従属節の中** | 使えない | 従属節内では が | [私**が**書いた]本（×私は書いた本） |
| 5 | **存在文の導入** | — | 新出の存在物 | あそこに猫**が**います。 |

### 2.1 旧情報 vs 新情報（最重要規則）

物語の展開で最も分かりやすい：

> むかしむかし、ある所におじいさん**が**いました。← 初登場 = 新情報 = が
> おじいさん**は**毎日山へ柴刈りに行きました。← 既出 = 旧情報 = は
> ある日、大きな桃**が**流れてきました。← 初登場 = 新情報 = が

### 2.2 対比の「は」

二つ以上の項目を対比する時、「は」を使う。「が」では対比のニュアンスが出ない。

> 肉**は**食べますが、魚**は**食べません。
> 日本語**は**難しいですが、面白いです。

### 2.3 排他の「が」

「他の誰でもなく、これが」という焦点を当てる時、「が」を使う。

> A：誰**が**犯人ですか？
> B：田中さん**が**犯人です。（他でもなく田中さん）

## 三、典型場面での使い分け

| 場面 | 使用 | 例文 | 理由 |
|------|------|------|------|
| 自己紹介 | は | 私**は**田中です。 | 自分を主題として提示 |
| 好き/嫌い | が | 猫**が**好きです。 | 感情の対象は が で標示 |
| 能力 | が | 日本語**が**話せます。 | 能力の対象は が で標示 |
| 天候 | が | 雨**が**降っています。 | 自然現象は主語を が で導入 |
| 疑問詞+答え | が | 何**が**ありますか？ | 疑問詞は常に が |
| 描写 | は | この部屋**は**広いです。 | 既知の対象について描写 |

**反直觉：感情・能力の対象は が**

中文说"我喜欢猫"，"猫"是宾语。但日语「猫**が**好きです」中，「猫」用 が 標示。这是因为「好き」「嫌い」「できる」「わかる」「ほしい」等词的语法结构不是"主语→宾语→动词"，而是"主题は＋対象が＋状態"。

## 四、「象は鼻が長い」問題

この文は日本語学の有名な難題。は と が が同時に出現する。

```
象は     鼻が     長い
↓        ↓        ↓
主題     主語      述語
(象について言えば)  (鼻が)  (長い)
```

> **解析**：「象」は主題（これから象について話す）、「鼻」は主語（長いのは鼻）。「は」と「が」は別の文法層で機能するため共存できる。

同じ構造の文：
- 日本**は**物価**が**高い。（日本について→物価が→高い）
- 私**は**頭**が**痛い。（私について→頭が→痛い）
- この店**は**ラーメン**が**おいしい。（この店について→ラーメンが→おいしい）

## 五、実践決策樹

```
助詞を選ぶ
  └─ 従属節の中（〜が…した + 名詞）？
       └─ Yes → が（従属節内では は 使用不可）
  └─ 疑問詞（誰/何/どれ）が含まれる？
       └─ Yes → が（疑問詞は常に が）
  └─ 対比している（A は…、B は…）？
       └─ Yes → は
  └─ その名詞は初登場（新情報）？
       └─ Yes → が
       └─ No（既出/既知） → は
  └─ 感情・能力の対象（好き/できる/わかる）？
       └─ Yes → が
  └─ 迷ったら → は（主題として提示するのが安全）
```

> **実践のコツ**：初級段階では「迷ったら は」が安全。は は広く使えるが、が は特定の条件でしか使えない。ただし、新情報の導入と疑問詞文では必ず が。
