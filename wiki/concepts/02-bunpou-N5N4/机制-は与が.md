# は与が

> 「は」标记主题（在谈论什么），「が」标记主语（谁/什么在做动作）。这个区别是日语语法最难关之一，中文・英文中不存在这套概念体系。

## 一、主题与主语：根本不同的概念

```
は = 主题标记（Topic）
  「关于〜」「说到〜」
  → 拿出旧信息（听者已知的事物）

が = 主语标记（Subject）
  「〜才是」「正是〜」
  → 提示新信息（对听者来说未知的事物）
```

> **核心区别**：
> - 「田中さん**は**学生です」= 说到田中 → 是学生（田中是已知信息）
> - 「田中さん**が**学生です」=（谁是学生？）→ 是田中（田中是新信息）

## 二、五条判别规则

| # | 规则 | は | が | 例句 |
|---|------|---|---|------|
| 1 | **旧信息 vs 新信息** | 已在话题中的人/物 | 首次引入的人/物 | 昔々、おじいさん**が**いました。おじいさん**は**山へ行きました。 |
| 2 | **对比** | 对比用「は」 | — | コーヒー**は**飲みますが、紅茶**は**飲みません。 |
| 3 | **排他（总记）** | — | 「不是别的，正是这个」 | 誰**が**来ましたか？ — 田中さん**が**来ました。 |
| 4 | **从属节内部** | 不能用 | 从属节内用 が | [私**が**書いた]本（×私は書いた本） |
| 5 | **存在句的引入** | — | 新出现的事物 | あそこに猫**が**います。 |

### 2.1 旧信息 vs 新信息（最重要规则）

用故事展开看最清楚：

> むかしむかし、ある所におじいさん**が**いました。← 首次登场 = 新信息 = が
> おじいさん**は**毎日山へ柴刈りに行きました。← 已出现 = 旧信息 = は
> ある日、大きな桃**が**流れてきました。← 首次登场 = 新信息 = が

### 2.2 对比的「は」

对比两个以上的项目时用「は」。用「が」则没有对比的语感。

> 肉**は**食べますが、魚**は**食べません。（肉吃，鱼不吃。）
> 日本語**は**難しいですが、面白いです。（日语虽难但有趣。）

### 2.3 排他的「が」

强调「不是别人/别的，正是这个」时用「が」。

> A：誰**が**犯人ですか？（谁是犯人？）
> B：田中さん**が**犯人です。（田中才是犯人。）

## 三、典型场景的使用区分

| 场景 | 用法 | 例句 | 原因 |
|------|------|------|------|
| 自我介绍 | は | 私**は**田中です。 | 将自己作为主题提出 |
| 喜欢/讨厌 | が | 猫**が**好きです。 | 感情的对象用 が 标记 |
| 能力 | が | 日本語**が**話せます。 | 能力的对象用 が 标记 |
| 天气 | が | 雨**が**降っています。 | 自然现象用 が 引入主语 |
| 疑问词+回答 | が | 何**が**ありますか？ | 疑问词一律用 が |
| 描写 | は | この部屋**は**広いです。 | 对已知对象进行描写 |

**反直觉：感情・能力的对象用 が**

中文说"我喜欢猫"，"猫"是宾语。但日语「猫**が**好きです」中，「猫」用 が 标记。这是因为「好き」「嫌い」「できる」「わかる」「ほしい」等词的语法结构不是"主语→宾语→动词"，而是"主题は＋对象が＋状态"。

## 四、「象は鼻が長い」问题

这个句子是日语语法学的著名难题。は 与 が 同时出现。

```
象は     鼻が     長い
↓        ↓        ↓
主题     主语      谓语
(说到大象)  (鼻子)  (长)
```

> **解析**：「象」是主题（接下来要谈论大象），「鼻」是主语（长的是鼻子）。「は」与「が」在不同的语法层面发挥作用，因此可以共存。

相同结构的句子：
- 日本**は**物価**が**高い。（说到日本→物价→高。）
- 私**は**頭**が**痛い。（说到我→头→痛。）
- この店**は**ラーメン**が**おいしい。（说到这家店→拉面→好吃。）

## 五、实战决策树

```
选择助词
  └─ 在从属节内（〜が…した + 名词）？
       └─ 是 → が（从属节内不能用 は）
  └─ 含有疑问词（誰/何/どれ）？
       └─ 是 → が（疑问词一律用 が）
  └─ 在做对比（A は…、B は…）？
       └─ 是 → は
  └─ 该名词是首次登场（新信息）？
       └─ 是 → が
       └─ 否（已出现/已知） → は
  └─ 是感情・能力的对象（好き/できる/わかる）？
       └─ 是 → が
  └─ 犹豫不决 → は（作为主题提出比较安全）
```

> **实战技巧**：初级阶段「犹豫就用 は」比较安全。は 适用范围广，而 が 只在特定条件下使用。但新信息引入和疑问词句中必须用 が。
