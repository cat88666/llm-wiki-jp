# 语法-变化表达

> 变化表达跨动词、形容词、名词。先区分自然变化 `なる` 和人为改变 `する`，再看 `ようになる/ようにする` 的能力、习惯和主动维持。

## 一、变化表达总览

```
変化表达
├── 1、〜になる / 〜くなる    自然变化，主语自己变
│   ├── い形容词   い→く + なる      例：暖(あたた)かくなる。（变暖。）
│   ├── な形容词   + に + なる       例：静(しず)かになる。（变安静。）
│   └── 名词       + に + なる       例：先生(せんせい)になる。（成为老师。）
│
├── 2、〜にする / 〜くする    人为处理，让某物变成某状态
│   ├── い形容词   い→く + する      例：部屋(へや)を明(あか)るくする。（把房间弄亮。）
│   ├── な形容词   + に + する       例：音(おと)を静(しず)かにする。（把声音调安静。）
│   └── 名词       + に + する       例：子供(こども)を医者(いしゃ)にする。（让孩子成为医生。）
│
├── 3、〜ようになる    能力/习惯/状态逐渐形成（非主动控制的变化）
│   接续：辞书形 / ない形 + ようになる
│   例：毎日(まいにち)練習(れんしゅう)して、泳(およ)げるようになった。（每天练习后变得会游泳了。）
│
└── 4、〜ようにする    主动努力维持某行为（有意识地做到）
    接续：辞书形 / ない形 + ようにする
    例：寝(ね)る前(まえ)に単語(たんご)を復習(ふくしゅう)するようにしている。（睡前尽量复习单词。）
```

---

## 二、なる vs する

| 易混组 | 判断口令 |
| --- | --- |
| なる / する | 自然变化=なる，人为改变=する |
| ようになる / ようにする | 逐渐能做到/变成习惯=ようになる，主动坚持做到=ようにする |
