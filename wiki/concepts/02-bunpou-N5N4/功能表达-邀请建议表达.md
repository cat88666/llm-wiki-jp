# 语法-邀请建议表达

> 邀请和建议的差别主要在“是否给对方选择空间”。`ませんか` 更温和，`ましょう` 更像推动一起做，`ましょうか` 常用于主动提出帮助。

| 快速导航 | 概述 |
| --- | --- |
| [一、常用形式](#一常用形式) | ませんか・ましょう・ましょうか・意志形+か |
| [二、易混对比](#二易混对比) | 给选择 vs 推动一起做 |

---

## 一、常用形式

```
邀请与建议
├── 1、ませんか    礼貌度：高  温和邀请，对方可拒绝
│   例：一緒(いっしょ)に行(い)きませんか。（一起去吗？）
│
├── 2、ましょう    礼貌度：中  推动一起做
│   例：行(い)きましょう。（走吧。）
│
├── 3、ましょうか  礼貌度：高  主动提出帮忙
│   例：手伝(てつだ)いましょうか。（需要我帮忙吗？）
│
└── 4、意志形+か   礼貌度：低/口语  熟人间随意邀请
    例：行(い)こうか。（去吗？）
```

---

## 二、易混对比

| 易混组 | 判断口令 |
| --- | --- |
| ませんか / ましょう | 给对方选择=ませんか，推动一起做=ましょう |
| ましょう / ましょうか | 一起做=ましょう，我来帮你做吗=ましょうか |
