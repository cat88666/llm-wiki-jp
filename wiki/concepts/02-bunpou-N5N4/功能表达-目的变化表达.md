# 功能表达-目的变化表达

> 目的表达说明“为了什么做”，变化表达说明“变成什么状态”。`ように` 与 `ようになる/ようにする` 都围绕目标状态达成，适合放在同一页比较。

| 快速导航 | 概述 |
| --- | --- |
| [一、目的表达](#一目的表达) | Vます形+に行く・ために・ように |
| [二、变化表达](#二变化表达) | なる・する・ようになる・ようにする |
| [三、易混对比](#三易混对比) | ために/ように，なる/する |

---

## 一、目的表达

| 形式 | 接续 | 语感 | 例 |
| --- | --- | --- | --- |
| Vます形 + に行く | ます形去ます + に行く | 移动目的 | 本(ほん)を買(か)いに行(い)く。 |
| ために | 辞书形/名词の + ために | 意志目的，自己能控制 | 合格(ごうかく)するために勉強(べんきょう)する。 |
| ように | 辞书形/ない形 + ように | 状态/能力/结果目标 | 聞(き)こえるように大(おお)きく話(はな)す。 |

---

## 二、变化表达

| 形式 | 接续 | 语感 | 例 |
| --- | --- | --- | --- |
| 〜になる / 〜くなる | な形/名词 + に、い形 + く | 自然变化，主语自己变 | 静(しず)かになる。暖(あたた)かくなる。 |
| 〜にする / 〜くする | な形/名词 + に、い形 + く | 人为处理，让某物变成某状态 | 部屋(へや)を明(あか)るくする。 |
| ようになる | 辞书形/ない形 + ようになる | 能力/习惯/状态逐渐形成 | 泳(およ)げるようになった。 |
| ようにする | 辞书形/ない形 + ようにする | 主动努力维持某行为 | 復習(ふくしゅう)するようにしている。 |

---

## 三、易混对比

| 易混组 | 判断口令 |
| --- | --- |
| ために / ように | 意志动作用 ために，状态达成用 ように |
| なる / する | 自然变化=なる，人为改变=する |
| ようになる / ようにする | 逐渐能做到=ようになる，主动坚持做到=ようにする |
