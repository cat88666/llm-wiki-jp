# 副助词体系

> 副助词叠加在格助词上（或替换格助词），为名词附加「限定・添加・程度・极端」等语义色彩。は/も/だけ/しか/ばかり/さえ/まで/など 是核心成员。

## 一、副助词的本质与格助词的关系

**副助词**（副助词）接在名词（或格助词）后，附加限定・添加・程度等语义，本身不担任格语（不表示主格/目的格等），而是在格关系基础上叠加意义。

| 与格助词的关系 | 说明 | 例 |
|--------------|------|-----|
| 替换が・を | が/を 被は/も/こそ等完全替换 | 猫**が**好き→猫**が**好き（はが替换→は）× → 正确：猫**は**好きだ |
| 叠加其他格助词 | 格助词＋副助词并存 | に**は**、に**も**、で**は**、から**も** |
| が・を 的替换规则 | は/も 可替换が・を，但不能直接替换に/で等 | 駅**に**は(に+は)、学校**でも**(で+も) |

**关键**：は・も 替换 が/を，叠加 に/で/から/と 等。

## 二、添加系：も・さえ・まで

### 2.1 も（添加・并列）（N5）

基础功能：「A も B も」（A 和 B 都〜），替换 が/を，叠加其他格助词。

> 私**も**行きます。（我也去。）  
> 学校**にも**図書館**にも**行った。（学校和图书馆都去了。）

否定句中：「〜も〜ない」= 连一个都没有（全否定）。

> 何**も**食べなかった。（什么也没吃。）  
> 誰**も**来なかった。（一个人也没来。）

### 2.2 さえ（极端例示）（N3）

极端→举出极端例子暗示其他更不成问题（或更成问题）。

> 彼は漢字**さえ**読めない。（他连汉字都不会读。——暗示：更简单的也不会）  
> **〜さえ〜ば**：「お金**さえ**あれ**ば**、何でも買える。」（只要有钱，什么都能买。）——条件缩减到极限

### 2.3 まで（意外的范围延伸）（N4）

超出预期的范围扩展，有"连〜也"的惊讶含义。

> 先生**まで**泣いてしまった。（连老师都哭了。）  
> こんな簡単なことが子供**にまで**できる。（这么简单的事连孩子都能做。）

**も vs まで vs さえ**：

| 副助词 | 感情色彩 | 例 |
|--------|---------|-----|
| も | 中性添加 | 田中さんも来た |
| まで | 意外（范围超出预期） | 田中さんまで来た（没想到） |
| さえ | 极端（最不可能的例子） | 田中さんさえ来た（最难请到的他都来了） |

## 三、限定系：だけ・しか・ばかり・のみ

| 副助词 | 谓语极性 | 语感 | 例 |
|--------|---------|------|-----|
| だけ（N5） | 肯定/否定皆可 | 中性限定 | 一杯**だけ**飲んだ |
| しか（N4） | **必须否定** | 排他（只有这个，其余都没有） | 一杯**しか**飲まなかった |
| ばかり（N4） | 肯定/否定皆可 | 偏向/指责/过度 | ゲーム**ばかり**している |
| のみ（N3） | 肯定/否定皆可 | 书面语≈だけ | 関係者**のみ**入場可 |

**反直觉**：「しか」是肯定含义（只有）但必须接否定谓语。「一杯しか飲まなかった」表示"只喝了一杯"是肯定意思，但「飲まなかった」是否定形。

> ×一杯しか飲んだ（错误，しか不能接肯定）  
> ○一杯しか飲まなかった（正确）

**ばかり 的多义**：

| 用法 | 意义 | 例 |
|------|------|-----|
| 限定+偏向 | 老是・一个劲地（指责/叙述） | 文句**ばかり**言っている |
| 限定+强调 | 全是〜（过度） | 海外旅行**ばかり**している |
| 「〜たばかり」| 刚刚〜（时间近）| 食べた**ばかり**で眠い |
| 「〜ばかりか」| 不仅〜（递进）| N2文型 → [[N2重要句型]] |

## 四、程度・例示系：ほど・くらい・など・なんか

| 副助词 | 功能 | 文体 | 例 |
|--------|------|------|-----|
| ほど（N4） | 程度基准（到这种程度） | 中性〜书面 | 死ぬ**ほど**疲れた |
| くらい/ぐらい（N4） | 程度估算（大约这种程度）| 口语 | そのくらいできる |
| など（N4） | 例示（举例/谦逊/轻视） | 中性 | 本**など**を読む |
| なんか（N4） | 例示（口语/轻视/谦逊） | 口语 | 私**なんか**無理だ |

**ほど vs くらい**：

> 泣きたい**ほど**嬉しい。（书面/强调程度）  
> それ**くらい**はできる。（口语/估算程度）

**など vs なんか**：

| 用法 | など（礼貌・中性） | なんか（口语・轻视） |
|------|---------|---------|
| 例示 | 資料**など**ご用意します | 資料**なんか**いらない |
| 自谦 | 私**など**まだまだです | 私**なんか**ダメだ |

## 五、副助词的组合与语序

**格助词＋副助词**的固定语序（格助词在前，副助词在后）：

| 组合 | 例 |
|------|-----|
| に + は | 学校**には**行かない（在学校这件事上不去） |
| に + も | 学校**にも**行く |
| に + だけ | 彼**にだけ**話した |
| から + しか | 東京**からしか**来られない |
| まで + も | 試験**までも**心配している |
| で + さえ | 大人**でさえ**難しい |

**は・も がが/を を置换後**叠加格助词的模式理解：

```
助词位置理解
  名词 + [格助词(に/で/から等)] + [副助词(は/も/だけ/しか等)]
  例：東京(名) + から(格) + しか(副) = 東京からしか
  例：学校(名) + に(格)  + は(副)  = 学校には
```
