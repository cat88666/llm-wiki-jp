# 词性(动词)

> 日语动词通过词尾变化实现否定・礼貌・过去・条件・命令等语法功能。活用形的准确产出是所有语法构建的起点。

## 一、动词三类

```
动词三分类
├── 一类（五段动词）    词干以子音结尾，词尾在あいうえお五段间变化 例：書(か)く、読(よ)む、話(はな)す
├── 二类（一段动词）    词干以母音结尾，去掉「る」后直接接续
│   ├── 上一段：词干末尾为 i    例：見(み)る、起(お)きる
│   └── 下一段：词干末尾为 e    例：食(た)べる、寝(ね)る
└── 三类（不规则动词）
    ├── する：しない / します / する / すれば / しろ
    └── 来(く)る：来(こ)ない / 来(き)ます / 来(く)る / 来(く)れば / 来(こ)い
```

## 二、动词变形

- 六种活用：未然、连用、终止、连体、假定、命令
- 功能变形：意志、可能、被动、使役

| 序号 | 活用形 | 接续对象 | 一类变形规则 | 二类变形规则 | 三类变形规则 |
| --- | --- | --- | --- | --- | --- |
| 1 | 未然形 | ない / れる / せる | あ段；う→わ | 去る | する：しない / される / させる<br>来る：来(こ)ない / 来(こ)られる / 来(こ)させる |
| 2 | 连用形 | ます / たい / て / た | い段；て/た 音便 | 去る | する：します / したい / して / した<br>来る：来(き)ます / 来(き)たい / 来(き)て / 来(き)た |
| 3 | 终止形 | 结句 | 辞书形 | 辞书形 | する / 来(く)る |
| 4 | 连体形 | 修饰名词 | 同终止形 | 同终止形 | 同终止形：する / 来(く)る |
| 5 | 假定形 | ば | え段 + ば | 去る + れば | する：すれば<br>来る：来(く)れば |
| 6 | 命令形 | 句末命令 | え段 | 去る + ろ/よ | する：しろ / せよ<br>来る：来(こ)い |
| 7 | 禁止形 | 不许 | 辞书形 + な | 辞书形 + な | するな / 来(く)るな |
| 8 | 意志形 | 意志 / 邀请 | お段 + う | 去る + よう | する：しよう<br>来る：来(こ)よう |
| 9 | 可能形 | 能够 | え段 + る | 去る + られる | する：できる<br>来る：来(こ)られる |
| 10 | 被动形 | 被做 / 受影响 | あ段 + れる；う→わ | 去る + られる | する：される<br>来る：来(こ)られる |
| 11 | 使役形 | 让 / 允许 | あ段 + せる；う→わ | 去る + させる | する：させる<br>来る：来(こ)させる |
| 12 | 使役被动形 | 被迫 | あ段 + される；う→わ | 去る + させられる | する：させられる<br>来る：来(こ)させられる |

## 三、动词分类的判别法

```
1. する / 来(く)る？
   └─ 是 → 三类

2. 辞书形不是「る」结尾？
   └─ 是 → 一类
      例：書く、読む、話す、買う

3. 「る」前不是い段/え段？
   └─ 是 → 一类
      例：取る、売る、作る

4. 「い段/え段 + る」？
   └─ 多数是二类，但高频词里有一类例外
      用ない形确认：
      ├─ 去る + ない → 二类  例：食べる → 食べない
      └─ る→らない   → 一类  例：帰る → 帰らない
```

**看似二类、实际一类的高频词**：

| 动词 | ない形 | 类型 |
| --- | --- | --- |
| 帰(かえ)る | 帰(かえ)らない | 一类 |
| 走(はし)る | 走(はし)らない | 一类 |
| 入(はい)る | 入(はい)らない | 一类 |
| 切(き)る | 切(き)らない | 一类 |
| 知(し)る | 知(し)らない | 一类 |
| 要(い)る | 要(い)らない | 一类 |
| 減(へ)る | 減(へ)らない | 一类 |
