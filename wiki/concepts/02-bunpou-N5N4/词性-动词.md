## 一、动词三类

```
动词三分类
├── 一类（五段动词）    词干以子音结尾，词尾在あいうえお五段间变化 例：書(か)く、読(よ)む、話(はな)す
├── 二类（一段动词）    词干以母音结尾，去掉「る」后直接接续
│   ├── 上一段：词干末尾为 i    例：見(み)る、起(お)きる
│   └── 下一段：词干末尾为 e    例：食(た)べる、寝(ね)る
└── 三类（不规则动词）
    ├── する：しない / します / する / すれば / しろ
    └── 来(く)る：来(こ)ない / 来(き)ます / 来(く)る / 来(く)れば / 来(こ)い
```

## 二、动词变形

- 六种活用：未然、连用、终止、连体、假定、命令
- 功能变形：意志、可能、被动、使役

| --- | --- | --- | --- | --- | --- |

| 3 | 终止形 | 结句 | 辞书形 | 辞书形 | する / 来(く)る |
| 4 | 连体形 | 修饰名词 | 同终止形 | 同终止形 | 同终止形：する / 来(く)る |
| 5 | 假定形 | ば | え段 + ば | 去る + れば | する：すれば<br>来る：来(く)れば |
| 6 | 命令形 | 句末命令 | え段 | 去る + ろ/よ | する：しろ / せよ<br>来る：来(こ)い |

## 三、动词分类判别

```
1. する / 来(く)る？
   └─ 是 → 三类

2. 辞书形不是「る」结尾？
   └─ 是 → 一类
      例：書く、読む、話す、買う

3. 「る」前不是い段/え段？
   └─ 是 → 一类
      例：取る、売る、作る

4. 「い段/え段 + る」？
   └─ 多数是二类，但高频词里有一类例外
      用ない形确认：
      ├─ 去る + ない → 二类  例：食べる → 食べない
      └─ る→らない   → 一类  例：帰る → 帰らない
```

**看似二类、实际一类的高频词**：

| 动词 | 中文意思 | ない形 | 类型 |
| --- | --- | --- | --- |
| 帰(かえ)る | 回去；回家 | 帰(かえ)らない | 一类 |
| 走(はし)る | 跑；行驶 | 走(はし)らない | 一类 |
| 入(はい)る | 进入；加入 | 入(はい)らない | 一类 |
| 切(き)る | 切；断开 | 切(き)らない | 一类 |
| 知(し)る | 知道；认识 | 知(し)らない | 一类 |
| 要(い)る | 需要 | 要(い)らない | 一类 |
| 減(へ)る | 减少 | 減(へ)らない | 一类 |

## 四、动词五段功能簇

| 五段 | 段 | 性格 | 活用形名 | 语义内核              | 后接的东西 / 概述 |
| --- | --- | --- | --- |-------------------| --- |
| [あ段：未然形](#あ段还没发生) | あ段 | 还没发生 | 未然形 | 未然 = 还没成为事实       | 否定 ない、被动 れる、使役 せる、使役被动 される |
| [い段：連用形](#い段连接后续) | い段 | 连接后续 | 連用形 | 连接后面的用言/助动词       | 礼貌 ます、希望 たい、て/た、ながら |
| [う段：終止形 / 連体形](#う段动作本身) | う段 | 动作本身 | 終止形 / 連体形 | 动作本身；句子到此结束；也可修饰名词 | 辞书形、禁止 な、らしい、つもり |
| [え段：假定形](#え段假设与能力) | え段 | 假设与能力 | 假定形 | 假设/假定，也引出能力和命令    | 条件 ば、可能形、命令形 |
| [お段：意志形](#お段意志) | お段 | 意志 | 意志形 | 想做/打算做            | 意志 う、邀请 |

**口令：あ没发生、い连后续、う句结束、え假设能、お要去做。**

<a id="あ段还没发生"></a>

### 五、あ段：未然形

"还没成为事实"，所以否定和被动/使役在同一段，三者共享"还没实际发生"的语义基底——否定是"没做"，被动是"被别人做（自己没主动做）"，使役是"让别人去做（自己还没做）"。

| 后缀    | 功能 | 一类变形结果                 | 二类变形结果             | する | 来(く)る |
|-------| --- |--------------------|----------------| --- | --- |
| + ない	  | 否定  | 書(か)**か**ない（不写）    | 食(た)べない        | **し**ない | 来(**こ**)ない |
| + れる | 被动  | 書(か)**か**れる （被写）   | 食(た)べ**られる**   | **さ**れる | 来(**こ**)られる |
| + せる | 使役  | 書(か)**か**せる   （让写） | 食(た)べ**させる**   | **さ**せる | 来(**こ**)させる |
| + される | 使役被动 | 書(か)**か**される   （被迫写）  | 食(た)べ**させられる** | **さ**せられる | 来(**こ**)させられる |




**反直觉**：う結尾动词的あ段是 **わ** 不是 あ。買(か)う → 買(か)**わ**ない，不是 ×買あない。

---

<a id="い段连接后续"></a>

### 六、い段：連用形

連用形。"连接后面的内容"——所以礼貌体、希望、て形、た形都从这里出发。五段动词在接て/た时发生**音便(おんびん)**——い段词干的子音脱落或同化，不是直接拼。

| 后缀 | 功能 | 一类变形结果 | 二类变形结果 | する | 来(く)る |
| --- | --- | --- | --- | --- | --- |
| + ます | 礼貌体 | 書(か)**き**ます（写） | 食(た)べます（吃） | **し**ます | 来(**き**)ます |
| + たい | 希望 | 書(か)**き**たい（想写） | 食(た)べたい（想吃） | **し**たい | 来(**き**)たい |
| + ながら | 同时做 | 書(か)**き**ながら（一边写） | 食(た)べながら（一边吃） | **し**ながら | 来(**き**)ながら |
| + て | 接续 / 请求 / ている | 書(か)い**て**（写了之后 / 请写） | 食(た)べて（吃了之后 / 请吃） | **し**て | 来(**き**)て |
| + た | 过去 | 書(か)い**た**（写了） | 食(た)べた（吃了） | **し**た | 来(**き**)た |


**い段还能单独用**：連用形本身可以做名词或连接句子。

> 書(か)き方(かた)（写法）——連用形 + 方 = 做法
> 朝起(あさお)き、顔(かお)を洗(あら)い、出(で)かけた。（早上起来，洗脸，出门了。）——連用形中止法（书面语）

<a id="う段动作本身"></a>

### 七、う段：終止形 / 連体形

終止形(しゅうしけい) / 連体形(れんたいけい)。现代日语两者同形。

以 書(か)く → 書(か)**く** 为例：

| 用法 | 例 | 功能 |
| --- | --- | --- |
| 句子结束 | 毎日(まいにち)書(か)く。（每天写。） | 终止 |
| 修饰名词 | 書(か)く人(ひと)（写的人） | 连体 |
| +な | 書(か)くな（不许写！） | 禁止 |
| +らしい | 書(か)くらしい（好像会写） | 推量 |
| +つもり | 書(か)くつもり（打算写） | 意志/计划 |

う段最简单——就是辞书形本身，不需要变化。

---

<a id="え段假设与能力"></a>

### 八、え段：假定形

仮定形(かていけい)。"假设能做到"——条件和可能都从这里出发。

以 書(か)く → 書(か)**け** 为例：

| 后缀 | 变形结果 | 功能 | 级别 |
| --- | --- | --- | --- |
| + ば | 書(か)けば | 条件：如果写的话 | N4 |
| + る（可能形） | 書(か)ける | 可能：能写 | N4 |
| （单独/命令） | 書(か)け！ | 命令：写！ | N4 |

**为什么可能形也在え段？** 五段动词的可能形 = え段 + る，本质上是仮定形词干 + る 构成的一段动词。"假设"和"能力"的语义相通——"如果做得到"。

**反直觉**：命令形也是え段。書(か)け！＝"写！"。命令形和仮定形同形，靠有没有接 ば 来区分。

---

<a id="お段意志"></a>

### 九、お段：意志形

以 書(か)く → 書(か)**こ** 为例：

| 后缀 | 变形结果 | 功能 | 级别 |
| --- | --- | --- | --- |
| + う | 書(か)こう | 意志/邀请：写吧 | N4 |

お段最简单，只有一个用法。

> 一緒(いっしょ)に書(か)こう。（一起写吧。——邀请）
> 明日(あした)書(か)こう。（明天写吧。——意志）

<a id="四类动词全对齐表"></a>

### 十、动词对齐



#### い段簇（連用形系列）

| 功能 | 五段：書(か)く | 一段：食(た)べる | する | 来(く)る |
| --- | --- | --- | --- | --- |
| 礼貌 ます | 書(か)**き**ます | 食(た)べます | **し**ます | 来(**き**)ます |
| 希望 たい | 書(か)**き**たい | 食(た)べたい | **し**たい | 来(**き**)たい |
| て形 | 書(か)い**て**⚠️ | 食(た)べて | **し**て | 来(**き**)て |
| た形 | 書(か)い**た**⚠️ | 食(た)べた | **し**た | 来(**き**)た |
| ながら | 書(か)**き**ながら | 食(た)べながら | **し**ながら | 来(**き**)ながら |

**関键差异**：
- 一段：去る后直接接（食べ+ます）
- する/来る：し/き + 后缀
- 五段的て/た有音便（⚠️），其他三类没有

#### う段（終止形）

| 功能 | 五段：書(か)く | 一段：食(た)べる | する | 来(く)る |
| --- | --- | --- | --- | --- |
| 辞书形 | 書(か)く | 食(た)べる | する | 来(く)る |
| 禁止 +な | 書(か)くな | 食(た)べるな | するな | 来(く)るな |

所有动词都用辞书形本身，四类无差异。

#### え段簇（仮定形 + 可能）

| 功能 | 五段：書(か)く | 一段：食(た)べる | する | 来(く)る |
| --- | --- | --- | --- | --- |
| 条件 ば | 書(か)**け**ば | 食(た)べ**れ**ば | す**れ**ば | 来(く)**れ**ば |
| 可能形 | 書(か)**け**る | 食(た)べ**られる** | **できる** | 来(こ)**られる** |
| 命令 | 書(か)**け** | 食(た)べ**ろ** | し**ろ**/せ**よ** | 来(こ)**い** |

**関键差异**：
- 五段可能形 = え段+る（書ける），和被动形（書かれる，あ段）完全不同
- 一段可能形 = 食べ**られる**，和被动形**同形** → 靠语境判断
- する 的可能形是独立词「できる」，不是 ×すれる
- 命令形：四类各不相同（け/ろ/しろ/こい），只能直接记

#### お段（意志）

| 功能 | 五段：書(か)く | 一段：食(た)べる | する | 来(く)る |
| --- | --- | --- | --- | --- |
| 意志 う/よう | 書(か)**こ**う | 食(た)べ**よう** | し**よう** | 来(こ)**よう** |

- 五段：お段 + う
- 其他三类：词干 + よう

<a id="て形た形音便五段的唯一不规则"></a>

### 十一、て形/た形音便：五段的唯一不规则

五段动词接て/た时，い段词干的子音和て/た发生同化，叫音便(おんびん)。一段/する/来る 没有音便。

| 词尾 | て形 | た形 | 音便类型 | 口令 |
| --- | --- | --- | --- | --- |
| く | いて | いた | イ音便 | く/ぐ → い |
| ぐ | いで | いだ | イ音便(浊) | |
| す | して | した | 无音便 | す → し（唯一规则的） |
| つ | って | った | 促音便 | つ/る/う → っ |
| る | って | った | 促音便 | |
| う | って | った | 促音便 | |
| ぬ | んで | んだ | 撥音便 | ぬ/ぶ/む → ん |
| ぶ | んで | んだ | 撥音便 | |
| む | んで | んだ | 撥音便 | |

**三组口令**：

```
く/ぐ → いて/いで
つ/る/う → って
ぬ/ぶ/む → んで
す → して（唯一不变的）
```

**反直觉**：行(い)く → 行(い)**って**（不是 ×行いて）。全语言唯一例外。

<a id="高频混淆点"></a>

### 十二、高频混淆点

#### 一段动词：可能形 = 被动形

| | 五段（不同形） | 一段（同形！） |
| --- | --- | --- |
| 可能 | 書(か)**ける**（え段+る） | 食(た)べ**られる** |
| 被动 | 書(か)**かれる**（あ段+れる） | 食(た)べ**られる** |

判断法：有 `に` 标动作主 → 被动。描述能力 → 可能。

口语中可能的 られる 常缩为 れる（ら抜(ぬ)き言葉(ことば)）：食(た)べ**れる**。

#### する：否定用し、被动使役用さ

```
✓ しない  します  して  した  したい
✓ される  させる                    ← 只有这两个换成 さ
✗ しれる  しせる                    ← 不存在
```

#### 看似一段的五段动词

词尾是「い段/え段+る」但实际是五段的动词。验证法：看ます形。

| 动词 | ます形 | 五段证据 |
| --- | --- | --- |
| 帰(かえ)る | 帰(かえ)**り**ます | り = い段变化 → 五段 |
| 走(はし)る | 走(はし)**り**ます | り = い段变化 → 五段 |
| 入(はい)る | 入(はい)**り**ます | り = い段变化 → 五段 |
| 切(き)る | 切(き)**り**ます | り = い段变化 → 五段 |
| 知(し)る | 知(し)**り**ます | り = い段变化 → 五段 |
| 要(い)る | 要(い)**り**ます | り = い段变化 → 五段 |
| 減(へ)る | 減(へ)**り**ます | り = い段变化 → 五段 |

对比真正的一段：食(た)べる → 食(た)べます（词干不变 → 一段）。

#### 来(く)る 的读音

```
こ = 未然形系列（还没来）：来(こ)ない、来(こ)られる、来(こ)させる、来(こ)い、来(こ)よう
き = 連用形系列（来的过程）：来(き)ます、来(き)て、来(き)た、来(き)たい
く = 終止形系列（来这个动作）：来(く)る、来(く)るな、来(く)れば
```

口令：こ→き→く 对应 あ→い→う，和五段的段序一致。
