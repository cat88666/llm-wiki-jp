## 一、动词三类

```
动词三分类
├── 一类（五段动词）    词干以子音结尾，词尾在あいうえお五段间变化 例：書(か)く、読(よ)む、話(はな)す
├── 二类（一段动词）    词干以母音结尾，去掉「る」后直接接续
│   ├── 上一段：词干末尾为 i    例：見(み)る、起(お)きる
│   └── 下一段：词干末尾为 e    例：食(た)べる、寝(ね)る
└── 三类（不规则动词）
    ├── する：しない / します / する / すれば / しろ
    └── 来(く)る：来(こ)ない / 来(き)ます / 来(く)る / 来(く)れば / 来(こ)い
```

## 二、动词变形

- 六种活用：未然、连用、终止、连体、假定、命令
- 功能变形：意志、可能、被动、使役

| 序号 | 活用形 | 接续对象 | 一类变形规则 | 二类变形规则 | 三类变形规则 |
| --- | --- | --- | --- | --- | --- |
| 1 | 未然形 | ない（否定）<br>れる（被动）<br>せる（使役） | う段→あ段 + ない<br>う段→あ段 + れる<br>う段→あ段 + せる | 去る + ない<br>去る + られる<br>去る + させる | する：しない / される / させる<br>来る：来(こ)ない / 来(こ)られる / 来(こ)させる |
| 2 | 连用形 | ます（礼貌）<br>たい（希望）<br>て / た（接续/过去） | う段→い段 + ます<br>う段→い段 + たい<br>う段→い段 + て/た（音便） | 去る + ます<br>去る + たい<br>去る + て/た | する：します / したい / して / した<br>来る：来(き)ます / 来(き)たい / 来(き)て / 来(き)た |
| 3 | 终止形 | 结句 | 辞书形 | 辞书形 | する / 来(く)る |
| 4 | 连体形 | 修饰名词 | 同终止形 | 同终止形 | 同终止形：する / 来(く)る |
| 5 | 假定形 | ば | う段→え段 + ば | 去る + れば | する：すれば<br>来る：来(く)れば |
| 6 | 命令形 | 句末命令 | う段→え段 | 去る + ろ/よ | する：しろ / せよ<br>来る：来(こ)い |
| 7 | 禁止形 | 不许 | 辞书形 + な | 辞书形 + な | するな / 来(く)るな |
| 8 | 意志形 | 意志 / 邀请 | う段→お段 + う | 去る + よう | する：しよう<br>来る：来(こ)よう |
| 9 | 可能形 | 能够 | う段→え段 + る | 去る + られる | する：できる<br>来る：来(こ)られる |
| 10 | 被动形 | 被做 / 受影响 | う段→あ段 + れる | 去る + られる | する：される<br>来る：来(こ)られる |
| 11 | 使役形 | 让 / 允许 | う段→あ段 + せる | 去る + させる | する：させる<br>来る：来(こ)させる |
| 12 | 使役被动形 | 被迫 | う段→あ段 + される | 去る + させられる | する：させられる<br>来る：来(こ)させられる |

## 三、动词分类判别

```
1. する / 来(く)る？
   └─ 是 → 三类

2. 辞书形不是「る」结尾？
   └─ 是 → 一类
      例：書く、読む、話す、買う

3. 「る」前不是い段/え段？
   └─ 是 → 一类
      例：取る、売る、作る

4. 「い段/え段 + る」？
   └─ 多数是二类，但高频词里有一类例外
      用ない形确认：
      ├─ 去る + ない → 二类  例：食べる → 食べない
      └─ る→らない   → 一类  例：帰る → 帰らない
```

**看似二类、实际一类的高频词**：

| 动词 | 中文意思 | ない形 | 类型 |
| --- | --- | --- | --- |
| 帰(かえ)る | 回去；回家 | 帰(かえ)らない | 一类 |
| 走(はし)る | 跑；行驶 | 走(はし)らない | 一类 |
| 入(はい)る | 进入；加入 | 入(はい)らない | 一类 |
| 切(き)る | 切；断开 | 切(き)らない | 一类 |
| 知(し)る | 知道；认识 | 知(し)らない | 一类 |
| 要(い)る | 需要 | 要(い)らない | 一类 |
| 減(へ)る | 减少 | 減(へ)らない | 一类 |

## 四、动词五段功能簇

| 五段 | 段 | 性格 | 活用形名 | 语义内核              | 后接的东西 / 概述 |
| --- | --- | --- | --- |-------------------| --- |
| [あ段：未然形](#あ段还没发生) | あ段 | 还没发生 | 未然形 | 未然 = 还没成为事实       | 否定 ない、被动 れる、使役 せる、使役被动 される |
| [い段：連用形](#い段连接后续) | い段 | 连接后续 | 連用形 | 连接后面的用言/助动词       | 礼貌 ます、希望 たい、て/た、ながら |
| [う段：終止形 / 連体形](#う段动作本身) | う段 | 动作本身 | 終止形 / 連体形 | 动作本身；句子到此结束；也可修饰名词 | 辞书形、禁止 な、らしい、つもり |

<a id="あ段还没发生"></a>

### 五、あ段：未然形

"还没成为事实"，所以否定和被动/使役在同一段，三者共享"还没实际发生"的语义基底——否定是"没做"，被动是"被别人做（自己没主动做）"，使役是"让别人去做（自己还没做）"。

| 后缀 | 功能 | 一类变形结果 | 二类变形结果 | 三类变形结果 |
| --- | --- | --- | --- | --- |
| + ない | 否定 | 書(か)**か**ない（不写） | 食(た)べない | する：**し**ない<br>来(く)る：来(**こ**)ない |
| + れる | 被动 | 書(か)**か**れる（被写） | 食(た)べ**られる** | する：**さ**れる<br>来(く)る：来(**こ**)られる |
| + せる | 使役 | 書(か)**か**せる（让写） | 食(た)べ**させる** | する：**さ**せる<br>来(く)る：来(**こ**)させる |
| + される | 使役被动 | 書(か)**か**される（被迫写） | 食(た)べ**させられる** | する：**さ**せられる<br>来(く)る：来(**こ**)させられる |




**注意**：う的あ段是 **わ**（わ行：わ・ゐ・う・ゑ・を）。買(か)う → 買(か)**わ**ない，不是 ×買あない。

<a id="い段连接后续"></a>

### 六、い段：連用形

連用形。"连接后面的内容"——所以礼貌体、希望、て形、た形都从这里出发。五段动词在接て/た时发生**音便(おんびん)**——い段词干的子音脱落或同化，不是直接拼。

| 后缀 | 功能 | 一类变形结果 | 二类变形结果 | 三类变形结果 |
| --- | --- | --- | --- | --- |
| + ます | 礼貌体 | 書(か)**き**ます（写） | 食(た)べます（吃） | する：**し**ます<br>来(く)る：来(**き**)ます |
| + たい | 希望 | 書(か)**き**たい（想写） | 食(た)べたい（想吃） | する：**し**たい<br>来(く)る：来(**き**)たい |
| + ながら | 同时做 | 書(か)**き**ながら（一边写） | 食(た)べながら（一边吃） | する：**し**ながら<br>来(く)る：来(**き**)ながら |
| + て | 接续 / 请求 / ている | 書(か)い**て**（写了之后 / 请写） | 食(た)べて（吃了之后 / 请吃） | する：**し**て<br>来(く)る：来(**き**)て |
| + た | 过去 | 書(か)い**た**（写了） | 食(た)べた（吃了） | する：**し**た<br>来(く)る：来(**き**)た |


**い段还能单独用**：連用形本身可以做名词或连接句子。

> 書(か)き方(かた)（写法）  
> 说明：連用形 + 方 = 做法

> 朝起(あさお)き、顔(かお)を洗(あら)い、出(で)かけた。（早上起来，洗脸，出门了。）  
> 说明：連用形中止法（书面语）

<a id="う段动作本身"></a>

### 七、う段：終止形 / 連体形

| 用法 | 功能 | 一类变形结果 | 二类变形结果 | 三类变形结果 |
| --- | --- | --- | --- | --- |
| 句子结束 | 终止 | 毎日(まいにち)書(か)く。（每天写。） | 毎日(まいにち)食(た)べる。（每天吃。） | する：毎日(まいにち)する。<br>来(く)る：毎日(まいにち)来(く)る。 |
| 修饰名词 | 连体 | 書(か)く人(ひと)（写的人） | 食(た)べる人(ひと)（吃的人） | する：勉強(べんきょう)する人(ひと)<br>来(く)る：来(く)る人(ひと) |
| + な | 禁止 | 書(か)くな（不许写） | 食(た)べるな（不许吃） | する：するな<br>来(く)る：来(く)るな |
| + らしい | 推量 | 書(か)くらしい（好像会写） | 食(た)べるらしい（好像会吃） | する：するらしい<br>来(く)る：来(く)るらしい |
| + つもり | 意志 / 计划 | 書(か)くつもり（打算写） | 食(た)べるつもり（打算吃） | する：するつもり<br>来(く)る：来(く)るつもり |

<a id="え段假设与能力"></a>

### 八、え段：假定/可能形

仮定形。"假设能做到"——条件和可能都从这里出发。

| 后缀 | 功能 | 一类变形结果 | 二类变形结果 | 三类变形结果 |
| --- | --- | --- | --- | --- |
| + ば | 条件 | 書(か)**け**ば（如果写的话） | 食(た)べ**れ**ば（如果吃的话） | する：す**れ**ば<br>来(く)る：来(く)**れ**ば |
| + る | 可能 | 書(か)**け**る（能写） | 食(た)べ**られる**（能吃） | する：できる<br>来(く)る：来(こ)**られる** |
| 单独用 | 命令 | 書(か)**け**！（写！） | 食(た)べ**ろ**！（吃！） | する：し**ろ** / せよ<br>来(く)る：来(こ)**い** |

五段动词的可能形 = え段 + る，本质上是仮定形词干 + る 构成的一段动词。"假设"和"能力"的语义相通——"如果做得到"。

**反直觉**：命令形也是え段。書(か)け！＝"写！"。命令形和仮定形同形，靠有没有接 ば 来区分。

<a id="お段意志"></a>

### 九、お段：意志形

| 后缀 | 功能 | 一类变形结果 | 二类变形结果 | 三类变形结果 |
| --- | --- | --- | --- | --- |
| + う / よう | 意志 / 邀请 | 書(か)**こ**う（写吧 / 想写） | 食(た)べ**よう**（吃吧 / 想吃） | する：し**よう**<br>来(く)る：来(こ)**よう** |

お段最简单，只有一个用法。

> 一緒(いっしょ)に書(か)こう。（一起写吧。——邀请）
> 明日(あした)書(か)こう。（明天写吧。——意志）

<a id="て形た形音便五段的唯一不规则"></a>

### 十、て形/た形音便：五段的唯一不规则

五段动词接て/た时，い段词干的子音和て/た发生同化，叫音便。一段/する/来る 没有音便。

| 词尾 | て形 | た形 | 音便类型 | 口令 |
| --- | --- | --- | --- | --- |
| く | いて | いた | イ音便 | く/ぐ → い |
| ぐ | いで | いだ | イ音便(浊) | |
| す | して | した | 无音便 | す → し（唯一规则的） |
| つ | って | った | 促音便 | つ/る/う → っ |
| る | って | った | 促音便 | |
| う | って | った | 促音便 | |
| ぬ | んで | んだ | 撥音便 | ぬ/ぶ/む → ん |
| ぶ | んで | んだ | 撥音便 | |
| む | んで | んだ | 撥音便 | |

**三组口令**：

```
く/ぐ → いて/いで
つ/る/う → って
ぬ/ぶ/む → んで
す → して（唯一不变的）
```

**反直觉**：行(い)く → 行(い)**って**（不是 ×行いて）。全语言唯一例外。

<a id="高频混淆点"></a>

### 十一、高频混淆点

#### 一段动词：可能形 = 被动形

| | 五段（不同形） | 一段（同形！） |
| --- | --- | --- |
| 可能 | 書(か)**ける**（え段+る） | 食(た)べ**られる** |
| 被动 | 書(か)**かれる**（あ段+れる） | 食(た)べ**られる** |

判断法：有 `に` 标动作主 → 被动。描述能力 → 可能。

口语中可能的 られる 常缩为 れる（ら抜(ぬ)き言葉(ことば)）：食(た)べ**れる**。

#### する：否定用し、被动使役用さ

```
✓ しない  します  して  した  したい
✓ される  させる                    ← 只有这两个换成 さ
✗ しれる  しせる                    ← 不存在
```

#### 看似一段的五段动词

词尾是「い段/え段+る」但实际是五段的动词。验证法：看ます形。

| 动词 | ます形 | 五段证据 |
| --- | --- | --- |
| 帰(かえ)る | 帰(かえ)**り**ます | り = い段变化 → 五段 |
| 走(はし)る | 走(はし)**り**ます | り = い段变化 → 五段 |
| 入(はい)る | 入(はい)**り**ます | り = い段变化 → 五段 |
| 切(き)る | 切(き)**り**ます | り = い段变化 → 五段 |
| 知(し)る | 知(し)**り**ます | り = い段变化 → 五段 |
| 要(い)る | 要(い)**り**ます | り = い段变化 → 五段 |
| 減(へ)る | 減(へ)**り**ます | り = い段变化 → 五段 |

对比真正的一段：食(た)べる → 食(た)べます（词干不变 → 一段）。

#### 来(く)る 的读音

```
こ = 未然形系列（还没来）：来(こ)ない、来(こ)られる、来(こ)させる、来(こ)い、来(こ)よう
き = 連用形系列（来的过程）：来(き)ます、来(き)て、来(き)た、来(き)たい
く = 終止形系列（来这个动作）：来(く)る、来(く)るな、来(く)れば
```

口令：こ→き→く 对应 あ→い→う，和五段的段序一致。

### 十二、功能表达多维对比

变形是工具，功能表达是变形的应用场景。

#### 12.1 能力表达

| 表达 | 接续 | 语感 | 例 |
| --- | --- | --- | --- |
| 可能形 | 动词变可能 | 最自然，口语常用 | 日本語(にほんご)が話(はな)せる。（会说日语。） |
| ことができる | 辞书形+ことができる | 书面/正式 | ここで予約(よやく)することができる。（可以在这里预约。） |
| 名词+ができる | 名词 | 会/能做某事 | 料理(りょうり)ができる。（会做菜。） |

#### 12.2 目的表达

| 表达 | 接续 | 例 |
| --- | --- | --- |
| Vます形+に行(い)く | 动作目的 | 本(ほん)を買(か)いに行(い)く。（去买书。） |
| ために | 辞书形/名词の+ために | 合格(ごうかく)するために勉強(べんきょう)する。（为了合格而学习。） |
| ように | 辞书形/ない形+ように | 聞(き)こえるように大(おお)きく話(はな)す。（说大声点让人能听到。） |

**为什么 ために 和 ように 不能互换？** ために 前面必须是意志动词（自己能控制的动作），ように 前面是状态/能力动词或否定（自己不能直接控制的结果）。

| 对比 | ために | ように |
| --- | --- | --- |
| 前件 | 意志动作 | 状态/能力/可能/否定 |
| 例 | 合格(ごうかく)する**ために**勉強(べんきょう)する（为了合格而学习） | 合格(ごうかく)できる**ように**勉強(べんきょう)する（学习以便能合格） |

#### 12.3 邀请与建议

| 表达 | 礼貌度 | 语感 | 例 |
| --- | --- | --- | --- |
| ませんか | 高 | 温和邀请，对方可拒绝 | 一緒(いっしょ)に行(い)きませんか。（一起去吗？） |
| ましょう | 中 | 提议一起做 | 行(い)きましょう。（走吧。） |
| ましょうか | 高 | 我来帮你做吗 | 手伝(てつだ)いましょうか。（需要我帮忙吗？） |
| 意志形+か | 低/口语 | 熟人间邀请 | 行(い)こうか。（去吗？） |

#### 12.4 意志、计划、想法

| 表达 | 接续 | 确定度 | 例 |
| --- | --- | --- | --- |
| 意志形 | お段+う/よう | 当场意志/邀请 | 勉強(べんきょう)しよう。（学习吧。） |
| つもりだ | 辞书形+つもり | 个人打算 | 来年(らいねん)受(う)けるつもりだ。（打算明年考。） |
| 予定(よてい)だ | 辞书形+予定 | 外部计划/日程 | 来月(らいげつ)試験(しけん)を受(う)ける予定(よてい)だ。（计划下月考试。） |
| ことにする | 辞书形+ことにする | 自己决定 | 毎日(まいにち)聞(き)くことにした。（决定每天听。） |
| ことになる | 辞书形+ことになる | 外部决定/结果 | 転勤(てんきん)することになった。（（公司决定）要调动了。） |

**为什么 つもり 和 予定 不能互换？** つもり 是心里打算（主观意志），予定 是日程安排（客观事实）。

#### 12.5 被动、使役、使役被动

| 功能 | 主语视角 | 典型含义 | 例 |
| --- | --- | --- | --- |
| 被动 | 承受动作的人/物 | 被做、受到影响 | 先生(せんせい)に褒(ほ)められた。（被老师表扬了。） |
| 间接被动 | 受影响的人 | 遭遇麻烦/受害 | 雨(あめ)に降(ふ)られた。（被雨淋了。） |
| 使役 | 让别人做 | 强制/允许 | 子供(こども)を行(い)かせる。（让孩子去。） |
| 使役被动 | 被迫做 | 不情愿 | 残業(ざんぎょう)させられた。（被迫加班了。） |

**四态最小对比**：

| 态 | 句子 | 中文 |
| --- | --- | --- |
| 能动 | 子供(こども)が野菜(やさい)を食(た)べる。 | 孩子吃蔬菜。 |
| 被动 | 子供(こども)が母(はは)に褒(ほ)められる。 | 孩子被妈妈表扬。 |
| 使役 | 母(はは)が子供(こども)に野菜(やさい)を食(た)べさせる。 | 妈妈让孩子吃蔬菜。 |
| 使役被动 | 子供(こども)が母(はは)に野菜(やさい)を食(た)べさせられる。 | 孩子被妈妈逼着吃蔬菜。 |

#### 12.6 变化表达：变得〜 / 变成〜 / 使其变成〜

日语表达"变化"时，先判断变化结果是什么词性：

| 结果类型 | 变成〜：なる | 使其变成〜：する | 例 |
| --- | --- | --- | --- |
| い形容词 | い→く + なる | い→く + する | 暖(あたた)かくなる（变暖）/ 部屋(へや)を明(あか)るくする（把房间弄亮） |
| な形容词 | +に + なる | +に + する | 静(しず)かになる（变安静）/ 音(おと)を静(しず)かにする（把声音调安静） |
| 名词 | +に + なる | +に + する | 先生(せんせい)になる（成为老师）/ 子供(こども)を医者(いしゃ)にする（让孩子成为医生） |
| 动词能力/习惯 | 辞书形 + ようになる | 辞书形 + ようにする | 日本語(にほんご)が話(はな)せるようになる（变得会说日语）/ 毎日(まいにち)聞(き)くようにする（尽量每天听） |

**核心区别**：

| 形式 | 重点 | 例 |
| --- | --- | --- |
| 〜になる | 自然变化/结果变化，主语自己变 | 春(はる)になると暖(あたた)かくなる。（到春天就变暖。） |
| 〜にする / 〜くする | 人为处理，让某物变成某状态 | 電気(でんき)をつけて、部屋(へや)を明(あか)るくした。（开灯把房间弄亮了。） |
| 〜ようになる | 能力、习惯、状态逐渐形成 | 毎日(まいにち)練習(れんしゅう)して、泳(およ)げるようになった。（每天练习后变得会游泳了。） |
| 〜ようにする | 主动努力保持某行为 | 寝(ね)る前(まえ)に単語(たんご)を復習(ふくしゅう)するようにしている。（睡前会尽量复习单词。） |

#### 12.7 看起来像・像……一样

"像"有两类：一种是**比喻相似**，一种是**外观判断**。基础用法先记这张：

| 表达 | 接续 | 含义 | 例 |
| --- | --- | --- | --- |
| 名词 + のようだ | 名词 | 像……一样，偏书面 | 夢(ゆめ)のようだ。（像梦一样。） |
| 普通形 + ようだ | 句子 | 看样子/似乎，偏书面 | 雨(あめ)が降(ふ)ったようだ。（好像下过雨。） |
| 名词/普通形 + みたいだ | 口语 | 像……/好像…… | 子供(こども)みたいだ。（像小孩一样。） |
| い形词干/な形词干/Vます形 + そうだ | 外观迹象 | 看起来要/看起来很 | おいしそうだ。（看起来好吃。）/ 雨(あめ)が降(ふ)りそうだ。（像要下雨。） |

**ようだ vs みたいだ vs そうだ**：

| 想表达 | 选哪个 | 例 |
| --- | --- | --- |
| "像……一样"的比喻 | ようだ / みたいだ | 彼(かれ)は子供(こども)のようだ / 子供(こども)みたいだ。 |
| 根据样子判断"好像已经是某状态" | ようだ / みたいだ | 彼(かれ)は疲(つか)れているようだ。 |
| 眼前看起来"马上要发生/看起来很……" | そうだ | 泣(な)きそうだ。/ 楽(たの)しそうだ。 |

更完整的「そうだ・ようだ・らしい・みたいだ」区别见 [[引用传闻表达]]。

### 十三、易混组最小对比

| 易混组 | 判断口令 |
| --- | --- |
| ために / ように | 意志动作用 ために，状态达成用 ように |
| つもり / 予定(よてい) | 心里打算=つもり，日程安排=予定 |
| ませんか / ましょう | 给对方选择=ませんか，推动一起做=ましょう |
| 可能形 / ことができる | 日常能力=可能形，正式说明=ことができる |
| 被动 / 可能 | 有动作主に多为被动，有能力语境多为可能 |
| 使役 / 使役被动 | 让别人做=使役，被迫自己做=使役被动 |
| なる / する | 自然变化=なる，人为改变=する |
| ようだ / みたいだ / そうだ | 比喻相似=ようだ/みたいだ，眼前迹象=そうだ |
