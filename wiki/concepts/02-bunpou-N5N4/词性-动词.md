# 词性(动词)

> 日语动词通过词尾变化（活用(かつよう)）实现否定・礼貌・过去・条件・命令等语法功能。活用形的准确产出是所有语法构建的起点。

## 一、动词的三分类

日语动词按活用模式分为三组。分类的本质是**词干与词尾的边界位置**。

```
动词三分类
├── 第一类（五段动词）    词干以子音结尾，词尾在あいうえお五段间变化
│   例：書(か)く、読(よ)む、話(はな)す
├── 第二类（一段动词）    词干以母音结尾，去掉「る」后直接接续
│   ├── 上一段：词干末尾为 i    例：見(み)る、起(お)きる
│   └── 下一段：词干末尾为 e    例：食(た)べる、寝(ね)る
└── 第三类（不规则动词）  仅 する・来(く)る 两个词
```

**为什么要分三类？** 因为词干的构造不同，接续后缀的方式就不同。五段动词的词干以子音结尾（如「書く」词干 kak-），需要在子音后插入不同的母音来实现变化；一段动词词干以母音结尾（如「食べる」词干 tabe-），去掉「る」后直接接后缀，不需要母音变化。

**反直觉**：「帰(かえ)る」「走(はし)る」以「〜る」结尾但属于五段动词。看似一段的五段动词只能逐个记忆（详见[第六节](#六动词分类的判别法)）。

## 二、六种活用形

活用形(かつようけい)是动词词尾变化后的形态，每种形态对应特定的后续接续。

| 活用形 | 接续对象 | 五段「書(か)く」 | 一段「食(た)べる」 | 核心功能 |
|--------|--------|------------|-------------|---------|
| 未然形(みぜんけい) | ない/れる/せる | 書(か)か- | 食(た)べ- | 否定・被动・使役的起点 |
| 连用形(れんようけい) | ます/た/て | 書(か)き- | 食(た)べ- | 礼貌体・过去・接续的起点 |
| 终止形(しゅうしけい) | 。/と/から | 書(か)く | 食(た)べる | 句子终止，与辞书形同形 |
| 连体形(れんたいけい) | 名词 | 書(か)く | 食(た)べる | 名词修饰（现代语与终止形同形） |
| 假定形(かていけい) | ば | 書(か)け- | 食(た)べれ- | 条件表达 [[条件表达体系]] |
| 命令形(めいれいけい) | （句末） | 書(か)け | 食(た)べろ | 命令・指示 |

**为什么现代日语的终止形和连体形长得一样？** 在古典日语中两者形态不同（如「書く」终止 vs「書く」连体在古语中有区别），但随着语言演变合流了。考试中仍然区分两个概念，因为它们的**语法功能**不同：终止形结句，连体形修饰名词。

## 三、五段动词的活用规则

五段动词的词尾共享同一个子音，母音在あ段→い段→う段→え段间变化。

**为什么叫「五段」？** 名称来自词尾在あいうえお五段间变化这一特征。

| 词尾 | 未然(あ段) | 连用(い段) | 终止(う段) | 假定(え段) | 命令(え段) | 代表动词 |
|------|---------|---------|---------|---------|---------|---------|
| く | か | き | く | け | け | 書(か)く |
| ぐ | が | ぎ | ぐ | げ | げ | 泳(およ)ぐ |
| す | さ | し | す | せ | せ | 話(はな)す |
| つ | た | ち | つ | て | て | 持(も)つ |
| ぬ | な | に | ぬ | ね | ね | 死(し)ぬ |
| ぶ | ば | び | ぶ | べ | べ | 遊(あそ)ぶ |
| む | ま | み | む | め | め | 読(よ)む |
| る | ら | り | る | れ | れ | 取(と)る |
| う | わ | い | う | え | え | 買(か)う |

**反直觉**：「う」结尾动词的未然形是「わ」而非「あ」。如「買(か)う」→「買(か)わない」，不是 ×買あない。这是历史音变的残留。

## 四、一段动词的活用规则

一段动词去掉「る」后，词干直接接续各后缀。

**为什么一段动词的活用比五段简单得多？** 因为一段动词的词干以母音结尾，后缀可以直接拼接，不需要像五段那样通过母音变化来衔接子音词干和后缀。

```
食(た)べる → 食(た)べ（词干）
  └─ + ない    → 食(た)べない        ← 未然形（不吃）
  └─ + ます    → 食(た)べます        ← 连用形（吃/礼貌体）
  └─ + る      → 食(た)べる          ← 终止形（吃）
  └─ + れば    → 食(た)べれば        ← 假定形（如果吃的话）
  └─ + ろ/よ   → 食(た)べろ/食(た)べよ  ← 命令形（吃！）
```

## 五、不规则动词

日语的不规则动词仅有「する」和「来(く)る」两个。

| 活用形 | する | 来(く)る |
|--------|------|------|
| 未然形 | し-/せ-/さ- | 来(こ)- |
| 连用形 | し- | 来(き)- |
| 终止形 | する | 来(く)る |
| 假定形 | すれ- | 来(く)れ- |
| 命令形 | しろ/せよ | 来(こ)い |

**为什么「する」有三种未然形？** 因为接续对象不同：し+ない（否定）、せ+られる（被动・尊敬）、さ+せる（使役）。三种形态分工不同，不能互换。

**反直觉**：「来(く)る」是唯一一个随活用形改变读音的动词——未然形こ、连用形き、终止形く、假定形く、命令形こ。必须将读音和活用形一一对应记住。

> 勉強(べんきょう)する（学习）/ 友達(ともだち)が来(く)る（朋友来）

## 六、动词分类的判别法

```
辞书形以「〜る」结尾？
  └─ 否 → 五段动词（确定）
  └─ 是 →「る」前面的假名是「い段」或「え段」？
            └─ 否 → 五段动词（确定）
            └─ 是 → 可能是一段动词
                      └─ 用ない形验证：
                          词干+ない → 一段动词    例：食(た)べ+ない
                          词干+あ段+ない → 五段动词  例：帰(かえ)ら+ない
```

**为什么不能只看词尾判断？** 因为「〜いる」「〜える」的形式同时出现在五段和一段动词中。如「見(み)る」（一段）和「切(き)る」（五段）都是「い段+る」，但活用模式完全不同。最可靠的方法是做ない形测试。

**需特别注意的五段动词**（看似一段实为五段）：

| 动词 | ない形 | 为什么容易误判 |
|------|--------|------|
| 帰(かえ)る | 帰(かえ)ら+ない（回去）|「え段+る」但属五段 |
| 走(はし)る | 走(はし)ら+ない（跑）|「い段+る」但属五段 |
| 入(はい)る | 入(はい)ら+ない（进入）|「い段+る」但属五段 |
| 切(き)る | 切(き)ら+ない（切）|「い段+る」但属五段 |
| 知(し)る | 知(し)ら+ない（知道）|「い段+る」但属五段 |

> 这类词约 30 个，高频词居多，遇到时逐个确认ない形。
