# 词性(动词)

> 日语动词通过词尾变化实现否定・礼貌・过去・条件・命令等语法功能。活用形的准确产出是所有语法构建的起点。

## 一、动词的三分类

```
动词三分类
├── 第一类（五段动词）    词干以子音结尾，词尾在あいうえお五段间变化 例：書(か)く、読(よ)む、話(はな)す
├── 第二类（一段动词）    词干以母音结尾，去掉「る」后直接接续
│   ├── 上一段：词干末尾为 i    例：見(み)る、起(お)きる
│   └── 下一段：词干末尾为 e    例：食(た)べる、寝(ね)る
└── 第三类（不规则动词）
    ├── する：しない / します / する / すれば / しろ
    └── 来(く)る：来(こ)ない / 来(き)ます / 来(く)る / 来(く)れば / 来(こ)い
```

先记结论：**一类变词尾，二类去る，三类单独背**。

## 二、动词六种变形

| 活用形 | 接续对象 | 五段「書(か)く」 | 一段「食(た)べる」 | 核心功能 |
| --- | --- | --- | --- | --- |
| 未然形 | ない/れる/せる | 書(か)か- | 食(た)べ- | 否定・被动・使役的起点 |
| 连用形 | ます/たい/て/た | 書(か)き- | 食(た)べ- | 礼貌・希望・接续・过去 |
| 终止形 | 。/と/から | 書(か)く | 食(た)べる | 句子终止，与辞书形同形 |
| 连体形 | 名词 | 書(か)く | 食(た)べる | 名词修饰（现代语与终止形同形） |
| 假定形 | ば | 書(か)け- | 食(た)べれ- | 条件表达 [[条件表达体系]] |
| 命令形 | （句末） | 書(か)け | 食(た)べろ | 命令・指示 |

现代日语里，终止形和连体形形式相同，但功能不同：终止形结句，连体形修饰名词。

## 三、五段动词的活用规则

五段动词看最后一个假名。核心是：**あ段接ない/れる/せる，い段接ます/たい，う段原形，え段接ば/命令**。

| 词尾 | 未然(あ段) | 连用(い段) | 终止(う段) | 假定(え段) | 命令(え段) | 代表动词 |
| --- | --- | --- | --- | --- | --- | --- |
| く | か | き | く | け | け | 書(か)く |
| ぐ | が | ぎ | ぐ | げ | げ | 泳(およ)ぐ |
| す | さ | し | す | せ | せ | 話(はな)す |
| つ | た | ち | つ | て | て | 持(も)つ |
| ぬ | な | に | ぬ | ね | ね | 死(し)ぬ |
| ぶ | ば | び | ぶ | べ | べ | 遊(あそ)ぶ |
| む | ま | み | む | め | め | 読(よ)む |
| る | ら | り | る | れ | れ | 取(と)る |
| う | わ | い | う | え | え | 買(か)う |

注意：`う` 的未然形是 `わ`。買(か)う → 買(か)わない，不是 `買あない`。

## 四、一段动词的活用规则

一段动词只做一件事：**去掉「る」后接后缀**。

```
食(た)べる → 食(た)べ（词干）
  └─ + ない    → 食(た)べない        ← 未然形（不吃）
  └─ + ます    → 食(た)べます        ← 连用形（吃/礼貌体）
  └─ + る      → 食(た)べる          ← 终止形（吃）
  └─ + れば    → 食(た)べれば        ← 假定形（如果吃的话）
  └─ + ろ/よ   → 食(た)べろ/食(た)べよ  ← 命令形（吃！）
```

## 五、不规则动词

日语的不规则动词仅有「する」和「来(く)る」两个。

| 活用形 | する | 来(く)る |
|--------|------|------|
| 未然形 | し-/せ-/さ- | 来(こ)- |
| 连用形 | し- | 来(き)- |
| 终止形 | する | 来(く)る |
| 假定形 | すれ- | 来(く)れ- |
| 命令形 | しろ/せよ | 来(こ)い |

`する` 的未然形按后缀分工：`しない`、`される`、`させる`。不要写成 `しれる`、`しせる`。

`来(く)る` 的读音按活用变化：未然 `こ`，连用 `き`，终止/连体/假定 `く`，命令 `こ`。

> 勉強(べんきょう)する（学习）/ 友達(ともだち)が来(く)る（朋友来）

## 六、动词分类的判别法

```
辞书形以「〜る」结尾？
  └─ 否 → 五段动词（确定）
  └─ 是 →「る」前面的假名是「い段」或「え段」？
            └─ 否 → 五段动词（确定）
            └─ 是 → 可能是一段动词
                      └─ 用ない形验证：
                          词干+ない → 一段动词    例：食(た)べ+ない
                          词干+あ段+ない → 五段动词  例：帰(かえ)ら+ない
```

最可靠的方法是做 `ない形` 测试：`食べない` 是二类，`帰らない` 是一类。

**需特别注意的五段动词**（看似一段实为五段）：

| 动词 | ない形 | 为什么容易误判 |
| --- | --- | --- |
| 帰(かえ)る | 帰(かえ)ら+ない（回去）|「え段+る」但属五段 |
| 走(はし)る | 走(はし)ら+ない（跑）|「い段+る」但属五段 |
| 入(はい)る | 入(はい)ら+ない（进入）|「い段+る」但属五段 |
| 切(き)る | 切(き)ら+ない（切）|「い段+る」但属五段 |
| 知(し)る | 知(し)ら+ない（知道）|「い段+る」但属五段 |
| 要(い)る | 要(い)ら+ない（需要）|「い段+る」但属五段 |
| 減(へ)る | 減(へ)ら+ない（减少）|「え段+る」但属五段 |

> 这类词高频，遇到 `い段/え段 + る` 不确定时，先查 `ない形`。
