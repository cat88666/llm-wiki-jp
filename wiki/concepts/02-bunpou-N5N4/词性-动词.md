## 动词三类

```
动词三分类
├── 一类（五段动词）    词尾在あいうえお五段间变化
│   ├── 非る结尾：く・む・す・つ・う・ぐ・ぬ・ぶ    例：書(か)く、読(よ)む、話(はな)す
│   └── る结尾也可能是一类                            例：帰(かえ)る、走(はし)る、切(き)る
├── 二类（一段动词）    全部「い段/え段 + る」结尾，去る直接接续
│   ├── 上一段：い段 + る    例：見(み)る、起(お)きる、借(か)りる
│   └── 下一段：え段 + る    例：食(た)べる、寝(ね)る、教(おし)える
└── 三类（不规则动词）    只有两个
    ├── する：しない / します / する / すれば / しろ
    └── 来(く)る：来(こ)ない / 来(き)ます / 来(く)る / 来(く)れば / 来(こ)い
    
判断口令：不是する/来る → 不是る结尾 → る前不是い/え段 → 以上全否，查ない形。二类：食(た)べない;  一类：帰(かえ)る → 帰(かえ)らない;
らない：二类动词去る + ない（食べない），一类动词う段→あ段 + ない（る→ら），出现 らない 是一类。

```

## 动词变形总表

六种活用：未然、连用、终止、连体、假定、命令。<br>
四种功能：意志、可能、被动、使役。<br>
记忆口令：あ没发生、い连后续、う句结束、え假设能、お要去做。

| 序号 | 段 | 活用形              | 接续对象 | 一类变形规则 | 二类变形规则 | 三类 する | 三类 来る |
| --- | --- |-----------------| - | --- | --- | --- | --- |
| 1 | あ段 | 未然形(还没发生)        | ない（否定）<br>れる（被动）<br>せる（使役）<br>される（使役被动） | う段→あ段 + ない<br>う段→あ段 + れる<br>う段→あ段 + せる<br>う段→あ段 + される | 去る + ない<br>去る + られる<br>去る + させる<br>去る + させられる | 否定：しない<br>被动：される<br>使役：させる<br>使役被动：させられる | 否定：来(こ)ない<br>被动：来(こ)られる<br>使役：来(こ)させる<br>使役被动：来(こ)させられる |
| 2 | い段 | 连用形(连接后续)        | ます（礼貌）<br>たい（希望）<br>て / た（接续/过去）<br>ながら（同时做） | う段→い段 + ます<br>う段→い段 + たい<br>う段→い段 + て/た（音便）<br>う段→い段 + ながら | 去る + ます<br>去る + たい<br>去る + て/た<br>去る + ながら | 礼貌：します<br>希望：したい<br>接续：して<br>过去：した<br>ながら：しながら | 礼貌：来(き)ます<br>希望：来(き)たい<br>接续：来(き)て<br>过去：来(き)た<br>ながら：来(き)ながら |
| 3 | う段 | 終止形 / 連体形(动作本身)  | 结句（終止）<br>修饰名词（連体）<br>+ な（禁止） | 辞书形（終止/連体）<br>辞书形 + な（禁止） | 辞书形（終止/連体）<br>辞书形 + な（禁止） | 終止/連体：する<br>禁止：するな | 終止/連体：来(く)る<br>禁止：来(く)るな |
| 4 | え段 | 假定形(假设与能力)       | + ば（条件）<br>单独（命令）<br>+ る（可能） | う段→え段 + ば<br>う段→え段（命令）<br>う段→え段 + る | 去る + れば<br>去る + ろ/よ<br>去る + られる | 条件：すれば<br>命令：しろ・せよ<br>可能：できる | 条件：来(く)れば<br>命令：来(こ)い<br>可能：来(こ)られる |
| 5 | お段 | 意志形(意志)          | 意志 | 意志 / 邀请 | う段→お段 + う | 去る + よう | 意志/邀请：しよう | 意志/邀请：来(こ)よう |

- あ段：う的あ段是 **わ**（わ行：わ・ゐ・う・ゑ・を）。買(か)う → 買(か)**わ**ない，不是 ×買あない。
- **い段：連用形单独用**：連用形本身可以做名词或连接句子。
  > 連用形：書(か)き方(かた)（写法） + 方(かた) = 做法 <br>
  > 連用形中止法: 朝(あさ)起(お)き、顔(かお)を洗(あら)い、出(で)かけた。（早上起来，洗脸，出门了。）
- **え段・命令形和仮定形同形**：共用え段，接 ば 是条件，单独用是命令。可能形 = え段 + る，本质是以え段词干新造的一段动词。
- **お段・意志/邀请例句**：
  > 邀请: 一緒(いっしょ)に書(か)こう。一起写吧。<br>
  > 意志: 明日(あした)書(か)こう。明天写吧

## て形/た形音便

五段动词接て/た时，い段词干的子音和て/た发生同化，叫音便。

| 词尾 | て形 | た形 | 音便类型 |
| --- | --- | --- | --- |
| く | いて | いた | イ音便 |
| ぐ | いで | いだ | イ音便(浊) |
| す | して | した | 无音便 |
| う・つ・る | って | った | 促音便 |
| む・ぶ・ぬ | んで | んだ | 撥音便 |

**反直觉**：行(い)く → 行(い)**って**（不是 ×行いて）。全语言唯一例外。

---
### 来(く)る 的读音规律

```
口令：こ→き→く 对应 あ→い→う，和五段的段序一致。
こ = 未然形系列（还没来）：来(こ)ない、来(こ)られる、来(こ)させる、来(こ)い、来(こ)よう
き = 連用形系列（来的过程）：来(き)ます、来(き)て、来(き)た、来(き)たい
く = 終止形系列（来这个动作）：来(く)る、来(く)るな、来(く)れば
```

### 一段动词：可能形 = 被动形

| | 五段（不同形） | 一段（同形） |判断法 |
| --- | --- | --- |--- |
| 可能 | 書(か)**ける**（え段+る） | 食(た)べ**られる** |描述能力|
| 被动 | 書(か)**かれる**（あ段+れる） | 食(た)べ**られる** |有 `に` 标动作主|

## 功能表达对比

### 能力表达

| 表达 | 接续 | 语感 | 例 |
| --- | --- | --- | --- |
| 可能形 | 动词变可能 | 最自然，口语常用 | 日本語(にほんご)が話(はな)せる。（会说日语。） |
| ことができる | 辞书形+ことができる | 书面/正式 | ここで予約(よやく)することができる。（可以在这里预约。） |
| 名词+ができる | 名词 | 会/能做某事 | 料理(りょうり)ができる。（会做菜。） |

### 目的表达

| 表达 | 接续 | 区别                          | 例 |
| --- | --- |-----------------------------| --- |
| Vます形+に行(い)く | 动作目的 |                             | 本(ほん)を買(か)いに行(い)く。（去买书。） |
| ために | 辞书形/名词の+ために | 前面必须是意志动词(自己能控制的动作)         | 合格(ごうかく)するために勉強(べんきょう)する。（为了合格而学习。） |
| ように | 辞书形/ない形+ように | 前面是状态/能力/可能/否定(自己不能直接控制的结果) | 聞(き)こえるように大(おお)きく話(はな)す。（说大声点让人能听到。） |


### 邀请与建议

| 表达 | 礼貌度 | 语感 | 例 |
| --- | --- | --- | --- |
| ませんか | 高 | 温和邀请，对方可拒绝 | 一緒(いっしょ)に行(い)きませんか。（一起去吗？） |
| ましょう | 中 | 提议一起做 | 行(い)きましょう。（走吧。） |
| ましょうか | 高 | 我来帮你做吗 | 手伝(てつだ)いましょうか。（需要我帮忙吗？） |
| 意志形+か | 低/口语 | 熟人间邀请 | 行(い)こうか。（去吗？） |

### 意志、计划、想法

| 表达 | 接续 | 确定度 | 例 |
| --- | --- | --- | --- |
| 意志形 | お段+う/よう | 当场意志/邀请 | 勉強(べんきょう)しよう。（学习吧。） |
| つもりだ | 辞书形+つもり | 心里打算（主观意志） | 来年(らいねん)受(う)けるつもりだ。（打算明年考。） |
| 予定(よてい)だ | 辞书形+予定 | 外部计划/日程（客观事实） | 来月(らいげつ)試験(しけん)を受(う)ける予定(よてい)だ。（计划下月考试。） |
| ことにする | 辞书形+ことにする | 自己决定 | 毎日(まいにち)聞(き)くことにした。（决定每天听。） |
| ことになる | 辞书形+ことになる | 外部决定/结果 | 転勤(てんきん)することになった。（（公司决定）要调动了。） |


### 被动、使役、使役被动

| 功能 | 主语视角 | 典型含义 | 例 |
| --- | --- | --- | --- |
| 被动 | 承受动作的人/物 | 被做、受到影响 | 先生(せんせい)に褒(ほ)められた。（被老师表扬了。） |
| 间接被动 | 受影响的人 | 遭遇麻烦/受害 | 雨(あめ)に降(ふ)られた。（被雨淋了。） |
| 使役 | 让别人做 | 强制/允许 | 子供(こども)を行(い)かせる。（让孩子去。） |
| 使役被动 | 被迫做 | 不情愿 | 残業(ざんぎょう)させられた。（被迫加班了。） |

**四态最小对比**：

| 态 | 句子 | 中文 |
| --- | --- | --- |
| 能动 | 子供(こども)が野菜(やさい)を食(た)べる。 | 孩子吃蔬菜。 |
| 被动 | 子供(こども)が母(はは)に褒(ほ)められる。 | 孩子被妈妈表扬。 |
| 使役 | 母(はは)が子供(こども)に野菜(やさい)を食(た)べさせる。 | 妈妈让孩子吃蔬菜。 |
| 使役被动 | 子供(こども)が母(はは)に野菜(やさい)を食(た)べさせられる。 | 孩子被妈妈逼着吃蔬菜。 |

### 变化表达

日语表达"变化"时，先判断变化结果是什么词性：

| 结果类型 | 变成〜：なる | 使其变成〜：する | 例 |
| --- | --- | --- | --- |
| い形容词 | い→く + なる | い→く + する | 暖(あたた)かくなる（变暖）/ 部屋(へや)を明(あか)るくする（把房间弄亮） |
| な形容词 | +に + なる | +に + する | 静(しず)かになる（变安静）/ 音(おと)を静(しず)かにする（把声音调安静） |
| 名词 | +に + なる | +に + する | 先生(せんせい)になる（成为老师）/ 子供(こども)を医者(いしゃ)にする（让孩子成为医生） |
| 动词能力/习惯 | 辞书形 + ようになる | 辞书形 + ようにする | 日本語(にほんご)が話(はな)せるようになる（变得会说日语）/ 毎日(まいにち)聞(き)くようにする（尽量每天听） |

**核心区别**：

| 形式 | 重点 | 例 |
| --- | --- | --- |
| 〜になる | 自然变化，主语自己变 | 春(はる)になると暖(あたた)かくなる。（到春天就变暖。） |
| 〜にする / 〜くする | 人为处理，让某物变成某状态 | 電気(でんき)をつけて、部屋(へや)を明(あか)るくした。（开灯把房间弄亮了。） |
| 〜ようになる | 能力、习惯、状态逐渐形成 | 毎日(まいにち)練習(れんしゅう)して、泳(およ)げるようになった。（每天练习后变得会游泳了。） |
| 〜ようにする | 主动努力保持某行为 | 寝(ね)る前(まえ)に単語(たんご)を復習(ふくしゅう)するようにしている。（睡前会尽量复习单词。） |

### 样态与相似表达

| 表达 | 接续 | 含义 | 例 |
| --- | --- | --- | --- |
| 名词 + のようだ | 名词 | 像……一样，偏书面 | 夢(ゆめ)のようだ。（像梦一样。） |
| 普通形 + ようだ | 句子 | 看样子/似乎，偏书面 | 雨(あめ)が降(ふ)ったようだ。（好像下过雨。） |
| 名词/普通形 + みたいだ | 口语 | 像……/好像…… | 子供(こども)みたいだ。（像小孩一样。） |
| い形词干/な形词干/Vます形 + そうだ | 外观迹象 | 看起来要/看起来很 | おいしそうだ。（看起来好吃。）/ 雨(あめ)が降(ふ)りそうだ。（像要下雨。） |

**ようだ vs みたいだ vs そうだ**：

| 想表达 | 选哪个 | 例 |
| --- | --- | --- |
| "像……一样"的比喻 | ようだ / みたいだ | 彼(かれ)は子供(こども)のようだ / 子供(こども)みたいだ。 |
| 根据样子判断"好像已经是某状态" | ようだ / みたいだ | 彼(かれ)は疲(つか)れているようだ。 |
| 眼前看起来"马上要发生/看起来很……" | そうだ | 泣(な)きそうだ。/ 楽(たの)しそうだ。 |

更完整的「そうだ・ようだ・らしい・みたいだ」区别见 [[引用传闻表达]]。

## 易混对比

| 易混组 | 判断口令 |
| --- | --- |
| ために / ように | 意志动作用 ために，状态达成用 ように |
| つもり / 予定(よてい) | 心里打算=つもり，日程安排=予定 |
| ませんか / ましょう | 给对方选择=ませんか，推动一起做=ましょう |
| 可能形 / ことができる | 日常能力=可能形，正式说明=ことができる |
| 被动 / 可能 | 有动作主に多为被动，有能力语境多为可能 |
| 使役 / 使役被动 | 让别人做=使役，被迫自己做=使役被动 |
| なる / する | 自然变化=なる，人为改变=する |
| ようだ / みたいだ / そうだ | 比喻相似=ようだ/みたいだ，眼前迹象=そうだ |
