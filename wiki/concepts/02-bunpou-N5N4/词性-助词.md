# 词性(助词)

> 助词接在名词、短语或句子后面，标记语法关系、话题、限定、添加、语气等意义。本页统一处理 N5-N4 核心助词，避免在 `は/が`、格助词、副助词之间来回跳。

## 一、助词和助动词

| 项目 | 助词 | 助动词 |
| --- | --- | --- |
| 接在哪里 | 名词、短语、句子后 | 动词、形容词、名词谓语后 |
| 主要作用 | 标记关系、话题、范围、语气 | 附加否定、希望、推量、被动、使役等 |
| 是否参与谓语活用 | 通常不参与 | 参与活用体系 |
| 例子 | は、が、を、に、で、も、だけ、しか | ない、たい、そうだ、らしい、れる、せる |

判断口令：接在名词后，说明名词在句子里的角色，多半是助词；接在谓语后，改变谓语意义，多半是助动词。助动词详见 [[词性(助动词)]]。

## 二、は 与 が

```text
は = 主题/对比
  关于〜、说到〜、至于〜
  常把已知信息拿出来继续说明

が = 主语/新信息/排他指定
  〜才是、正是〜
  常指出听者不知道的新信息
```

| 场景 | 用法 | 例 |
| --- | --- | --- |
| 自我介绍/已知话题 | は | 私(わたし)**は**田中(たなか)です。 |
| 首次登场/新信息 | が | 昔々(むかしむかし)、おじいさん**が**いました。 |
| 对比 | は | コーヒー**は**飲(の)みますが、紅茶(こうちゃ)**は**飲(の)みません。 |
| 排他指定 | が | 田中(たなか)さん**が**犯人(はんにん)です。 |
| 疑问词 | が | 誰(だれ)**が**来(き)ましたか。 |
| 从属节内部 | が | 私(わたし)**が**書(か)いた本(ほん)。 |

`は` 可以覆盖原本用 `が` 或 `を` 标记的位置，把那个名词提升为话题。

| 中性句 | 主题化 | 说明 |
| --- | --- | --- |
| 雨(あめ)**が**降(ふ)っています。 | 雨(あめ)**は**降(ふ)っています。 | 说到雨，正在下 |
| 日本語(にほんご)**を**勉強(べんきょう)します。 | 日本語(にほんご)**は**勉強(べんきょう)します。 | 日语会学，可能对比别的语言 |

双层结构：

> 象(ぞう)**は**鼻(はな)**が**長(なが)い。  
> 日本(にほん)**は**物価(ぶっか)**が**高(たか)い。  
> 私(わたし)**は**頭(あたま)**が**痛(いた)い。

## 三、格助词

格助词标示名词与谓语之间的意义关系。日语不靠语序决定语法关系，而靠助词决定每个名词的角色。

| 助词 | 核心功能 | 代表例句 |
| --- | --- | --- |
| が | 主语/存在物 | 雨(あめ)**が**降(ふ)る。 |
| を | 动作对象 | 本(ほん)**を**読(よ)む。 |
| に | 时间点、到达点、对象、存在地点 | 7時(しちじ)**に**起(お)きる。学校(がっこう)**に**行(い)く。 |
| で | 动作场所、手段、原因 | 図書館(としょかん)**で**勉強(べんきょう)する。 |
| へ | 方向 | 東京(とうきょう)**へ**向(む)かう。 |
| と | 共同、引用 | 友達(ともだち)**と**行(い)く。 |
| から | 起点 | 9時(くじ)**から**始(はじ)まる。 |
| まで | 终点 | 5時(ごじ)**まで**働(はたら)く。 |

### に vs で

| 情况 | 助词 | 例 | 判断 |
| --- | --- | --- | --- |
| 存在地点 | に | 公園(こうえん)**に**猫(ねこ)がいる。 | 静态存在点 |
| 动作地点 | で | 公園(こうえん)**で**遊(あそ)ぶ。 | 动作发生范围 |

判断口诀：`に = 静态点`，`で = 动态场`。

## 四、副助词

副助词不负责主语/宾语/地点这种骨架关系，而是在格关系上叠加限定、添加、程度、极端等语感。

| 系列 | 助词 | 语感 | 例 |
| --- | --- | --- | --- |
| 添加 | も | 也、都 | 私(わたし)**も**行(い)く。 |
| 添加 | さえ | 连〜都（极端例示） | 漢字(かんじ)**さえ**読(よ)めない。 |
| 添加 | まで | 连〜也（意外范围） | 先生(せんせい)**まで**泣(な)いた。 |
| 限定 | だけ | 只（中性） | 一杯(いっぱい)**だけ**飲(の)んだ。 |
| 限定 | しか | 只，必须接否定 | 一杯(いっぱい)**しか**飲(の)まなかった。 |
| 偏向 | ばかり | 老是、净是、刚刚 | ゲーム**ばかり**している。 |
| 程度 | ほど | 到〜程度 | 死(し)ぬ**ほど**疲(つか)れた。 |
| 程度 | くらい/ぐらい | 大约这种程度 | それ**くらい**できる。 |
| 例示 | など/なんか | 〜之类 | 本(ほん)**など**を読(よ)む。 |

`しか` 必须接否定：

> ×一杯(いっぱい)しか飲(の)んだ  
> ○一杯(いっぱい)しか飲(の)まなかった

## 五、组合与省略

### 5.1 格助词 + 副助词

`は/も` 会替换 `が/を`，但和 `に/で/から/と` 等通常叠加。

| 组合 | 例 |
| --- | --- |
| に + は | 学校(がっこう)**には**行(い)かない。 |
| に + も | 学校(がっこう)**にも**行(い)く。 |
| に + だけ | 彼(かれ)**にだけ**話(はな)した。 |
| から + しか | 東京(とうきょう)**からしか**来(こ)られない。 |
| で + さえ | 大人(おとな)**でさえ**難(むずか)しい。 |

### 5.2 口语省略

| 助词 | 可否省略 | 条件 |
| --- | --- | --- |
| を | 经常省略 | 日常口语中最常省略 |
| が | 可省略 | 疑问句、感叹句中多见 |
| は | 可省略 | 不言自明的主题可省 |
| に | 有限 | 方向的 に 可省，时间点的 に 不可 |
| で | 几乎不可 | 省略后意义容易不清 |

书面语和考试作文中基本不要省略助词。

## 六、易错点

| 误区 | 错误 | 正确 | 原因 |
| --- | --- | --- | --- |
| 把 `は/が` 都当主语 | ×誰**は**来ましたか | ○誰**が**来ましたか | 疑问词用が |
| 动作地点用に | ×図書館(としょかん)**に**勉強する | ○図書館(としょかん)**で**勉強する | 动作地点用で |
| 存在地点用で | ×机の上**で**本がある | ○机の上**に**本がある | 存在地点用に |
| 宾语漏を | ×日本語(にほんご)勉強する | ○日本語(にほんご)**を**勉強する | 对象要标を |
| しか接肯定 | ×一人しか来た | ○一人しか来なかった | しか必须接否定 |
