# 词性(助词)

> 助词(じょし)接在名词、短语或句子后面，标记语法关系、话题、限定、添加、语气等意义。它和助动词(じょどうし)不同：助动词接在动词・形容词后面，参与活用和语法意义变化。

## 一、助词和助动词不是一类东西

| 项目 | 助词 | 助动词 |
| --- | -- | --- |
| 接在哪里 | 名词、短语、句子后 | 动词、形容词、名词谓语后 |
| 主要作用 | 标记关系、话题、范围、语气 | 附加否定、希望、推量、被动、使役等 |
| 是否参与谓语活用 | 通常不参与 | 参与活用体系 |
| 例子 | は、が、を、に、で、も、だけ、しか | ない、たい、そうだ、らしい、れる、せる |

**判断口令**：  
接在名词后，告诉你“这个名词在句子里是什么角色” → 多半是助词。  
接在谓语后，改变“做不做、想不想、像不像、被不被做” → 多半是助动词。

> 私(わたし)**は**日本語(にほんご)**を**勉強(べんきょう)したい。  
> `は/を` 是助词；`たい` 是助动词。

## 二、助词的三大入口

### 2.1 主题助词：は / が

`は` 和 `が` 是日语最基础也最难的区分。

| 助词 | 核心功能 | 例 |
| --- | --- | --- |
| は | 提出话题：说到〜 | 私(わたし)**は**田中(たなか)です。 |
| が | 标出主语/新信息：正是〜 | 誰(だれ)**が**来(き)ましたか。 |

完整判别规则见 [[词性-助词(はが)]]。

### 2.2 格助词：名词在句子里的角色

格助词标示名词与谓语之间的关系。

| 助词 | 核心功能 | 例 |
| --- | --- | --- |
| が | 主语/存在物 | 雨(あめ)**が**降(ふ)る。 |
| を | 动作对象 | 本(ほん)**を**読(よ)む。 |
| に | 时间点、到达点、对象 | 7時(しちじ)**に**起(お)きる。 |
| で | 动作场所、手段、原因 | 図書館(としょかん)**で**勉強(べんきょう)する。 |
| へ | 方向 | 東京(とうきょう)**へ**行(い)く。 |
| と | 共同、引用 | 友達(ともだち)**と**行(い)く。 |
| から | 起点 | 9時(くじ)**から**始(はじ)まる。 |
| まで | 终点 | 5時(ごじ)**まで**働(はたら)く。 |

完整说明见 [[词性(格助词)]]。

### 2.3 副助词：在关系上叠加语义

副助词不负责“主语/宾语/地点”这种骨架关系，而是叠加限定、添加、程度、极端等语感。

| 助词 | 核心功能 | 例 |
| --- | --- | --- |
| も | 也、都 | 私(わたし)**も**行(い)く。 |
| だけ | 只 | 一杯(いっぱい)**だけ**飲(の)んだ。 |
| しか | 只，必须接否定 | 一杯(いっぱい)**しか**飲(の)まなかった。 |
| ばかり | 老是、刚刚、净是 | ゲーム**ばかり**している。 |
| さえ | 连〜都 | 漢字(かんじ)**さえ**読(よ)めない。 |
| ほど | 到〜程度 | 泣(な)きたい**ほど**嬉(うれ)しい。 |

完整说明见 [[词性(副助词)]]。

## 三、学习顺序

助词不要一上来全背，按句子骨架学习最稳：

```text
第一步：先会搭句子骨架
  は / が / を / に / で

第二步：再区分相似助词
  は vs が
  に vs で
  に vs へ

第三步：最后叠加语感
  も / だけ / しか / ばかり / さえ / ほど
```

推荐路径：

1. 先读 [[语法句型(はを)]]，建立最基础句型。
2. 再读 [[词性-助词(はが)]]，解决 `は/が`。
3. 再读 [[词性(格助词)]]，掌握 `を/に/で/へ/と/から/まで`。
4. 最后读 [[词性(副助词)]]，补 `も/だけ/しか/ばかり/さえ/ほど`。

## 四、最小对比表

| 想表达 | 用哪个 | 例 |
| --- | --- | --- |
| 说到某人/某物 | は | 私(わたし)**は**学生(がくせい)です。 |
| 谁/什么做动作 | が | 雨(あめ)**が**降(ふ)る。 |
| 做什么对象 | を | 本(ほん)**を**読(よ)む。 |
| 到某点/在某时间点 | に | 学校(がっこう)**に**行(い)く。 |
| 在某范围内做动作 | で | 学校(がっこう)**で**勉強(べんきょう)する。 |
| 也 | も | 私(わたし)**も**行(い)く。 |
| 只 | だけ | 一人(ひとり)**だけ**来(き)た。 |
| 才/仅仅 | しか + 否定 | 一人(ひとり)**しか**来(こ)なかった。 |

## 五、常见误区

| 误区 | 正确理解 |
| --- | --- |
| 把 `たい/ない/そうだ` 也归入助词 | 它们是助动词，详见 [[词性(助动词)]] |
| 把 `は` 和 `が` 都理解成主语 | `は` 是主题，`が` 才常标主语/新信息 |
| 把 `も/だけ/しか` 当格助词 | 它们是副助词，负责添加、限定、排他 |
| 学助词时只背中文翻译 | 要先看动词和名词关系，再选助词 |
