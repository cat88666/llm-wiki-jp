# 助動詞体系

> 助動詞は動詞・形容詞に後接して否定・希望・推量・伝聞・様態などの文法的意味を付加する。活用形への正確な接続が求められ、N5-N4 レベルの助動詞は上級文法の基盤。

| [一、助動詞の接続体系](#一助動詞の接続体系) | 未然形接続/連用形接続/終止形接続 |
| [二、否定の助動詞：ない](#二否定の助動詞ない) | 接続規則、ないで/なくて の使い分け |
| [三、希望の助動詞：たい・たがる](#三希望の助動詞たいたがる) | 自分/他者の希望表現 |
| [四、推量・様態の助動詞](#四推量様態の助動詞) | らしい・そうだ・ようだ・みたいだ |
| [五、伝聞と断定](#五伝聞と断定) | そうだ(伝聞) vs そうだ(様態)、だ/です |
| [六、受身・使役の助動詞](#六受身使役の助動詞) | れる/られる・せる/させる → L2 への橋渡し |
