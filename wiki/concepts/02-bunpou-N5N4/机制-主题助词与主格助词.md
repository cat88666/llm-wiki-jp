# 主题助词与主格助词

> 「は」标记主题（在谈论什么），「が」标记主语（谁/什么在做动作）。这两个助词在不同的语法层面运作，中文和英文中没有对应的区分。

## 一、主题与主语：两个不同层面的概念

```
は = 主题标记（Topic）
  「关于〜」「说到〜」
  → 提出旧信息（听者已知的事物）

が = 主语标记（Subject）
  「〜才是」「正是〜」
  → 提示新信息（对听者来说未知的事物）
```

> 田中(たなか)さん**は**学生(がくせい)です。（说到田中→是学生。——田中是已知信息。）
> 田中(たなか)さん**が**学生(がくせい)です。（谁是学生？→是田中。——田中是新信息。）

**为什么日语要区分这两个？** 中文靠语序和语境区分"在说谁"和"谁做了"，但日语语序灵活，需要助词明确标记。は 告诉听者"接下来要谈论这个东西"，が 告诉听者"做动作/处于状态的是这个"。两者解决的问题不同，所以不能互换。

## 二、五条判别规则

| # | 规则 | は | が | 例句 |
|---|------|---|---|------|
| 1 | **旧信息 vs 新信息** | 已在话题中的人/物 | 首次登场的人/物 | 昔(むかし)々(むかし)、おじいさん**が**いました。おじいさん**は**山(やま)へ行(い)きました。 |
| 2 | **对比** | 对比两项以上用は | — | コーヒー**は**飲(の)みますが、紅茶(こうちゃ)**は**飲(の)みません。（咖啡喝，红茶不喝。） |
| 3 | **排他** | — | 「不是别的，正是这个」 | 誰(だれ)**が**来(き)ましたか？ — 田中(たなか)さん**が**来(き)ました。（谁来了？——田中来了。） |
| 4 | **从属节内部** | 不能用は | 从属节内用が | [私(わたし)**が**書(か)いた]本(ほん)（× 私は書いた本）（我写的书。） |
| 5 | **存在句引入** | — | 新出现的事物 | あそこに猫(ねこ)**が**います。（那边有猫。） |

### 2.1 旧信息 vs 新信息（最重要的规则）

用故事展开看最清楚：

> むかしむかし、ある所(ところ)におじいさん**が**いました。（从前，有一位老爷爷。——首次登场＝新信息＝が）
> おじいさん**は**毎日(まいにち)山(やま)へ柴刈(しばか)りに行(い)きました。（老爷爷每天去山上砍柴。——已出现＝旧信息＝は）
> ある日(ひ)、大(おお)きな桃(もも)**が**流(なが)れてきました。（有一天，一个大桃子漂了过来。——首次登场＝新信息＝が）

**为什么"新信息用が"？** が 的核心功能是"指定"——在多个可能性中指出"就是这个"。首次登场的事物对听者来说是未知的，需要用 が 明确指出它的存在。一旦双方都知道了（旧信息），就切换到 は 作为话题继续谈论。

### 2.2 对比的「は」

对比两个以上的项目时用「は」。用「が」则没有对比语感。

> 肉(にく)**は**食(た)べますが、魚(さかな)**は**食(た)べません。（肉吃，但鱼不吃。）
> 日本語(にほんご)**は**難(むずか)しいですが、面白(おもしろ)いです。（日语虽难但有趣。）

### 2.3 排他的「が」

强调「不是别人/别的，正是这个」时用「が」。

> A：誰(だれ)**が**犯人(はんにん)ですか？（谁是犯人？）
> B：田中(たなか)さん**が**犯人(はんにん)です。（田中才是犯人。——排他：不是别人，是田中。）

### 2.4 从属节内必须用「が」

**为什么从属节不能用は？** は 的作用范围是整个句子（设定全句的话题），但从属节只是句子的一部分。在从属节内部用 は 会导致话题范围溢出，造成歧义。

> [私(わたし)**が**作(つく)った]料理(りょうり)（我做的菜。——从属节内，が 正确）
> × [私**は**作った]料理（は 的范围溢出从属节，语法错误）

## 三、典型场景对照

| 场景 | 用法 | 例句 | 原因 |
|------|------|------|------|
| 自我介绍 | は | 私(わたし)**は**田中(たなか)です。（我是田中。） | 将自己作为主题提出 |
| 感情对象 | が | 猫(ねこ)**が**好(す)きです。（喜欢猫。） | 感情的对象用 が 标记 |
| 能力对象 | が | 日本語(にほんご)**が**話(はな)せます。（会说日语。） | 能力的对象用 が 标记 |
| 天气 | が | 雨(あめ)**が**降(ふ)っています。（在下雨。） | 自然现象＝新出现的事态 |
| 疑问词句 | が | 何(なに)**が**ありますか？（有什么？） | 疑问词一律用 が |
| 描写已知对象 | は | この部屋(へや)**は**広(ひろ)いです。（这个房间很宽敞。） | 对已知对象进行描写 |

**反直觉：感情・能力的对象用 が，不用 を**

中文说"我喜欢猫"，"猫"是宾语。但日语「猫(ねこ)**が**好(す)きです」中，「猫」用 が 标记。

**为什么？** 「好(す)き」「嫌(きら)い」「できる」「わかる」「ほしい」等词不是动词，而是形容词或状态表达。它们的句型不是"主语→宾语→动词"，而是"主题は＋对象が＋状态"。对象处于引起感情/能力的位置，用 が 标记。

## 四、「象は鼻が長い」——は与が共存

这个句子是日语语法学的著名难题。は 与 が 同时出现，各司其职。

```
象(ぞう)は      鼻(はな)が      長(なが)い
  ↓              ↓              ↓
  主题            主语            谓语
(说到大象)       (鼻子)          (长)
```

**为什么能共存？** 「は」设定全句话题（接下来谈大象），「が」标记谓语的主语（长的是鼻子）。两者在不同层面运作：は 管信息结构，が 管语法关系，因此不冲突。

相同结构的句子：

> 日本(にほん)**は**物価(ぶっか)**が**高(たか)い。（说到日本，物价高。）
> 私(わたし)**は**頭(あたま)**が**痛(いた)い。（我头痛。）
> この店(みせ)**は**ラーメン**が**おいしい。（这家店拉面好吃。）

## 五、判定流程

```
选择 は 还是 が
  └─ 在从属节内（〜が…した + 名词）？
       └─ 是 → が（从属节内不能用 は）
  └─ 含有疑问词（誰/何/どれ）？
       └─ 是 → が（疑问词一律用 が）
  └─ 在做对比（A は…、B は…）？
       └─ 是 → は
  └─ 该名词首次登场（新信息）？
       └─ 是 → が
       └─ 否（已出现/已知） → は
  └─ 是感情・能力的对象（好き/できる/わかる）？
       └─ 是 → が
  └─ 犹豫不决 → は ← は 适用范围广，が 只在特定条件下使用；
                        但新信息引入和疑问词句中必须用 が
```
