## 二、变形规则

て形和た形的变化基本相同：把 `て/で` 换成 `た/だ` 即可。

| 动词类型 | 原形 | て形 | た形 | 规则 |
| --- | --- | --- | --- | --- |
| 一类 く | 書(か)く | 書(か)いて | 書(か)いた | く → いて/いた |
| 一类 ぐ | 泳(およ)ぐ | 泳(およ)いで | 泳(およ)いだ | ぐ → いで/いだ |
| 一类 す | 話(はな)す | 話(はな)して | 話(はな)した | す → して/した |
| 一类 う・つ・る | 買(か)う / 待(ま)つ / 帰(かえ)る | 買(か)って / 待(ま)って / 帰(かえ)って | 買(か)った / 待(ま)った / 帰(かえ)った | 促音便 |
| 一类 む・ぶ・ぬ | 読(よ)む / 遊(あそ)ぶ / 死(し)ぬ | 読(よ)んで / 遊(あそ)んで / 死(し)んで | 読(よ)んだ / 遊(あそ)んだ / 死(し)んだ | 撥音便 |
| 二类 | 食(た)べる | 食(た)べて | 食(た)べた | 去る + て/た |
| 三类 する | する | して | した | 不规则 |
| 三类 来(く)る | 来(く)る | 来(き)て | 来(き)た | 不规则 |

**例外**：行(い)く → 行(い)って / 行(い)った，不是 ×行いて / ×行いた。

## 三、て形常用文型

| 文型 | 含义 | 例句 | 详细去向 |
| --- | --- | --- | --- |
| Vて、V | 动作连接/先后 | 朝(あさ)起(お)きて、顔(かお)を洗(あら)う。 | [[接续表达体系]] |
| Vてください | 请求 | ここに名前(なまえ)を書(か)いてください。 | 本页 |
| Vてもいい | 许可 | 写真(しゃしん)を撮(と)ってもいいですか。 | 本页 |
| Vてはいけない | 禁止 | ここで泳(およ)いではいけません。 | 本页 |
| Vてから | 做完前项再做后项 | ご飯(はん)を食(た)べてから出(で)かける。 | [[接续表达体系]] |
| Vている | 正在做/结果状态 | 今(いま)食(た)べている。 / 窓(まど)が開(あ)いている。 | [[体貌]] |
| Vてみる | 试着做 | 一度(いちど)食(た)べてみる。 | 本页 |
| Vておく | 事先做好 | 資料(しりょう)を準備(じゅんび)しておく。 | [[体貌]] |
| Vてしまう | 做完/不小心做了 | 財布(さいふ)を忘(わす)れてしまった。 | [[体貌]] |
| Vてあげる/くれる/もらう | 行为授受 | 友達(ともだち)に手伝(てつだ)ってもらった。 | [[功能表达-授受表达]] |

### 3.1 〜ている：不是永远等于“正在”

| 动词类型 | 〜ている 的意思 | 例 |
| --- | --- | --- |
| 持续动作 | 正在做 | 走(はし)っている。正在跑。 |
| 瞬间变化 | 结果状态 | 結婚(けっこん)している。已婚。 |
| 自动词结果 | 某物处于某状态 | ドアが開(あ)いている。门开着。 |

## 四、た形常用文型

| 文型/用法 | 含义 | 例句 | 详细去向 |
| --- | --- | --- | --- |
| Vた | 过去发生 | 昨日(きのう)映画(えいが)を見(み)た。 | [[语法-基本句型]] |
| もう Vた | 已经完成 | もう宿題(しゅくだい)をやった。 | 本页 |
| Vたことがある | 曾经做过 | 日本(にほん)へ行(い)ったことがある。 | 本页 |
| Vたことがない | 从没做过 | 納豆(なっとう)を食(た)べたことがない。 | 本页 |
| Vたら | 如果/做完之后 | 駅(えき)に着(つ)いたら電話(でんわ)して。 | [[条件表达体系]] |
| Vたほうがいい | 最好做 | 早(はや)く寝(ね)たほうがいい。 | 本页 |
| Vたり、Vたりする | 列举代表动作 | 本(ほん)を読(よ)んだり、映画(えいが)を見(み)たりする。 | 本页 |
| Vたまま | 保持动作后的状态 | 窓(まど)を開(あ)けたまま寝(ね)た。 | [[体貌]] |

### 4.1 た形不只是真实过去

| 用法 | 例 | 说明 |
| --- | --- | --- |
| 过去 | 昨日(きのう)見(み)た。 | 过去发生 |
| 完成 | もう食(た)べた。 | 已经完成 |
| 发现 | あ、ここにあった。 | 发现原来在这里 |
| 想起 | 明日(あした)は会議(かいぎ)だった。 | 想起来某事，不一定是过去事件 |

## 五、易错点

### 5.1 中文“了”不等于日语た形

| 中文想法 | 容易误判 | 正确理解 |
| --- | --- | --- |
| “吃了” | 一定用た形 | 多数情况用た形，但“已经吃完”常说 もう食(た)べた。 |
| “门开了” | 用他动词动作进行 | 状态通常用自动词ている：ドアが開(あ)いている。 |
| “正在结婚” | 結婚(けっこん)している = 正在结婚 | 多数是“已婚状态”。 |

### 5.2 て形不能单独当“如果”

| 误用 | 正确 | 原因 |
| --- | --- | --- |
| 食(た)べて、おいしい。 | 食(た)べたら、おいしかった。 | 条件/发现要用「たら」，不是单独て形。 |
| 駅(えき)に着(つ)いて電話(でんわ)して。 | 駅(えき)に着(つ)いたら電話(でんわ)して。 | “到了之后就打电话”更自然用「たら」。 |

### 5.3 て形并列不等于随便连接

| 句子 | 判断 |
| --- | --- |
| 朝(あさ)起(お)きて、顔(かお)を洗(あら)って、学校(がっこう)へ行(い)く。 | 动作按顺序发生，自然。 |
| 暑(あつ)くて、窓(まど)を開(あ)けた。 | 前项是原因，自然。 |
| 本(ほん)を読(よ)んで、映画(えいが)が好きです。 | 前后关系弱，不自然。应拆句或改连接方式。 |
