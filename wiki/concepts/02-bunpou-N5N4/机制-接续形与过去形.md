# 接续形(せつぞくけい)与过去形(かこけい)

> テ形(けい)是动词最核心的接续形态——进行、完成、请求、并列等 20+ 语法模式都从テ形出发。タ形(けい)与テ形遵循相同的音便规则，用于表达过去、完成、经验、发现。

## 一、テ形与タ形的关系

テ形和タ形是**同一套音便规则**的两个出口——末尾是「て/で」就是テ形，是「た/だ」就是タ形。

```
テ形 → 接续枢纽
  ├─ 补助动词接续：〜ている / 〜てしまう / 〜てみる ...
  ├─ 并列动作：食(た)べて、飲(の)んで、寝(ね)た
  └─ 轻因果：遅(おく)れてすみません

タ形 → 时间・完了标记
  ├─ 过去：昨日(きのう)食(た)べた
  ├─ 完成：もう食(た)べた
  ├─ 发现：あ、ここにあった！
  └─ 条件：食(た)べたら → [[条件表达体系]]
```

**为什么要一起学？** 因为五段动词的音便变化完全相同，只是最后一个音不同（て↔た、で↔だ）。记住テ形就自动会タ形。

## 二、一段动词与不规则动词

规则：去掉词尾「る」，直接加「て/た」。

| 分类 | 辞书形(じしょけい) | テ形 | タ形 | 规则 |
|------|--------|------|------|------|
| 一段 | 食(た)べる | 食(た)べて | 食(た)べた | る→て/た |
| 一段 | 見(み)る | 見(み)て | 見(み)た | る→て/た |
| 不规则 | する | して | した | 死记 |
| 不规则 | 来(く)る | 来(き)て | 来(き)た | 死记 |

## 三、五段动词的音便规则

五段动词的テ形/タ形伴随**音便(おんびん)**（发音变化）。音便类型由词尾假名决定。

```
テ形变化判别流程
  └─ する/来(く)る？           → して/来(き)て          ← 先排除不规则
  └─ 一段动词？               → 词干+て               ← 去る即可
  └─ 五段动词 → 看词尾：
       ├─ う/つ/る → って/った                          ← 促音便
       ├─ む/ぬ/ぶ → んで/んだ                          ← 拨音便（注意浊音 で/だ）
       ├─ く → いて/いた                                 ← イ音便
       ├─ ぐ → いで/いだ                                 ← イ音便（浊音）
       └─ す → して/した                                 ← 无音便
```

| 词尾 | 音便类型 | 辞书形 | テ形 | タ形 | 记忆口诀 |
|------|---------|--------|------|------|---------|
| う | 促音便 | 買(か)う | 買(か)って | 買(か)った | う・つ・る → **っ** |
| つ | 促音便 | 持(も)つ | 持(も)って | 持(も)った | |
| る | 促音便 | 取(と)る | 取(と)って | 取(と)った | |
| む | 拨音便 | 読(よ)む | 読(よ)んで | 読(よ)んだ | む・ぬ・ぶ → **ん**（で/だ） |
| ぬ | 拨音便 | 死(し)ぬ | 死(し)んで | 死(し)んだ | |
| ぶ | 拨音便 | 遊(あそ)ぶ | 遊(あそ)んで | 遊(あそ)んだ | |
| く | イ音便 | 書(か)く | 書(か)いて | 書(か)いた | く → **い**て |
| ぐ | イ音便 | 泳(およ)ぐ | 泳(およ)いで | 泳(およ)いだ | ぐ → **い**で（浊音） |
| す | 无音便 | 話(はな)す | 話(はな)して | 話(はな)した | す → して（最规则） |

**反直觉：行(い)く 的例外**

「行(い)く」是く结尾，但不遵循イ音便：行(い)く → **行(い)って**（× 行(い)いて）。

**为什么 行(い)く 是例外？** 古语中「行く」的テ形是「行きて」，经过促音便化变成「行って」，而非イ音便的「行いて」。这条历史音变路径与其他く结尾动词不同，只能死记。

## 四、テ形的接续模式

| 文型 | 意义 | 例句 | JLPT |
|------|------|------|------|
| 〜ている | 进行/结果状态 | 今(いま)食(た)べている。（正在吃。）/ 窓(まど)が開(あ)いている。（窗户开着。） | N5 |
| 〜てください | 礼貌请求 | ここに書(か)いてください。（请写在这里。） | N5 |
| 〜てもいい | 许可 | 写真(しゃしん)を撮(と)ってもいいですか。（可以拍照吗？） | N5 |
| 〜てはいけない | 禁止 | ここで泳(およ)いではいけません。（这里禁止游泳。） | N5 |
| 〜てから | 时间先后 | ご飯(はん)を食(た)べてから出(で)かける。（吃完饭再出门。） | N4 |
| 〜てしまう | 完成/遗憾 | 全部(ぜんぶ)食(た)べてしまった。（全吃完了。/ 不小心全吃完了。） | N4 |
| 〜てあげる/もらう/くれる | 授受 | 友達(ともだち)に教(おし)えてあげた。（教了朋友。）→ [[授受表达]] | N4 |
| 〜ておく | 事前准备 | 明日(あした)の資料(しりょう)を準備(じゅんび)しておく。（提前准备明天的材料。） | N4 |
| 〜てみる | 尝试 | 一度(いちど)食(た)べてみてください。（请尝尝看。） | N4 |

> **〜ている 的双重性**：持续动词（食(た)べる、走(はし)る）+ ている → 进行态；瞬间动词（死(し)ぬ、結婚(けっこん)する）+ ている → 结果状态。判断依据是动词的时间特性，不是〜ている本身。详见 [[体貌]]。

## 五、タ形的用法

| 用法 | 例句 | 要点 |
|------|------|------|
| 过去 | 昨日(きのう)映画(えいが)を見(み)た。（昨天看了电影。） | 最基本用法 |
| 完成 | もう宿題(しゅくだい)をやった。（作业已经做了。） | 强调已完成 |
| 发现 | あ、ここにあった！（啊，在这里！） | 非过去，表当场发现 |
| 条件 | 駅(えき)に着(つ)いたら電話(でんわ)して。（到车站了就打电话。） | 〜たら = タ形+ら |
| 想起 | 明日(あした)は会議(かいぎ)だった。（明天有会议来着。） | 非过去，表想起已知信息 |

## 六、中文母语者误区

| 误用 | 正确 | 原因 |
|------|------|------|
| × 彼(かれ)は結婚(けっこん)している = "正在结婚" | ○ 結婚(けっこん)している = **已婚状态** | 瞬间动词的ている表结果状态，不是进行 |
| × 窓(まど)を開(あ)けている = "窗开着" | ○ 窓(まど)が開(あ)いている（窗开着） | 他动词ている → 动作进行中；自动词ている → 结果状态。详见 [[语态体系]] |
| × 食(た)べて、おいしい | ○ 食(た)べたら、おいしかった（吃了之后发现很好吃） | テ形单独不能表条件发现，需要〜たら |
| × 中文"了" = タ形 | ○ 两者有交集但不等同 | 「明日(あした)は会議(かいぎ)だった」（想起）和「ここにあった」（发现）用タ形但不对应中文"了" |

**为什么"了"≠タ形？** 中文的"了"标记变化或完成，日语タ形标记过去时或完了体。タ形还能表达发现、想起等与"过去"无关的语义——这些是中文"了"覆盖不到的。
