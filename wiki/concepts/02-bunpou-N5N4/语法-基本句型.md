## 一、三种基本句型

```
三种基本句型
├── 1、名词句    ──→ A は B です
│   例：私(わたし)は学生(がくせい)です。（我是学生。）
│
├── 2、形容词句  ──→ A は 形容词 です
│   例：この部屋(へや)は静(しず)かです。（这个房间很安静。）
│
└── 3、动词句    ──→ A は B を Vます
    例：私(わたし)は本(ほん)を読(よ)みます。（我读书。）

变形规则归属
├── 名词 / な形容词   ──→ 靠 です 变化
├── い形容词         ──→ 靠词尾 い 变化
└── 动词            ──→ 靠 ます 变化
```

## 二、名词句

```
名词句
├── 1、肯定  ──→   A は B です
│   例：私(わたし)は学生(がくせい)です。（我是学生。）
│
├── 2、否定
│   ├── ではありません    礼貌体（です 底层是 で＋ある，否定即 で（は）あり＋ません）
│   ├── じゃありません    口语缩略（では → じゃ）
│   └── じゃない         普通体/随意场合
│   例：田中(たなか)さんは先生(せんせい)ではありません。（田中不是老师。）
│
└── 3、疑问  ──→ A は B ですか
    例：これは本(ほん)ですか。（这是书吗？）
    ├── はい、そうです。（是的。）
    └── いいえ、違(ちが)います。（不对。）
```
---

## 三、形容词句

```
形容词句  ──→ A は 形容词です
├── 1、い形容词   直接接 です
│   例：この本(ほん)はおもしろいです。（这本书很有趣。）
│
└── 2、な形容词   词干 + です（修饰名词时才带 な，谓语位置不带）
    例：この部屋(へや)は静(しず)かです。（这个房间很安静。）
```

## 四、动词句

```
动词句
├── 1、基本句型   ──→ A は B を Vます
│   例：私(わたし)は本(ほん)を読(よ)みます。（我读书。）
│   を 读 o（历史遗留：古代读 wo，现代变 o，只做宾语助词）
│
├── 2、地点
│   ├── 場所 + に + 移動動詞    目的地 / 存在地点（标"静态的点"）
│   │   例：学校(がっこう)に行(い)きます。（去学校。）
│   └── 場所 + で + 動作動詞   动作发生的场所（标"有事在发生的舞台"）
│       例：図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。）
│
└── 3、时间点   ──→ 時間 + に + 動詞
    ├── 加に：7時(しちじ)に、月曜日(げつようび)に、3日(みっか)に  ←── 时间轴上的具体刻度
    └── 不加に：今日(きょう)・明日(あした)・昨日(きのう)・毎日(まいにち)・今(いま)
    例：7時(しちじ)に起(お)きます。（7点起床。）
```

## 五、时态变化

日语 2 个时态（现在/将来・过去）× 2 个极性（肯定・否定）= 4 种变化。

### 丁寧体（礼貌体）

```
时态变化（丁寧体）
├── 名词句
│   ├── 现在肯定  学生(がくせい)です（是学生）
│   ├── 现在否定  学生(がくせい)ではありません（不是学生）
│   ├── 过去肯定  学生(がくせい)でした（以前是学生）
│   └── 过去否定  学生(がくせい)ではありませんでした（以前不是学生）
│
├── い形容词
│   ├── 现在肯定  高(たか)いです（贵）
│   ├── 现在否定  高(たか)くないです（不贵）
│   ├── 过去肯定  高(たか)かったです（以前贵）
│   └── 过去否定  高(たか)くなかったです（以前不贵）
│
├── な形容词
│   ├── 现在肯定  静(しず)かです（安静）
│   ├── 现在否定  静(しず)かではありません（不安静）
│   ├── 过去肯定  静(しず)かでした（以前安静）
│   └── 过去否定  静(しず)かではありませんでした（以前不安静）
│
└── 动词
    ├── 现在肯定  行(い)きます（去）
    ├── 现在否定  行(い)きません（不去）
    ├── 过去肯定  行(い)きました（去了）
    └── 过去否定  行(い)きませんでした（没去）
```

### 普通体（基本形）

| 谓语类型 | 现在肯定 | 现在否定 | 过去肯定 | 过去否定 |
| --- | --- | --- | --- | --- |
| 名词 | 学生(がくせい)だ | 学生(がくせい)じゃない | 学生(がくせい)だった | 学生(がくせい)じゃなかった |
| な形容词 | 静(しず)かだ | 静(しず)かじゃない | 静(しず)かだった | 静(しず)かじゃなかった |
| い形容词 | 高(たか)い | 高(たか)くない | 高(たか)かった | 高(たか)くなかった |
| 动词 | 行(い)く | 行(い)かない | 行(い)った | 行(い)かなかった |

**为什么名词和な形容词变化方式相同？** な形容词本身没有活用能力，靠「だ」系统变化——与名词谓语机制完全一致。

**压缩记忆**：

| 类型 | 否定 | 过去 | 过去否定 |
| --- | --- | --- | --- |
| 名词/な形 | じゃない | だった | じゃなかった |
| い形 | くない | かった | くなかった |
| 动词 | ない形 | た形 | ない→なかった |

## 六、指示词与疑问词

### こそあど（指示词体系）

日语指示词按「近・中・远・疑问」四列展开。

| | 近（说话者一侧） | 中（听话者一侧） | 远（双方都远） | 疑问 |
| --- | --- | --- | --- | --- |
| 事物（独立） | **これ** | **それ** | **あれ** | **どれ** |
| 修饰名词 | **この + N** | **その + N** | **あの + N** | **どの + N** |
| 场所 | **ここ** | **そこ** | **あそこ** | **どこ** |
| 方向/礼貌 | **こちら** | **そちら** | **あちら** | **どちら** |

> これは本(ほん)です。（这是书。）
> その映画(えいが)はおもしろいですか。（那部电影有趣吗？）
> あそこに図書館(としょかん)があります。（那边有图书馆。）

**反直觉**：この/その/あの 是连体修饰形，**必须后接名词**，不能独立使用。独立指代用 これ/それ/あれ。
> ×この（单独）→ ○これ（单独指代事物）

### 疑问词句型

| 疑問詞 | 含义 | 例句 | JLPT |
| --- | --- | --- | --- |
| 何(なに/なん) | 什么 | これは**何(なん)**ですか。（这是什么？） | N5 |
| 誰(だれ) | 谁 | **誰(だれ)が**来(き)ましたか。（谁来了？） | N5 |
| どこ | 哪里 | 駅(えき)は**どこ**ですか。（车站在哪里？） | N5 |
| いつ | 什么时候 | **いつ**来(き)ますか。（什么时候来？） | N5 |
| どれ / どの + N | 哪个 | **どれ**が田中(たなか)さんのですか。（哪个是田中的？） | N5 |
| どう | 怎么（状态/感受） | 日本語(にほんご)は**どう**ですか。（日语怎么样？） | N5 |
| なぜ / どうして | 为什么 | **なぜ**遅(おく)れましたか。（为什么迟到？） | N4/N5 |
| いくら | 多少钱 | これは**いくら**ですか。（这多少钱？） | N5 |
| いくつ | 几个/几岁 | りんごは**いくつ**ありますか。（苹果有几个？） | N5 |

**中文母语者误区**：疑问词作主语时必须用「が」，不用「は」。
> ×誰**は**来ましたか → ○誰**が**来ましたか（→ [[词性-助词(はが)]]）

## 七、语序与助词

```
基本语序
[时间] → [主题は] → [地点] → [对象を] → [动词]

例：私(わたし)は 毎日(まいにち) 家(いえ)で 日本語(にほんご)を 勉強(べんきょう)します。
    主题は       时间           地点で       对象を              动词（永远最后）

助词标角色（语序调换意思不变，靠助词识别成分）
├── は    主题（提出话题："说到 A"）
├── が    主语 / 存在物
├── を    动作对象（宾语）
├── に    时间点 / 目的地 / 存在地点
└── で    动作场所 / 手段工具
```

## 八、终助词

```
终助词（置句末，标记说话者态度）
├── か    疑问（显式问句标记，礼貌体靠 か 而非语调）
│   例：これは本(ほん)ですか。（这是书吗？）
│
├── ね    确认/共鸣（"你应该也有同感吧"）
│   例：今日(きょう)は暑(あつ)いですね。（今天很热呢。）
│   注：去掉 ね 只是陈述，加上 ね 是建立共鸣
│
├── よ    告知/提醒（"你可能不知道，我来告诉你"）
│   例：明日(あした)は休(やす)みですよ。（明天休息哦。）
│   注：对长辈慎用，底层含"我知道你不知道"，显得居高临下
│
└── な/なあ  感叹（自言自语，感慨/感动，不指向听话者）
    例：きれいだなあ。（好漂亮啊。）/ 懐(なつ)かしいな。（真怀念啊。）
    注：比 ね 更内向，不期待对方回应；男性用语较多
```
