## 一、三种基本句型

```
三种基本句型
├── 1、名词句    ──→ A は B です
│   变形归属：名词靠 です 变化
│   ├── 肯定  A は B です
│   │   例：私(わたし)は学生(がくせい)です。（我是学生。）
│   ├── 否定
│   │   ├── ではありません    礼貌体（です 底层是 で＋ある，否定即 で（は）あり＋ません）
│   │   ├── じゃありません    口语缩略（では → じゃ）
│   │   └── じゃない         普通体/随意场合
│   │   例：田中(たなか)さんは先生(せんせい)ではありません。（田中不是老师。）
│   └── 疑问  A は B ですか
│       例：これは本(ほん)ですか。（这是书吗？）
│       ├── はい、そうです。（是的。）
│       └── いいえ、違(ちが)います。（不对。）
│
├── 2、形容词句  ──→ A は 形容词です
│   ├── い形容词   直接接 です
│   │   例：この本(ほん)はおもしろいです。（这本书很有趣。）
│   └── な形容词   词干 + です（修饰名词时才带 な，谓语位置不带）
│       例：この部屋(へや)は静(しず)かです。（这个房间很安静。）
│
└── 3、动词句    ──→ A は B を Vます
    ├── 基本句型
    │   例：私(わたし)は本(ほん)を読(よ)みます。（我读书。）
    │   を 读 o（历史遗留：古代读 wo，现代变 o，只做宾语助词）
    ├── 地点
    │   ├── 場所 + に + 移動動詞    目的地 / 存在地点（标"静态的点"）
    │   │   例：学校(がっこう)に行(い)きます。（去学校。）
    │   └── 場所 + で + 動作動詞   动作发生的场所（标"有事在发生的舞台"）
    │       例：図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。）
    └── 时间点   ──→ 時間 + に + 動詞
        ├── 加に：7時(しちじ)に、月曜日(げつようび)に、3日(みっか)に  ←── 时间轴上的具体刻度
        └── 不加に：今日(きょう)・明日(あした)・昨日(きのう)・毎日(まいにち)・今(いま)
        例：7時(しちじ)に起(お)きます。（7点起床。）
```

### 高频句型骨架

```
高频句型骨架
├── 1、名词句扩展：ある / いる 系
│   ├── A は B が あります/います   A 有 B
│   │   例：私(わたし)は時間(じかん)があります。（我有时间。）
│   └── 場所 に N が あります/います 某处有 N
│       例：机(つくえ)の上(うえ)に本(ほん)があります。（桌上有书。）
│
├── 2、形容词句：A は 形容词です 系
│   ├── A は い形容词です           A 很...
│   │   例：この料理(りょうり)はおいしいです。（这道菜好吃。）
│   ├── A は い形容词くないです     A 不...
│   │   例：今日は寒(さむ)くないです。（今天不冷。）
│   ├── A は な形容词です           A 很...
│   │   例：この町(まち)は静(しず)かです。（这个城市安静。）
│   ├── A は な形容词ではありません A 不...
│   │   例：この仕事(しごと)は簡単(かんたん)ではありません。（这份工作不简单。）
│   └── A より B のほうが 形容词です B 比 A 更...
│       例：大阪(おおさか)より東京(とうきょう)のほうが大(おお)きいです。（东京比大阪大。）
│
└── 3、动词句：A は B を Vます 系
    ├── A は B を Vます             做某事
    │   例：私(わたし)は日本語(にほんご)を勉強(べんきょう)します。（我学日语。）
    ├── 場所 で Vます               在某地做动作
    │   例：図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。）
    ├── 場所 に 行きます/来ます/帰ります 去/来/回某地
    │   例：学校(がっこう)に行(い)きます。（去学校。）
    ├── Vて います                  正在做 / 状态持续
    │   例：雨(あめ)が降(ふ)っています。（正在下雨。）
    ├── Vて ください                请做...
    │   例：名前(なまえ)を書(か)いてください。（请写名字。）
    ├── Vてもいいです               可以做...
    │   例：写真(しゃしん)を撮(と)ってもいいです。（可以拍照。）
    ├── Vないでください             请不要做...
    │   例：ここでたばこを吸(す)わないでください。（请不要在这里抽烟。）
    ├── Vたいです                   想做...
    │   例：日本(にほん)に行(い)きたいです。（想去日本。）
    ├── Vたことがあります           曾经做过...
    │   例：京都(きょうと)へ行(い)ったことがあります。（去过京都。）
    ├── Vることができます           能够做...
    │   例：日本語(にほんご)を読(よ)むことができます。（能读日语。）
    ├── Vなければなりません         必须做...
    │   例：薬(くすり)を飲(の)まなければなりません。（必须吃药。）
    └── Vましょう / Vませんか       一起做吧 / 要不要做
        例：一緒(いっしょ)に行(い)きましょう。（一起去吧。）
        例：映画(えいが)を見(み)ませんか。（要不要看电影？）
```

## 二、时态变化

### 丁寧体（礼貌体）

```
时态变化（丁寧体）
├── 名词句
│   ├── 现在肯定  学生(がくせい)です（是学生）
│   ├── 现在否定  学生(がくせい)ではありません（不是学生）
│   ├── 过去肯定  学生(がくせい)でした（以前是学生）
│   └── 过去否定  学生(がくせい)ではありませんでした（以前不是学生）
│
├── い形容词
│   ├── 现在肯定  高(たか)いです（贵）
│   ├── 现在否定  高(たか)くないです（不贵）
│   ├── 过去肯定  高(たか)かったです（以前贵）
│   └── 过去否定  高(たか)くなかったです（以前不贵）
│
├── な形容词
│   ├── 现在肯定  静(しず)かです（安静）
│   ├── 现在否定  静(しず)かではありません（不安静）
│   ├── 过去肯定  静(しず)かでした（以前安静）
│   └── 过去否定  静(しず)かではありませんでした（以前不安静）
│
└── 动词
    ├── 现在肯定  行(い)きます（去）
    ├── 现在否定  行(い)きません（不去）
    ├── 过去肯定  行(い)きました（去了）
    └── 过去否定  行(い)きませんでした（没去）
```

### 普通体（基本形）

```
普通体（基本形）
├── 名词
│   ├── 现在肯定  学生(がくせい)だ（是学生）
│   ├── 现在否定  学生(がくせい)じゃない（不是学生）
│   ├── 过去肯定  学生(がくせい)だった（以前是学生）
│   ├── 过去否定  学生(がくせい)じゃなかった（以前不是学生）
│   └── 压缩记忆  否定＝じゃない / 过去＝だった / 过去否定＝じゃなかった
│
├── な形容词
│   ├── 现在肯定  静(しず)かだ（安静）
│   ├── 现在否定  静(しず)かじゃない（不安静）
│   ├── 过去肯定  静(しず)かだった（以前安静）
│   ├── 过去否定  静(しず)かじゃなかった（以前不安静）
│   └── 压缩记忆  否定＝じゃない / 过去＝だった / 过去否定＝じゃなかった
│
├── い形容词
│   ├── 现在肯定  高(たか)い（贵）
│   ├── 现在否定  高(たか)くない（不贵）
│   ├── 过去肯定  高(たか)かった（以前贵）
│   ├── 过去否定  高(たか)くなかった（以前不贵）
│   └── 压缩记忆  否定＝くない / 过去＝かった / 过去否定＝くなかった
│
└── 动词
    ├── 现在肯定  行(い)く（去）
    ├── 现在否定  行(い)かない（不去）
    ├── 过去肯定  行(い)った（去了）
    ├── 过去否定  行(い)かなかった（没去）
    └── 压缩记忆  否定＝ない形 / 过去＝た形 / 过去否定＝ない→なかった
```

## 三、指示词与疑问词


| | 近（说话者一侧） | 中（听话者一侧） | 远（双方都远） | 疑问 |
| --- | --- | --- | --- | --- |
| 事物（独立） | **これ** | **それ** | **あれ** | **どれ** |
| 修饰名词 | **この + N** | **その + N** | **あの + N** | **どの + N** |
| 场所 | **ここ** | **そこ** | **あそこ** | **どこ** |
| 方向/礼貌 | **こちら** | **そちら** | **あちら** | **どちら** |



```
指示词・疑问词
├── 指示词：こ・そ・あ・ど
│   ├── 距离轴
│   │   ├── こ  近说话者：これ / この / ここ / こちら
│   │   ├── そ  近听话者：それ / その / そこ / そちら
│   │   ├── あ  双方都远：あれ / あの / あそこ / あちら
│   │   └── ど  疑问：どれ / どの / どこ / どちら
│   │
│   ├── 用法轴
│   │   ├── 事物独立  これ / それ / あれ / どれ
│   │   │   例：これは本(ほん)です。（这是书。）
│   │   ├── 修饰名词  このN / そのN / あのN / どのN
│   │   │   例：その映画(えいが)はおもしろいですか。（那部电影有趣吗？）
│   │   ├── 场所      ここ / そこ / あそこ / どこ
│   │   │   例：あそこに図書館(としょかん)があります。（那边有图书馆。）
│   │   └── 方向礼貌  こちら / そちら / あちら / どちら
│   │
│   └── 易错点
│       ├── この / その / あの / どの 必须后接名词
│       └── ×この（单独）→ ○これ（独立指代事物）
│
└── 疑问词：问什么就选什么
    ├── 什么    何(なに/なん)
    │   例：これは何(なん)ですか。（这是什么？）
    ├── 谁      誰(だれ)
    │   例：誰(だれ)が来(き)ましたか。（谁来了？）
    ├── 哪里    どこ
    │   例：駅(えき)はどこですか。（车站在哪里？）
    ├── 什么时候  いつ
    │   例：いつ来(き)ますか。（什么时候来？）
    ├── 哪个    どれ / どの + N
    │   例：どれが田中(たなか)さんのですか。（哪个是田中的？）
    ├── 怎么样  どう
    │   例：日本語(にほんご)はどうですか。（日语怎么样？）
    ├── 为什么  なぜ / どうして
    │   例：なぜ遅(おく)れましたか。（为什么迟到？）
    ├── 多少钱  いくら
    │   例：これはいくらですか。（这多少钱？）
    ├── 几个/几岁  いくつ
    │   例：りんごはいくつありますか。（苹果有几个？）
    └── 易错点
        └── 疑问词作主语用 が：×誰は来ましたか → ○誰が来ましたか（→ [[词性(助词)]]）
```

## 四、语序与助词

```
基本语序
[时间] → [主题は] → [地点] → [对象を] → [动词]

例：私(わたし)は 毎日(まいにち) 家(いえ)で 日本語(にほんご)を 勉強(べんきょう)します。
    主题は       时间           地点で       对象を              动词（永远最后）

助词标角色（语序调换意思不变，靠助词识别成分）
├── は    主题（提出话题："说到 A"）
├── が    主语 / 存在物
├── を    动作对象（宾语）
├── に    时间点 / 目的地 / 存在地点
└── で    动作场所 / 手段工具
```

## 五、终助词

```
终助词（置句末，标记说话者态度）
├── か    疑问（显式问句标记，礼貌体靠 か 而非语调）
│   例：これは本(ほん)ですか。（这是书吗？）
│
├── ね    确认/共鸣（"你应该也有同感吧"）
│   例：今日(きょう)は暑(あつ)いですね。（今天很热呢。）
│   注：去掉 ね 只是陈述，加上 ね 是建立共鸣
│
├── よ    告知/提醒（"你可能不知道，我来告诉你"）
│   例：明日(あした)は休(やす)みですよ。（明天休息哦。）
│   注：对长辈慎用，底层含"我知道你不知道"，显得居高临下
│
└── な/なあ  感叹（自言自语，感慨/感动，不指向听话者）
    例：きれいだなあ。（好漂亮啊。）/ 懐(なつ)かしいな。（真怀念啊。）
    注：比 ね 更内向，不期待对方回应；男性用语较多
```

## 六、存在表达

```
存在表达（有/在）
├── 先判断对象
│   ├── 无生物/植物/抽象事物 → ある
│   │   例：机(つくえ)、本(ほん)、木(き)、時間(じかん)、問題(もんだい) = 桌子、书、树、时间、问题
│   └── 人/动物 → いる
│       例：人(ひと)、子供(こども)、猫(ねこ)、魚(さかな) = 人、孩子、猫、鱼
│
├── 再判断句子重点
│   ├── 重点是"某处有什么/有谁" → 場所に N が ある/いる
│   │   例：机(つくえ)の上(うえ)に本(ほん)がある。= 桌子上有书。
│   │   例：教室(きょうしつ)に学生(がくせい)がいる。= 教室里有学生。
│   ├── 重点是"某物/某人在哪里" → N は 場所に ある/いる
│   │   例：本(ほん)は机(つくえ)の上(うえ)にある。= 书在桌子上。
│   │   例：田中(たなか)さんは教室(きょうしつ)にいる。= 田中在教室里。
│   └── 重点是"拥有/存在数量" → N が ある/いる
│       例：今(いま)、時間(じかん)がある。= 现在有时间。
│       例：兄(あに)が二人(ふたり)いる。= 我有两个哥哥。
│
└── 最后判断地点助词
    ├── 存在地点 → に
    │   例：図書館(としょかん)に本(ほん)がある。= 图书馆里有书。
    └── 动作地点 → で
        例：図書館(としょかん)で本(ほん)を読(よ)む。= 在图书馆读书。
```

| 判断点 | 用法 | 句型 | 例句 | 中文 |
| --- | --- | --- | --- | --- |
| 某处有什么 | 场所是已知信息，存在物是新信息 | 場所に N が ある/いる | 机(つくえ)の上(うえ)に本(ほん)がある。 | 桌子上有书。 |
| 某物在哪里 | 对象是已知信息，位置是新信息 | N は 場所に ある/いる | 本(ほん)は机(つくえ)の上(うえ)にある。 | 书在桌子上。 |
| 有某物 | 有物品、时间、问题、机会 | N がある | 時間(じかん)がある。 | 有时间。 |
| 有某人/动物 | 有亲属、朋友、宠物、人员数量 | N がいる | 猫(ねこ)がいる。 | 有猫。 |

| 形式 | ある | いる |
| --- | --- | --- |
| 普通否定 | ない | いない |
| 普通过去 | あった | いた |
| 礼貌肯定 | あります | います |
| 礼貌否定 | ありません | いません |
| 礼貌过去 | ありました | いました |

| 对比 | 助词 | 例句 | 中文 | 记忆点 |
| --- | --- | --- | --- | --- |
| 存在 | に | 図書館(としょかん)**に**本(ほん)がある。 | 图书馆里有书。 | 书“在”图书馆 |
| 动作 | で | 図書館(としょかん)**で**本(ほん)を読(よ)む。 | 在图书馆读书。 | 在图书馆“读书” |
