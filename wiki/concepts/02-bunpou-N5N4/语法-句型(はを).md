## 一、三种基本句型

| 类型 | 句型 | 例句 |
| --- | --- | --- |
| 名词句 | A は B です | 私(わたし)は学生(がくせい)です。（我是学生。） |
| 形容词句 | A は 形容词です | この部屋(へや)は静(しず)かです。（这个房间很安静。） |
| 动词句 | A は B を Vます | 私(わたし)は本(ほん)を読(よ)みます。（我读书。） |

**为什么分三类？** 因为句末谓语决定了整句的变形方式——名词和な形容词靠 `です` 变化，い形容词靠词尾 `い` 变化，动词靠 `ます` 变化。三套规则互不通用。

### 1.1 本页只讲造句骨架

中文主要靠语序区分"我吃饭"和"饭吃我"。日语靠助词给每个词贴角色标签，所以学习基本句型时先抓住四个位置：

```text
A は B です。          A 是 B。
A は B を Vます。      A 做 B。
場所 で Vます。        在某地做动作。
時間 に Vます。        在某个时间点做动作。
```

| 助词 | 本页中的最低理解 | 典型句型 | 专题页 |
| --- | --- | --- | --- |
| は | 提出话题："说到 A" | 私(わたし)**は**学生(がくせい)です。 | [[主题助词(はが)]] |
| が | 标出主语/存在物 | 机(つくえ)の上(うえ)に本(ほん)**が**あります。 | [[主题助词(はが)]] |
| を | 标出动作对象 | 本(ほん)**を**読(よ)みます。 | [[词性(格助词)]] |
| に | 时间点、目的地、存在地点 | 7時(しちじ)**に**起(お)きます。 | [[词性(格助词)]] |
| で | 动作发生的地点、手段 | 図書館(としょかん)**で**勉強(べんきょう)します。 | [[词性(格助词)]] |

本页不展开 `は/が` 的旧信息、新信息、排他、从属节等判别规则；那些规则集中放在 [[主题助词(はが)]]。这里先保证你能把最基础的句子搭出来。

**句末决定一切**：日语谓语放最后，句末的 `です / ます / ない / た` 决定礼貌度、时态、肯否定。中文"我**不**去"第二个字就出结论，日语 `行(い)きません` 要听到末尾 `ません` 才知道是否定。

## 二、名词句：A は B です

### 2.1 肯定：A は B です

> 私(わたし)は学生(がくせい)です。（我是学生。）

| 词 | 作用 |
| --- | --- |
| 私(わたし) | 我 |
| は | 主题助词。写作「は」但读 wa——历史遗留：古代助词 は 确实读 ha，后来发音演变为 wa，写法没改。同理助词 `へ` 读 e。只有做助词时如此，普通假名仍读 ha |
| 学生(がくせい) | 学生 |
| です | 礼貌判断助动词，相当于"是" |

`A は B です` 整体理解为"至于 A，是 B"，不要逐字对应中文。

### 2.2 否定：A は B ではありません

> 田中(たなか)さんは先生(せんせい)ではありません。（田中不是老师。）

| 表达 | 说明 |
| --- | --- |
| ではありません | `です` 的礼貌否定 |
| じゃありません | 口语缩略，常见 |
| じゃない | 普通体，随意场合 |

**为什么否定形这么长？** `です` 的底层结构是 `で＋ある`（ある = 存在/是），礼貌否定就是 `で（は）あり＋ません`。口语嫌长，把 `では` 缩成 `じゃ` → `じゃありません`，再简化 → `じゃない`。

### 2.3 疑问：A は B ですか

> これは本(ほん)ですか。（这是书吗？）

| 词 | 作用 |
| --- | --- |
| これ | 这个 |
| は | 主题 |
| 本(ほん) | 书 |
| ですか | 是……吗？ |

**为什么加 `か` 就变问句？** 日语礼貌体不靠语调升降来表示疑问，而是用 `か` 做显式的疑问标记，贴在句末即可，语序不变。

## 三、形容词句

形容词句用来描述一个主题的性质。基本模板是：

```text
A は 形容词です。       A 很……
A は 形容词ではありません。 A 不……
```

本页只把形容词放进句型里；い形容词/な形容词的分类、否定、过去、名词修饰和例外词，集中看 [[词性(形容词)]]。

### 3.1 い形容词句

> この本(ほん)はおもしろいです。（这本书很有趣。）

```text
この本は   おもしろいです。
主题        性质 + 礼貌结尾
```

い形容词可以直接放在 `です` 前面。否定和过去变化见 [[词性(形容词)]]。

### 3.2 な形容词句

> この部屋(へや)は静(しず)かです。（这个房间很安静。）

```text
この部屋は   静かです。
主题          性质 + 礼貌结尾
```

な形容词在礼貌句里也可以直接接 `です`。名词修饰、否定和过去变化见 [[词性(形容词)]]。

## 四、动词句

### 4.1 基本句型：A は B を Vます

> 私(わたし)は本(ほん)を読(よ)みます。（我读书。）

| 词 | 作用 |
| --- | --- |
| 私(わたし) | 我 |
| は | 主题 |
| 本(ほん) | 书 |
| を | 宾语助词。写作「を」读 o——和 `は` 读 wa 一样是历史遗留，古代读 wo，现代已变为 o。`を` 现在只做宾语助词，看到它就知道前面的词是动作对象 |
| 読(よ)みます | 读（礼貌体） |

```text
私は   本を   読みます。
主题    对象    动词
```

### 4.2 动作地点：場所(ばしょ)で V ます

> 図書館(としょかん)で勉強(べんきょう)します。（在图书馆学习。）

**に vs で**：都可以接地点，但含义不同。

| 助词 | 用于 | 例 |
| --- | --- | --- |
| に | 静止存在的地点 / 移动的目的地 | 学校(がっこう)に行(い)きます。（去学校。） |
| で | 动作发生的场所 / 工具手段 | 学校(がっこう)で勉強(べんきょう)します。（在学校学习。） |

**为什么同样是"地点"却用不同助词？** `に` 标记"存在位置"或"到达终点"，强调静态的点；`で` 标记"动作展开的舞台"，强调有事在发生。简记：`に` = 在哪里待着 / 往哪里去，`で` = 在哪里干活。

### 4.3 时间点：時間(じかん) + に

> 7時(しちじ)に起(お)きます。（7点起床。）

具体刻度（几点、星期几、几号）加 `に`：7時(しちじ)に、月曜日(げつようび)に。

今日(きょう)、明日(あした)、昨日(きのう)、毎日(まいにち) **不加** `に`。

**为什么？** `に` 标记的是"时间坐标轴上的一个刻度"。而"今天、明天、每天"这些词本身已经定位了时间，不需要再标记。类似的还有 `今(いま)`（现在）也不加 `に`。

## 五、时态变化总表

日语礼貌体只有两个时态（现在/将来、过去）× 两个极性（肯定、否定）= 四种变化。

| 谓语类型 | 现在肯定 | 现在否定 | 过去肯定 | 过去否定 |
| --- | --- | --- | --- | --- |
| 名词 | 学生(がくせい)です（是学生） | 学生(がくせい)ではありません（不是学生） | 学生(がくせい)でした（以前是学生） | 学生(がくせい)ではありませんでした（以前不是学生） |
| い形容词 | 高(たか)いです（贵） | 高(たか)くないです（不贵） | 高(たか)かったです（以前贵） | 高(たか)くなかったです（以前不贵） |
| な形容词 | 静(しず)かです（安静） | 静(しず)かではありません（不安静） | 静(しず)かでした（以前安静） | 静(しず)かではありませんでした（以前不安静） |
| 动词 | 行(い)きます（去） | 行(い)きません（不去） | 行(い)きました（去了） | 行(い)きませんでした（没去） |

**动词礼貌体变形最规律**：`ます`（肯定）→ `ません`（否定）→ `ました`（过去）→ `ませんでした`（过去否定）。`ます` 前面的部分（連用形）始终不变，所有变化集中在 `ます` 这个词尾上。

### 5.1 疑问句

任何陈述句末尾加 `か` 即变为疑问句。

> 明日(あした)、学校(がっこう)に行(い)きますか。（明天去学校吗？）

回答：

| 回答 | 中文 |
| --- | --- |
| はい、行(い)きます。 | 是的，去。 |
| いいえ、行(い)きません。 | 不，不去。 |

## 六、存在句

→ 详见 [[存在(あるいる)]]

### 6.1 あります（无生命物体、植物）

> 机(つくえ)の上(うえ)に本(ほん)があります。（桌上有书。）

| 词 | 作用 |
| --- | --- |
| 机(つくえ) | 桌子 |
| の | 的 |
| 上(うえ) | 上面 |
| に | 存在地点 |
| 本(ほん) | 书 |
| が | 标出存在物 |
| あります | 有/在（无生命物） |

### 6.2 います（人、动物）

> 教室(きょうしつ)に学生(がくせい)がいます。（教室里有学生。）

**あります vs います**：

| 动词 | 对象 | 例 |
| --- | --- | --- |
| あります | 无生命物：书、桌子、钱、时间、植物 | 本(ほん)があります。（有书。） |
| います | 有生命且能自主移动：人、猫、狗 | 猫(ねこ)がいます。（有猫。） |

**为什么"有"分两个词？** 日语按"能否自主移动"区分。人和动物会自己动 → `います`；书、桌子不会自己动 → `あります`。

**植物用 `あります`**：虽然植物有生命，但不会自主移动。日语的 `います` 看重的是"自主移动能力"，不是生物学上的"生命"。

## 七、语序与助词

### 7.1 助词标角色，语序可调

```text
私(わたし)は ご飯(はん)を 食(た)べます。（我吃饭。）
  主题         对象        动词
```

因为助词标记了每个词的角色，`ご飯(はん)を 私(わたし)は 食(た)べます` 语序变了但意思不变——`を` 标着"饭是对象"，`は` 标着"我是主题"。

**铁律：动词永远放最后。**

### 7.2 基本语序

```text
[时间] [主题] [地点] [对象] [动词]
```

> 私(わたし)は毎日(まいにち)、家(いえ)で日本語(にほんご)を勉強(べんきょう)します。（我每天在家学习日语。）

| 词 | 作用 |
| --- | --- |
| 私(わたし)は | 主题 |
| 毎日(まいにち) | 时间 |
| 家(いえ)で | 动作地点 |
| 日本語(にほんご)を | 学习对象 |
| 勉強(べんきょう)します | 动词 |

### 7.3 常见错误

| 错误 | 原因 | 正确 |
| --- | --- | --- |
| 私(わたし)は学生(がくせい)。 | 缺 `です`，礼貌体必须有判断词 | 私(わたし)は学生(がくせい)です。（我是学生。） |
| 図書館(としょかん)**に**勉強(べんきょう)します。 | 动作地点用 `で`，不用 `に` | 図書館(としょかん)**で**勉強(べんきょう)します。（在图书馆学习。） |
| 本(ほん)読(よ)む。 | 缺宾语助词 `を` | 本(ほん)**を**読(よ)みます。（读书。） |

## 八、终助词

终助词放句末，标记语气。

| 助词 | 功能 | 例 |
| --- | --- | --- |
| か | 疑问 | これは本(ほん)ですか。（这是书吗？） |
| ね | 确认/共鸣 | 今日(きょう)は暑(あつ)いですね。（今天很热呢。） |
| よ | 告知/提醒 | これは大切(たいせつ)ですよ。（这个很重要哦。） |

**为什么需要终助词？** 中文靠语调和语气词（"吧""啊""呢"）传递态度，日语也类似，但更依赖句末显式标记：`ね` = 我觉得你也同意；`よ` = 你可能不知道，我来告诉你；`か` = 我在提问。

### 8.1 ね

> 今日(きょう)は寒(さむ)いですね。（今天很冷呢。）

`ね` 表示"你应该也有同感吧？"，用来建立共鸣。不加 `ね` 的 `今日(きょう)は寒(さむ)いです` 只是在陈述事实。

### 8.2 よ

> 明日(あした)は休(やす)みですよ。（明天休息哦。）

`よ` 表示"这件事你可能不知道，我告诉你"。**注意**：对上司或不熟的人频繁用 `よ` 会显得强硬，因为底层含义是"我知道你不知道"，有居高临下的感觉。对朋友自然，对长辈慎用。
