# 语法-能力表达

> 能力表达不是只有可能形。N5-N4 先掌握「可能形」「ことができる」「名词 + ができる」三条路，再区分口语自然度和正式度。

| 快速导航 | 概述 |
| --- | --- |
| [一、三种能力表达](#一三种能力表达) | 可能形・ことができる・名词ができる |
| [二、可能形变形入口](#二可能形变形入口) | 一类え段+る，二类去る+られる |
| [三、可能形和被动形](#三可能形和被动形) | 一段动词同形，靠语境判断 |

---

## 一、三种能力表达

```
能力表达
├── 1、可能形          口语首选，最自然
│   接续：动词变可能形
│   例：日本語(にほんご)が話(はな)せる。（会说日语。）
│
├── 2、ことができる    书面/正式
│   接续：辞书形 + ことができる
│   例：ここで予約(よやく)することができる。（可以在这里预约。）
│
└── 3、名詞 + ができる  会/能做某事
    例：料理(りょうり)ができる。（会做菜。）
```

---

## 二、可能形变形入口

| 动词类型 | 规则 | 例 |
| --- | --- | --- |
| 一类动词 | う段 → え段 + る | 書(か)く → 書(か)ける、話(はな)す → 話(はな)せる |
| 二类动词 | 去る + られる | 食(た)べる → 食(た)べられる、見(み)る → 見(み)られる |
| する | できる | 勉強(べんきょう)する → 勉強(べんきょう)できる |
| 来(く)る | 来(こ)られる | 来(く)る → 来(こ)られる |

---

## 三、可能形和被动形

| | 五段（不同形） | 一段（同形） | 判断法 |
| --- | --- | --- | --- |
| 可能 | 書(か)**ける**（え段+る） | 食(た)べ**られる** | 描述能力 |
| 被动 | 書(か)**かれる**（あ段+れる） | 食(た)べ**られる** | 有 `に` 标动作主 |

例：

> 日本語(にほんご)が話(はな)せる。（会说日语。）
> 先生(せんせい)に褒(ほ)められる。（被老师表扬。）
