# テ形・タ形

> テ形は動詞の最重要接続形：後ろに補助動詞・助詞・文型を接続するハブとして機能し、進行・完了・依頼・並列など 20 以上の文法パターンの起点となる。タ形はテ形と同じ音便規則に従い、過去・完了・経験を表す。

## 一、テ形・タ形の本質

テ形とタ形は**同じ音便規則**で変化する兄弟形。違いは末尾が「て/で」か「た/だ」かだけ。

```
テ形の役割
  └─ 接続ハブ：〜ている / 〜てください / 〜てから / 〜ても / 〜てあげる ...
  └─ 並列接続：食べて、飲んで、寝た（動作の列挙）
  └─ 原因理由：遅れて、すみません（軽い因果）

タ形の役割
  └─ 過去：昨日食べた
  └─ 完了：もう食べた
  └─ 条件：食べたら（[[条件表現]] 参照）
  └─ 発見：あ、ここにあった！
```

## 二、一段動詞・不規則動詞の変化

**規則は単純：語尾を取って「て/た」を付ける。**

| 分類 | 辞書形 | テ形 | タ形 | 規則 |
|------|--------|------|------|------|
| 一段 | 食べる | 食べて | 食べた | る→て/た |
| 一段 | 見る | 見て | 見た | る→て/た |
| 不規則 | する | して | した | 丸暗記 |
| 不規則 | 来る | 来（き）て | 来（き）た | 丸暗記 |

## 三、五段動詞の音便規則

五段動詞のテ形/タ形は**音便**（音の変化）を伴う。音便の種類は語尾の子音で決まる。

```
テ形変化判別フロー
  └─ 不規則（する/来る）？ → して/来て              ← 先に排除
  └─ 一段動詞？           → 語幹+て               ← る を取るだけ
  └─ 五段動詞 → 語尾で判定：
       ├─ う/つ/る → って/った                      ← 促音便
       ├─ む/ぬ/ぶ → んで/んだ                      ← 撥音便（濁音注意）
       ├─ く → いて/いた                             ← イ音便
       ├─ ぐ → いで/いだ                             ← イ音便（濁音）
       └─ す → して/した                             ← 音便なし
```

| 語尾 | 音便種類 | 辞書形 | テ形 | タ形 | 覚え方 |
|------|---------|--------|------|------|--------|
| う | 促音便 | 買う | 買って | 買った | う・つ・る→**っ** |
| つ | 促音便 | 持つ | 持って | 持った | |
| る | 促音便 | 取る | 取って | 取った | |
| む | 撥音便 | 読む | 読んで | 読んだ | む・ぬ・ぶ→**ん**（で/だ） |
| ぬ | 撥音便 | 死ぬ | 死んで | 死んだ | |
| ぶ | 撥音便 | 遊ぶ | 遊んで | 遊んだ | |
| く | イ音便 | 書く | 書いて | 書いた | く→**い**て |
| ぐ | イ音便 | 泳ぐ | 泳いで | 泳いだ | ぐ→**い**で（濁音） |
| す | 無音便 | 話す | 話して | 話した | す→して（規則的） |

**反直觉：唯一の例外**

「行く」は く行だがイ音便にならない：行く → **行って**（×行いて）。これは歴史的音変化の名残で、現代語では丸暗記。

## 四、テ形の主要接続パターン

| 文型 | 意味 | 例文 | JLPT |
|------|------|------|------|
| 〜ている | 進行/結果状態 | 今食べている（进行）/ 窓が開いている（结果） | N5 |
| 〜てください | 丁寧な依頼 | ここに書いてください。 | N5 |
| 〜てもいい | 許可 | 写真を撮ってもいいですか。 | N5 |
| 〜てはいけない | 禁止 | ここで泳いではいけません。 | N5 |
| 〜てから | 時間的先後 | ご飯を食べてから出かける。 | N4 |
| 〜てしまう | 完了/遺憾 | 全部食べてしまった。（吃完了/不小心吃完了） | N4 |
| 〜てあげる/もらう/くれる | 授受 | 友達に教えてあげた。 | N4 |
| 〜ておく | 事前準備 | 明日の会議の資料を準備しておく。 | N4 |
| 〜てみる | 尝试 | 一度食べてみてください。 | N4 |

> **〜ている的二重性**：持続動詞（食べる、走る）→ 进行态；瞬間動詞（死ぬ、結婚する）→ 结果状态态。判断依据是动词的时间特性，不是 〜ている 本身的意思。详见 [[アスペクト]]。

## 五、タ形の用法

| 用法 | 例文 | 中文 | 要点 |
|------|------|------|------|
| 過去 | 昨日映画を見た。 | 昨天看了电影。 | 最基本用法 |
| 完了 | もう宿題をやった。 | 作业已经做了。 | 強調完成 |
| 発見 | あ、ここにあった！ | 啊，在这里！ | 非过去，表发现 |
| 条件 | 駅に着いたら電話して。 | 到车站了就打电话。 | 〜たら = タ形+ら |
| 想起 | 明日は会議だった。 | 明天有会议来着。 | 非过去，表想起 |

## 六、中国語母語者の典型误区

| 误用 | 正确 | 原因分析 |
|------|------|---------|
| ×昨日買いました。とても好きです。 | ○昨日買いました。とても気に入りました。 | 中文"喜欢"是状态，日语「好き」也是状态，但叙述购买后感受时用「気に入る」的タ形更自然 |
| ×彼は結婚している（=正在结婚） | ○彼は結婚している（=已婚状态） | 瞬間動詞「結婚する」的テイル形表结果状态，不是进行 |
| ×窓を開けている（=正在开窗） | ○窓が開いている（窗户开着的状态） | 他動詞テイル→动作进行；自動詞テイル→结果状态 |
| ×食べて、おいしい | ○食べたら、おいしかった / 食べてみたら、おいしかった | テ形单独不能表条件发现，需要 〜たら |

> **核心原则**：中文的"了"≠タ形。中文"了"表示变化或完成，日语タ形表示过去时或完了体，两者有交集但不等同。「明日は会議だった」（想起）和「ここにあった」（发现）都用タ形但不对应中文的"了"。
