# テ形・タ形

> テ形是动词最重要的接续形：作为连接补助动词・助词・文型的枢纽，是进行・完成・请求・并列等 20 个以上语法模式的起点。タ形与テ形遵循相同的音便规则，表达过去・完成・经验。

## 一、テ形・タ形的本质

テ形与タ形是遵循**相同音便规则**变化的兄弟形。区别仅在于末尾是「て/で」还是「た/だ」。

```
テ形的角色
  └─ 接续枢纽：〜ている / 〜てください / 〜てから / 〜ても / 〜てあげる ...
  └─ 并列接续：食べて、飲んで、寝た（动作列举）
  └─ 原因理由：遅れて、すみません（轻因果）

タ形的角色
  └─ 过去：昨日食べた
  └─ 完成：もう食べた
  └─ 条件：食べたら（参照 [[条件表現]]）
  └─ 发现：あ、ここにあった！
```

## 二、一段动词・不规则动词的变化

**规则很简单：去掉词尾加「て/た」。**

| 分类 | 辞书形 | テ形 | タ形 | 规则 |
|------|--------|------|------|------|
| 一段 | 食べる | 食べて | 食べた | る→て/た |
| 一段 | 見る | 見て | 見た | る→て/た |
| 不规则 | する | して | した | 死记 |
| 不规则 | 来る | 来（き）て | 来（き）た | 死记 |

## 三、五段动词的音便规则

五段动词的テ形/タ形伴随**音便**（音的变化）。音便类型由词尾子音决定。

```
テ形变化判别流程
  └─ 不规则（する/来る）？ → して/来て              ← 先排除
  └─ 一段动词？           → 词干+て               ← 去掉る即可
  └─ 五段动词 → 根据词尾判定：
       ├─ う/つ/る → って/った                      ← 促音便
       ├─ む/ぬ/ぶ → んで/んだ                      ← 拨音便（注意浊音）
       ├─ く → いて/いた                             ← イ音便
       ├─ ぐ → いで/いだ                             ← イ音便（浊音）
       └─ す → して/した                             ← 无音便
```

| 词尾 | 音便类型 | 辞书形 | テ形 | タ形 | 记忆法 |
|------|---------|--------|------|------|--------|
| う | 促音便 | 買う | 買って | 買った | う・つ・る→**っ** |
| つ | 促音便 | 持つ | 持って | 持った | |
| る | 促音便 | 取る | 取って | 取った | |
| む | 拨音便 | 読む | 読んで | 読んだ | む・ぬ・ぶ→**ん**（で/だ） |
| ぬ | 拨音便 | 死ぬ | 死んで | 死んだ | |
| ぶ | 拨音便 | 遊ぶ | 遊んで | 遊んだ | |
| く | イ音便 | 書く | 書いて | 書いた | く→**い**て |
| ぐ | イ音便 | 泳ぐ | 泳いで | 泳いだ | ぐ→**い**で（浊音） |
| す | 无音便 | 話す | 話して | 話した | す→して（规则的） |

**反直觉：唯一的例外**

「行く」属于く行但不遵循イ音便：行く → **行って**（×行いて）。这是历史音变的遗留，现代语中需要死记。

## 四、テ形的主要接续模式

| 文型 | 意义 | 例句 | JLPT |
|------|------|------|------|
| 〜ている | 进行/结果状态 | 今食べている（进行）/ 窓が開いている（结果） | N5 |
| 〜てください | 礼貌请求 | ここに書いてください。（请写在这里。） | N5 |
| 〜てもいい | 许可 | 写真を撮ってもいいですか。（可以拍照吗？） | N5 |
| 〜てはいけない | 禁止 | ここで泳いではいけません。（这里禁止游泳。） | N5 |
| 〜てから | 时间先后 | ご飯を食べてから出かける。（吃完饭再出门。） | N4 |
| 〜てしまう | 完成/遗憾 | 全部食べてしまった。（全吃完了/不小心全吃完了。） | N4 |
| 〜てあげる/もらう/くれる | 授受 | 友達に教えてあげた。（教了朋友。） | N4 |
| 〜ておく | 事前准备 | 明日の会議の資料を準備しておく。（提前准备明天会议的材料。） | N4 |
| 〜てみる | 尝试 | 一度食べてみてください。（请尝尝看。） | N4 |

> **〜ている 的双重性**：持续动词（食べる、走る）→ 进行态；瞬间动词（死ぬ、結婚する）→ 结果状态。判断依据是动词的时间特性，不是 〜ている 本身。详见 [[アスペクト]]。

## 五、タ形的用法

| 用法 | 例句 | 中文 | 要点 |
|------|------|------|------|
| 过去 | 昨日映画を見た。 | 昨天看了电影。 | 最基本用法 |
| 完成 | もう宿題をやった。 | 作业已经做了。 | 强调完成 |
| 发现 | あ、ここにあった！ | 啊，在这里！ | 非过去，表发现 |
| 条件 | 駅に着いたら電話して。 | 到车站了就打电话。 | 〜たら = タ形+ら |
| 想起 | 明日は会議だった。 | 明天有会议来着。 | 非过去，表想起 |

## 六、中文母语者的典型误区

| 误用 | 正确 | 原因分析 |
|------|------|---------|
| ×昨日買いました。とても好きです。 | ○昨日買いました。とても気に入りました。 | 中文"喜欢"是状态，但叙述购买后感受时用「気に入る」的タ形更自然 |
| ×彼は結婚している（=正在结婚） | ○彼は結婚している（=已婚状态） | 瞬间动词「結婚する」的ている形表结果状态，不是进行 |
| ×窓を開けている（=正在开窗） | ○窓が開いている（窗户开着的状态） | 他动词ている→动作进行；自动词ている→结果状态 |
| ×食べて、おいしい | ○食べたら、おいしかった / 食べてみたら、おいしかった | テ形单独不能表条件发现，需要 〜たら |

> **核心原则**：中文的"了"≠タ形。中文"了"表示变化或完成，日语タ形表示过去时或完了体，两者有交集但不等同。「明日は会議だった」（想起）和「ここにあった」（发现）都用タ形但不对应中文的"了"。
