# 语法-连续型(てた)

> て形是动词最重要的接续形：作为连接补助动词・助词・文型的枢纽，是进行・完成・请求・并列等 20 个以上语法模式的起点。た形与て形遵循相同的音便规则，表达过去・完成・经验。

## 一、て形与た形的本质

て形与た形遵循**相同音便规则**变化，区别仅在于末尾是「て/で」还是「た/だ」。

**为什么て形如此重要？** 因为て形是日语最高频的接续点——进行态（〜ている）、请求（〜てください）、先后（〜てから）、许可（〜てもいい）、授受（〜てあげる）等大量文型都从て形接续。掌握了て形，就打开了 N5-N4 语法的大门。

```
て形的角色
  └─ 接续枢纽：〜ている / 〜てください / 〜てから / 〜ても / 〜てあげる ...
  └─ 并列接续：食(た)べて、飲(の)んで、寝(ね)た（动作列举）
  └─ 原因理由：遅(おく)れて、すみません（轻因果）

た形的角色
  └─ 过去：昨日(きのう)食(た)べた（昨天吃了）
  └─ 完成：もう食(た)べた（已经吃了）
  └─ 条件：食(た)べたら（如果吃了的话）→ [[条件表达体系]]
  └─ 发现：あ、ここにあった！（啊，在这里！）
```

## 二、一段动词・不规则动词的变化

规则很简单：去掉词尾加「て/た」。

| 分类 | 辞书形 | て形 | た形 | 规则 |
|------|--------|------|------|------|
| 一段 | 食(た)べる | 食(た)べて | 食(た)べた | る→て/た |
| 一段 | 見(み)る | 見(み)て | 見(み)た | る→て/た |
| 不规则 | する | して | した | 固定形式 |
| 不规则 | 来(く)る | 来(き)て | 来(き)た | 固定形式 |

## 三、五段动词的音便规则

五段动词的て形/た形伴随**音便**（音的变化）。音便类型由词尾决定。

**为什么五段动词要有音便？** 因为五段动词的词干以子音结尾，直接加「て」发音不自然。日语通过音便让发音更流畅，比如「書(か)く」+「て」→ 如果直接变成「書きて」，发音生硬，所以变成「書(か)いて」。

```
て形变化判别流程
  └─ 不规则（する/来る）？ → して/来(き)て                ← 先排除
  └─ 一段动词？           → 词干+て                      ← 去掉る即可
  └─ 五段动词 → 根据词尾判定：
       ├─ う/つ/る → って/った                            ← 促音便
       ├─ む/ぬ/ぶ → んで/んだ                            ← 拨音便（注意浊音）
       ├─ く → いて/いた                                   ← い音便
       ├─ ぐ → いで/いだ                                   ← い音便（浊音）
       └─ す → して/した                                   ← 无音便
```

| 词尾 | 音便类型 | 辞书形 | て形 | た形 |
|------|---------|--------|------|------|
| う | 促音便 | 買(か)う | 買(か)って | 買(か)った |
| つ | 促音便 | 持(も)つ | 持(も)って | 持(も)った |
| る | 促音便 | 取(と)る | 取(と)って | 取(と)った |
| む | 拨音便 | 読(よ)む | 読(よ)んで | 読(よ)んだ |
| ぬ | 拨音便 | 死(し)ぬ | 死(し)んで | 死(し)んだ |
| ぶ | 拨音便 | 遊(あそ)ぶ | 遊(あそ)んで | 遊(あそ)んだ |
| く | い音便 | 書(か)く | 書(か)いて | 書(か)いた |
| ぐ | い音便 | 泳(およ)ぐ | 泳(およ)いで | 泳(およ)いだ |
| す | 无音便 | 話(はな)す | 話(はな)して | 話(はな)した |

**反直觉：唯一的例外**

「行(い)く」属于く行但不遵循い音便：行(い)く → **行(い)って**（×行いて）。

**为什么 行く 是例外？** 这是历史音变的遗留。古日语中「行く」的て形确实是「行きて」，但在口语演变中变成了「行って」，并固定下来。现代语中只能硬记。

## 四、て形的主要接续模式

| 文型 | 意义 | 例句 | JLPT |
|------|------|------|------|
| 〜ている | 进行/结果状态 | 今(いま)食(た)べている。（正在吃。）/ 窓(まど)が開(あ)いている。（窗户开着。） | N5 |
| 〜てください | 礼貌请求 | ここに書(か)いてください。（请写在这里。） | N5 |
| 〜てもいい | 许可 | 写真(しゃしん)を撮(と)ってもいいですか。（可以拍照吗？） | N5 |
| 〜てはいけない | 禁止 | ここで泳(およ)いではいけません。（这里禁止游泳。） | N5 |
| 〜てから | 时间先后 | ご飯(はん)を食(た)べてから出(で)かける。（吃完饭再出门。） | N4 |
| 〜てしまう | 完成/遗憾 | 全部(ぜんぶ)食(た)べてしまった。（全吃完了/不小心全吃完了。） | N4 |
| 〜てあげる/もらう/くれる | 授受 | 友達(ともだち)に教(おし)えてあげた。（教了朋友。） | N4 |
| 〜ておく | 事前准备 | 明日(あした)の会議(かいぎ)の資料(しりょう)を準備(じゅんび)しておく。（提前准备明天会议的材料。） | N4 |
| 〜てみる | 尝试 | 一度(いちど)食(た)べてみてください。（请尝尝看。） | N4 |

> **〜ている 的双重性**：持续动词（食(た)べる、走(はし)る）→ 进行态；瞬间动词（死(し)ぬ、結婚(けっこん)する）→ 结果状态。判断依据是动词的时间特性，不是 〜ている 本身。详见 [[体貌]]。

## 五、た形的用法

| 用法 | 例句 | 中文 | 要点 |
|------|------|------|------|
| 过去 | 昨日(きのう)映画(えいが)を見(み)た。 | 昨天看了电影。 | 最基本用法 |
| 完成 | もう宿題(しゅくだい)をやった。 | 作业已经做了。 | 强调完成 |
| 发现 | あ、ここにあった！ | 啊，在这里！ | 非过去，表发现 |
| 条件 | 駅(えき)に着(つ)いたら電話(でんわ)して。 | 到车站了就打电话。 | 〜たら = た形+ら |
| 想起 | 明日(あした)は会議(かいぎ)だった。 | 明天有会议来着。 | 非过去，表想起 |

## 六、中文母语者的典型误区

**为什么中文"了"≠た形？** 中文的"了"表示变化或完成，日语た形表示过去时或完了体，两者有交集但不等同。

| 误用 | 正确 | 原因分析 |
|------|------|---------|
| ×彼(かれ)は結婚(けっこん)している（=正在结婚） | ○彼(かれ)は結婚(けっこん)している（=已婚状态） | 瞬间动词的ている形表结果状态，不是进行 |
| ×窓(まど)を開(あ)けている（=正在开窗） | ○窓(まど)が開(あ)いている（窗户开着的状态） | 他动词ている→动作进行；自动词ている→结果状态 |
| ×食(た)べて、おいしい | ○食(た)べたら、おいしかった | て形单独不能表条件发现，需要 〜たら |

> 「明日(あした)は会議(かいぎ)だった」（想起）和「ここにあった」（发现）都用た形但不对应中文的"了"。
