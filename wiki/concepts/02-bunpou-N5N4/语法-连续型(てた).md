# 语法-连续型(てた)

> て形是日语动词最重要的接续形，用来连接请求、许可、禁止、先后、进行、结果状态等文型。た形和て形的变形规则基本相同，只是把「て / で」换成「た / だ」。

| 快速导航 | 重点 |
| --- | --- |
| [一、て形和た形先分工](#て形和た形先分工) | て形负责连接后续文型，た形负责完成点和既发生状态 |
| [二、て形常用文型](#て形常用文型) | 〜ている、〜てください、〜てもいい、〜てから |
| [三、た形常用文型](#た形常用文型) | 过去、完成、发现、想起、〜たら |
| [四、中文母语者常见误区](#中文母语者常见误区) | 「了」不等于た形，ている不一定是正在 |

<a id="て形和た形先分工"></a>

## 一、て形和た形先分工

```
て形
  └─ 接续枢纽：〜ている / 〜てください / 〜てから / 〜ても / 〜てもいい / 〜てあげる ...
  └─ 请求：ここに名前(なまえ)を書(か)いてください（请做）
  └─ 进行：今(いま)勉強(べんきょう)している（正在做）
  └─ 结果状态：窓(まど)が開(あ)いている（做完后状态保留）
  └─ 许可：入(はい)ってもいいですか（可以做）
  └─ 禁止：ここで泳(およ)いではいけません（不可以做）
  └─ 动作先后：ご飯(はん)を食(た)べてから、薬(くすり)を飲(の)む（做完A再做B）
  └─ 并列接续：朝(あさ)起(お)きて、顔(かお)を洗(あら)って、学校(がっこう)へ行(い)く（动作按顺序列举）
  └─ 原因理由：遅(おく)れて、すみません（轻因果）
  └─ 习惯反复：毎日(まいにち)走(はし)っている（长期/反复做）
  └─ 禁止请求：写真(しゃしん)を撮(と)らないでください（请不要做）
  └─ 尝试：一度(いちど)食(た)べてみる（试着做）
  └─ 事先准备：資料(しりょう)を準備(じゅんび)しておく（预先做好）
  └─ 完成/遗憾：財布(さいふ)を忘(わす)れてしまった（做完/不小心做了）
  └─ 授受：友達(ともだち)に教(おし)えてあげた（为别人做/别人为我做/请别人做）
  └─ 让步：雨(あめ)が降(ふ)っても行(い)く（即使做A也B）
  └─ 补助移动：少(すこ)しずつ寒(さむ)くなってきた（变化过来/下去）
  └─ 人为结果留存：電気(でんき)がつけてある（有人做完后结果保留）

た形
  └─ 过去：昨日(きのう)映画(えいが)を見(み)た（过去做了）
  └─ 完成：もう宿題(しゅくだい)をやった（已经做完）
  └─ 经验：日本(にほん)へ行(い)ったことがある（曾经做过）
  └─ 条件：駅(えき)に着(つ)いたら電話(でんわ)して（如果/做完之后）→ [[条件表达体系]]
  └─ 否定经验：納豆(なっとう)を食(た)べたことがない（从没做过）
  └─ 发现：あ、ここにあった！（发现原来如此）
  └─ 想起：明日(あした)は会議(かいぎ)だった（想起来某事）
  └─ 建议评价：早(はや)く寝(ね)たほうがいい（最好做）
  └─ 列举代表：本(ほん)を読(よ)んだり、映画(えいが)を見(み)たりする（做A做B之类）
  └─ 保持原状：窓(まど)を開(あ)けたまま寝(ね)た（做完后状态不变）
  └─ 样态判断：雨(あめ)が降(ふ)ったようだ / 降(ふ)ったみたいだ（好像已经做了）
  └─ 传闻/说明：田中(たなか)さんはもう帰(かえ)ったそうだ（听说做了）
```

<a id="て形常用文型"></a>

## 二、て形常用文型

| 文型 | 含义 | 例句 | JLPT |
| --- | --- | --- | --- |
| 〜ている | 正在做；结果状态 | 今(いま)食(た)べている。正在吃。 / 窓(まど)が開(あ)いている。窗户开着。 | N5 |
| 〜てください | 请对方做 | ここに名前(なまえ)を書(か)いてください。请在这里写名字。 | N5 |
| 〜てもいい | 可以做 | 写真(しゃしん)を撮(と)ってもいいですか。可以拍照吗？ | N5 |
| 〜てはいけない | 不可以做 | ここで泳(およ)いではいけません。这里不能游泳。 | N5 |
| 〜てから | 做完前项再做后项 | ご飯(はん)を食(た)べてから出(で)かける。吃完饭再出门。 | N4 |
| 〜てみる | 试着做 | 一度(いちど)食(た)べてみてください。请尝尝看。 | N4 |
| 〜ておく | 事先做好 | 明日(あした)の資料(しりょう)を準備(じゅんび)しておく。提前准备明天的资料。 | N4 |
| 〜てしまう | 做完；不小心做了 | 財布(さいふ)を忘(わす)れてしまった。不小心忘了钱包。 | N4 |

### （一）〜ている：不是永远等于“正在”

| 动词类型 | 〜ている 的意思 | 例 |
| --- | --- | --- |
| 持续动作 | 正在做 | 走(はし)っている。正在跑。 |
| 瞬间变化 | 结果状态 | 結婚(けっこん)している。已婚。 |
| 自动词结果 | 某物处于某状态 | ドアが開(あ)いている。门开着。 |

<a id="た形常用文型"></a>

## 三、た形常用文型

| 用法 | 含义 | 例句 | 注意 |
| --- | --- | --- | --- |
| 过去 | 过去发生 | 昨日(きのう)映画(えいが)を見(み)た。昨天看了电影。 | 最基本用法 |
| 完成 | 已经做完 | もう宿題(しゅくだい)をやった。作业已经做了。 | 常和「もう」一起出现 |
| 经验 | 曾经做过 | 日本(にほん)へ行(い)ったことがある。去过日本。 | 「た形 + ことがある」 |
| 发现 | 发现原来在这里 | あ、ここにあった！啊，在这里！ | 中文不一定翻成“了” |
| 想起 | 想起来某事 | 明日(あした)は会議(かいぎ)だった。明天有会议来着。 | 非真实过去，而是“想起” |
| 条件 | 如果；做完之后 | 駅(えき)に着(つ)いたら電話(でんわ)して。到车站后给我打电话。 | 「た形 + ら」 |

<a id="中文母语者常见误区"></a>

## 四、中文母语者常见误区

### （一）中文“了”不等于日语た形

| 中文想法 | 容易误判 | 正确理解 |
| --- | --- | --- |
| “吃了” | 一定用た形 | 多数情况用た形，但“已经吃完”更强调完成：もう食(た)べた。 |
| “门开了” | 用他动词动作进行 | 状态通常用自动词ている：ドアが開(あ)いている。门开着。 |
| “正在结婚” | 結婚(けっこん)している = 正在结婚 | 結婚(けっこん)している 多数是“已婚状态”。 |

### （二）て形不能单独当“如果”

| 误用 | 正确 | 原因 |
| --- | --- | --- |
| 食(た)べて、おいしい。 | 食(た)べたら、おいしかった。 | 条件/发现要用「たら」，不是单独て形。 |
| 駅(えき)に着(つ)いて電話(でんわ)して。 | 駅(えき)に着(つ)いたら電話(でんわ)して。 | “到了之后就打电话”更自然用「たら」。 |

### （三）て形并列不等于随便连接

| 句子 | 判断 |
| --- | --- |
| 朝(あさ)起(お)きて、顔(かお)を洗(あら)って、学校(がっこう)へ行(い)く。 | 动作按顺序发生，自然。 |
| 暑(あつ)くて、窓(まど)を開(あ)けた。 | 前项是原因，自然。 |
| 本(ほん)を読(よ)んで、映画(えいが)が好きです。 | 前后关系弱，不自然。应拆句或改连接方式。 |
