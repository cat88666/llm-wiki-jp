# 语法-授受表达

> 授受表达用「あげる・くれる・もらう」表示物品或行为的给予与接受。核心不是中文的“给”，而是从谁的视点看、恩惠流向谁。

## 一、方向性总览

中文常用一个“给”，日语要说明恩惠的方向。说话者自己这一方是「内」，对方或外部人物是「外」。

```
方向性图解（说话者 = ★）

  ★（自己/内侧）──あげる──→  外（对方）      从我方给出去
  ★（自己/内侧）←──くれる──  外（对方）      对方给到我方
  ★（自己/内侧）←──もらう──  外（对方）      我方从对方收到
```

| 动词 | 主语 | 对方 | 方向 | 中文近似 |
| --- | --- | --- | --- | --- |
| あげる | 给的人 | 收的人 | 内 → 外 / 第三者 → 第三者 | 给出去 |
| くれる | 给的人 | 收的人=我方 | 外 → 内 | 给我/给我们 |
| もらう | 收的人 | 给的人 | 内 ← 外 | 从别人那里收到 |
| やる | 给的人 | 下位者/动植物 | 上 → 下 | 给（较随意） |

**为什么 くれる 和 もらう 都是"给我"却不能互换？** 视角不同。くれる 的主语是对方（"对方主动给了我"），もらう 的主语是自己（"我从对方那里得到"）。

## 二、物品授受

### 2.1 基本句型（N5）

| 句型 | 含义 | 例 |
| --- | --- | --- |
| A が B に C をあげる | A 给 B 物品 C | 私(わたし)が友達(ともだち)にプレゼントを**あげた**。 |
| A が B に C をくれる | A 给我方 B 物品 C | 友達(ともだち)が私(わたし)にプレゼントを**くれた**。 |
| A が B に/から C をもらう | A 从 B 那里收到 C | 私(わたし)が友達(ともだち)にプレゼントを**もらった**。 |
| A が B に C をやる | A 给下位者/动植物 C | 兄(あに)が犬(いぬ)に水(みず)を**やった**。 |

`もらう` 的来源可以用 `に`，也可以用 `から`。强调“来源”时 `から` 更清楚。

> 先生(せんせい)に本(ほん)をもらった。  
> 先生(せんせい)から本(ほん)をもらった。

### 2.2 方向最小对比

| 句子 | 主语 | 视点 |
| --- | --- | --- |
| 私(わたし)が友達(ともだち)に本(ほん)を**あげた**。 | 我 | 我给出去 |
| 友達(ともだち)が私(わたし)に本(ほん)を**くれた**。 | 朋友 | 朋友给到我方 |
| 私(わたし)が友達(ともだち)に本(ほん)を**もらった**。 | 我 | 我从朋友那里收到 |

## 三、行为授受

行为授受用 **动词て形 + あげる/くれる/もらう**。这里给的不是物品，而是“为某人做某事”的恩惠。

| 形式 | 含义 | 例 |
| --- | --- | --- |
| 〜てあげる | 我方/某人替别人做 | 友達(ともだち)の荷物(にもつ)を持(も)って**あげた**。 |
| 〜てくれる | 别人替我方做 | 友達(ともだち)が荷物(にもつ)を持(も)って**くれた**。 |
| 〜てもらう | 我方请/让别人替我方做 | 友達(ともだち)に荷物(にもつ)を持(も)って**もらった**。 |

**〜てもらう vs 〜てくれる 的视角差**：

> 彼(かれ)が手伝(てつだ)って**くれた**。（他帮了我。——他主动，我被动受益。）
> 彼(かれ)に手伝(てつだ)って**もらった**。（让他帮了我。——我主导请求，他协助。）

两者中文都可能翻成“他帮了我”，但日语的主语和视点不同。

### 3.1 〜てあげる 的礼貌限制

`〜てあげる` 暗含“我施恩于你”的语气。对长辈、上司或不熟的人直接说，容易显得居高临下。

> ×先生(せんせい)、私(わたし)が教(おし)えて**あげます**。  
> ○先生(せんせい)、何(なに)かお手伝(てつだ)いできることはありますか。

## 四、视点与内外关系

### 4.1 私方优先

「私」「私の家族」「私の会社」等通常算内侧。外部人物给内侧人物时，用 `くれる` 很自然。

| 场景 | 自然表达 |
| --- | --- |
| 朋友给我书 | 友達(ともだち)が私(わたし)に本(ほん)を**くれた**。 |
| 朋友给我弟弟书 | 友達(ともだち)が弟(おとうと)に本(ほん)を**くれた**。 |
| 我从朋友那里收到书 | 私(わたし)が友達(ともだち)に本(ほん)を**もらった**。 |
| 弟弟从朋友那里收到书 | 弟(おとうと)が友達(ともだち)に本(ほん)を**もらった**。 |

### 4.2 第三者之间

两个人都不是我方时，一般用 `あげる` 描述“给出去”。如果说话者把受方当作内侧，也可以用 `くれる`。

> 田中(たなか)さんが山田(やまだ)さんに本(ほん)を**あげた**。（中立叙述）  
> 田中(たなか)さんが妹(いもうと)に本(ほん)を**くれた**。（妹妹属于我方）

## 五、敬语入口

敬语授受属于更高一层的敬语系统，本页只保留入口，详细用法见 [[敬语体系]]。

| 普通形 | 敬语形 | 用法 |
| --- | --- | --- |
| あげる | 差(さ)し上(あ)げる | 给长辈/上司，谦让 |
| くれる | くださる | 长辈/上司给我方，尊敬对方 |
| もらう | いただく | 我方从长辈/上司处接受，谦让 |

> 先生(せんせい)が本(ほん)を**くださいました**。（老师给了我书。）
> 先生(せんせい)に本(ほん)を**いただきました**。（从老师那里得到了书。）

## 六、中文母语者易错点

| 错误类型 | 错误例 | 正确形式 | 原因 |
| --- | --- | --- | --- |
| あげる 方向用反 | ×友達(ともだち)が私(わたし)に本(ほん)をあげた | ○友達(ともだち)が私(わたし)に本(ほん)を**くれた** | 朋友→我方，用くれる |
| くれる/もらう 主语混乱 | ×私(わたし)が友達(ともだち)に本(ほん)をくれた | ○私(わたし)が友達(ともだち)に本(ほん)を**もらった** | くれる 的主语是给的人 |
| もらう 漏来源助词 | ×友達(ともだち)本(ほん)をもらった | ○友達(ともだち)**に/から**本(ほん)をもらった | 来源要用 に/から |
| てあげる 用于长辈 | ×先生(せんせい)を助(たす)けてあげた | ○先生(せんせい)をお手伝(てつだ)いした | てあげる 容易显得施恩 |
| いただく 主语搞错 | ×先生(せんせい)が本(ほん)をいただいた | ○先生(せんせい)が本(ほん)を**くださった** | いただく 是谦让，主语通常是我方 |
