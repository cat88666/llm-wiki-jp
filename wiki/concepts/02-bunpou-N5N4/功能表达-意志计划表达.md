# 功能表达-意志计划表达

> 意志形表达当场意志或邀请；`つもり` 是心里打算，`予定` 更像已排好的日程，`ことにする/ことになる` 区分自己决定和外部决定。

## 一、意志表达总览

```
意志型
├── 1、意志形（お段+う/よう）          当场意志/邀请
│   例：勉強(べんきょう)しよう。（学习吧。）
│
├── 2、つもりだ（辞书形+つもり）        心里打算（主观意志）
│   例：来年(らいねん)受(う)けるつもりだ。（打算明年考。）
│
├── 3、予定だ（辞书形+予定）           外部计划/日程（客观事实）
│   例：来月(らいげつ)試験(しけん)を受(う)ける予定(よてい)だ。（计划下月考试。）
│
├── 4、ことにする（辞书形+ことにする）   自己决定
│   例：毎日(まいにち)聞(き)くことにした。（决定每天听。）
│
└── 5、ことになる（辞书形+ことになる）   外部决定/结果
    例：転勤(てんきん)することになった。（公司决定要调动了。）
```

---

## 二、易混对比

| 易混组 | 判断口令 |
| --- | --- |
| つもり / 予定(よてい) | 心里打算=つもり，日程安排=予定 |
| ことにする / ことになる | 自己决定=ことにする，外部决定/自然结果=ことになる |
