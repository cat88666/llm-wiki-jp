# 助词(はが)

> 「は」标记主题（在谈论什么），「が」标记主语（谁/什么在做动作）。这个区别是日语语法最大难关之一，中文・英文中不存在这套概念体系。

## 一、主题与主语：根本不同的概念

```
は = 主题标记（Topic）
  「关于〜」「说到〜」
  → 拿出旧信息（听者已知的事物）

が = 主语标记（Subject）
  「〜才是」「正是〜」
  → 提示新信息（对听者来说未知的事物）
```

**为什么日语要区分主题和主语？** 中文通过语境隐含主题，日语用 は 明确标记"我现在要谈论的对象"。が 则标记"执行动作的人/事物"。两者在不同的语法层面工作，所以可以同时出现在一个句子里。

> 田中(たなか)さん**は**学生(がくせい)です。（说到田中 → 是学生。田中是已知信息。）
> 田中(たなか)さん**が**学生(がくせい)です。（谁是学生？→ 是田中。田中是新信息。）

## 二、五条判别规则

| # | 规则 | は | が | 例句 |
|---|------|---|---|------|
| 1 | **旧信息 vs 新信息** | 已在话题中 | 首次引入 | 昔々(むかしむかし)、おじいさん**が**いました。おじいさん**は**山(やま)へ行(い)きました。 |
| 2 | **对比** | 对比用は | — | コーヒー**は**飲(の)みますが、紅茶(こうちゃ)**は**飲(の)みません。（咖啡喝，红茶不喝。） |
| 3 | **排他（正是这个）** | — | 不是别的，正是这个 | 誰(だれ)**が**来(き)ましたか。（谁来了？）— 田中(たなか)さん**が**来(き)ました。 |
| 4 | **从属节内部** | 不能用 | 从属节内用が | [私(わたし)**が**書(か)いた]本(ほん)（我写的书。×私は書いた本） |
| 5 | **存在句引入** | — | 新出现的事物 | あそこに猫(ねこ)**が**います。（那里有猫。） |

### 2.1 旧信息 vs 新信息（最重要规则）

用故事展开看最清楚：

> むかしむかし、ある所(ところ)におじいさん**が**いました。（很久很久以前，有一个老爷爷。← 首次登场 = 新信息 = が）
> おじいさん**は**毎日(まいにち)山(やま)へ柴刈(しばか)りに行(い)きました。（老爷爷每天去山上砍柴。← 已出现 = 旧信息 = は）
> ある日(ひ)、大(おお)きな桃(もも)**が**流(なが)れてきました。（有一天，一个大桃子漂了过来。← 首次登场 = 新信息 = が）

**为什么新信息用 が？** 因为 が 的功能是「指定」——在众多可能性中指出「就是这个」。新出现的事物正是听者不知道的，需要被明确指出。

### 2.2 对比的「は」

对比两个以上的项目时用「は」。用「が」则没有对比语感。

> 肉(にく)**は**食(た)べますが、魚(さかな)**は**食(た)べません。（肉吃，鱼不吃。）
> 日本語(にほんご)**は**難(むずか)しいですが、面白(おもしろ)いです。（日语虽难但有趣。）

### 2.3 排他的「が」

强调「不是别人/别的，正是这个」时用「が」。

> A：誰(だれ)**が**犯人(はんにん)ですか。（谁是犯人？）
> B：田中(たなか)さん**が**犯人(はんにん)です。（田中才是犯人。）

## 三、典型场景的使用区分

| 场景 | 用法 | 例句 | 原因 |
|------|------|------|------|
| 自我介绍 | は | 私(わたし)**は**田中(たなか)です。（我是田中。） | 将自己作为主题提出 |
| 喜欢/讨厌 | が | 猫(ねこ)**が**好(す)きです。（我喜欢猫。） | 感情的对象用が标记 |
| 能力 | が | 日本語(にほんご)**が**話(はな)せます。（会说日语。） | 能力的对象用が标记 |
| 天气 | が | 雨(あめ)**が**降(ふ)っています。（在下雨。） | 自然现象用が引入主语 |
| 疑问词+回答 | が | 何(なに)**が**ありますか。（有什么？） | 疑问词一律用が |
| 描写 | は | この部屋(へや)**は**広(ひろ)いです。（这个房间很大。） | 对已知对象进行描写 |

**反直觉：感情・能力的对象用 が**

中文说"我喜欢猫"，"猫"是宾语。但日语「猫(ねこ)**が**好(す)きです」中，「猫」用が标记。

**为什么？** 因为「好(す)き」「嫌(きら)い」「できる」「わかる」「ほしい」等词的语法结构不是"主语→宾语→动词"，而是"主题は＋对象が＋状态"。感情・能力的对象在日语中不被当作宾语，而是用が标记的"状态触发者"。

## 四、「象は鼻が長い」问题

这个句子是日语语法学的著名难题。は 与 が 同时出现。

```
象(ぞう)は     鼻(はな)が     長(なが)い
↓              ↓              ↓
主题            主语            谓语
(说到大象)      (鼻子)          (长)
```

**为什么 は 和 が 能共存？** 因为它们在不同的语法层面工作。「は」设定话题范围（"关于大象"），「が」在这个范围内标记主语（"鼻子长"）。

相同结构的句子：

> 日本(にほん)**は**物価(ぶっか)**が**高(たか)い。（说到日本→物价→高。）
> 私(わたし)**は**頭(あたま)**が**痛(いた)い。（说到我→头→痛。）
> この店(みせ)**は**ラーメン**が**おいしい。（说到这家店→拉面→好吃。）

## 五、判定决策树

```
选择助词
  └─ 在从属节内（〜が…した + 名词）？
       └─ 是 → が（从属节内不能用は）
  └─ 含有疑问词（誰/何/どれ）？
       └─ 是 → が（疑问词一律用が）
  └─ 在做对比（A は…、B は…）？
       └─ 是 → は
  └─ 该名词是首次登场（新信息）？
       └─ 是 → が
       └─ 否（已出现/已知） → は
  └─ 是感情・能力的对象（好き/できる/わかる）？
       └─ 是 → が
  └─ 犹豫不决 → は（作为主题提出比较安全）
```

> 犹豫时用「は」比较安全——は 适用范围广，而 が 只在特定条件下使用。但新信息引入和疑问词句中必须用 が。
