# 功能表达-存在表达

> 存在表达用 `ある` 和 `いる` 表示“有/在”。选择哪个动词取决于对象是否有生命；句型选择取决于你要说明“某处有什么”，还是“某物/某人在哪里”。

## 二、ある 与 いる

日语对“有生性”敏感。有自主生命的对象用 `いる`，没有生命或不按生命体看待的对象用 `ある`。

| 动词 | 适用对象 | 判断基准 | 例 |
| --- | --- | --- | --- |
| ある | 无生物、植物、抽象事物 | 没有自主移动/意志 | 本(ほん)、机(つくえ)、木(き)、花(はな)、時間(じかん)、問題(もんだい) |
| いる | 人、动物 | 有生命/意志 | 人(ひと)、子供(こども)、猫(ねこ)、魚(さかな)、虫(むし) |

边界情况：

| 对象 | 常用动词 | 原因 |
| --- | --- | --- |
| 赤(あか)ちゃん | いる | 有生命 |
| ロボット | いる / ある | 拟人化时用いる，作为机器看待时用ある |
| 死(し)んだ魚(さかな) | ある | 作为食材/物体看待 |
| 花(はな)・木(き) | ある | 植物通常用ある |

> 公園(こうえん)に子供(こども)が**いる**。（公园里有小孩。）
> 机(つくえ)の上(うえ)に本(ほん)が**ある**。（桌上有书。）
> 庭(にわ)に花(はな)が**ある**。（院子里有花。）

## 三、基本句型

### 3.1 某处有什么

场所是已知信息，重点说明“有什么/有谁”。

```text
场所 に N が ある/いる
```

| 句子 | 说明 |
| --- | --- |
| 机(つくえ)の上(うえ)に本(ほん)がある。 | 桌上有书。 |
| 公園(こうえん)に猫(ねこ)がいる。 | 公园里有猫。 |
| 教室(きょうしつ)に学生(がくせい)が三人(さんにん)いる。 | 教室里有三名学生。 |

### 3.2 某物/某人在哪里

对象是已知话题，重点说明位置。

```text
N は 场所 に ある/いる
```

| 句子 | 说明 |
| --- | --- |
| 本(ほん)は机(つくえ)の上(うえ)にある。 | 书在桌上。 |
| 山田(やまだ)さんは会議室(かいぎしつ)にいる。 | 山田在会议室。 |
| 駅(えき)は銀行(ぎんこう)の隣(となり)にある。 | 车站在银行旁边。 |

## 四、所有表达

日语常把“拥有”表达成“某物/某人在我这边存在”。

| 语义 | 句型 | 例 |
| --- | --- | --- |
| 有物品/时间/问题 | N がある | 今(いま)、時間(じかん)がある。 |
| 有亲属/朋友/宠物 | N がいる | 兄(あに)が二人(ふたり)いる。 |
| 没有物品/时间 | N がない | お金(かね)がない。 |
| 没有亲属/朋友/宠物 | N がいない | ペットがいない。 |

> 私(わたし)は車(くるま)がある。（我有车。）
> 私(わたし)は兄(あに)が二人(ふたり)いる。（我有两个哥哥。）

## 五、时态・否定・疑问

| 形式 | ある | いる |
| --- | --- | --- |
| 普通肯定 | ある | いる |
| 普通否定 | ない | いない |
| 普通过去 | あった | いた |
| 普通过去否定 | なかった | いなかった |
| 礼貌肯定 | あります | います |
| 礼貌否定 | ありません | いません |
| 礼貌过去 | ありました | いました |
| 礼貌过去否定 | ありませんでした | いませんでした |

疑问句：

| 问法 | 答法 |
| --- | --- |
| 机(つくえ)の上(うえ)に本(ほん)がありますか。 | はい、あります。 / いいえ、ありません。 |
| 教室(きょうしつ)に先生(せんせい)がいますか。 | はい、います。 / いいえ、いません。 |
| 山田(やまだ)さんはどこにいますか。 | 会議室(かいぎしつ)にいます。 |

## 六、易错点

### 6.1 存在地点用 に，动作地点用 で

| 助词 | 用法 | 例 |
| --- | --- | --- |
| に + ある/いる | 存在地点 | 図書館(としょかん)**に**本(ほん)がある。 |
| で + 动作动词 | 动作发生地点 | 図書館(としょかん)**で**本(ほん)を読(よ)む。 |

### 6.2 は 和 が 的信息结构不同

| 句型 | 重点 | 例 |
| --- | --- | --- |
| 场所に N がある/いる | 新信息=N | 机(つくえ)の上(うえ)に本(ほん)がある。 |
| N は 场所にある/いる | 新信息=位置 | 本(ほん)は机(つくえ)の上(うえ)にある。 |

### 6.3 数量词位置

数量词通常放在 `が` 后、动词前。

> 教室(きょうしつ)に学生(がくせい)が三人(さんにん)います。  
> 机(つくえ)の上(うえ)に本(ほん)が二冊(にさつ)あります。

### 6.4 经验的 ある 不属于存在表达本体

`日本に行ったことがある` 的 `ある` 是“经验存在”，本页不展开，详见 [[语法-て形た形]]。

### 6.5 〜てある / 〜ている 属于体貌

`窓が開いている`、`窓が開けてある` 是结果状态/体貌表达，详见 [[体貌]]。
