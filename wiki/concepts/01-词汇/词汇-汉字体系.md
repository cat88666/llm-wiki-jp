## 一、音读与训读

| 读法 | 本质 | 常见位置 | 例 |
| --- | --- | --- | --- |
| 音读 | 借入汉字时保留的汉语音系统 | 汉字熟语、正式词、抽象词 | 経済（けいざい）、社会（しゃかい） |
| 训读 | 用汉字表示原有日语词 | 单字词、动词形容词、带送假名 | 山（やま）、行く（いく）、美しい |

同一个汉字可以有多种读法：行 = こう（旅行）/ ぎょう（行列）/ いく（行く）。读音不能只靠字形，要看词的构成。

### 混读

| 类型 | 模式 | 例 |
| --- | --- | --- |
| 重箱读法 | 音读 + 训读 | 本棚（ほん+たな）、台所（だい+どころ） |
| 汤桶读法 | 训读 + 音读 | 場合（ば+あい）、手帳（て+ちょう） |

## 二、读音判断

```
未知汉字词
├── 有送假名 -> 多为训读
│       例：走る、書く、高い
├── 二字以上汉字熟语 -> 多为音读
│       例：電話、経済、確認
├── 单字名词 -> 音训都可能，按常用词记
│       例：本（ほん）、山（やま）、人（ひと/じん/にん）
└── 人名地名 -> 例外多，优先按固定读法记
        例：田中（たなか）、東京（とうきょう）
```

| 线索 | 读音倾向 | 反例 |
| --- | --- | --- |
| 汉字后有假名 | 训读 | 勉強する 的 `する` 不影响 `勉強` 音读 |
| 二字熟语 | 音读 | 友達（ともだち）、手紙（てがみ） |
| 抽象名词/社会词 | 音读 | 話（はなし）是常用训读名词 |
| 身体/自然/日常词 | 训读 | 頭痛（ずつう）是汉语熟语 |

## 三、字形与部首

部首不能直接给出读音，但能快速缩小意义范围，适合阅读中猜词。

| 部首 | 意义范围 | 例 |
| --- | --- | --- |
| 氵 | 水、液体、流动 | 海、泳、洗 |
| 亻 | 人、动作、身份 | 住、働、係 |
| 言 | 语言、说明、交流 | 話、語、説明 |
| 心/忄 | 心理、情感、意志 | 思、忘、忙 |
| 手/扌 | 手部动作、操作 | 持、押、授 |
| 木 | 树木、材料、结构 | 林、机、構 |
| 貝 | 金钱、价值、交易 | 買、費、貯 |

判断顺序：先看部首定大类，再拆熟语部件，最后用上下文确认。例：貯金 = 貝（金钱）+ 貯（储存）+ 金，大意是“存钱/储蓄”。

## 四、N2 易错点

### 同音异义

| 读音 | 词 | 区别 |
| --- | --- | --- |
| こうか | 効果 / 高価 / 硬貨 | 效果 / 昂贵 / 硬币 |
| せいかく | 正確 / 性格 | 准确 / 性格 |
| かんしん | 関心 / 感心 | 关心兴趣 / 佩服 |

### 近形字

| 易混字 | 区别 |
| --- | --- |
| 未 / 末 | 未 = 尚未；末 = 末尾 |
| 士 / 土 | 士 = 人/职业；土 = 土地 |
| 持 / 待 | 持つ = 拿；待つ = 等 |
| 緑 / 縁 | 緑 = 绿色；縁 = 缘分/边缘 |

### 中日差异

| 类型 | 例 | 注意 |
| --- | --- | --- |
| 字形不同 | 国 / 國、図 / 圖 | 日语常用新字体 |
| 字义偏移 | 手紙 | 日语是“信”，不是卫生纸 |
| 读音不可套中文 | 勉強（べんきょう） | 音读不是普通话读音 |

N2 做题时，汉字题不要只凭中文直觉；优先确认日语读音、固定搭配和语境功能。
