## 一、构词的三层结构

```
构词体系的三层结构
├── 单纯词（たんじゅんご）   词基不可分割
│   例：山（やま）、海（うみ）、食べる、大きい
├── 派生词（はせいご）       词基 + 接辞（接头辞/接尾辞）
│   例：不満（ふ+まん）、美化（び+か）、〜的（〜てき）
└── 复合词（ふくごうご）     两个以上词基结合
    例：山道（やまみち）、読書（どくしょ）、焼き肉
```

**派生**与**复合**的区分：接辞（前缀・后缀）不能单独使用，复合词的各部件可单独使用。

## 三、复合词的构成模式

### 3.1 汉词复合词（汉语熟词）

| 模式 | 关系 | 例 |
|------|------|-----|
| A修饰B | A 修饰 B | 電話（電+話）、高校（高等+学校的省略） |
| A＋B（并列） | A 和 B 并列 | 父母（ちちはは→ふぼ）、東西（ひがしにし→とうざい） |
| A动词B名词 | A 动作作用于 B | 読書（読む本）、入学（学校に入る） |
| A使役B | 使 B 发生 A | 感動（感情を動かす） |

### 3.2 和语复合词

| 模式 | 结合方式 | 例 |
|------|---------|-----|
| 名词+名词 | 前词修饰后词 | 山道（やまみち）、夕日（ゆうひ） |
| 动词连用形+名词 | 动词语素+名词 | 飲み物（のみもの）、書き方（かきかた） |
| 动词连用形+动词 | 复合动词 | 飛び込む、書き直す → [[复合动词体系]] |
| 形容词词幹+名词 | 形容词语素+名词 | 白雪（しらゆき）、近道（ちかみち） |

## 四、复合词的连浊

**连浊（れんだく）**：复合词中，后部要素的语头清音变为浊音。

```
复合词连浊规则
  └─ 和语复合词：连浊频率高
       例：花（はな）+火（ひ）→ 花火（はなび）
           人（ひと）+時（とき）→ 人時（ひとどき，古语）
  └─ 汉语复合词：原则上不连浊
       例：学（がく）+校（こう）→ 学校（がっこう，不变）
  └─ 外來词：不连浊
       例：コーヒー+カップ → コーヒーカップ（不变）
```

**莱曼定律（Lyman 定律）**：如果后部要素内部**已有浊音**，则不发生连浊。

> 例：山（やま）+风（かぜ）→ 山風（やまかぜ，不连浊，因「かぜ」内有浊音「ぜ」）  
> 但：山（やま）+川（かわ）→ 山川（やまがわ，连浊，「かわ」内无浊音）

## 五、未知词汇的意义推测法

**阅读时遇到未知词汇的分析步骤**：

```
未知词汇の推測
  └─ 步骤1：词種判断（汉字→汉语、片假名→外来语）
  └─ 步骤2：接辞分解
       「不満足な状態」= 不（否定接头）+ 満足（满足）+ な（形容动词化）
  └─ 步骤3：词基拆解
       「再利用」= 再（再次）+ 利用（使用）→ 再次使用
  └─ 步骤4：部首推测（单字含义不明时）
       「憧れる」= 忄（心）+ 童（孩子般纯净）→ 憧憬
  └─ 步骤5：上下文验证
       推测含义放回句子，是否逻辑通顺
```

**N2 高频可推测构词型**：

| 词 | 推测步骤 | 含义 |
|----|---------|------|
| 不可欠（ふかけつ） | 不+可+欠= 不可缺少 | 不可缺少的 |
| 非効率（ひこうりつ） | 非+効率 = 不高效 | 非效率的 |
| 未解決（みかいけつ） | 未+解決 = 尚未解决 | 未解决的 |
| 再確認（さいかくにん） | 再+確認 = 再次确认 | 再次确认 |
| 高齢化（こうれいか） | 高齢+化 = 老年化趋势 | 老龄化 |
