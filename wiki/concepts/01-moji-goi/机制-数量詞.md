# 量词体系

> 日语数名词时必须使用助数词（量词），根据对象的形状・性质选择不同的助数词。数词+助数词的音变规则也是 N2 以前的重要考点。

## 一、助数词体系

| 助数词 | 读法 | 计数对象 | 例 |
|--------|------|---------|-----|
| 〜人 | にん/ひとり・ふたり | 人 | 3人（さんにん）、1人（ひとり） |
| 〜つ | ひとつ〜ここのつ | 和語通用量词（无对应助数词时） | 一つ、二つ、三つ |
| 〜個（こ） | こ | 小物件 | リンゴ3個、ボタン2個 |
| 〜枚（まい） | まい | 薄平物 | 紙3枚、切手5枚、シャツ2枚 |
| 〜本（ほん） | ほん/ぽん/ぼん | 细长物 | 鉛筆3本、ビール2本、道1本 |
| 〜冊（さつ） | さつ | 书・册子 | 本5冊、ノート3冊 |
| 〜台（だい） | だい | 机器・车辆 | 車2台、パソコン1台 |
| 〜匹（ひき） | ひき/ぴき/びき | 小动物 | 猫2匹、犬3匹、魚5匹 |
| 〜頭（とう） | とう | 大型动物 | 牛3頭、象1頭 |
| 〜羽（わ） | わ/ぱ | 鸟・兔 | 鳥5羽、ウサギ2羽 |
| 〜杯（はい） | はい/ぱい/ばい | 容器盛物 | コーヒー1杯、水2杯 |
| 〜枚（まい）→〜枚組 | — | 张数+套 | 5枚組のCD |
| 〜階（かい） | かい/がい | 楼层 | 3階（さんがい）、1階（いっかい） |

**通用量词「〜つ」的范围**：从 1（ひとつ）到 9（ここのつ），10 之后用「〜個」等。用于无专用助数词的场合。

## 二、数词的音变规则

助数词与数词结合时，发生促音・浊音化等音变，是 N2 以前的必考点。

### 2.1 〜本（ほん）的音变

| 数字 | 读法 | 变化类型 |
|------|------|---------|
| 1本 | いっぽん | 促音（っ）+半浊音（ぽ） |
| 2本 | にほん | 无变化 |
| 3本 | さんぼん | 浊音（ぼ） |
| 4本 | よんほん | 无变化 |
| 5本 | ごほん | 无变化 |
| 6本 | ろっぽん | 促音（っ）+半浊音（ぽ） |
| 8本 | はっぽん | 促音（っ）+半浊音（ぽ） |
| 10本 | じゅっぽん | 促音（っ）+半浊音（ぽ） |

**规律**：1・6・8・10 前，清音助数词出现促音変（k/p→っ+p）。

### 2.2 〜枚（まい）・〜台（だい）：无音变

> 1枚（いちまい）、6枚（ろくまい）、8枚（はちまい）— まい行无音变。

### 2.3 〜匹（ひき）的音变

| 数字 | 读法 |
|------|------|
| 1匹 | いっぴき |
| 3匹 | さんびき |
| 6匹 | ろっぴき |
| 8匹 | はっぴき |
| 10匹 | じゅっぴき |

### 2.4 〜杯（はい）的音变

| 数字 | 读法 |
|------|------|
| 1杯 | いっぱい |
| 3杯 | さんばい |
| 6杯 | ろっぱい |
| 8杯 | はっぱい |

## 三、N2 高频抽象助数词

N2 阶段词汇中，抽象事物的助数词大量出现：

| 助数词 | 读法 | 计数对象 | 例句 |
|--------|------|---------|------|
| 〜件（けん） | けん | 事件・事务 | 事故3件、苦情2件 |
| 〜通（つう） | つう | 信・文件 | 手紙5通、メール3通 |
| 〜割（わり） | わり | 10%单位 | 売上が3割増えた |
| 〜倍（ばい） | ばい | 倍数 | 去年の2倍 |
| 〜回（かい） | かい | 次数 | 3回試験がある |
| 〜度（ど） | ど | 次数・温度 | 何度も確認した |
| 〜棟（とう） | とう | 建筑物栋数 | 3棟のビル |
| 〜泊（はく） | はく/ぱく | 住宿夜数 | 2泊3日の旅行 |
| 〜語（ご） | ご | 语言种数 | 4カ国語 |
| 〜社（しゃ） | しゃ | 公司数 | 3社に応募した |
| 〜種（しゅ） | しゅ | 种类数 | 5種類のサービス |
| 〜割引（わりびき） | — | 折扣 | 3割引（=70%价格） |

> **「〜割」的运用**：「3割」= 30%，「7割」= 70%。  
> 「売上が前年比1割増し」= 比前一年增加了10%。

## 四、数量词在句中的位置

日语中数量词有两个主要位置，语义略有不同：

### 4.1 名词前（连体修饰位置）

**「数量詞+の+名詞」**：强调数量是名词的修饰语。

> 3冊の本を買った。（买了3本书——着重说明3本的属性）

### 4.2 动词前（副词性位置）

**「名詞を+数量詞+動詞」**：数量修饰动作，更自然的口语语序。

> 本を3冊買った。（买了书，3本——更自然）

**规律**：日常口语中副词性语序（动词前）更自然；书面语和强调数量时用「〜のN」形式。

| 比较 | 语感 |
|------|------|
| 本を3冊買った | 自然口语 |
| 3冊の本を買った | 稍书面，着重说明3本 |
| りんごを2個食べた | 自然 |
| 2個のりんごを食べた | 着重说2个（不常用） |
