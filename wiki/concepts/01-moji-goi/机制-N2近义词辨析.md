# N2 類義語使い分け

> N2 問5（言い換え類義）と問4（文脈規定）の攻略核心。意味が近い語の「使える場面・文体・ニュアンス」の違いが出題される。中文母語者の盲点は「汉语でほぼ同义でも日语では使い分ける」ペアに多い。

## 一、動詞の類義ペア

### 1.1 「変化」系

| 語 | ニュアンス | 文体 | 例 |
|----|----------|------|-----|
| 変わる | 一般的変化 | 口語〜書面 | 気温が**変わった** |
| 変化する | 客観的・科学的変化 | 書面 | 状況が急速に**変化した** |
| 改まる | 改善方向の変化・新しくなる | 書面 | 態度が**改まった** |
| 変貌する | 外観が大きく変わる（+意外感）| 書面 | 街が**変貌した** |

### 1.2 「言う・話す」系

| 語 | ニュアンス | 接续 |
|----|----------|------|
| 言う | 最も汎用 | ～と言う |
| 述べる | 順序立てて説明（書面・改まり）| 意見を**述べる** |
| 主張する | 強く訴える・立場を明確にする | 権利を**主張する** |
| 訴える | 感情に働きかける・申し立て | 裁判所に**訴える** / 痛みを**訴える** |
| 申し上げる | 谦让语（言うの謙譲）| →[[尊敬语谦让语转换]] |

****反直觉****：「訴える」有两个完全不同的用法：法律申诉 vs 感情诉说。N2 問6 用法题常考这种多义性。

### 1.3 「増える・減る」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| 増える | 数量が増加 | 人口が**増えた** |
| 増す | 程度・強さが増す | 不安が**増した** |
| 増加する | 統計的・書面的 | 出生率が**増加した** |
| 高まる | 質・度合いが高くなる | 意識が**高まった** |
| 膨らむ | 膨張・期待感も含む | 夢が**膨らんだ** |

### 1.4 「考える」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| 考える | 一般的思考 | どうすべきか**考えた** |
| 検討する | 多角的に調べて考える | 案を**検討した** |
| 判断する | 結論を出す | 状況を**判断した** |
| 評価する | 価値・成果を測る | 研究を**評価した** |
| 見なす | ～とみなす・認定する | 既読と**見なす** |

### 1.5 「やめる」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| やめる | 一般的停止 | 仕事を**やめた** |
| 中止する | 予定していた行事を止める | 会議を**中止した** |
| 断念する | 夢・目標を諦める | 留学を**断念した** |
| 撤回する | 言ったこと・決定を取り消す | 発言を**撤回した** |
| 廃止する | 制度・法律を廃する | 制度が**廃止された** |

## 二、形容詞・形容動詞の類義ペア

### 2.1 「難しい」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| 難しい | 一般的困難 | 問題が**難しい** |
| 困難だ | やや書面・強調困難 | 実現が**困難だ** |
| 複雑だ | 構造・関係が入り組む | 状況が**複雑だ** |
| 厄介だ | 面倒・扱いにくい | 手続きが**厄介だ** |

### 2.2 「大切な」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| 大切だ | 感情的に大事 | 家族が**大切だ** |
| 重要だ | 客観的に重要 | 書面でよく使う |
| 重大だ | 結果が深刻 | **重大な**問題 |
| 貴重だ | 価値が高く少ない | **貴重な**経験 |
| 不可欠だ | なくてはならない | 水は生命に**不可欠だ** |

### 2.3 「意外な」系

| 語 | ニュアンス | 例 |
|----|----------|-----|
| 意外だ | 予想外・びっくり | **意外な**結果だった |
| 予想外だ | 予測を超えた | **予想外**の展開 |
| 突然だ | 時間的に急 | **突然の**出来事 |
| 唐突だ | 脈絡もなく急に（違和感強い）| **唐突な**質問 |

## 三、副詞・接续詞の使い分け

### 3.1 逆接・転換系接续詞

| 語 | 強さ | ニュアンス | 文体 |
|----|------|---------|------|
| でも | 弱 | 口語的逆接 | 口語 |
| しかし | 中 | 汎用逆接 | 中性 |
| けれども/けれど | 中 | やや口語 | 中性〜口語 |
| だが | 中-強 | 簡潔・硬 | 書面 |
| ところが | 強 | 予想外の展開 | 中性 |
| にもかかわらず | 強 | 逆の事態が続く | 書面 |
| とはいえ | 中 | 認めつつも逆 | 書面 |

**ところが vs しかし**：
> 電車で行くつもりだった。**しかし**、遅刻した。（単純な逆接）  
> 電車で行くつもりだった。**ところが**、電車が止まっていた。（意外な展開）

### 3.2 添加・強調系副詞

| 語 | 含义 | 文体 | 例 |
|----|------|------|-----|
| さらに | 更加・追加 | 書面 | **さらに**問題が複雑になった |
| その上 | 加之・而且 | 中性 | 遅刻した。**その上**謝らなかった |
| しかも | 何と・而且（強調）| 中性 | 無料で、**しかも**高品質だ |
| おまけに | 还有（口語的）| 口語 | 負けた。**おまけに**けがもした |

### 3.3 程度副詞の使い分け

| 語 | 程度 | ニュアンス | 搭配限制 |
|----|------|---------|---------|
| とても | 強 | 口語・肯定のみ | とても美しい ○ / ×とても〜ない |
| 非常に | 強 | 書面・客観 | 非常に重要だ |
| かなり | やや強 | 予想より多い | かなり難しい |
| かなり | — | 話者の驚き含む | — |
| ずいぶん | やや強 | 話者の感情含む | ずいぶん変わった |
| だいぶ | 中〜強 | 程度が相当 | だいぶよくなった |
| やや | 弱 | 少し・書面 | やや複雑だ |
| 少し/ちょっと | 弱 | 口語 | 少し待って |

> **反直觉**：「とても〜ない」は不自然。程度が強い否定には「全然〜ない・まったく〜ない・全く〜ない」を使う。

### 3.4 N2 頻出副詞まとめ

| 副詞 | 含义 | 搭配 | 例 |
|------|------|------|-----|
| むしろ | 倒不如・宁可说 | （意外な逆の評価）| **むしろ**難しい方がいい |
| かえって | 反而・适得其反 | （逆の結果）| 薬が**かえって**悪化させた |
| せめて | 至少・起码 | （最低限の期待）| **せめて**連絡してほしかった |
| どうせ | 反正・横竖 | （諦め・自嘲）| **どうせ**無理だよ |
| わざわざ | 特地・不辞辛苦 | （不必要な手間に感謝/批判）| **わざわざ**来てくれた |
| あえて | 特意・刻意 | （困難を承知で）| **あえて**難しい道を選んだ |
| たとえ | 即使・哪怕 | たとえ〜ても/でも | **たとえ**失敗しても諦めない |

## 四、名詞の類義ペア

| 类义组 | 各词差异 |
|--------|---------|
| 理由 vs 原因 vs 原因 | 理由＝説明（主観的）；原因＝出来事の発端（客観）|
| 結果 vs 成果 | 結果＝中立的な結末；成果＝ポジティブな達成 |
| 問題 vs 課題 | 問題＝解決すべき困難；課題＝取り組むべきテーマ |
| 目的 vs 目標 | 目的＝最終的なゴール；目標＝具体的な数値・中間点 |
| 方法 vs 手段 | 方法＝やり方全般；手段＝目的のための具体的手立て |
| 状況 vs 状態 | 状況＝外部環境の様子；状態＝物・人の内部状態 |

## 五、問5 解答テクニック

**問5 の出題形式**：
> 下線部の言葉に意味が最も近いものを選べ

**4ステップ解法**：

```
解法フロー
  ├── Step1：下線語の意味確定
  │    → 品詞・文体（書面か口語か）を確認
  ├── Step2：文脈から用法を絞る
  │    → 多義語なら前後のヒントで用法特定
  ├── Step3：選択肢を品詞・文体でフィルター
  │    → 書面語↔口語のミスマッチを消去
  └── Step4：ニュアンス差で最終選択
       → 「強さ」「感情色彩」「使える主語」を比較
```

**よくある罠**：
- 同じ漢字を含む語を選んでしまう（例：「変化する」→ 「改善する」×）
- 文体が違う（書面語を口語に置き換え→意味は近くても不正解）
- 多義語の別義を選ぶ（「訴える」の法的意味 vs 感情的意味）
