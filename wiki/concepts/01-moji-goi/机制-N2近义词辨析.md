# N2 近义词辨析

> N2 题 5（近义替换）和题 4（语境判定）的核心，是区分近义词在使用场景、文体、语感上的差异。中文母语者的盲点在于：汉语里近似同义的词，日语中往往必须按语境分开使用。

## 一、动词近义词组

### 1.1 「変化」系

| 词 | 语感 | 文体 | 例 |
|----|----------|------|-----|
| 変わる | 一般变化 | 口语〜书面 | 気温が**変わった** |
| 変化する | 客观・科学变化 | 书面 | 状況が急速に**変化した** |
| 改まる | 向改善方向变化・变新 | 书面 | 態度が**改まった** |
| 変貌する | 外观发生明显变化（+意外感）| 书面 | 街が**変貌した** |

### 1.2 「言う・話す」系

| 词 | 语感 | 接续 |
|----|----------|------|
| 言う | 最通用 | ～と言う |
| 述べる | 有条理地说明（书面・正式）| 意見を**述べる** |
| 主張する | 强烈主张・明确立场 | 権利を**主張する** |
| 訴える | 诉诸感情・申诉 | 裁判所に**訴える** / 痛みを**訴える** |
| 申し上げる | 谦让语（「言う」的谦让表达）| →[[尊敬语谦让语转换]] |

****反直觉****：「訴える」有两个完全不同的用法：法律申诉 vs 感情诉说。N2 题6 用法题常考这种多义性。

### 1.3 「増える・減る」系

| 词 | 语感 | 例 |
|----|----------|-----|
| 増える | 数量增加 | 人口が**増えた** |
| 増す | 程度・强度增加 | 不安が**増した** |
| 増加する | 统计性・书面性 | 出生率が**増加した** |
| 高まる | 质量・程度提高 | 意識が**高まった** |
| 膨らむ | 膨胀，也包含期待感 | 夢が**膨らんだ** |

### 1.4 「考える」系

| 词 | 语感 | 例 |
|----|----------|-----|
| 考える | 一般思考 | どうすべきか**考えた** |
| 検討する | 多角度调查后思考 | 案を**検討した** |
| 判断する | 得出结论 | 状況を**判断した** |
| 評価する | 衡量价值・成果 | 研究を**評価した** |
| 見なす | 视为〜・认定 | 既読と**見なす** |

### 1.5 「やめる」系

| 词 | 语感 | 例 |
|----|----------|-----|
| やめる | 一般停止 | 仕事を**やめた** |
| 中止する | 停止原定活动 | 会議を**中止した** |
| 断念する | 放弃梦想・目标 | 留学を**断念した** |
| 撤回する | 撤回说过的话・决定 | 発言を**撤回した** |
| 廃止する | 废除制度・法律 | 制度が**廃止された** |

## 二、形容词・形容动词近义词组

### 2.1 「難しい」系

| 词 | 语感 | 例 |
|----|----------|-----|
| 難しい | 一般的困難 | 問題が**難しい** |
| 困難だ | 偏书面・强调困难 | 実現が**困難だ** |
| 複雑だ | 结构・关系复杂交错 | 状況が**複雑だ** |
| 厄介だ | 麻烦・难处理 | 手続きが**厄介だ** |

### 2.2 「大切な」系

| 词 | 语感 | 例 |
|----|----------|-----|
| 大切だ | 情感上重要 | 家族が**大切だ** |
| 重要だ | 客观上重要 | 书面常用 |
| 重大だ | 结果严重 | **重大な**問題 |
| 貴重だ | 价值高且稀少 | **貴重な**経験 |
| 不可欠だ | 不可或缺 | 水は生命に**不可欠だ** |

### 2.3 「意外な」系

| 词 | 语感 | 例 |
|----|----------|-----|
| 意外だ | 出乎意料・惊讶 | **意外な**結果だった |
| 予想外だ | 超出预测 | **予想外**の展開 |
| 突然だ | 时间上突然 | **突然の**出来事 |
| 唐突だ | 毫无脉络地突然（违和感强）| **唐突な**質問 |

## 三、副词・接续词辨析

### 3.1 转折・转换类接续词

| 词 | 强弱 | 语感 | 文体 |
|----|------|---------|------|
| でも | 弱 | 口语化逆接 | 口语 |
| しかし | 中 | 通用转折 | 中性 |
| けれども/けれど | 中 | 偏口语 | 中性〜口语 |
| だが | 中-强 | 简洁・偏硬 | 书面 |
| ところが | 强 | 出乎意料的发展 | 中性 |
| にもかかわらず | 强 | 相反事态持续 | 书面 |
| とはいえ | 中 | 承认前提后转折 | 书面 |

**ところが vs しかし**：
> 電車で行くつもりだった。**しかし**、遅刻した。（单纯转折）  
> 電車で行くつもりだった。**ところが**、電車が止まっていた。（意外发展）

### 3.2 添加・强调类副词

| 词 | 含义 | 文体 | 例 |
|----|------|------|-----|
| さらに | 更加・追加 | 书面 | **さらに**問題が複雑になった |
| その上 | 加之・而且 | 中性 | 遅刻した。**その上**謝らなかった |
| しかも | 竟然・而且（强调）| 中性 | 無料で、**しかも**高品質だ |
| おまけに | 还有（口语化）| 口语 | 負けた。**おまけに**けがもした |

### 3.3 程度副词辨析

| 词 | 程度 | 语感 | 搭配限制 |
|----|------|---------|---------|
| とても | 强 | 口语・只接肯定 | とても美しい ○ / ×とても〜ない |
| 非常に | 强 | 书面・客观 | 非常に重要だ |
| かなり | 稍强 | 比预想更多 | かなり難しい |
| かなり | — | 包含说话者惊讶 | — |
| ずいぶん | 稍强 | 包含说话者感情 | ずいぶん変わった |
| だいぶ | 中〜強 | 程度相当 | だいぶよくなった |
| やや | 弱 | 稍微・书面 | やや複雑だ |
| 少し/ちょっと | 弱 | 口语 | 少し待って |

> **反直觉**：「とても〜ない」不自然。强程度否定要用「全然〜ない・まったく〜ない・全く〜ない」。

### 3.4 N2 高频副词汇总

| 副词 | 含义 | 搭配 | 例 |
|------|------|------|-----|
| むしろ | 倒不如・宁可说 | （意外的反向评价）| **むしろ**難しい方がいい |
| かえって | 反而・适得其反 | （相反结果）| 薬が**かえって**悪化させた |
| せめて | 至少・起码 | （最低限度期待）| **せめて**連絡してほしかった |
| どうせ | 反正・横竖 | （放弃感・自嘲）| **どうせ**無理だよ |
| わざわざ | 特地・不辞辛苦 | （对额外费心表示感谢/批评）| **わざわざ**来てくれた |
| あえて | 特意・刻意 | （明知困难仍然）| **あえて**難しい道を選んだ |
| たとえ | 即使・哪怕 | たとえ〜ても/でも | **たとえ**失敗しても諦めない |

## 四、名词近义词组

| 类义组 | 各词差异 |
|--------|---------|
| 理由 vs 原因 vs 原因 | 理由＝说明（主观）；原因＝事件起因（客观）|
| 結果 vs 成果 | 結果＝中性的结果；成果＝正向成果 |
| 問題 vs 課題 | 問題＝需要解决的困难；課題＝需要处理的课题 |
| 目的 vs 目標 | 目的＝最终目标；目標＝具体数值・中间节点 |
| 方法 vs 手段 | 方法＝做法整体；手段＝为了目的采取的具体手段 |
| 状況 vs 状態 | 状況＝外部环境状态；状態＝物・人的内部状态 |

## 五、题5 解题技巧

**题5 的出题形式**：
> 选择与画线词含义最接近的一项

**4步骤解法**：

```
解题流程
  ├── Step1：确定画线词含义
  │    → 确认词性・文体（书面还是口语）
  ├── Step2：根据语境缩小用法范围
  │    → 多义词要根据前后提示确定用法
  ├── Step3：按词性・文体筛选选项
  │    → 排除书面语与口语不匹配项
  └── Step4：根据语感差异最终选择
       → 比较「强弱」「感情色彩」「可搭配主语」
```

**常见陷阱**：
- 误选含有相同汉字的词（例：「変化する」→ 「改善する」×）
- 文体不一致（把书面词替换成口语词，含义接近也可能不正确）
- 误选多义词的其他义项（「訴える」的法律意义 vs 感情意义）
