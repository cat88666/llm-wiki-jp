## 一、身体部位慣用句

### 1.1 手（て）

| 慣用句 | 含义 | 例文 |
|--------|------|------|
| 手を抜く | 偷工减料・敷衍 | 仕事の**手を抜く**な。 |
| 手が足りない | 人手不足 | 今、**手が足りない**から助けてほしい。 |
| 手を貸す | 帮忙・助一把 | 引越しに**手を貸して**もらえますか。 |
| 手に負えない | 无法控制・管不住 | あの子は**手に負えない**。 |
| 手を打つ | 采取措施・达成协议 | 早めに**手を打って**おこう。 |
| 手が込む | 做工精细・费工夫 | **手の込んだ**料理だ。 |
| 手を焼く | 感到棘手 | 子どもの反抗期に**手を焼いている**。 |
| 手放す | 转让・卖掉 | 大事にしていた本を**手放した**。 |

### 1.2 目（め）

| 慣用句 | 含义 | 例文 |
|--------|------|------|
| 目を通す | 过目・浏览 | 書類に**目を通して**ください。 |
| 目が肥える | 眼力高・有鉴赏力 | 美術館に通って**目が肥えた**。 |
| 目をつぶる | 睁一只眼闭一只眼・默许 | 今回は**目をつぶる**ことにした。 |
| 目に余る | 触目惊心・实在看不下去 | 彼の態度は**目に余る**。 |
| 目が高い | 有眼光 | **目が高い**ですね、それは名品です。 |
| 目をそらす | 回避・转移视线 | 現実から**目をそらして**いる。 |
| 目処が立つ | 有眉目・看到希望 | やっと解決の**目処が立った**。 |

### 1.3 口（くち）

| 慣用句 | 含义 | 例文 |
|--------|------|------|
| 口を出す | 多嘴・干涉 | 余計な**口を出すな**。 |
| 口を割る | 招供・说出秘密 | ようやく容疑者が**口を割った**。 |
| 口が堅い | 守口如瓶 | 彼女は**口が堅い**から信頼できる。 |
| 口が軽い | 嘴巴不严 | **口が軽い**人には秘密は言えない。 |
| 口先だけ | 口头上・说说而已 | **口先だけ**で行動が伴わない。 |
| 言葉を濁す | 说话含糊其辞 | 彼は**言葉を濁した**。 |

### 1.4 足・頭・顔・耳

| 慣用句 | 含义 | 例文 |
|--------|------|------|
| 足を引っ張る | 拖后腿 | チームの**足を引っ張る**な。 |
| 足がつく | 露出马脚・被追踪 | 証拠から犯人の**足がついた**。 |
| 頭を抱える | 抱头苦思・为难 | 難問に**頭を抱えた**。 |
| 頭が下がる | 由衷佩服 | 彼の努力には**頭が下がる**。 |
| 顔が広い | 交际广 | 社長は業界で**顔が広い**。 |
| 顔を立てる | 给面子 | 先輩の**顔を立てて**あげた。 |
| 耳を疑う | 难以置信 | その話を聞いて**耳を疑った**。 |
| 耳が痛い | 听着刺耳（说中了自己弱点）| 言われると**耳が痛い**。 |
| 耳にはさむ | 无意中听到 | うわさを**耳にはさんだ**。 |

| 慣用句 | 含义 | 例文 |
|--------|------|------|
| 気を遣う | 费心・细心照顾 | 人に**気を遣う**のが得意だ。 |
| 気にかける | 放在心上・担心 | ずっと君のことを**気にかけていた**。 |
| 気が引ける | 不好意思・心里过意不去 | お世話になって**気が引ける**。 |
| 気を抜く | 放松警惕 | 大事な場面で**気を抜く**な。 |
| 腹が立つ | 生气・愤怒 | 不公平な扱いに**腹が立った**。 |

## 四、慣用句 vs ことわざ の見分け方

**出题方式**（N2 問4・問6）：

```
出題パターン
├── 文脈規定（問4）：填入语境中最合适的慣用句
│   → 看上下文情感色彩：negative（足を引っ張る）or positive（頭が下がる）
├── 用法（問6）：判断慣用句在四个例句中哪个用法正确
│   → 注意：慣用句的主语・宾语限制比普通词严格
└── 言い換え類義（問5）：慣用句的同义换说
    例：「手を貸す」→「助ける」「協力する」
```

**解题原则**：
1. 优先排除字面意思解读（慣用句整体意义 ≠ 各词之和）
2. 确认感情色彩（褒め/貶し/中性）
3. 确认使用主语限制：「耳が痛い」只能用自己听到别人批评时，不能用于说别人

## 五、中文母语者易混淆

| 日语慣用句 | 中文字面联想 | 实际含义 | 易错方向 |
|---------|-----------|---------|---------|
| 情けは人のためならず | ×「善意对别人没好处」| ○「善意最终对自己有益」| 误读为负面 |
| 出る杭は打たれる | ×「冒尖是好事」| ○「枪打出头鸟（警告）」| 误读为褒义 |
| 猫をかぶる | ×与猫有关 | ○「装乖・藏锋露拙」| 完全不同 |
| 馬が合う | ×「马合拍」| ○「两人合得来・投缘」| 字面无关 |

> **反直觉**：「情けは人のためならず」常被误解为"善意对别人无益"，实际相反——帮助别人最终回到自己身上。N2 阅读题有时会考查对这句谚语的正确理解。
