## 未言之悟

> 言葉(ことば)にしなくても分(わ)かると思(おも)っていたけど、それだけでは足(た)りなかったと今(いま)になって気(き)づいた。

我曾以为不用说出口也能互相理解，但现在才意识到，仅靠那样是不够的。

| 语法 | 说明 |
|------|------|
| ～にしなくても | 即使不做～也…… |
| ～だけでは足りない | 仅靠……是不够的 |
| 今になって気づいた | 事到如今才意识到（后悔语气） |

**记忆点**：`今になって` 表示"到了现在这个时点才……"，常带后悔。搭配：今になって分かる/気づく/後悔する

## 无形的距离

> 同(おな)じ場所(ばしょ)にいても、少(すこ)しずつ心(こころ)の距離(きょり)が変(か)わっていくことに気(き)づくことがある。

即使待在同一个地方，也会发现彼此心的距离在慢慢变化。

| 语法 | 说明 |
|------|------|
| ～にいても | 即使在……（逆接） |
| 少しずつ～ていく | 逐渐……（变化持续） |
| ～ことに気づく | 注意到……这件事 |

**记忆点**：`～ていく` 表示从现在向未来持续变化；`～てくる` 是从过去到现在的变化。方向相反，常考辨析

## 思念随时间生长

> 会(あ)えない時間(じかん)が長(なが)くなると、その人(ひと)のことを思(おも)い出(だ)す時間(じかん)も自然(しぜん)と増(ふ)えていく。

无法见面的时间越长，想起那个人的时间也自然增加。

| 语法 | 说明 |
|------|------|
| ～が長くなると | 当……变长时（条件） |
| ～も自然と増えていく | ……也自然地增加（随之变化） |

**记忆点**：`自然(しぜん)と` = 自然而然地，副词用法；`思い出す` = 想起（瞬间），区别 `思い出` = 回忆（名词）

## 无法选择的路

> あのとき選(えら)べなかった道(みち)を、今(いま)でもときどき静(しず)かな時間(じかん)の中(なか)で何度(なんど)も思(おも)い出(だ)す。

那时没能选择的道路，如今偶尔仍会在安静的时候反复想起。

| 语法 | 说明 |
|------|------|
| 選べなかった | 可能形 + なかった（没能选择） |
| 今でも | 即使到现在仍然 |
| 何度も | 反复多次 |

**记忆点**：`ときどき` = 偶尔；`何度も` = 反复多次。两个频率词叠用 → 虽然不是总想，但每次想起就反复回味

## 短暂却深刻

> 一緒(いっしょ)に過(す)ごした短(みじか)い時間(じかん)でも、その記憶(きおく)は今(いま)でもはっきり心(こころ)の中(なか)に残(のこ)っている。

即使只是短暂共度的时光，那份记忆至今依然清晰留在心里。

| 语法 | 说明 |
|------|------|
| ～た短い時間でも | 即使是……的短暂时间 |
| はっきり残っている | 清晰地留存着（状态持续） |

**记忆点**：`はっきり` = 清晰地（副词）；搭配：はっきり覚えている/はっきり見える/はっきり言う

## 不经意的想起

> 忘(わす)れようとしても、ふとした何気(なにげ)ない瞬間(しゅんかん)にまた思(おも)い出(だ)してしまうことがある。

即使想忘记，也会在某个不经意瞬间再次想起。

| 语法 | 说明 |
|------|------|
| ～ようとしても | 即使试图做……也（意志逆接） |
| ～てしまう | 不由自主地……（非本意） |
| ことがある | 有时会……（偶发） |

**记忆点**：`ふとした` = 偶然的、不经意的；`何気ない` = 不在意的、无心的。两词常搭配，强调"没有预兆地突然"

## 同一片天空

> どんなに遠(とお)く離(はな)れていても、同(おな)じ空(そら)の下(した)にいると思(おも)うだけで少(すこ)し安心(あんしん)できる。

无论相隔多远，只要想到仍在同一片天空下，就会稍微安心。

| 语法 | 说明 |
|------|------|
| どんなに～ても | 无论多么……也（极端逆接） |
| ～と思うだけで | 仅仅想到……就…… |
| 安心できる | 能够安心 |

**记忆点**：`どんなに～ても` = N2 高频逆接；类似：いくら～ても/たとえ～ても。三者语气递增

## 想走却停下

> 前(まえ)に進(すす)みたいと思(おも)いながらも、ときどき立(た)ち止(ど)まってしまうことがある。

虽然想继续前进，但有时还是会停下脚步。

| 语法 | 说明 |
|------|------|
| ～たいと思いながらも | 虽然想……但是（心理逆接） |
| 立ち止まってしまう | 不由自主地停下（非本意） |

**记忆点**：`ながらも` = 虽然……但是（N2 语法）；接续：V-ます形去ます + ながらも / い形容詞 + ながらも / N + ながらも
