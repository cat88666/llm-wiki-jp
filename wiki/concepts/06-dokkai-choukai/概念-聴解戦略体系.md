> N2 听力包含课题理解→要点理解→概要理解→即时应答→综合理解五种题型。「先读选项」「关键词笔记」「听说话者意图」三大策略提升正答率。

## 一、N2 听力的题型与分值

| 题型 | 题数 | 特征 | 选项位置 |
|------|------|------|---------|
| 課題理解 | 5題 | 听取需要做什么（行动选题） | 试卷上有选项图 |
| ポイント理解 | 6題 | 听取特定信息点 | 试卷上有选项 |
| 概要理解 | 5題 | 听整体主旨/态度（无选项印刷） | **选项只在音频中念**|
| 即時応答 | 12題 | 只有一句话，选合适应答 | 试卷无内容，音频中选择 |
| 統合理解 | 4題 | 多个人物对话→综合判断 | 试卷上有选项 |

**最难点**：概要理解的选项只读一次，需要在听音频的同时就要判断。

---

## 二、课题理解的攻略法

**课题理解** = 听取「接下来要做什么/应该做什么」。

**攻略步骤**：

```
課題理解の手順
  └─ 音频开始前：快速扫一眼选项图片（把握选项内容）
  └─ 音频播放时：
       └─ まず/最初に/次に/最後に に注意（顺序条件）
       └─ 否定条件（〜ではなく/〜はいらない）に注意
       └─ 最終的に何をするか に絞る
  └─ 完成后立即选答（不纠结）
```

**常见陷阱**：

| 陷阱 | 对策 |
|------|------|
| 先说A，后来改为B | 以**最终说法**为准，不以最初说法为准 |
| 条件否定（まず〜でなくて〜） | 仔细听否定范围 |
| 多步骤（まず〜次に〜） | 笔记关键步骤 |

## 三、要点理解的攻略法

**要点理解** = 听取特定信息（理由/方法/时间/地点等）。

**攻略步骤**：

```
ポイント理解の手順
  └─ 音频前：读题目（问的是什么？），确定听取焦点
  └─ 音频前：快速扫选项（预测可能的答案范围）
  └─ 音频播放：只集中听与题目相关的信息
       └─ 表示理由：〜から/〜ので/〜ため
       └─ 表示方法：〜方法/〜やり方/〜によって
       └─ 表示结论：つまり/要するに
  └─ 完成后对照选项
```

---

## 四、概要理解的攻略法

**概要理解** = 听整体主旨・说话者态度（无文字选项）。

**特点**：没有任何文字提示，选项只在音频中朗读，需要高度集中和快速判断。

**攻略策略**：

| 策略 | 说明 |
|------|------|
| 听话者立场 | 说话者是赞成还是反对？积极还是消极？ |
| 听转折点 | しかし/でも/だが 后面是说话者真实观点 |
| 听句末情态 | 〜べきだ/〜はずだ/〜と思う 等表达态度 |
| 不追细节 | 不记具体数据，把握整体语气和方向 |

> **例**：某人长篇分析了A方案的优点，但最后说「でも、結局Bのほうがいいんじゃないかと思います」  
> → 说话者支持的是**B方案**，不是A。

## 五、即时应答的攻略法

**即时应答** = 听一句话，选合适的应答（速度最快，大量题目）。

**关键**：听句末表达快速判断应答类型。

| 句末表达 | 应答类型 | 例 |
|---------|---------|-----|
| 〜ませんか / 〜ましょうか | 邀请/提议→接受/拒绝 | 一緒に行きませんか→ええ、ぜひ / すみません、ちょっと…|
| 〜てくれない? / 〜てもらえる? | 请求→承诺/拒绝 | 手伝ってくれない?→いいよ / ちょっと今は… |
| 〜どう思う? / 〜どうかな? | 意见请求→表达看法 | どう思う?→いいと思う / ちょっと難しいかも |
| 〜してもいいですか? | 许可请求→许可/拒绝 | |
| ～じゃないですか? | 确认/同意请求→确认/否认 | |

**时间策略**：每题只有约 8 秒，听完即选，不能回放。

## 六、听力的语音特征

N2 听力音频使用自然口语速度，包含以下特征：

### 6.1 缩约形（重要）

| 标准形 | 缩约形（口语） |
|--------|--------------|
| 〜ている | 〜てる/〜てん |
| 〜てしまう | 〜ちゃう/〜じゃう |
| 〜てしまった | 〜ちゃった/〜じゃった |
| 〜ておく | 〜とく |
| 〜ている | 〜てる |
| ということは | てことは |
| それは | それ→省略多 |

### 6.2 填充词（フィラー）

说话者在思考时使用的无意义词：

> えーと、あのー、そのー、まあ — 这些词无语法/语义功能，可忽略。

### 6.3 口语特有表达

| 口语 | 意义 |
|------|------|
| 〜んだけど | 〜のですが（委婉说明） |
| 〜じゃない? | 〜ではないですか?（反问确认） |
| やっぱり/やっぱ | やはり（果然/还是）|
| なんか | なんとなく/なにか（模糊表达）|
