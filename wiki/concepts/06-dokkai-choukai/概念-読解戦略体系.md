## 一、N2 阅读的题型与分值

| 题型 | 字数 | 题数 | 内容 |
|------|------|------|------|
| 短文阅读（内容理解・短）| 約200字 | 3問×3文 | 日常話題・メール |
| 中文阅读（内容理解・中）| 約500字 | 3問×2文 | 解説文・随筆 |
| 长文阅读（内容理解・长）| 約1000字 | 4問 | 論説文 |
| 综合理解 | 複数文章 | 3問 | 2つの文章の比較・統合 |
| 信息检索（情報検索） | 広告・案内等 | 2問 | 实用类文本中的信息提取 |

**时间分配参考**（语言知识＋阅读共 105 分钟）：

> 文字・词汇（15 分钟）→ 语法（20 分钟）→ 阅读（70 分钟）  
> 阅读内部：短文（10分）→ 中文（15分）→ 长文（20分）→ 综合（10分）→ 信息检索（10分）＋余裕（5分）

## 二、文章结构模式

识别文章结构是快速定位作者主张的关键：

| 结构类型 | 特征 | 主张位置 |
|---------|------|---------|
| 头括型（頭括型）| 开头给结论，后续展开论据 | 第一段 |
| 尾括型（尾括型）| 先展开论据，结尾给结论 | 最后一段 |
| 双括型（双括型）| 开头预告+结尾确认结论 | 开头和结尾 |
| 列举型 | 多个并列论点（まず・次に・最後に）| 分散在各节 |
| 对比型 | A 与 B 对比，最后给出判断 | 最终对比部分 |

**实用规则**：

> N2 长文多为論説文（议论文）→ **尾括型**频繁，主张在最后一段。  
> 「しかし・ところが・だが」后面是作者真正的主张。

## 三、通过接续词追踪逻辑

接续词是文章逻辑的路标，特别重要：

| 接续词 | 功能 | 阅读策略 |
|--------|------|---------|
| しかし/ところが/だが | 转折 | 后面是作者主张或关键转换点 |
| だから/したがって/よって | 结论 | 后面是核心论点 |
| つまり/すなわち/要するに | 换言/总结 | 后面是整段的精华总结 |
| 一方（で）/それに対して | 对比 | 信号：出现对比，注意两者差异 |
| まず/次に/最後に | 列举顺序 | 信号：步骤类内容，注意顺序题 |
| 例えば/たとえば | 例示 | 例子本身不重要，重要的是例子之前的主张 |

**例示の前の主張が答え**：

> 「〜。**例えば**、Aという事例がある。」→ 问"筆者の主張は？"时，答案在「例えば」的**前面**，不在例子本身。

## 四、指示词・省略的解析

**指示词的指示对象**是出题重点（"これ/それ 指的是什么？"）：

| 指示词系列 | 位置关系 | 阅读规则 |
|-----------|---------|---------|
| こ系（これ/この/こんな） | 说话者近側 / **文章中指紧前面的内容** | 往前一句找指示对象 |
| そ系（それ/その/そんな） | 听话者侧 / **文章中指已出现、共有知识** | 往前几句找前面提到的内容 |
| あ系（あれ/あの/あんな） | 共有知識・遠い記憶 | 双方共有的知识，或较远的前文 |

**解题步骤**：

```
指示词の解析
  └─ これ/こういう → 该文中：往上一句找指示对象
  └─ その〜 → 该文中：往上几句找明确出现的名词
  └─ 找到候选词 → 代入原文验证逻辑是否通顺
```

## 五、选项分析与消去法

N2 选项的常见陷阱类型：

| 陷阱类型 | 说明 | 对策 |
|---------|------|------|
| 部分正确 | 选项的前半部分正确，后半部分歪曲 | 把选项每一部分对照原文核实 |
| 极端表达 | 必ず・絶対・すべて・全く などの言い過ぎ | 原文是否有此程度的断言？若无则×|
| 原文没有的信息 | 选项内容原文未提及（不能"推断" 填充） | 找不到对应原文→× |
| 同义换词 | 选项用不同词表达相同含义（较难） | 意义理解，不仅仅看词形 |
| 逻辑方向错 | 原因→结果关系被搞反 | 仔细确认因果关系的方向 |

**消去步骤**：

```
选项消去法
  └─ 先读题目（什么问题？）
  └─ 找原文对应段落（关键词定位）
  └─ 逐一比对选项
       └─ 明显×（原文没有/极端）→消去
       └─ 可能○ → 精读原文确认
  └─ 最终在2个选项中 → 回到原文比较细节
```

## 六、速读与精读的使用区分

**三阶段阅读法**：

| 阶段 | 方法 | 目的 |
|------|------|------|
| 第一遍（略読）| 只读各段首句 + 接续词 | 把握文章整体结构（头/尾/对比等）|
| 第二遍（扫読）| 读题目，定位相关段落 | 确定答题区域 |
| 第三遍（精読）| 精读答题区域 + 结尾段 | 确认细节，比对选项 |

**各题型的读法重点**：

| 题型 | 策略 |
|------|------|
| 短文（200字） | 全文精读，无略读余地 |
| 中文（500字） | 读首尾+题目相关段 |
| 长文（1000字） | 各段首句略读→结尾精读→题目定位精读 |
| 综合理解 | 先读一篇，再读第二篇，找共同点/差异点 |
| 信息检索 | 直接读题目→在文本中搜索关键词 |

## 七、文章类型别攻略

N2 阅读按文章性质分为四类，各有专属解题策略：

### 7.1 論説文（问题と主张型）

**特征**：作者对某一议题提出主张、論据を展開する。N2 长文（1000字）・中文の多くはこれ。

**常见问题类型**：
- 「筆者の主張・意見は何か」
- 「筆者はなぜ〜と言っているか（理由）」
- 「〜とはどういう意味か（語句の意味）」

**攻略法**：

```
論説文の解法
  ├── 1. 各段冒頭の一文だけ読む（スキミング）
  ├── 2. しかし/ところが の後の文を拾う（主張候補）
  ├── 3. 最終段落を精読（尾括型が多い）
  └── 4. 設問の「なぜ」→原因・理由の段落へ戻る
```

**反直觉**：N2 では「筆者の意見に同意するか」は問われない。あくまで「文章の中で筆者が何を言っているか」を探す。

### 7.2 説明文・随筆（情報伝達型）

**特征**：客観的な事実説明または個人の経験・感想。主張が明確でなく、情報整理が重要。

**常见问题类型**：
- 「〜について正しいものはどれか（内容一致）」
- 「〜の特徴として述べられていないものは」

**攻略法**：
- 「述べられていないもの」問題 → 4選項のうち原文に**対応文がない**ものを選ぶ
- 内容一致 → 原文の言い換えを探す（同義換言に注意）

### 7.3 情報文（広告・案内・お知らせ）

**特征**：実用的文書（チラシ・掲示板・メール・規則）。情報検索（問15）で出る。

**常见问题类型**：
- 「Aさんはどれを利用できるか」（条件の絞り込み）
- 「〜の場合、何をすればよいか」（手順確認）

**攻略法**：

```
情報文の解法
  ├── 1. まず設問を読んで条件を確認
  │    （対象者・条件・日時・価格 etc.）
  ├── 2. 文書内をキーワードで検索（斜め読み）
  ├── 3. 条件に合う箇所を精読
  └── 4. 「〜を除く」「〜の場合のみ」の除外条件に注意
```

**中文母語者の注意点**：日本語の「〜を除く」「〜に限り」「ただし〜の場合は」などの限定・除外表現を見落とさない。

### 7.4 複数文章の統合（综合理解）

**特征**：2つの文章（AとB）を読み比べる。筆者の立場・意見の一致点と相違点を問う。

**常见问题类型**：
- 「AとBの共通する考えはどれか」
- 「Aと異なるBの主張はどれか」
- 「AはBについてどう思うか」

**攻略法**：

```
複数文章の解法
  ├── 1. 先にA文章の主張（筆者の立場）を把握
  ├── 2. B文章を読みながら、Aとの一致/相違をメモ
  ├── 3. 問われているのは「どちらの筆者の主張か」を確認
  └── 4. 両者の立場表を頭の中で作る（賛成/反対/一致）
```

**よくある罠**：「AはBの意見をどう評価するか」という問いで、A文章の中にB文章への言及がない場合は選択肢を論理推論で選ぶ。A の主張からB への態度を推論する。
