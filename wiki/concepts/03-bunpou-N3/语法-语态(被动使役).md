# 语态体系

> 语态不是背四个变形表，而是在句子里换“镜头”。能動(のうどう)看做动作的人，受身(うけみ)看受影响的人，使役(しえき)看发命令/允许的人，使役受身(しえきうけみ)看被迫做事的人。

## 一、一句话心法

先不要背“被动、使役、使役被动”这些名字。先问三个问题：

1. **谁做了动作？**  
   先生(せんせい)が学生(がくせい)を叱(しか)る。  
   老师批评学生。镜头在老师。

   - 助词：`が` 标动作主体 = 先生(せんせい)；`を` 标动作对象 = 学生(がくせい)。
   - 动词：叱(しか)る = 原形，能動(のうどう)。
   - 句型：A が B を V = A 对 B 做动作。

2. **谁受到了影响？**  
   学生(がくせい)が先生(せんせい)に叱(しか)られる。  
   学生被老师批评。镜头移到学生。

   - 助词：`が` 标受影响的主语 = 学生(がくせい)；`に` 标动作来源/动作主 = 先生(せんせい)。
   - 动词变形：叱(しか)る → 叱(しか)られる，五段(ごだん)动词受身(うけみ)。
   - 句型：B が A に V受身(うけみ) = B 被 A 做 / B 受到 A 的动作影响。

3. **谁让谁做？**  
   先生(せんせい)が学生(がくせい)に本(ほん)を読(よ)ませる。  
   老师让学生读书。镜头在发命令/允许的人。

   - 助词：`が` 标使役者 = 先生(せんせい)；`に` 标被要求做事的人 = 学生(がくせい)；`を` 标动作对象 = 本(ほん)。
   - 动词变形：読(よ)む → 読(よ)ませる，五段(ごだん)动词使役(しえき)。
   - 句型：A が B に C を V使役(しえき) = A 让 B 做 C。

把语态记成一个镜头切换：

```
能動(のうどう)       做的人在前面
受身(うけみ)         受影响的人在前面
使役(しえき)         让别人做的人在前面
使役受身(しえきうけみ) 被迫做的人在前面
```

## 二、四态角色图

用同一个动词 `読(よ)む` 看四种语态：

| 语态 | 日语 | 中文直觉 | 镜头 |
| --- | --- | --- | --- |
| 能動(のうどう) | 学生(がくせい)が本(ほん)を読(よ)む。 | 学生读书。 | 做动作的人 |
| 受身(うけみ) | 本(ほん)が学生(がくせい)に読(よ)まれる。 | 书被学生读。 | 被动作影响的东西 |
| 使役(しえき) | 先生(せんせい)が学生(がくせい)に本(ほん)を読(よ)ませる。 | 老师让学生读书。 | 让别人做的人 |
| 使役受身(しえきうけみ) | 学生(がくせい)が先生(せんせい)に本(ほん)を読(よ)まされる。 | 学生被老师逼着读书。 | 被迫做的人 |

最重要的格助词：

| 结构 | 记忆方式 |
| --- | --- |
| A が B を V | A 做 B |
| B が A に V受身(うけみ) | B 被 A 做 |
| A が B に/を V使役(しえき) | A 让 B 做 |
| B が A に V使役受身(しえきうけみ) | B 被 A 逼着做 |

### 2.1 四态逐句拆解

| 句子 | 助词用法 | 动词变形 | 语感 |
| --- | --- | --- | --- |
| 学生(がくせい)が本(ほん)を読(よ)む。 | `が`=做动作的人；`を`=被读的对象 | 読(よ)む=原形 | 学生主动读书 |
| 本(ほん)が学生(がくせい)に読(よ)まれる。 | `が`=受影响的主语；`に`=动作主 | 読(よ)む→読(よ)まれる | 书被学生读 |
| 先生(せんせい)が学生(がくせい)に本(ほん)を読(よ)ませる。 | `が`=让别人做的人；`に`=被安排读的人；`を`=读的对象 | 読(よ)む→読(よ)ませる | 老师让学生读 |
| 学生(がくせい)が先生(せんせい)に本(ほん)を読(よ)まされる。 | `が`=被迫做的人；`に`=逼他做的人；`を`=读的对象 | 読(よ)む→読(よ)まされる | 学生被迫读 |

## 三、受身：不是只有中文的“被”

受身(うけみ)的核心不是“被”，而是：**主语受到了影响**。

### 3.1 直接受身：真的被做了

动作直接打到主语身上，中文通常能翻译成“被”。

> 学生(がくせい)が先生(せんせい)に褒(ほ)められた。  
> 学生被老师表扬了。

- 助词：`が` 标受影响的人 = 学生(がくせい)；`に` 标动作主 = 先生(せんせい)。
- 动词变形：褒(ほ)める → 褒(ほ)められる → 褒(ほ)められた，一段(いちだん)动词受身(うけみ)过去形。
- 语感：学生是句子的主角，重点是“学生受到了表扬”。

> 私(わたし)は友達(ともだち)に助(たす)けられた。  
> 我被朋友帮助了。

- 助词：`は` 标话题 = 私(わたし)；`に` 标动作主 = 友達(ともだち)。
- 动词变形：助(たす)ける → 助(たす)けられる → 助(たす)けられた，一段(いちだん)动词受身(うけみ)过去形。
- 语感：`は` 比 `が` 更像“至于我呢”，强调后面的经历。

句型：

```
受影响的人/物 + が/は + 做动作的人 + に + V受身(うけみ)
```

### 3.2 受害受身：不是被做，而是被影响到

这是 N3 的难点。中文没有完全对应结构。

> 雨(あめ)に降(ふ)られた。  
> 不是“雨把我降了”，而是“我遇上下雨，很倒霉/受影响”。

- 助词：`に` 标造成影响的事件/来源 = 雨(あめ)。
- 省略：主语通常是 私(わたし)，完整理解为 私(わたし)は雨(あめ)に降(ふ)られた。
- 动词变形：降(ふ)る → 降(ふ)られる → 降(ふ)られた，五段(ごだん)动词受身(うけみ)过去形。
- 语感：不是“雨被谁下”，而是“我被下雨这件事影响了”。

> 子供(こども)に泣(な)かれた。  
> 不是“我被孩子哭了”，而是“孩子哭了，搞得我很困扰”。

- 助词：`に` 标造成困扰的人 = 子供(こども)。
- 省略：主语通常是 私(わたし)，即“我被孩子哭这件事影响”。
- 动词变形：泣(な)く → 泣(な)かれる → 泣(な)かれた，五段(ごだん)动词受身(うけみ)过去形。
- 语感：孩子在主动哭，但说话人受到影响。

> 夜中(よなか)に電話(でんわ)をかけられた。  
> 半夜有人打电话来，我受到了打扰。

- 助词：`に` 标时间点 = 夜中(よなか)；`を` 标动作对象 = 電話(でんわ)。
- 省略：打电话的人可省略；主语通常是 私(わたし)。
- 动词变形：電話(でんわ)をかける → 電話(でんわ)をかけられる → かけられた，一段(いちだん)动词受身(うけみ)过去形。
- 语感：重点不是电话本身，而是“我被半夜来电打扰”。

判断方法：如果中文硬翻“被”很奇怪，但句子有“倒霉、麻烦、困扰”的感觉，多半是受害受身(うけみ)。

### 3.3 所有者受身：我的东西被动了，所以我受影响

日语常把“人”放主语，而不是把“东西”放主语。

> 私(わたし)は財布(さいふ)を盗(ぬす)まれた。  
> 我钱包被偷了。

- 助词：`は` 标受害者/话题 = 私(わたし)；`を` 标被偷的东西 = 財布(さいふ)。
- 动词变形：盗(ぬす)む → 盗(ぬす)まれる → 盗(ぬす)まれた，五段(ごだん)动词受身(うけみ)过去形。
- 语感：钱包是动作对象，但“我”是受害者，所以 私(わたし) 放前面。

> 私(わたし)は足(あし)を踏(ふ)まれた。  
> 我脚被踩了。

- 助词：`は` 标受害者/话题 = 私(わたし)；`を` 标被踩的身体部位 = 足(あし)。
- 动词变形：踏(ふ)む → 踏(ふ)まれる → 踏(ふ)まれた，五段(ごだん)动词受身(うけみ)过去形。
- 语感：不是“脚”自己受害，而是“我”因为脚被踩而受害。

不是：

> 財布(さいふ)が盗(ぬす)まれた。  
> 钱包被偷了。

- 助词：`が` 标主语 = 財布(さいふ)。
- 动词变形：盗(ぬす)む → 盗(ぬす)まれる → 盗(ぬす)まれた。
- 语感：只客观描述“钱包被偷”，不突出“我很倒霉”。

这句也对，但重点只是“钱包”。`私(わたし)は財布(さいふ)を盗(ぬす)まれた` 的重点是“我倒霉/我受害”。

## 四、使役：让别人做

使役(しえき)的核心是：**A 让 B 做某事**。

```
A が B に/を Vさせる
```

### 4.1 强制：命令别人做

> 親(おや)が子供(こども)を学校(がっこう)に行(い)かせた。  
> 父母让孩子去学校。

- 助词：`が` 标使役者 = 親(おや)；`を` 标被要求移动的人 = 子供(こども)；`に` 标移动目的地 = 学校(がっこう)。
- 动词变形：行(い)く → 行(い)かせる → 行(い)かせた，五段(ごだん)动词使役(しえき)过去形。
- 语感：父母安排/要求孩子去学校，偏强制。

> 先生(せんせい)が学生(がくせい)に作文(さくぶん)を書(か)かせた。  
> 老师让学生写作文。

- 助词：`が` 标使役者 = 先生(せんせい)；`に` 标被要求做事的人 = 学生(がくせい)；`を` 标动作对象 = 作文(さくぶん)。
- 动词变形：書(か)く → 書(か)かせる → 書(か)かせた，五段(ごだん)动词使役(しえき)过去形。
- 语感：老师让学生完成“写作文”这个动作。

### 4.2 允许：让别人去做想做的事

同样是 `させる`，也可以表示“允许”。

> 子供(こども)を外(そと)で遊(あそ)ばせる。  
> 让孩子在外面玩。

- 助词：`を` 标被允许/被放出去活动的人 = 子供(こども)；`で` 标动作发生地点 = 外(そと)。
- 省略：使役者常省略，通常是 私(わたし) 或 親(おや)。
- 动词变形：遊(あそ)ぶ → 遊(あそ)ばせる，五段(ごだん)动词使役(しえき)。
- 语感：这里更像“允许孩子玩”，不是强迫孩子玩。

> 好(す)きにさせてください。  
> 请让我按自己喜欢的方式做。

- 助词：`に` 标状态/方式 = 好(す)きに，“按喜欢的方式”。
- 省略：完整理解为 私(わたし)を好(す)きにさせてください。
- 动词变形：する → させる → させてください，使役(しえき)的请求形。
- 语感：请求对方允许我自由行动。

判断强制还是允许，看上下文：

| 语气 | 常见感觉 | 例 |
| --- | --- | --- |
| 强制 | 不做也得做 | 宿題(しゅくだい)をやらせる |
| 允许 | 想做，所以让他做 | 遊(あそ)ばせる |

## 五、使役受身：被迫做

使役受身(しえきうけみ) = 使役(しえき) + 受身(うけみ)。

核心感觉：**别人让我做，我不太愿意，但不得不做**。

> 学生(がくせい)が先生(せんせい)に作文(さくぶん)を書(か)かされた。  
> 学生被老师逼着写作文。

- 助词：`が` 标被迫做事的人 = 学生(がくせい)；`に` 标逼他做的人 = 先生(せんせい)；`を` 标动作对象 = 作文(さくぶん)。
- 动词变形：書(か)く → 書(か)かせる → 書(か)かせられる → 書(か)かされる → 書(か)かされた。
- 语感：学生不一定想写，但被老师要求写。

> 私(わたし)は上司(じょうし)に残業(ざんぎょう)させられた。  
> 我被上司要求加班。

- 助词：`は` 标话题/受害者 = 私(わたし)；`に` 标强迫来源 = 上司(じょうし)。
- 动词变形：残業(ざんぎょう)する → 残業(ざんぎょう)させる → 残業(ざんぎょう)させられる → 残業(ざんぎょう)させられた。
- 语感：说话人带有“不情愿、无奈”的感觉。

> 子供(こども)の頃(ころ)、野菜(やさい)を食(た)べさせられた。  
> 小时候被迫吃蔬菜。

- 助词：`の` 连接名词，子供(こども)の頃(ころ)=小时候；`を` 标动作对象 = 野菜(やさい)。
- 省略：主语通常是 私(わたし)；强迫的人可从上下文理解为父母等。
- 动词变形：食(た)べる → 食(た)べさせる → 食(た)べさせられる → 食(た)べさせられた。
- 语感：不是单纯“吃了蔬菜”，而是“被迫吃了蔬菜”。

记忆公式：

```
使役(しえき)：A 让 B 做
使役受身(しえきうけみ)：B 被 A 逼着做
```

## 六、变形最小记忆表

先不要一次背太多。先记住下面这张表。

| 动词类型 | 原形 | 受身(うけみ) | 使役(しえき) | 使役受身(しえきうけみ) |
| --- | --- | --- | --- | --- |
| 五段(ごだん) | 書(か)く | 書(か)かれる | 書(か)かせる | 書(か)かされる |
| 五段(ごだん) | 読(よ)む | 読(よ)まれる | 読(よ)ませる | 読(よ)まされる |
| 一段(いちだん) | 食(た)べる | 食(た)べられる | 食(た)べさせる | 食(た)べさせられる |
| 不规则 | する | される | させる | させられる |
| 不规则 | 来(く)る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

### 6.1 五段动词

受身(うけみ)：あ段(だん) + れる

> 書(か)く → 書(か)かれる  
> 読(よ)む → 読(よ)まれる  
> 言(い)う → 言(い)われる

使役(しえき)：あ段(だん) + せる

> 書(か)く → 書(か)かせる  
> 読(よ)む → 読(よ)ませる  
> 言(い)う → 言(い)わせる

使役受身(しえきうけみ)：高频口语常缩短成 `〜される`

> 書(か)かせられる → 書(か)かされる  
> 読(よ)ませられる → 読(よ)まされる  
> 飲(の)ませられる → 飲(の)まされる

### 6.2 一段动词

去掉 `る`，再接：

| 形式 | 例 |
| --- | --- |
| 受身(うけみ) | 食(た)べる → 食(た)べられる |
| 使役(しえき) | 食(た)べる → 食(た)べさせる |
| 使役受身(しえきうけみ) | 食(た)べる → 食(た)べさせられる |

## 七、和尊敬语的区别

`れる/られる` 不只有受身(うけみ)，也可以表示尊敬語(そんけいご)。

| 句子 | 意思 | 判断 |
| --- | --- | --- |
| 先生(せんせい)に叱(しか)られた。 | 被老师批评了。 | `に` 标动作主，是受身(うけみ) |
| 先生(せんせい)が話(はな)された。 | 老师说了。 | 老师做主语，是尊敬語(そんけいご) |

逐句拆解：

| 句子 | 助词用法 | 动词变形 | 语感 |
| --- | --- | --- | --- |
| 先生(せんせい)に叱(しか)られた。 | `に`=动作主；主语“我”被省略 | 叱(しか)る→叱(しか)られる→叱(しか)られた | 我被老师批评 |
| 先生(せんせい)が話(はな)された。 | `が`=动作主体/尊敬对象 | 話(はな)す→話(はな)される→話(はな)された | 老师讲话，尊敬表达 |

简单判断：

```
先生(せんせい)に Vられる → 多半是“被老师……”
先生(せんせい)が Vられる → 可能是“老师尊敬地……”
```

正式场合更常用 `お〜になる`，例如：

> 先生(せんせい)がお話(はな)しになる。  
> 老师讲话。

- 助词：`が` 标动作主体/尊敬对象 = 先生(せんせい)。
- 动词形式：話(はな)す → お話(はな)しになる，尊敬語(そんけいご)固定形式。
- 语感：比 `話(はな)される` 更正式、更自然。

### 卡片 1：四态一句话

### 卡片 2：最容易混的三个受身

| 类型 | 例 | 重点 |
| --- | --- | --- |
| 直接受身(うけみ) | 学生(がくせい)が褒(ほ)められた | 学生直接被表扬 |
| 受害受身(うけみ) | 雨(あめ)に降(ふ)られた | 我受困扰 |
| 所有者受身(うけみ) | 財布(さいふ)を盗(ぬす)まれた | 我的东西出事，所以我受害 |

### 卡片 3：中文母语者最该记

| 易错点 | 记忆 |
| --- | --- |
| `雨(あめ)に降(ふ)られた` | 不是“被雨降”，而是“遇上下雨，很倒霉” |
| `財布(さいふ)を盗(ぬす)まれた` | 主语通常是“我”，钱包用 `を` |
| `させる` | 既可强制，也可允许 |
| `書(か)かされる` | 五段使役受身常用短形 |

## 九、小练习

先看中文，判断该用哪种语态。

| 中文 | 应该想到 |
| --- | --- |
| 学生读书。 | 能動(のうどう) |
| 学生被老师表扬。 | 受身(うけみ) |
| 老师让学生读书。 | 使役(しえき) |
| 学生被老师逼着读书。 | 使役受身(しえきうけみ) |
| 下雨了，我很倒霉。 | 受害受身(うけみ) |
| 我钱包被偷了。 | 所有者受身(うけみ) |

答案例：

> 学生(がくせい)が本(ほん)を読(よ)む。  
> 学生(がくせい)が先生(せんせい)に褒(ほ)められる。  
> 先生(せんせい)が学生(がくせい)に本(ほん)を読(よ)ませる。  
> 学生(がくせい)が先生(せんせい)に本(ほん)を読(よ)まされる。  
> 雨(あめ)に降(ふ)られた。  
> 財布(さいふ)を盗(ぬす)まれた。

答案拆解：

| 答案 | 助词用法 | 动词变形 |
| --- | --- | --- |
| 学生(がくせい)が本(ほん)を読(よ)む。 | `が`=读的人；`を`=读的对象 | 読(よ)む=原形 |
| 学生(がくせい)が先生(せんせい)に褒(ほ)められる。 | `が`=被表扬的人；`に`=表扬的人 | 褒(ほ)める→褒(ほ)められる |
| 先生(せんせい)が学生(がくせい)に本(ほん)を読(よ)ませる。 | `が`=让别人做的人；`に`=被要求读的人；`を`=读的对象 | 読(よ)む→読(よ)ませる |
| 学生(がくせい)が先生(せんせい)に本(ほん)を読(よ)まされる。 | `が`=被迫读的人；`に`=逼他读的人；`を`=读的对象 | 読(よ)む→読(よ)まされる |
| 雨(あめ)に降(ふ)られた。 | `に`=造成影响的事件；主语“我”省略 | 降(ふ)る→降(ふ)られる→降(ふ)られた |
| 財布(さいふ)を盗(ぬす)まれた。 | `を`=被偷的东西；主语“我”省略 | 盗(ぬす)む→盗(ぬす)まれる→盗(ぬす)まれた |

最后只背这一句：

> **受身(うけみ)看受影响的人，使役(しえき)看让别人做的人，使役受身(しえきうけみ)看被迫做的人。**
