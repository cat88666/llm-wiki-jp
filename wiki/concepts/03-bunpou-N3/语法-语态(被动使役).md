## 一、先分清一类二类三类动词

日语动词变形的第一步不是背公式，而是先判断它是哪一类。

| 类型 | 日语名 | 常见结尾 | 例子 | 记忆点 |
| --- | --- | --- | --- | --- |
| 一类动词 | 五段動詞(ごだんどうし) | う、く、ぐ、す、つ、ぬ、ぶ、む、る | 書(か)く、読(よ)む、話(はな)す、行(い)く、帰(かえ)る | 词尾要变到 あ段 |
| 二类动词 | 一段動詞(いちだんどうし) | 多为 る 结尾，前面常是 い段/え段 | 食(た)べる、見(み)る、起(お)きる、借(か)りる | 去掉 る 再接 |
| 三类动词 | 不規則動詞(ふきそくどうし) | する、来(く)る | 勉強(べんきょう)する、来(く)る | 单独背 |

### 一类动词的あ段入口

一类动词做被动、使役时，都要先把词尾改到「あ段」。

| 原形 | あ段入口 | 说明 |
| --- | --- | --- |
| 書(か)く | 書(か)か | く → か |
| 読(よ)む | 読(よ)ま | む → ま |
| 話(はな)す | 話(はな)さ | す → さ |
| 待(ま)つ | 待(ま)た | つ → た |
| 言(い)う | 言(い)わ | う → わ，不能变成 あ |
| 帰(かえ)る | 帰(かえ)ら | る → ら |

**重点**：`言う` 的入口是 `言わ`，所以是 `言われる`、`言わせる`。

### 二类动词的去る入口

二类动词不要变音，只去掉最后的 `る`。

| 原形 | 词干 | 说明 |
| --- | --- | --- |
| 食(た)べる | 食(た)べ | 食べ + られる / させる |
| 見(み)る | 見(み) | 見 + られる / させる |
| 起(お)きる | 起(お)き | 起き + られる / させる |

### 三类动词单独背

| 原形 | 被动 | 使役 | 使役受身 |
| --- | --- | --- | --- |
| する | される | させる | させられる |
| 来(く)る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

## 二、被动形：被做/受到影响

被动形叫 受身(うけみ)。中文常翻成“被……”，但日语里更准确的感觉是：**主语受到某个动作的影响**。

### 一类动词：あ段 + れる

公式：

```text
書く → 書か + れる → 書かれる
読む → 読ま + れる → 読まれる
話す → 話さ + れる → 話される
```

| 原形 | 被动形 | 例句 |
| --- | --- | --- |
| 書(か)く | 書(か)かれる | 名前(なまえ)を書(か)かれる。（名字被写上。） |
| 読(よ)む | 読(よ)まれる | 日記(にっき)を読(よ)まれた。（日记被看了。） |
| 話(はな)す | 話(はな)される | 秘密(ひみつ)を話(はな)された。（秘密被说出去了。） |
| 言(い)う | 言(い)われる | 先生(せんせい)に注意(ちゅうい)された / そう言(い)われた。（被老师提醒 / 被那样说了。） |

### 二类动词：去る + られる

公式：

```text
食べる → 食べ + られる → 食べられる
見る → 見 + られる → 見られる
```

| 原形 | 被动形 | 例句 |
| --- | --- | --- |
| 食(た)べる | 食(た)べられる | ケーキを食(た)べられた。（蛋糕被吃了。） |
| 見(み)る | 見(み)られる | 写真(しゃしん)を見(み)られた。（照片被看到了。） |
| 褒(ほ)める | 褒(ほ)められる | 先生(せんせい)に褒(ほ)められた。（被老师表扬了。） |

### 三类动词：する→される，来る→来られる

| 原形 | 被动形 | 例句 |
| --- | --- | --- |
| する | される | 説明(せつめい)された。（被说明了。） |
| 来(く)る | 来(こ)られる | 友達(ともだち)に来(こ)られて困(こま)った。（朋友来了，搞得我很困扰。） |

### 被动句的基本助词框架

```text
A が B を V。        A 做某事
B が A に Vられる。 B 被 A 做某事
```

> 先生(せんせい)が学生(がくせい)を褒(ほ)めた。  
> 老师表扬了学生。

> 学生(がくせい)が先生(せんせい)に褒(ほ)められた。  
> 学生被老师表扬了。

**注意**：动作发出者常用 `に`。正式文章里也会用 `によって`。

## 三、使役形：让/允许别人做

使役(しえき)表示“让别人做”或“允许别人做”。同一个形式，语感要看上下文。

### 一类动词：あ段 + せる

公式：

```text
書く → 書か + せる → 書かせる
読む → 読ま + せる → 読ませる
話す → 話さ + せる → 話させる
```

| 原形 | 使役形 | 例句 |
| --- | --- | --- |
| 書(か)く | 書(か)かせる | 先生(せんせい)が学生(がくせい)に作文(さくぶん)を書(か)かせた。（老师让学生写作文。） |
| 読(よ)む | 読(よ)ませる | 母(はは)が子供(こども)に本(ほん)を読(よ)ませた。（妈妈让孩子读书。） |
| 行(い)く | 行(い)かせる | 親(おや)が子供(こども)を学校(がっこう)に行(い)かせた。（父母让孩子去学校。） |
| 言(い)う | 言(い)わせる | 最後(さいご)まで言(い)わせてください。（请让我说到最后。） |

### 二类动词：去る + させる

公式：

```text
食べる → 食べ + させる → 食べさせる
見る → 見 + させる → 見させる
```

| 原形 | 使役形 | 例句 |
| --- | --- | --- |
| 食(た)べる | 食(た)べさせる | 母(はは)が子供(こども)に野菜(やさい)を食(た)べさせた。（妈妈让孩子吃蔬菜。） |
| 見(み)る | 見(み)させる | 写真(しゃしん)を見(み)させてください。（请让我看照片。） |
| 起(お)きる | 起(お)きさせる | 子供(こども)を早(はや)く起(お)きさせた。（让孩子早起。） |

### 三类动词：する→させる，来る→来させる

| 原形 | 使役形 | 例句 |
| --- | --- | --- |
| する | させる | 部下(ぶか)に準備(じゅんび)させた。（让下属准备。） |
| 来(く)る | 来(こ)させる | 先生(せんせい)が学生(がくせい)を教室(きょうしつ)に来(こ)させた。（老师让学生来教室。） |

### 使役句的 に / を

最容易错的是“被使役的人”用 `に` 还是 `を`。

| 原动词 | 框架 | 例句 |
| --- | --- | --- |
| 自動詞(じどうし) | A が B を Vさせる | 親(おや)が子供(こども)を行(い)かせる。（父母让孩子去。） |
| 他動詞(たどうし) | A が B に C を Vさせる | 先生(せんせい)が学生(がくせい)に作文(さくぶん)を書(か)かせる。（老师让学生写作文。） |

为什么他动词用 `に`？因为 `を` 已经给了动作对象：作文を、野菜を、本を。

## 四、使役受身：被迫做

使役受身(しえきうけみ) = 使役 + 被动。意思是：**别人让我做，我不情愿，但不得不做**。中文常翻成“被迫……”。

### 一类动词：あ段 + せられる，也常缩成 あ段 + される

正式形：

```text
書く → 書か + せられる → 書かせられる
読む → 読ま + せられる → 読ませられる
```

口语常用缩约形：

```text
書かせられる → 書かされる
読ませられる → 読まされる
待たせられる → 待たされる
```

| 原形 | 正式形 | 常用缩约形 | 例句 |
| --- | --- | --- | --- |
| 書(か)く | 書(か)かせられる | 書(か)かされる | 反省文(はんせいぶん)を書(か)かされた。（被迫写检讨。） |
| 読(よ)む | 読(よ)ませられる | 読(よ)まされる | 長(なが)い文章(ぶんしょう)を読(よ)まされた。（被迫读长文章。） |
| 行(い)く | 行(い)かせられる | 行(い)かされる | 会議(かいぎ)に行(い)かされた。（被迫去开会。） |
| 言(い)う | 言(い)わせられる | 言(い)わされる | 本当(ほんとう)のことを言(い)わされた。（被迫说出真话。） |

### 二类动词：去る + させられる，不能乱缩

| 原形 | 使役受身 | 例句 |
| --- | --- | --- |
| 食(た)べる | 食(た)べさせられる | 嫌(きら)いな野菜(やさい)を食(た)べさせられた。（被迫吃讨厌的蔬菜。） |
| 見(み)る | 見(み)させられる | つまらない映画(えいが)を見(み)させられた。（被迫看无聊的电影。） |
| 起(お)きる | 起(お)きさせられる | 朝(あさ)早(はや)く起(お)きさせられた。（被迫早起。） |

不能说：~~食べさされる~~、~~見さされる~~。

### 三类动词：する→させられる，来る→来させられる

| 原形 | 使役受身 | 例句 |
| --- | --- | --- |
| する | させられる | 残業(ざんぎょう)させられた。（被迫加班。） |
| 来(く)る | 来(こ)させられる | 休(やす)みの日(ひ)に会社(かいしゃ)に来(こ)させられた。（休息日被迫来公司。） |

不能说：~~さされる~~、~~来(こ)さされる~~。

先横向背这一张。每次看到动词，按“一类/二类/三类”套进去。

| 类型 | 原形 | 被动 | 使役 | 使役受身 |
| --- | --- | --- | --- | --- |
| 一类 | 書(か)く | 書(か)かれる | 書(か)かせる | 書(か)かせられる / 書(か)かされる |

| 一类 | 買(か)う | 買(か)われる | 買(か)わせる | 買(か)わせられる / 買(か)わされる |
| 一类 | 言(い)う | 言(い)われる | 言(い)わせる | 言(い)わせられる / 言(い)わされる |
| 二类 | 食(た)べる | 食(た)べられる | 食(た)べさせる | 食(た)べさせられる |
| 二类 | 見(み)る | 見(み)られる | 見(み)させる | 見(み)させられる |
| 二类 | 起(お)きる | 起(お)きられる | 起(お)きさせる | 起(お)きさせられる |

| 三类 | 来(く)る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

### 最短公式

| 类型 | 被动 | 使役 | 使役受身 |
| --- | --- | --- | --- |
| 一类 | あ段 + れる | あ段 + せる | あ段 + せられる / あ段 + される |
| 二类 | 去る + られる | 去る + させる | 去る + させられる |
| 三类：する | される | させる | させられる |
| 三类：来る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

## 六、最容易混淆的地方

### 1. 食べられる：可能还是被动

`食べられる` 既可以是“能吃”，也可以是“被吃”。要看上下文。

| 句子 | 意思 | 判断 |
| --- | --- | --- |
| この魚(さかな)は生(なま)で食(た)べられる。 | 这鱼可以生吃。 | 可能 |
| 私(わたし)のケーキが弟(おとうと)に食(た)べられた。 | 我的蛋糕被弟弟吃了。 | 被动，有 `に` 标动作人 |

### 2. 先生が話された：尊敬，不一定是被动

`れる/られる` 还有尊敬用法。

| 句子 | 意思 |
| --- | --- |
| 先生(せんせい)に話(はな)された。 | 被老师说了。 |
| 先生(せんせい)が話(はな)された。 | 老师说了。尊敬表达。 |

判断口诀：

```text
先生に Vられる → 多半是被动
先生が Vられる → 多半是尊敬
```

详见 [[敬语体系]]。

### 3. させる：强制和允许都可以

使役不一定是“逼迫”。它也可以是“允许”。

| 句子 | 语感 |
| --- | --- |
| 先生(せんせい)が学生(がくせい)に宿題(しゅくだい)をやらせた。 | 让学生做作业，偏强制 |
| 子供(こども)を自由(じゆう)に遊(あそ)ばせた。 | 让孩子自由玩，偏允许 |
| 少(すこ)し休(やす)ませてください。 | 请让我休息一下 |

### 4. させられる：不是“让别人”，而是“自己被迫”

| 形 | 主语感觉 | 例句 |
| --- | --- | --- |
| 書(か)かせる | 我让别人写 | 先生(せんせい)が学生(がくせい)に書(か)かせた。 |
| 書(か)かされる | 我被迫写 | 学生(がくせい)が先生(せんせい)に書(か)かされた。 |

记忆法：看到 `させられる / される`，先想“被迫”。

### 5. 一类的 う → わ

`言う`、`買う`、`会う` 这一类不要变成 `あ`。

| 原形 | 被动 | 使役 | 使役受身 |
| --- | --- | --- | --- |
| 言(い)う | 言(い)われる | 言(い)わせる | 言(い)わされる |
| 買(か)う | 買(か)われる | 買(か)わせる | 買(か)わされる |
| 誘(さそ)う | 誘(さそ)われる | 誘(さそ)わせる | 誘(さそ)わされる |

### 6. 二类不要用一类规则

`見る`、`起きる` 虽然以 `る` 结尾，但它们是二类。

| 错误 | 正确 |
| --- | --- |
| ~~見らせる~~ | 見させる |
| ~~起きらせる~~ | 起きさせる |
| ~~食べされる~~ | 食べられる |

## 七、强化记忆练习

### 练习 A：先判断类型

| 动词 | 类型 | 提示 |
| --- | --- | --- |
| 書(か)く | 一类 | く 结尾 |
| 食(た)べる | 二类 | べる，去る |
| 見(み)る | 二类 | 短词，但仍是二类 |
| する | 三类 | 不规则 |
| 来(く)る | 三类 | 不规则，读音变化为 こ |
| 言(い)う | 一类 | う → わ |

### 练习 B：把原形变成三种形式

| 原形 | 被动 | 使役 | 使役受身 |
| --- | --- | --- | --- |
| 書(か)く | 書(か)かれる | 書(か)かせる | 書(か)かされる |
| 読(よ)む | 読(よ)まれる | 読(よ)ませる | 読(よ)まされる |
| 食(た)べる | 食(た)べられる | 食(た)べさせる | 食(た)べさせられる |
| 見(み)る | 見(み)られる | 見(み)させる | 見(み)させられる |
| する | される | させる | させられる |
| 来(く)る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

### 练习 C：用一句话记住

```text
一类：先变あ段。被动 +れる，使役 +せる，使役受身 +される。
二类：去る。被动 +られる，使役 +させる，使役受身 +させられる。
三类：する/来る 单独背。
```

例句连读：

> 書(か)く → 書(か)かれる → 書(か)かせる → 書(か)かされる  
> 写 → 被写 → 让写 → 被迫写

> 食(た)べる → 食(た)べられる → 食(た)べさせる → 食(た)べさせられる  
> 吃 → 被吃/能吃 → 让吃 → 被迫吃

> する → される → させる → させられる  
> 做 → 被做 → 让做 → 被迫做
