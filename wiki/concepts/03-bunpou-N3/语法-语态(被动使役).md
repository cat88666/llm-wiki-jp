# 语态体系

> 语态(ヴォイス)决定"句子以谁的视角叙述"。同一个事件，选不同语态 = 选不同主语 = 选不同助词框架。核心关注：主语是谁、助词怎么配、动词怎么变。

## 四态总览与助词框架

用同一事件 `先生(せんせい) / 学生(がくせい) / 作文(さくぶん)を書(か)く` 对齐四种语态：

| 语态 | 例句 | 主语视角 |
| --- | --- | --- |
| 能動(のうどう) | 先生(せんせい)が学生(がくせい)に作文(さくぶん)を書(か)かせた。 | — (基准) |
| 受身(うけみ) | 学生(がくせい)が先生(せんせい)に叱(しか)られた。（学生被老师批评了。） | 受影响的人 |
| 使役(しえき) | 先生(せんせい)が学生(がくせい)に作文(さくぶん)を書(か)かせた。（老师让学生写作文。） | 让别人做的人 |
| 使役受身(しえきうけみ) | 学生(がくせい)が先生(せんせい)に作文(さくぶん)を書(か)かされた。（学生被老师逼着写作文。） | 被迫做的人 |

助词框架速查：

```
能動：    A が B を V
受身：    B が A に V(ら)れる
使役：    A が B に/を Vさせる      ← に/を 的选择见下文
使役受身： B が A に V(さ)せられる
```

## 受身的三种类型

受身(うけみ)的核心不是中文的"被"，而是**主语受到了影响**。

### 直接受身：动作直接作用于主语

中文可以翻译成"被"，最容易理解。

> 学生(がくせい)が先生(せんせい)に褒(ほ)められた。（学生被老师表扬了。）

> 私(わたし)は友達(ともだち)に助(たす)けられた。（我被朋友帮助了。）

### 受害受身：主语并非动作对象，但受到了影响 【N3】

日语特有，中文没有对应结构。主语感到困扰、倒霉。

> 雨(あめ)に降(ふ)られた。（遇上下雨，很倒霉。）

不是"雨被谁下"。降(ふ)る 是自動詞(じどうし)，本身没有动作对象，但说话人（省略的 私）受到了影响。

> 子供(こども)に泣(な)かれて、仕事(しごと)ができなかった。（孩子一直哭，搞得我没法工作。）

> 隣(となり)の人(ひと)にタバコを吸(す)われた。（旁边的人抽烟，我被熏到了。）

**判断法**：中文硬翻"被"很奇怪，但句子有倒霉/困扰的语感 → 受害受身。

### 所有者受身：我的东西/身体被动了

日语习惯把"受害的人"放主语，而非把"东西"放主语。

> 私(わたし)は財布(さいふ)を盗(ぬす)まれた。（我钱包被偷了。）

注意助词：私は…を盗まれた。`を` 标被偷的东西，`は` 标受害者。对比客观叙述：

> 財布(さいふ)が盗(ぬす)まれた。（钱包被偷了。——只描述事实，不突出"我倒霉"。）

> 私(わたし)は足(あし)を踏(ふ)まれた。（我脚被踩了。）

**反直觉**：中文说"我的钱包被偷了"，主语是钱包。日语说「私は財布を盗まれた」，主语是"我"——因为日语关注的是"谁受害"，不是"什么东西出事"。

## れる/られる 的四重身份

`れる/られる` 不只表示受身。它有四种功能，必须靠上下文判断：

| 功能 | 例句 | 判断线索 |
| --- | --- | --- |
| 受身 | 先生(せんせい)**に**叱(しか)られた。（被老师批评了。） | 有 `に` 标动作主 |
| 尊敬 | 先生(せんせい)**が**話(はな)された。（老师讲话了。——尊敬表达） | 尊敬对象做主语（が），无 に 标动作主 |
| 可能 | この魚(さかな)は生(なま)で食(た)べ**られる**。（这鱼能生吃。） | 表示能力/可能性 |
| 自発(じはつ) | 故郷(こきょう)のことが思(おも)い出(だ)される。（不由得想起故乡。） | 思う/感じる/偲ぶ 等感知动词，非意志性 |

### 受身 vs 尊敬：最高频混淆点

```
先生(せんせい)に Vられる → 受身（被老师……）
先生(せんせい)が Vられる → 尊敬（老师做了……）
```

正式场合的尊敬表达更常用 `お〜になる`，不容易和受身混淆：

> 先生(せんせい)がお話(はな)しになった。（老师讲话了。）

→ 详见 [[敬语体系]]

### 自発：不由得…… 【N3-N2】

自発(じはつ)表示非意志性的心理活动自然发生，只用于感知・思考类动词。

> 亡(な)くなった祖母(そぼ)のことが偲(しの)ばれる。（不由得怀念去世的祖母。）

> そう考(かんが)えられる。（自然会这么认为。）

**为什么自発只接感知动词？** 因为自発表达的是"不受意志控制、自然浮现的心理状态"。移動・作業など意志動詞不能用自発。

## 书面语被动 【N3-N2】

新闻、论文、正式文书中大量使用被动句。这是 N3→N2 的关键过渡。

### 〜によって：强调动作主体

口语受身用 `に` 标动作主，书面语用 `によって`，多见于客观叙述。

> この寺(てら)は14世紀(せいき)に足利義満(あしかがよしみつ)によって建(た)てられた。（这座寺庙由足利义满于14世纪建造。）

> 新(あたら)しい法律(ほうりつ)が国会(こっかい)によって承認(しょうにん)された。（新法律由国会批准。）

### 〜と思(おも)われる / 〜と考(かんが)えられる：客观判断

表示"一般认为……"，避免主观断言。学术论文高频表达。

> この問題(もんだい)は深刻(しんこく)だと思(おも)われる。（这个问题被认为很严重。）

> 原因(げんいん)は気候変動(きこうへんどう)にあると考(かんが)えられている。（原因被认为在于气候变化。）

### 〜とされている：定论/通说

表示"被普遍认定为……"。

> 日本語(にほんご)の敬語(けいご)は五(いつ)つに分類(ぶんるい)されるとされている。（日语敬语被认定分为五类。）

## 使役的 に と を

使役(しえき)的核心：**A 让 B 做某事**。难点在于 B 用 `に` 还是 `を`。

### 规则：看原动词是自動詞还是他動詞

```
自動詞 → B を Vさせる    （を 空着，给 B 用）
他動詞 → B に C を Vさせる （を 已经被 C 占了，B 只能用 に）
```

**为什么？** 一个句子里 `を` 原则上只出现一次。他動詞已经有一个 `を`（动作对象），所以被使役的人只能用 `に`。

| 动词类型 | 例句 | 助词分析 |
| --- | --- | --- |
| 自動詞：行(い)く | 親(おや)が子供(こども)**を**学校(がっこう)に行(い)かせた。（父母让孩子去学校。） | 行く 无宾语，子供 用 を |
| 自動詞：泣(な)く | 彼(かれ)は彼女(かのじょ)**を**泣(な)かせた。（他把她弄哭了。） | 泣く 无宾语，彼女 用 を |
| 他動詞：書(か)く | 先生(せんせい)が学生(がくせい)**に**作文(さくぶん)**を**書(か)かせた。（老师让学生写作文。） | を 给了作文，学生 用 に |
| 他動詞：食(た)べる | 母(はは)が子供(こども)**に**野菜(やさい)**を**食(た)べさせた。（妈妈让孩子吃蔬菜。） | を 给了野菜，子供 用 に |

### 强制 vs 允许：同一形式，靠语境区分

| 语感 | 例句 |
| --- | --- |
| 强制（不做也得做） | 先生(せんせい)が学生(がくせい)に宿題(しゅくだい)をやらせた。（老师让学生做作业。） |
| 允许（想做，所以让做） | 子供(こども)を好(す)きなように遊(あそ)ばせた。（让孩子随意玩。） |
| 请求允许 | 好(す)きにさせてください。（请让我按自己喜欢的方式做。） |

**反直觉**：`させてください` 是请求允许（"请让我做"），不是命令别人。`させていただく` 更谦恭 → [[敬语体系]]。

## 使役受身与缩约形

使役受身(しえきうけみ) = 使役 + 受身。语感：**别人让我做，我不愿意但不得不做**。

> 私(わたし)は上司(じょうし)に残業(ざんぎょう)させられた。（我被上司要求加班了。）

> 子供(こども)の頃(ころ)、野菜(やさい)を食(た)べさせられた。（小时候被迫吃蔬菜。）

### 五段动词的缩约形

五段动词的使役受身有正式形和缩约形两种，口语中缩约形更常用：

| 正式形 | 缩约形 | 例 |
| --- | --- | --- |
| Vあせられる | Vあされる | 書(か)かせられる → 書(か)か**される** |
| | | 読(よ)ませられる → 読(よ)ま**される** |
| | | 飲(の)ませられる → 飲(の)ま**される** |

缩约规则：`〜せられる` → `〜される`（去掉 `ら`）。

### 一段動詞・する は缩约不可

**反直觉**：缩约只限五段动词。一段動詞と する 不能缩约。

| 动词 | 使役受身 | 能否缩约 |
| --- | --- | --- |
| 書(か)く（五段） | 書(か)かせられる → 書(か)かされる | ○ |
| 食(た)べる（一段） | 食(た)べさせられる | ✕（不能说 ~~食べさされる~~） |
| する（不规则） | させられる | ✕（不能说 ~~さされる~~） |
| 来(く)る（不规则） | 来(こ)させられる | ✕（不能说 ~~来さされる~~） |

## 活用总表

变形的接续原理详见 [[语法(变形)]] 和 [[词性(助动词)]]。

| 动词类型 | 原形 | 受身 | 使役 | 使役受身（正式/缩约） |
| --- | --- | --- | --- | --- |
| 五段(ごだん) | 書(か)く | 書(か)かれる | 書(か)かせる | 書(か)かせられる / 書(か)かされる |
| 五段 | 読(よ)む | 読(よ)まれる | 読(よ)ませる | 読(よ)ませられる / 読(よ)まされる |
| 五段 | 言(い)う | 言(い)われる | 言(い)わせる | 言(い)わせられる / 言(い)わされる |
| 一段(いちだん) | 食(た)べる | 食(た)べられる | 食(た)べさせる | 食(た)べさせられる |
| 不规则 | する | される | させる | させられる |
| 不规则 | 来(く)る | 来(こ)られる | 来(こ)させる | 来(こ)させられる |

五段动词规则：

```
受身：  未然形(あ段) + れる    書く → 書か + れる
使役：  未然形(あ段) + せる    書く → 書か + せる
```

一段動詞規則：

```
受身：  語幹 + られる    食べ + られる
使役：  語幹 + させる    食べ + させる
```
