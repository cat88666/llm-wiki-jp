# 复合动词体系

> 复合动词由「V连用形+V」构成，后项动词为前项动作附加体态・方向・程度等意义。N3-N2 阶段大量增加，理解后项动词的意义体系是扩展词汇量的关键。

## 一、复合动词的结构

**接续**：前項動詞の**連用形（ます形去ます）** + 後項動詞

```
複合動詞の形成
  食べ（食べる連用形）+ 始める → 食べ始める
  読み（読む連用形）  + 終わる → 読み終わる
  飛び（飛ぶ連用形）  + 込む   → 飛び込む
```

**与テ形的区别**：

| 形式 | 结合程度 | 语义 | 例 |
|------|---------|------|-----|
| 複合動詞（連用+V） | 一个单词，写在一起 | 融合为一个语义单位 | 読み終わる |
| テ形接续（テ+V） | 独立两个动词 | 顺序/手段关系 | 読んで終わる（读完后结束）|

连用形形成的复合动词是**词汇层面**的，テ形是句法层面的。

## 二、体态系后项动词

| 后项动词 | 含义 | 例 |
|---------|------|-----|
| 〜始める | 开始〜（有意识地开始） | 話し**始める**、泣き**始める** |
| 〜出す | 突然开始〜（无预兆） | 泣き**出す**、走り**出す**、降り**出す** |
| 〜続ける | 持续〜（不间断） | 歩き**続ける**、働き**続ける** |
| 〜終わる | 完成〜结束 | 読み**終わる**、食べ**終わる** |
| 〜かける | 开始但未完成（中途） | 食べ**かける**（吃了一半）、言い**かける** |
| 〜かかる | 接近完成/开始（被动倾向） | 死に**かかる**（快要死了） |

**始める vs 出す**：

> 雨が降り**始めた**。（雨开始下了——预期中的开始）  
> 急に雨が降り**出した**。（突然下起雨来——意外、突然）

## 三、方向・完遂系后项动词

| 后项动词 | 含义 | 例 |
|---------|------|-----|
| 〜上げる | 完成（他动词感） | 仕上げる、書き**上げる**、作り**上げる** |
| 〜上がる | 完成（自动词感）/向上 | できあがる、出来**上がる**、盛り**上がる** |
| 〜込む | 向内/深入/彻底 | 飛び**込む**、持ち**込む**、落ち**込む** |
| 〜切る | 彻底完成/极限 | 使い**切る**（用完）、疲れ**切る**（精疲力竭） |
| 〜抜く | 坚持到底/贯穿 | 走り**抜く**、生き**抜く**、やり**抜く** |
| 〜通す | 从头到尾坚持 | 読み**通す**、やり**通す** |
| 〜出す | 向外/取出 | 取り**出す**、持ち**出す**、引き**出す** |

**〜切る vs 〜抜く**：

> 食べ**切る** = 全部吃完（全量消耗）  
> やり**抜く** = 坚持做完（克服困难，到终点）

## 四、程度・评价系后项动词

| 后项动词 | 含义 | 例 |
|---------|------|-----|
| 〜過ぎる | 过度（N4） | 食べ**過ぎる**、飲み**過ぎる**、考え**過ぎる** |
| 〜合う | 相互・互相 | 話し**合う**、助け**合う**、分かり**合う** |
| 〜直す | 重新做・改正 | 書き**直す**、やり**直す**、作り**直す** |
| 〜損なう | 没做成/失误 | 言い**損なう**（想说没说成）、見**損なう** |
| 〜忘れる | 忘了做 | 言い**忘れる**（忘了说）、持ち**忘れる** |

---

## 五、词汇性复合动词 vs 句法性复合动词

| 类型 | 特征 | 例 |
|------|------|-----|
| 句法性（統語的） | 后项动词保持独立意义，意义透明 | 食べ始める（始める=开始 意义清晰）|
| 词汇性（語彙的） | 前后融合为新意义，不透明 | 飛び込む（≠飛ぶ+込む的字面）→跳进去 |

**意义透明度决定学习策略**：
- 句法性（体态系：始める/続ける/終わる 等）：掌握后项动词语义→系统推导
- 词汇性（飛び込む/追いかける 等）：需逐个记忆

## 六、N2 高频复合动词速查

| 复合动词 | 含义 | 例句 |
|---------|------|------|
| 取り組む | 致力于・积极处理 | 問題に取り組む |
| 受け入れる | 接受・接纳 | 意見を受け入れる |
| 打ち明ける | 坦诚・袒露 | 秘密を打ち明ける |
| 思い込む | 坚信（有时错误） | 彼が正しいと思い込んでいた |
| 落ち着く | 冷静・稳定 | 落ち着いて考えよう |
| 呼びかける | 呼吁・号召 | 節電を呼びかける |
| 引き受ける | 承担・接手 | 仕事を引き受ける |
| 乗り越える | 克服・渡过 | 困難を乗り越える |
| 立ち上がる | 站起来/起步 | 新会社が立ち上がる |
| 追い付く | 赶上 | 先頭グループに追い付く |
