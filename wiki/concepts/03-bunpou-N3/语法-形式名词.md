# 形式名词表达(N3)

> 形式名词是“名词外壳 + 语法功能”。N3 重点不是背单词义，而是识别它把前面的句子变成什么：事实、理由、预期、局面、状态保持。

## 一、形式名词的判断法

| 词 | 实义 | N3 语法功能 |
| --- | --- | --- |
| こと | 事情 | 把动作/句子名词化，表示经验、决定、忠告 |
| もの | 物 | 一般道理、回忆感慨、带解释感的理由 |
| わけ | 理由 | 逻辑归结、并非、不可能、不能那样做 |
| はず | 应当如此的根据 | 有依据的预期、预期落空 |
| ところ | 地方 | 动作局面：正要、正在、刚刚 |
| うち | 内部/期间 | 趁某状态还在、在某期间发生变化 |
| まま | 原样 | 保持某状态不变 |

形式名词前面通常接普通形；名词和な形容词接续时常见 `の/な/である`，具体要按文型记。

## 二、こと 系

| 文型 | 接续 | 含义 | 例 |
| --- | --- | --- | --- |
| Vこと | 辞书形/ない形 + こと | 名词化 | 泳(およ)ぐことが好きだ。 |
| Vたことがある | た形 + ことがある | 曾经做过 | 京都(きょうと)へ行(い)ったことがある。 |
| Vることがある | 辞书形 + ことがある | 有时会做/发生 | 週末(しゅうまつ)は出勤(しゅっきん)することがある。 |
| Vることにする | 辞书形/ない形 + ことにする | 自己决定 | 毎日(まいにち)復習(ふくしゅう)することにした。 |
| Vることになる | 辞书形/ない形 + ことになる | 外部决定/自然结果 | 来月(らいげつ)転勤(てんきん)することになった。 |
| Vることだ | 辞书形/ない形 + ことだ | 忠告 | 早(はや)く寝(ね)ることだ。 |

### 2.1 例句拆解

| 句子 | 结构 | 语感 |
| --- | --- | --- |
| 日本語(にほんご)を話(はな)すことができる。 | V辞书形 + こと + ができる | 把“说日语”整体名词化成能力对象 |
| 毎朝(まいあさ)走(はし)ることにした。 | V辞书形 + ことにする | 说话者自己定下习惯 |
| 会議(かいぎ)は来週(らいしゅう)行(おこな)うことになった。 | V辞书形 + ことになる | 会议安排由外部决定 |

## 三、もの 系

| 文型 | 接续 | 含义 | 例 |
| --- | --- | --- | --- |
| Vるものだ | 辞书形 + ものだ | 一般道理/本来如此 | 約束(やくそく)は守(まも)るものだ。 |
| Vたものだ | た形 + ものだ | 回忆感慨 | 子供(こども)のころ、よく川(かわ)で遊(あそ)んだものだ。 |
| 普通形 + ものだから | 普通形 + ものだから | 理由，带解释/辩解 | 道(みち)が込(こ)んでいたものだから、遅(おく)れた。 |
| Vれるものなら | 可能形 + ものなら | 如果能做到的话，常含反语 | できるものなら、やってみたい。 |

### 3.1 ものだ 的两种核心含义

| 形式 | 含义 | 判断 |
| --- | --- | --- |
| 辞书形 + ものだ | 社会常识/一般规律 | “本来就该/通常就是” |
| た形 + ものだ | 过去回忆 | “以前常常……啊” |

> 人(ひと)は助(たす)け合(あ)うものだ。（人本来就该互相帮助。）  
> 昔(むかし)はよくこの店(みせ)に来(き)たものだ。（以前常来这家店啊。）

## 四、わけ 系

| 文型 | 含义 | 例 |
| --- | --- | --- |
| わけだ | 逻辑归结，怪不得/也就是说 | 10年(じゅうねん)も勉強(べんきょう)した。上手(じょうず)なわけだ。 |
| わけではない | 并不是说…… | 嫌(きら)いなわけではない。 |
| わけがない | 不可能/没道理 | 彼(かれ)がそんなことを言(い)うわけがない。 |
| わけにはいかない | 不能那样做，受情理限制 | ここで諦(あきら)めるわけにはいかない。 |

### 4.1 わけ 的判断口令

```
わけだ             看到理由后做结论：怪不得
わけではない       否定整体判断：并不是说
わけがない         否定可能性：不可能
わけにはいかない   否定行动许可：不能那么做
```

## 五、はず 系

| 文型 | 含义 | 例 |
| --- | --- | --- |
| はずだ | 有依据的预期，应该 | 彼(かれ)はもう着(つ)いたはずだ。 |
| はずがない | 不可能，不应该会 | そんなはずがない。 |
| はずだった | 原本应该 | 今日(きょう)は休(やす)みのはずだった。 |
| はずなのに | 按理应该却没有 | 簡単(かんたん)なはずなのに、できなかった。 |

`はずだ` 的重点是“说话者有根据”。它不是单纯愿望，也不是命令。

## 六、ところ・うち・まま

### 6.1 ところ

| 接续 | 含义 | 例 |
| --- | --- | --- |
| Vるところだ | 正要做 | これから出(で)かけるところだ。 |
| Vているところだ | 正在做 | 今(いま)、調(しら)べているところだ。 |
| Vたところだ | 刚做完 | ちょうど帰(かえ)ったところだ。 |

### 6.2 うち

| 文型 | 含义 | 例 |
| --- | --- | --- |
| Vる/ないうちに | 趁还没变化前 | 忘(わす)れないうちにメモする。 |
| い形/な形な/名词の + うちに | 在某状态期间 | 若(わか)いうちに経験(けいけん)したほうがいい。 |
| Vているうちに | 在做的过程中逐渐变化 | 聞(き)いているうちに分(わ)かってきた。 |

### 6.3 まま

| 接续 | 含义 | 例 |
| --- | --- | --- |
| Vたまま | 做完后保持状态 | 電気(でんき)をつけたまま寝(ね)た。 |
| Vないまま | 没做某事就保持后续 | 朝(あさ)ご飯(はん)を食(た)べないまま出(で)かけた。 |
| 名词の/な形な/い形 + まま | 保持原样 | 静(しず)かなまま終(お)わった。 |

## 七、易混对比

| 易混组 | 区别 |
| --- | --- |
| わけだ / はずだ | わけだ=看到事实后归结；はずだ=根据已知信息预期 |
| たところ / たばかり | たところ=刚刚做完，时间感更近；たばかり=主观觉得刚做完 |
| うちに / 間に | うちに=趁状态还在，常有“别错过”；間に=在期间某点发生 |
| まま / ながら | まま=状态保持；ながら=两个动作同时进行 |
| ことにする / ことになる | 自己决定=ことにする；外部决定=ことになる |
