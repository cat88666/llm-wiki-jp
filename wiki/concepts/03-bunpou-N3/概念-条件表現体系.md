# 条件表現体系

> 日语的条件表达是「と・ば・たら・なら」的四形式体系。各形式在接续规则・后件限制・时间关系上各不相同，不能用「如果〜的话」一对一翻译。

## 一、四种条件形式的全景

| 形式 | 接续 | 后件限制 | 时间关系 | 特征 |
|------|------|---------|---------|------|
| と（N4） | 辞书形/ない形 | **意志/命令/依頼×** | 同步/恒常 | 自然法则、反复・习惯、意外发现 |
| ば（N4） | 仮定形 | **意志/命令/依頼△** | 未実現假定 | 假定中的状态条件、〜ば〜ほど |
| たら（N4） | た形 + ら | 限制最少（最通用） | 前件完成后→后件 | 通用、偶然发现、假定 |
| なら（N4） | 名詞/普通形 + なら | 後件が前件より前に実施可 | 话题拾起、前件不一定实现 | 基于对方信息的建议 |

---

## 二、と：恒常条件・发现

**接续**：動詞辞書形/ない形 + と

**核心**：条件与结果之间存在**必然关系或反复关系**，不受说话者意志控制。

| 用法 | 例 | 说明 |
|------|-----|------|
| 自然法则 | 春になる**と**、桜が咲く | 必然规律 |
| 操作结果 | 右に曲がる**と**、駅が見える | 按操作步骤的必然结果 |
| 偶然发现 | 部屋に入る**と**、彼がいた | 意外发现（动作已完成） |
| 习惯反复 | 夜になる**と**、静かになる | 每次都如此 |

**后件限制**（重要）：后件**不能用意志・命令・依赖・勧誘等**。

> ×春になると、花見をしよう。（×意志形）  
> ×この薬を飲むと、早く寝てください。（×命令形）  
> ○春になると、暖かくなる。（○状态描述）

## 三、ば：假定条件

**接续**：動詞仮定形（え段 + ば）+ 形容词（〜ければ）+ 名詞/ナ形（〜であれば/〜なら）

**五段動詞仮定形**：词尾母音→え段 + ば

| 动词 | 仮定形 | + ば |
|------|-------|------|
| 書く | 書け- | 書け**ば** |
| 読む | 読め- | 読め**ば** |
| 行く | 行け- | 行け**ば** |
| 一段・する・来る | 食べ- | 食べれ**ば** |

**核心用法**：
1. **假定条件**：もし〜ば〜（书面/正式，口语よりたら多）
2. **〜ば〜ほど**：越〜越〜（N3 重要文型）

> 勉強すれ**ば**、成績が上がる。（如果学习，成绩会提高。）  
> 読め**ば**読む**ほど**、日本語が上手になる。（越读越好。）

**后件限制**：意志・命令・依赖等在一定语境下可接，但比たら限制稍多。

---

## 四、たら：完成条件・假定

**接续**：タ形（た/だ）+ ら

**最通用的条件形式**，限制最少，口语使用频率最高。

| 用法 | 例 | 说明 |
|------|-----|------|
| 未来假定（前件完成后） | 宿題が終わっ**たら**、遊んでいいよ | 前件完成→后件实现 |
| 确定条件（已成立的前件） | 春になっ**たら**、花見をしよう | 春天到来后打算 |
| 偶然发现 | 帰っ**たら**、誰もいなかった | 前件完成时意外发现 |
| 假定建议 | もし私だっ**たら**、断る | 反事实假设 |

**偶然发现**是たら的独特用法（と也可以，但と更强调意外性）：

> 冷蔵庫を開け**たら**、何もなかった。（打开冰箱，结果什么都没有。）  
> 冷蔵庫を開ける**と**、何もなかった。（打开冰箱，发现什么都没有。——更强调意外）

## 五、なら：话题条件

**接续**：名詞/普通形 + なら（なのなら/のなら也可）

**本质**：以对方的发言或话题中的信息为前提，给出建议・意见・判断。

> A:「明日、京都に行くんですが……」  
> B:「京都に行く**なら**、伏見稲荷に寄ってみてください。」  
> （你要去京都的话，顺便去伏见稻荷看看吧。）

**特殊性**：なら 的后件**可以在前件之前发生**（建议在前件实现之前提出）。

> 京都に行く**なら**、新幹線を予約しておいてください。（去之前先预约）  
> ×京都に行ったなら、新幹線を予約してください。（不自然：已经去了再预约没意义）

## 六、四形式的使用区分决策树

```
条件を表したい
  └─ 自然法則・操作結果・習慣？
       └─ 是 → と
  └─ 假定条件（未実現）？
       └─ 後件に意志・命令あり？
            └─ 是 → たら（最通用）
            └─ 否 → ば/たら 均可（ば偏书面，たら偏口语）
  └─ 确定条件（時間順序あり）？ → たら
  └─ 接住对方信息给出建议？ → なら
  └─ 「越〜越〜」（程度强调）？ → 〜ば〜ほど
```

## 七、中文母语者易错点

| 错误类型 | 错误例 | 正确形式 | 原因 |
|---------|-------|---------|------|
| と后接意志形 | ×明日になると、行こう | ○明日になったら、行こう | と不接意志 |
| ば后接命令 | ×早く来れば、急いでください | ○早く来たら、急いでください | たら限制最少 |
| なら误作时间序列 | ×食べたなら、帰った | ○食べてから、帰った | なら是话题条件 |
| 一律用たら代替全部 | たら虽通用，但自然法则/越〜越有专用形式 | と / ば〜ほど | たら过度使用 |
