# 复合动词体系

> 复合动词由「动词连用形(れんようけい) + 动词」构成，后项动词为前项动作附加体态・方向・程度等意义。理解后项动词的意义体系是扩展词汇量的关键。

## 一、复合动词的结构

### 1.1 接续规则

前项动词的**连用形**（ます形去掉ます）+ 后项动词

```
食(た)べ（食べる的连用形）+ 始(はじ)める → 食べ始める（开始吃）
読(よ)み（読む的连用形）  + 終(お)わる → 読み終わる（读完）
飛(と)び（飛ぶ的连用形）  + 込(こ)む   → 飛び込む（跳进去）
```

**为什么用连用形而不是其他形？** 连用形的本质功能就是"连接后续成分"——接 `ます` 变礼貌体，接名词变复合名词（読(よ)み物(もの)），接动词变复合动词。它是动词的"连接口"。

### 1.2 与て形接续的区别

| 形式 | 结合程度 | 语义 | 例 |
| --- | --- | --- | --- |
| 复合动词（连用形+V） | 融合成一个单词 | 一个语义单位 | 読(よ)み終(お)わる（读完） |
| て形接续（テ+V） | 两个独立动词 | 顺序/手段关系 | 読(よ)んで終(お)わる（读了之后结束） |

**为什么意思不同？** 复合动词是词汇层面的结合——两个动词融合成一个新词，不可拆；て形是句法层面的连接——两个动作保持独立，"先做 A 再做 B"。`読み終わる` 是"读这个动作完结了"，`読んで終わる` 是"读完之后（某件事）结束了"。

## 二、体态系后项动词

标记动作在时间轴上的阶段：开始、持续、完成、中断。

| 后项动词 | 含义 | 例 |
| --- | --- | --- |
| 〜始(はじ)める | 开始（有意识/可预期） | 話(はな)し始める（开始说话）、雨(あめ)が降(ふ)り始める（开始下雨） |
| 〜出(だ)す | 突然开始（无预兆） | 泣(な)き出す（突然哭起来）、走(はし)り出す（突然跑起来） |
| 〜続(つづ)ける | 不间断地持续 | 歩(ある)き続ける（一直走）、働(はたら)き続ける（一直工作） |
| 〜終(お)わる | 动作完结 | 読(よ)み終わる（读完）、食(た)べ終わる（吃完） |
| 〜かける | 刚开始/做到一半（未完成） | 食(た)べかける（吃了一半）、言(い)いかける（话到嘴边） |
| 〜かかる | 接近某状态（被动倾向） | 死(し)にかかる（快要死了）、倒(たお)れかかる（快要倒了） |

### 2.1 始(はじ)める vs 出(だ)す

两者都表示"开始"，区别在于**是否可预期**。

> 雨(あめ)が降(ふ)り**始(はじ)めた**。（雨开始下了。）— 天阴了一阵，预期中开始下  
> 急(きゅう)に雨(あめ)が降(ふ)り**出(だ)した**。（突然下起雨来。）— 毫无预兆，突然发生

**为什么 `出す` 表示"突然"？** `出す` 的原义是"从内向外冒出"，引申为"（某状态）从无到有突然冒出来"。而 `始める` 的原义是"着手开始"，隐含有准备、有过程。

**判断法**：能加 `急に`（突然）→ 用 `出す`；有渐进过程或意志性 → 用 `始める`。

### 2.2 かける vs かかる

| | かける | かかる |
| --- | --- | --- |
| 含义 | 主动做到一半/刚着手 | 被动地接近某状态 |
| 例 | 食(た)べかけのパン（吃了一半的面包） | 死(し)にかかる（快要死了） |
| 语感 | "做了但没做完" | "差点就……了" |

**为什么 `かける` 是"未完成"？** `かける` 原义是"挂上去"——动作挂在那里没落地，引申为"中途状态"。`かかる` 是其自动词，"被挂上"→ 被动地陷入某状态的边缘。

## 三、方向・完遂系后项动词

标记动作的空间方向或完成程度。

| 后项动词 | 含义 | 例 |
| --- | --- | --- |
| 〜上(あ)げる | 完成（他动词感，自己做完） | 書(か)き上げる（写完）、作(つく)り上げる（做出来）、仕上(しあ)げる（完成） |
| 〜上(あ)がる | 完成（自动词感，自然完成）/向上 | 出来上(できあ)がる（做好了）、盛(も)り上がる（气氛高涨） |
| 〜込(こ)む | 向内/深入/彻底 | 飛(と)び込む（跳进去）、持(も)ち込む（带进去）、落(お)ち込む（消沉） |
| 〜切(き)る | 彻底完成/到极限 | 使(つか)い切る（用光）、疲(つか)れ切る（精疲力竭）、売(う)り切る（卖光） |
| 〜抜(ぬ)く | 坚持到底/贯穿 | 走(はし)り抜く（跑完全程）、生(い)き抜く（顽强活下去）、やり抜く（做到底） |
| 〜通(とお)す | 从头到尾不中断 | 読(よ)み通す（一口气读完）、やり通す（贯彻到底） |
| 〜出(だ)す | 向外/取出 | 取(と)り出す（取出来）、持(も)ち出す（带出去）、引(ひ)き出す（引出/抽屉） |

### 3.1 上(あ)げる vs 上(あ)がる

他动词/自动词的对应，和其他自他动词对一样：

> 論文(ろんぶん)を書(か)き**上(あ)げた**。（把论文写完了。）— 自己主动完成，用他动词  
> ケーキが焼(や)き**上(あ)がった**。（蛋糕烤好了。）— 自然完成，用自动词

### 3.2 切(き)る vs 抜(ぬ)く vs 通(とお)す

三者都表示"做到底"，侧重不同：

| 后项 | 侧重 | 例 |
| --- | --- | --- |
| 切(き)る | 全量消耗/到极限 | 使(つか)い切る（用光了，一点不剩） |
| 抜(ぬ)く | 克服困难坚持到终点 | やり抜く（再难也做到底） |
| 通(とお)す | 从头到尾不中断 | 読(よ)み通す（一口气读完，没停下来） |

**为什么意思不同？** 看原义——`切る` = 切断，全部割完 → 全量耗尽；`抜く` = 拔出，贯穿障碍 → 克服困难到底；`通す` = 穿过，从一端到另一端 → 中间不断。

### 3.3 込(こ)む 的三层含义

`込む` 原义"塞进去"，作为后项动词引申出三层：

| 层次 | 含义 | 例 |
| --- | --- | --- |
| 物理方向 | 向内移动 | 飛(と)び込む（跳进去）、持(も)ち込む（带进去） |
| 程度深入 | 深入/彻底 | 考(かんが)え込む（陷入沉思）、話(はな)し込む（深聊起来） |
| 状态陷入 | 陷入某种状态 | 落(お)ち込む（消沉）、黙(だま)り込む（沉默不语） |

## 四、程度・评价系后项动词

对动作的结果做评价：过度、相互、重做、失败、遗忘。

| 后项动词 | 含义 | 例 |
| --- | --- | --- |
| 〜過(す)ぎる | 过度（N4） | 食(た)べ過ぎる（吃太多）、飲(の)み過ぎる（喝太多）、考(かんが)え過ぎる（想太多） |
| 〜合(あ)う | 互相 | 話(はな)し合う（商量/互相讨论）、助(たす)け合う（互相帮助） |
| 〜直(なお)す | 重新做/改正 | 書(か)き直す（重写）、やり直す（重做）、作(つく)り直す（重做） |
| 〜損(そこ)なう | 没做成/失误（N2） | 言(い)い損なう（想说没说成）、見(み)損なう（看走眼） |
| 〜忘(わす)れる | 忘了做 | 言(い)い忘れる（忘了说）、持(も)ち忘れる（忘了带） |

### 4.1 過(す)ぎる 的接续

`過ぎる` 不只接动词，还能接形容词，接续方式不同：

| 接续对象 | 规则 | 例 |
| --- | --- | --- |
| 动词 | 连用形 + 過ぎる | 食(た)べ過ぎる（吃太多） |
| イ形容词 | 去 い + 過ぎる | 高(たか)過ぎる（太贵）、大(おお)き過ぎる（太大） |
| ナ形容词 | 词干 + 過ぎる | 静(しず)か過ぎる（太安静）、便利(べんり)過ぎる（太方便） |

### 4.2 損(そこ)なう vs 忘(わす)れる

| 后项 | 含义 | 例 |
| --- | --- | --- |
| 損なう | 想做但没做成（失误/错失） | 言(い)い損なう（话到嘴边没说出来） |
| 忘れる | 该做但忘了做（遗忘） | 言(い)い忘れる（有件事忘了说） |

**区别**：`損なう` 是有意识地想做但失败了；`忘れる` 是根本没想起来要做。

## 五、词汇性 vs 句法性

| 类型 | 特征 | 后项动词意义 | 例 |
| --- | --- | --- | --- |
| 句法性 | 意义透明，可推导 | 保持独立意义 | 食(た)べ始(はじ)める（始める = 开始，清晰） |
| 词汇性 | 意义不透明，需单独记 | 与前项融合为新义 | 飛(と)び込(こ)む（≠ 飛ぶ+込む 的字面叠加） |

**为什么要区分？** 因为学习策略完全不同：

- **句法性**（体态系为主：始める/続ける/終わる/過ぎる 等）→ 掌握后项动词的含义后，遇到新的前项动词也能推导出意思
- **词汇性**（飛び込む/取り組む/打ち明ける 等）→ 前后融合产生新义，必须作为一个词逐个记忆

**判断法**：把前后两个动词的意思直接相加，如果等于复合动词的意思 → 句法性；如果不等于 → 词汇性。

## 六、N2 高频复合动词速查

| 复合动词 | 含义 | 例句 |
| --- | --- | --- |
| 取(と)り組(く)む | 致力于/积极处理 | 問題(もんだい)に取り組む。（致力于解决问题。） |
| 受(う)け入(い)れる | 接受/接纳 | 意見(いけん)を受け入れる。（接受意见。） |
| 打(う)ち明(あ)ける | 坦白/袒露 | 秘密(ひみつ)を打ち明ける。（坦白秘密。） |
| 思(おも)い込(こ)む | 深信不疑（常为误信） | 彼(かれ)が正(ただ)しいと思い込んでいた。（一直深信他是对的。） |
| 落(お)ち着(つ)く | 冷静/稳定 | 落ち着いて考(かんが)えよう。（冷静下来想想吧。） |
| 呼(よ)びかける | 呼吁/号召 | 節電(せつでん)を呼びかける。（呼吁节电。） |
| 引(ひ)き受(う)ける | 承担/接手 | 仕事(しごと)を引き受ける。（接手工作。） |
| 乗(の)り越(こ)える | 克服/跨越 | 困難(こんなん)を乗り越える。（克服困难。） |
| 立(た)ち上(あ)がる | 站起来/创立 | 新(あたら)しい会社(かいしゃ)が立ち上がる。（新公司成立。） |
| 追(お)い付(つ)く | 赶上 | 先頭(せんとう)グループに追い付く。（赶上领先的队伍。） |
