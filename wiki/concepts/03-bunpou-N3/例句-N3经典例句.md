# N3 经典例句集

## 一、形式名词

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| Vることにする | 毎日(まいにち)日本語(にほんご)を聞(き)くことにした。 | 我决定每天听日语。 |
| Vることになる | 来月(らいげつ)大阪(おおさか)へ転勤(てんきん)することになった。 | 下个月被安排调去大阪。 |
| Vることだ | 試験(しけん)に受(う)かりたいなら、毎日(まいにち)復習(ふくしゅう)することだ。 | 想考过的话，就要每天复习。 |
| Vるものだ | 約束(やくそく)は守(まも)るものだ。 | 约定本来就该遵守。 |
| Vたものだ | 子供(こども)のころ、よくこの公園(こうえん)で遊(あそ)んだものだ。 | 小时候常在这个公园玩。 |
| ものだから | 道(みち)が込(こ)んでいたものだから、遅(おく)れました。 | 因为路上堵，所以迟到了。 |
| わけだ | 10年(じゅうねん)も練習(れんしゅう)した。上手(じょうず)なわけだ。 | 练了十年，怪不得这么厉害。 |
| わけではない | 嫌(きら)いなわけではないが、毎日(まいにち)は食(た)べたくない。 | 不是讨厌，但不想每天吃。 |
| わけがない | 彼(かれ)がそんなことを言(い)うわけがない。 | 他不可能说那种话。 |
| わけにはいかない | ここで諦(あきら)めるわけにはいかない。 | 不能在这里放弃。 |
| はずだ | 田中(たなか)さんはもう着(つ)いたはずだ。 | 田中应该已经到了。 |
| はずがない | そんな簡単(かんたん)な問題(もんだい)を間違(まちが)えるはずがない。 | 不可能把那么简单的题做错。 |
| Vるところだ | これから出(で)かけるところだ。 | 正要出门。 |
| Vているところだ | 今(いま)、資料(しりょう)を確認(かくにん)しているところだ。 | 现在正在确认资料。 |
| Vたところだ | ちょうど会議(かいぎ)が終(お)わったところだ。 | 会议刚刚结束。 |
| うちに | 忘(わす)れないうちにメモしておく。 | 趁还没忘先记下来。 |
| まま | 電気(でんき)をつけたまま寝(ね)てしまった。 | 开着灯就睡着了。 |

## 二、限定强调

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| だけ | 水(みず)だけ飲(の)んだ。 | 只喝了水。 |
| しか + 否定 | 水(みず)しか飲(の)まなかった。 | 只喝了水，没喝别的。 |
| きり | 一度(いちど)会(あ)ったきり、連絡(れんらく)がない。 | 只见过一次之后就没联系。 |
| ばかり | 弟(おとうと)はゲームばかりしている。 | 弟弟老是在打游戏。 |
| たばかり | 日本(にほん)に来(き)たばかりで、まだ慣(な)れていない。 | 刚来日本，还不习惯。 |
| ばかりに | 急(いそ)いだばかりに、大事(だいじ)な書類(しょるい)を忘(わす)れた。 | 就因为太着急，忘了重要文件。 |
| こそ | 今年(ことし)こそ合格(ごうかく)したい。 | 今年一定要考过。 |
| さえ | 自分(じぶん)の名前(なまえ)さえ書(か)けない。 | 连自己的名字都不会写。 |
| さえ〜ば | 時間(じかん)さえあれば、もっと練習(れんしゅう)できる。 | 只要有时间，就能多练习。 |
| まで | 子供(こども)までそのニュースを知(し)っている。 | 连小孩都知道那条新闻。 |
| なんか | 私(わたし)なんかには無理(むり)です。 | 像我这样的人做不到。 |
| なんて | 一人(ひとり)で海外(かいがい)へ行(い)くなんて、すごい。 | 一个人去国外，真厉害。 |

## 三、接续与条件

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| ために/ため | 大雨(おおあめ)のため、試合(しあい)は中止(ちゅうし)になった。 | 由于大雨，比赛取消了。 |
| ので | 熱(ねつ)があるので、今日は休(やす)みます。 | 因为发烧，今天请假。 |
| のに | たくさん勉強(べんきょう)したのに、合格(ごうかく)できなかった。 | 明明学了很多，却没考过。 |
| ても | 雨(あめ)が降(ふ)っても、試合(しあい)は行(おこな)われる。 | 即使下雨，比赛也会举行。 |
| 上に | この店(みせ)は安(やす)い上(うえ)に、おいしい。 | 这家店便宜而且好吃。 |
| と | このボタンを押(お)すと、ドアが開(ひら)く。 | 一按这个按钮，门就会开。 |
| ば | 時間(じかん)があれば、手伝(てつだ)います。 | 如果有时间，我会帮忙。 |
| ば〜ほど | 読(よ)めば読(よ)むほど、面白(おもしろ)くなる。 | 越读越有意思。 |
| たら | 駅(えき)に着(つ)いたら、電話(でんわ)してください。 | 到车站后请打电话。 |
| なら | 日本語(にほんご)を勉強(べんきょう)するなら、この本(ほん)がいい。 | 如果学日语，这本书不错。 |

## 四、体貌与复合动词

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| ている | 窓(まど)が開(あ)いている。 | 窗户开着。 |
| てある | 机(つくえ)の上(うえ)に資料(しりょう)が置(お)いてある。 | 资料被事先放在桌上。 |
| ておく | 会議(かいぎ)の前(まえ)に資料(しりょう)を読(よ)んでおく。 | 会前先把资料读好。 |
| てしまう | 財布(さいふ)を忘(わす)れてしまった。 | 把钱包忘了。 |
| 始める | 雨(あめ)が降(ふ)り始(はじ)めた。 | 开始下雨了。 |
| 出す | 子供(こども)が急(きゅう)に泣(な)き出(だ)した。 | 孩子突然哭了起来。 |
| 続ける | 彼(かれ)は三時間(さんじかん)も話(はな)し続(つづ)けた。 | 他连续说了三个小时。 |
| 切る | マラソンを最後(さいご)まで走(はし)り切(き)った。 | 马拉松坚持跑完了。 |
| 込む | 電車(でんしゃ)が込(こ)んでいる。 | 电车很拥挤。 |
| 過ぎる | 食(た)べ過(す)ぎて、お腹(なか)が痛(いた)い。 | 吃太多，肚子疼。 |

## 五、引用传闻与样态

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| 伝聞そうだ | 明日(あした)は雨(あめ)だそうだ。 | 听说明天下雨。 |
| 様態そうだ | 雨(あめ)が降(ふ)りそうだ。 | 看起来快下雨了。 |
| ようだ | 彼(かれ)は疲(つか)れているようだ。 | 他好像很累。 |
| みたいだ | この店(みせ)は人気(にんき)があるみたいだ。 | 这家店好像很有人气。 |
| らしい | 田中(たなか)さんは来(こ)ないらしい。 | 田中好像不来。 |
| 名词 + らしい | 春(はる)らしい天気(てんき)だ。 | 是很有春天气息的天气。 |
| という | これは何(なん)という花(はな)ですか。 | 这叫什么花？ |
| とのこと | 先生(せんせい)は明日(あした)休(やす)むとのことです。 | 据说老师明天休息。 |

## 六、语态与比较

| 文型 | 例句 | 中文 |
| --- | --- | --- |
| 被动 | 先生(せんせい)に褒(ほ)められた。 | 被老师表扬了。 |
| 迷惑被动 | 雨(あめ)に降(ふ)られて、服(ふく)が濡(ぬ)れた。 | 被雨淋了，衣服湿了。 |
| 使役 | 先生(せんせい)は学生(がくせい)に作文(さくぶん)を書(か)かせた。 | 老师让学生写作文。 |
| 使役受身 | 子供(こども)のころ、野菜(やさい)を食(た)べさせられた。 | 小时候被迫吃蔬菜。 |
| より | 電車(でんしゃ)よりバスのほうが速(はや)い。 | 比起电车，巴士更快。 |
| ほど〜ない | 今年(ことし)の夏(なつ)は去年(きょねん)ほど暑(あつ)くない。 | 今年夏天没有去年热。 |
| ほど | 泣(な)きたいほど嬉(うれ)しい。 | 高兴到想哭。 |
| ように | 鳥(とり)のように空(そら)を飛(と)びたい。 | 想像鸟一样飞。 |
