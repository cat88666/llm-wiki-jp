## 一、伴随・非伴随

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜つつ（N2） | 動詞連用形 + つつ | 一边〜一边〜（两个动作同时进行，或心理矛盾）| 音楽を聴き**つつ**、勉強する |
| 〜つつも（N2） | 動詞連用形 + つつも | 明知〜却〜（内心矛盾转折）| 悪いと知り**つつも**止められない |
| 〜ながら（も）（N4） | 動詞連用形 + ながら | 一边〜一边（可用于转折）| 泣き**ながら**笑う |
| 〜一方（で）（N3） | 普通形 + 一方（で） | 另一方面（对比・并行）| 便利な**一方で**、問題も多い |
| 〜反面（N3） | 普通形 + 反面 | 反面〜（正反两面）| 自由な**反面**、責任も重い |

**つつ vs ながら**：

| 对比 | ながら | つつ |
|------|--------|------|
| 文体 | 口语〜中性 | 书面・硬 |
| 动作性质 | 两个持续动作 | 持续+心理矛盾 |
| 例 | 歩き**ながら**電話する | 反省し**つつ**も繰り返す |

## 二、限度・程度

| 文型 | 含义 | 例 |
|------|------|-----|
| 〜だけあって（N3） | 不愧是〜（相称的）| 日本人**だけあって**、発音がきれいだ |
| 〜だけに（N3） | 正因为〜（程度相称+强调）| 有名人**だけに**、注目を集める |
| 〜ばかりに（N3） | 就因为〜（导致不好结果）| 欲張った**ばかりに**、失敗した |
| 〜きり/っきり（N3） | 只〜（限定+否定感）| 二人**きり**で話す / 出て行った**きり**帰らない |

**ばかりに vs だけに**：

> 試験があった**だけに**、合格して嬉しい。（正因为有考试，考上了更开心——相称）  
> 緊張した**ばかりに**、失敗した。（就因为紧张，搞砸了——负面结果）

## 三、倾向・习惯

| 文型 | 含义 | 接续 | 例 |
|------|------|------|-----|
| 〜がちだ（N3） | 容易〜（负面倾向）| 動詞連用形/名詞 + がち | 遅刻**がち**な人 |
| 〜っぽい（N3） | 有点儿〜（外观/倾向/轻微）| 名詞/動詞連用形 + っぽい | 子供**っぽい**、忘れ**っぽい** |
| 〜気味（きみ）（N3） | 稍微有点儿〜（程度轻微）| 動詞連用形/名詞 + 気味 | 風邪**気味**、疲れ**気味** |
| 〜向きだ（N3） | 适合〜（客观适合性）| 名詞 + 向きだ | 初心者**向き**の教材 |
| 〜向けだ（N3） | 面向〜（主观设计意图）| 名詞 + 向けだ | 子供**向け**の番組 |

**向き vs 向け**：

| 对比 | 〜向き | 〜向け |
|------|-------|-------|
| 基准 | 客观上适合 | 主观上为〜而设计 |
| 例 | 初心者**向き**（客观来说适合初学者）| 子供**向け**（特意为孩子设计的）|

## 四、无关・不变

| 文型 | 含义 | 文体 | 例 |
|------|------|------|-----|
| 〜にかかわらず（N2） | 不管〜（无论如何）| 书面 | 天気**にかかわらず**実施する |
| 〜を問わず（N2） | 不问〜（不区分）| 书面 | 年齢**を問わず**参加できる |
| 〜によらず（N2） | 不管哪种〜（不依赖）| 书面 | 方法**によらず**、結果が大事 |
| 〜にかかわりなく（N2） | 同にかかわらず | 书面硬 | — |

**三者都表达"不管〜"，但强调角度不同**：

| 文型 | 强调角度 | 例 |
|------|---------|-----|
| にかかわらず | 无论怎样都一样 | 結果**にかかわらず**努力が大事 |
| を問わず | 不区分类别 | 性別**を問わず**採用する |
| によらず | 不依赖/不按照 | 経験**によらず**判断する |

## 五、起点・基准

| 文型 | 含义 | 例 |
|------|------|-----|
| 〜をきっかけに（N3） | 以〜为契机（口语）| 転職**をきっかけに**日本語を勉強した |
| 〜を契機（けいき）に（N3） | 以〜为契机（书面）| 事件**を契機に**法律が改正された |
| 〜を皮切り（かわきり）に（N2） | 以〜为开端（连续性起点）| 東京**を皮切りに**全国ツアーが始まった |
| 〜をはじめ（N3） | 以〜为首（代表性列举）| 田中先生**をはじめ**、多くの方が参加した |
| 〜を中心に（N3） | 以〜为中心 | 東京**を中心に**イベントが開催される |

## 六、强调・极端

| 文型 | 含义 | 文体 | 例 |
|------|------|------|-----|
| 〜さえ〜ば（N3） | 只要〜就〜（最小条件）| 中性 | お金**さえあれば**、何でもできる |
| 〜すら（N2） | 连〜都〜（极端例示，同さえ）| 书面/硬 | 自分の名前**すら**書けない |
| 〜だに（N2） | 连〜都〜（极端，非常硬）| 文語的 | 想像する**だに**恐ろしい |
| 〜ともなると（N2） | 一旦到达〜（级别/数量达到某点）| 中性 | 社長**ともなると**、責任も重い |

**さえ vs すら vs だに**：文体硬度 だに > すら > さえ（最口语）

## 七、动态变化・比例

「随着〜而〜」的动态变化表现——N2 问9 文章语法高频考点。

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜につれて（N3） | 動詞辞書形/名詞 + につれて | 随着〜（同步变化）| 年をとる**につれて**知恵がつく |
| 〜にしたがって（N2） | 動詞辞書形/名詞 + にしたがって | 随着〜・按照〜 | 時代が変わる**にしたがって**常識も変わる |
| 〜に伴って（N2） | 名詞/動詞辞書形 + に伴って | 伴随〜（两者同步）| 経済発展**に伴って**環境問題が深刻化した |
| 〜に応じて（N2） | 名詞 + に応じて | 根据〜（按照比例调整）| 能力**に応じて**給料が決まる |

**四者の違い**：

| 対比 | 強調点 | 例 |
|------|--------|-----|
| につれて | 时间経過に伴う変化 | 暖かくなる**につれて**花が咲く |
| にしたがって | 基准に従う変化（規則性）| マニュアル**にしたがって**作業する |
| に伴って | 两事件の連動（書面）| 開発**に伴って**自然が失われた |
| に応じて | 比例的配分・対応 | 需要**に応じて**生産を増やす |

## 八、逆接・让步

| 文型 | 接续 | 含义 | 文体 |
|------|------|------|------|
| 〜ものの（N2） | 普通形 + ものの | 虽然〜但是〜（前提成立但有問題）| 書面 |
| 〜ものを（N2） | 普通形 + ものを | 明明应该〜却〜（遗憾・埋怨）| 書面 |
| 〜とはいえ（N2） | 普通形 + とはいえ | 虽说〜但〜（承认前提再转折）| 書面 |
| 〜にしては（N3） | 普通形 + にしては | 就〜来说〜（预期不符）| 中性 |
| 〜ながら（も）（N3） | 動詞連用形 + ながらも | 一面〜一面〜（内心矛盾）| 中性 |

**ものの vs とはいえ vs にしては**：

| 对比 | ものの | とはいえ | にしては |
|------|--------|---------|---------|
| 前件 | 事实成立 | 一定程度承认 | 基准/身份 |
| 后件 | 有问题/不足 | 但有限制 | 超出/低于预期 |
| 例 | 合格した**ものの**自信がない | 春**とはいえ**まだ寒い | 子ども**にしては**上手だ |

> 試験に受かった**ものの**、体調が悪くて素直に喜べない。  
> 暖かくなった**とはいえ**、まだ油断できない。  
> 1年しか練習していない**にしては**、かなりうまい。

## 九、条件・前提

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜上で（N3） | 動詞た形/名詞の + 上で | 在〜基础上・之后 | 相談した**上で**決める |
| 〜上に（N2） | 普通形 + 上に | 不仅〜还〜（累加）| 遅刻した**上に**謝らなかった |
| 〜上は（N2） | 動詞た形/普通形 + 上は | 既然〜就〜（已成事实的义务）| こうなった**上は**やるしかない |
| 〜てからでないと（N2） | 動詞て形 + からでないと | 不先〜就不能〜 | 確認し**てからでないと**進められない |
| 〜次第だ（N2） | 動詞連用形 + 次第だ | 〜就马上〜（事情原委/决定于）| 詳細は後ほど連絡する**次第です** |
| 〜次第で（N2） | 名詞 + 次第で | 根据〜而定 | 努力**次第で**結果が変わる |

**上で vs 上に vs 上は 三者の用法**：

```
上で → 前置条件（〜した後で = 前提を踏まえた上で）
上に → 累加（さらに・しかも）
上は → 既然（已成事実への対応義務）
```

## 十、时机・転折

| 文型 | 含义 | 例 |
|------|------|-----|
| 〜ところが（N3） | 意外な転換（接续詞）| 行くつもりだった。**ところが**、電車が止まった |
| 〜ところに（N2） | 正在〜的时候（追加事件来临）| 食べている**ところに**電話が来た |
| 〜ところへ（N2） | 正在〜的时候（追加事件进入）| 休んでいる**ところへ**客が来た |
| 〜ところを（N2） | 正在〜的时候被〜（被目击/介入）| 寝ている**ところを**起こされた |
| 〜ところだった（N3） | 差点儿就〜了 | もう少しで忘れる**ところだった** |

**ところに vs ところへ vs ところを**：

| 対比 | 後件の性質 | 例 |
|------|----------|-----|
| ところに | 追加事件が「起きる」 | 困っている**ところに**助けが来た |
| ところへ | 人・物が「来る・入る」 | 疲れている**ところへ**上司が来た |
| ところを | 当事者が「目撃/介入される」 | サボっている**ところを**見られた |

## 十一、理由・原因

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜ことから（N2） | 普通形 + ことから | 以〜为由・从〜来判断 | 発音がきれいな**ことから**日本育ちだとわかる |
| 〜ものだから（N3） | 普通形 + ものだから | 因为〜所以〜（辩解）| 急いでいた**ものだから**忘れてしまった |
| 〜からといって（N3） | 普通形 + からといって | 就算〜也不能〜 | 忙しい**からといって**約束を破ってはいけない |
| 〜だけに（N3） | 普通形 + だけに | 正因为〜（相称性强调）| 期待していた**だけに**失望が大きい |
| 〜以上（は）（N2） | 動詞た形/普通形 + 以上は | 既然〜就必须〜 | 約束した**以上は**守るべきだ |
| 〜からには（N3） | 動詞た形/普通形 + からには | 既然〜就应该〜 | 始めた**からには**最後までやる |

**以上は vs からには**：
両者ほぼ同義だが、「以上は」の方が若干書面的・義務感が強い。

## 十二、可能・限制

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜かねる（N2） | 動詞連用形 + かねる | 难以做到〜（客气拒绝）| お答えし**かねます** |
| 〜かねない（N2） | 動詞連用形 + かねない | 有可能（不好的事）| 過労で倒れ**かねない** |
| 〜に限り（N2） | 名詞 + に限り | 仅限〜（限定条件）| 本日**に限り**半額です |
| 〜に限らず（N2） | 名詞 + に限らず | 不限于〜 | 日本**に限らず**世界中で問題だ |
| 〜っぱなし（N3） | 動詞連用形 + っぱなし | 一直〜（放置状态）| ドアを開け**っぱなし**にするな |
| 〜ないことには（N2） | ない形 + ことには | 不〜就不能〜 | やってみない**ことには**わからない |

**かねる vs かねない**（重要な対比）：

| 対比 | 意味 | 例 |
|------|------|-----|
| 〜かねる | 难以做（委婉拒绝）| 責任は負い**かねます** |
| 〜かねない | 可能会做（危険な可能性）| このままでは失敗し**かねない** |

> **反直觉**：二者看起来相反的否定关系，但含义截然不同。かねる＝できない；かねない＝するかもしれない（危険）。

## 十三、关系・基准

| 文型 | 接续 | 含义 | 例 |
|------|------|------|-----|
| 〜をもとに（N2） | 名詞 + をもとに | 以〜为基础・以〜为材料 | 実体験**をもとに**小説を書いた |
| 〜をめぐって（N2） | 名詞 + をめぐって | 围绕〜（争论・问题）| 遺産**をめぐって**争いが起きた |
| 〜に反して（N2） | 名詞 + に反して | 违背〜・与〜相反 | 予想**に反して**失敗した |
| 〜をとおして（N2） | 名詞 + をとおして | 通过〜・通过整个过程 | 努力**をとおして**成長した |
| 〜に対して（N3） | 名詞 + に対して | 对于〜・针对〜 | 問題**に対して**解決策を示す |
| 〜にとって（N3） | 名詞 + にとって | 对〜来说 | 私**にとって**大切な人 |

**をもとに vs にもとづいて**：

| 対比 | 含义 | 例 |
|------|------|-----|
| をもとに | 以〜为素材・原型（創作・応用）| 実話**をもとに**映画を作った |
| にもとづいて | 以〜为根据・依据（規則・事実）| 法律**にもとづいて**処罰した |

| 動詞た形 | 〜たものだ、〜たところだ、〜たばかりに、〜上は、〜上で |
| 動詞連用形 | 〜つつ、〜っぱなし、〜かねる、〜かねない、〜次第だ |
| 動詞ナイ形語幹 | 〜ないことには、〜ないことはない |
| 普通形 | 〜ものの、〜とはいえ、〜ことから、〜以上は、〜ものだから |
| 名詞 + を | 〜をきっかけに、〜をめぐって、〜をもとに、〜をとおして |
| 名詞 + に | 〜に伴って、〜に応じて、〜に反して、〜に限り、〜にとって |
| 名詞 + の/な | 〜にもかかわらず、〜次第で、〜に限らず |
