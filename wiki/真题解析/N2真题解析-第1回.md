## 一、文字・词汇（20分）

### 問1 汉字读音（5题 × 1分 = 5分）

1. 会議の資料を**配布**した。
   1. はいふ 2. はいぶ 3. はぶ 4. くばふ

> **答案：1** はいふ
> **考点**：汉字读音 — 配布(はいふ)
> **解析**：`配布` 固定读作 `はいふ`。`配` 是 `はい`，`布` 在这个词中读 `ふ`。如「会議(かいぎ)の資料(しりょう)を配布(はいふ)した」（分发了会议资料）。2 的 `はいぶ` 是浊音误导，3/4 不是这个词的读法。
> **记忆点**：配布(はいふ) = 分发；搭配：資料を配布する

2. この薬は水に**溶ける**。
   1. とける 2. ほどける 3. うける 4. にげる

> **答案：1** とける
> **考点**：汉字读音 — 溶(と)ける
> **解析**：`溶ける` 读 `とける`，表示固体在液体中融化/溶解，如「薬(くすり)は水(みず)に溶(と)ける」（药会溶于水）。`ほどける` 是"解开"，`うける` 是"接受"，`にげる` 是"逃跑"。
> **记忆点**：溶(と)ける = 溶解；搭配：水に溶ける；区别：解(ほど)ける(解开)

3. 彼の説明はとても**具体的**だった。
   1. ぐたいでき 2. ぐたいてき 3. こたいでき 4. こたいてき

> **答案：2** ぐたいてき
> **考点**：汉字读音 — 具体的(ぐたいてき)
> **解析**：`具体的` 读 `ぐたいてき`，如「彼(かれ)の説明(せつめい)はとても具体的(ぐたいてき)だった」（他的说明非常具体）。注意 `的` 在形容动词词尾中读 `てき`，不是 `でき`。
> **记忆点**：具体的(ぐたいてき)；注意：「的」读てき不读でき；反义：抽象的(ちゅうしょうてき)

4. 新しい制度を**導入**する。
   1. どうにゅう 2. とうにゅう 3. どういれ 4. みちにゅう

> **答案：1** どうにゅう
> **考点**：汉字读音 — 導入(どうにゅう)
> **解析**：`導入` 读 `どうにゅう`，常用于制度、设备、技术的"引入"，如「新(あたら)しい制度(せいど)を導入(どうにゅう)する」（引入新制度）。2 `とうにゅう` 是清浊错误（`投入` 才读 `とうにゅう`）。
> **记忆点**：導入(どうにゅう) = 引入；易混：投入(とうにゅう)；搭配：制度を導入する

5. 雨のため、試合は**延期**になった。
   1. えんき 2. えんぎ 3. のびき 4. えんご

> **答案：1** えんき
> **考点**：汉字读音 — 延期(えんき)
> **解析**：`延期` 读 `えんき`，`期` 在这里读 `き`，如「試合(しあい)は延期(えんき)になった」（比赛延期了）。`延期になる` 是固定搭配。
> **记忆点**：延期(えんき) = 延期；搭配：延期になる／延期する；注意：「期」读き不读ぎ

### 問2 表记（5题 × 1分 = 5分）

6. 商品の品質を（ほしょう）する。
   1. 保証 2. 補償 3. 保障 4. 保章

> **答案：1** 保証
> **考点**：同音异字 — 保証/補償/保障
> **解析**：`保証` 是保证质量/身份/内容，如「商品(しょうひん)の品質(ひんしつ)を保証(ほしょう)する」（保证商品质量）。`補償` 是赔偿损失，`保障` 是保障权利/安全。这里和 `品質` 搭配，选 `保証`。
> **记忆点**：保証 = 保证(质量/内容)；補償 = 赔偿(损失)；保障 = 保障(权利/安全)

7. この計画はまだ（けんとう）中だ。
   1. 検討 2. 見当 3. 健闘 4. 検当

> **答案：1** 検討
> **考点**：同音异字 — 検討/見当/健闘
> **解析**：`検討中` 表示"正在研究是否可行"，如「計画(けいかく)はまだ検討(けんとう)中(ちゅう)だ」（计划还在研究中）。`見当` 是大致方向/估计，`健闘` 是努力奋斗。
> **记忆点**：検討 = 研究探讨；見当 = 估计(見当がつく)；健闘 = 奋斗(健闘を祈る)

8. 駅前の道路は朝とても（こんざつ）する。
   1. 混雑 2. 困雑 3. 混札 4. 昆雑

> **答案：1** 混雑
> **考点**：汉字表记 — 混雑(こんざつ)
> **解析**：`混雑する` 是交通、人群拥挤，如「駅前(えきまえ)の道路(どうろ)は朝(あさ)とても混雑(こんざつ)する」（车站前的道路早上非常拥挤）。其他选项不是正确表记。
> **记忆点**：混雑(こんざつ) = 拥挤混杂；搭配：道が混雑する／混雑を避ける

9. 彼女は新しい案を（ていあん）した。
   1. 提案 2. 定案 3. 呈案 4. 題案

> **答案：1** 提案
> **考点**：汉字表记 — 提案(ていあん)
> **解析**：`提案する` 是"提出方案/建议"，如「新(あたら)しい案(あん)を提案(ていあん)した」（提出了新方案）。`案` 是方案，前面用 `提`。
> **记忆点**：提案(ていあん) = 提出方案；搭配：提案する／提案を出す

10. 工場の安全を（かくほ）する。
    1. 確保 2. 獲保 3. 格保 4. 確歩

> **答案：1** 確保
> **考点**：汉字表记 — 確保(かくほ)
> **解析**：`確保する` 表示确保资源/安全/人员，如「工場(こうじょう)の安全(あんぜん)を確保(かくほ)する」（确保工厂安全）。`獲` 多用于获取，不能写作 `獲保`。
> **记忆点**：確保(かくほ) = 确保；搭配：安全を確保する／人員を確保する

### 問3 词汇・语境（5题 × 1分 = 5分）

11. 今日は湿度が高くて、空気が（　）している。
    1. じめじめ 2. きらきら 3. すらすら 4. どきどき

> **答案：1** じめじめ
> **考点**：拟声拟态词 — じめじめ
> **解析**：`じめじめ` 描写湿气重、不爽快的感觉，如「湿度(しつど)が高(たか)くて、空気(くうき)がじめじめしている」（湿度高，空气潮湿黏腻）。`きらきら` 闪亮，`すらすら` 顺畅，`どきどき` 心跳。
> **记忆点**：じめじめ = 潮湿黏腻；类：べたべた(黏)/さらさら(干爽)/むしむし(闷热)

12. この仕事は経験がない人には少し（　）かもしれない。
    1. きつい 2. ずるい 3. まぶしい 4. くやしい

> **答案：1** きつい
> **考点**：近义词辨析 — きつい
> **解析**：`きつい` 可表示辛苦、吃力、严格，如「経験(けいけん)がない人(ひと)には少(すこ)しきついかもしれない」（对没经验的人可能有点吃力）。`ずるい` 狡猾，`まぶしい` 刺眼，`くやしい` 不甘心。
> **记忆点**：きつい = 辛苦/吃力/严格(多义)；搭配：仕事がきつい／性格がきつい

13. 予算が足りないので、計画を（　）する必要がある。
    1. 見直す 2. 見送る 3. 見下ろす 4. 見上げる

> **答案：1** 見直す
> **考点**：复合动词 — 見直す
> **解析**：`見直す` 是重新检查并修改，如「予算(よさん)が足(た)りないので、計画(けいかく)を見直(みなお)す必要(ひつよう)がある」（因预算不足，需要重新审视计划）。`見送る` 是暂缓/送别，`見下ろす` 俯视，`見上げる` 仰望。
> **记忆点**：見直す = 重新审视；見送る = 暂缓/送别；見下ろす = 俯视；見上げる = 仰望

14. 電車の遅れについて、駅員が乗客に（　）した。
    1. 案内 2. 説明 3. 紹介 4. 注文

> **答案：2** 説明
> **考点**：近义词辨析 — 説明/案内
> **解析**：晚点原因/情况要用 `説明する`，如「駅員(えきいん)が乗客(じょうきゃく)に説明(せつめい)した」（站务员向乘客说明了情况）。`案内` 是引导/通知路线，语义不如 `説明` 准确。
> **记忆点**：説明 = 说明(原因/内容)；案内 = 引导/介绍(场所/路线)；搭配：遅れを説明する

15. 彼は失敗しても、すぐに気持ちを（　）できる。
    1. 切り替える 2. 切り取る 3. 切り抜く 4. 切り離す

> **答案：1** 切り替える
> **考点**：复合动词 — 切り替える
> **解析**：`気持ちを切り替える` 是固定搭配，表示转换心情，如「失敗(しっぱい)しても、すぐに気持(きも)ちを切(き)り替(か)えることができる」（即使失败也能马上转换心情）。`切り取る` 剪下，`切り抜く` 克服/剪裁，`切り離す` 分离。
> **记忆点**：気持ちを切り替える = 转换心情(固定搭配)；切り取る = 剪下；切り抜く = 克服；切り離す = 分离

### 問4 言い換え（5题 × 1分 = 5分）

16. この機械は**扱いやすい**。
    1. 操作しやすい 2. 壊れやすい 3. 運びやすい 4. 見つけやすい

> **答案：1** 操作しやすい
> **考点**：言い換え — 扱う=操作する
> **解析**：`扱う` 是操作/处理，所以 `扱いやすい` 最接近 `操作しやすい`，如「この機械(きかい)は扱(あつか)いやすい」（这台机器容易操作）。不是容易坏、容易搬或容易找到。
> **记忆点**：扱(あつか)う = 操作/处理；扱いやすい = 操作しやすい

17. 彼の意見には**納得できない**。
    1. 受け入れられない 2. 思い出せない 3. 伝えられない 4. 比べられない

> **答案：1** 受け入れられない
> **考点**：言い換え — 納得できない
> **解析**：`納得できない` 是无法理解并接受其道理，等于 `受け入れられない`，如「彼(かれ)の意見(いけん)には納得(なっとく)できない」（我不能接受/信服他的意见）。
> **记忆点**：納得(なっとく)できない = 受け入れられない = 无法信服接受

18. その話は**事実に基づいている**。
    1. 本当のことをもとにしている 2. 未来の予定を示している 3. 人の希望を表している 4. 古い考えに反対している

> **答案：1** 本当のことをもとにしている
> **考点**：言い換え — 事実に基づく
> **解析**：`〜に基づく` 表示"以……为基础"，如「その話(はなし)は事実(じじつ)に基(もと)づいている」（那个说法基于事实）。1 的 `本当のことをもとにしている` 正好对应。
> **记忆点**：〜に基(もと)づく = 〜をもとにする = 基于；搭配：事実に基づく／法律に基づく

19. 彼女は会議で**率直に**意見を述べた。
    1. 正直に 2. 静かに 3. 急に 4. 詳しく

> **答案：1** 正直に
> **考点**：言い換え — 率直に=正直に
> **解析**：`率直に` 是坦率地、直率地，接近 `正直に`，如「会議(かいぎ)で率直(そっちょく)に意見(いけん)を述(の)べた」（在会议上坦率地表达了意见）。不是静静地、突然地、详细地。
> **记忆点**：率直(そっちょく)に = 正直に = 坦率地；搭配：率直に言う／率直な意見

20. この店は駅から**わずか**3分だ。
    1. たった 2. かなり 3. およそ 4. ちょうど

> **答案：1** たった
> **考点**：言い換え — わずか=たった
> **解析**：`わずか` 表示"仅仅、只有"，等于 `たった`，如「駅(えき)からわずか三分(さんぷん)だ」（离车站仅三分钟）。`およそ` 是大约，`ちょうど` 是正好。
> **记忆点**：わずか = たった = 仅仅；易混：およそ(大约)／ちょうど(正好)／かなり(相当)

## 二、语法（40分）

> 语法选择 Q21-30：每题 2 分（20分）/ 文の組み立て Q31-35：每题 4 分（20分）

21. この問題は一人で考える（　）、先生に相談したほうがいい。
    1. くらいなら 2. ものの 3. につれて 4. ばかりに

> **答案：1** くらいなら
> **考点**：N2语法句型 — くらいなら
> **解析**：`AくらいならBほうがいい` 表示"与其A，不如B"，如「一人(ひとり)で考(かんが)えるくらいなら、先生(せんせい)に相談(そうだん)したほうがいい」（与其一个人想，不如找老师商量）。`ものの` 是虽然，`につれて` 随着，`ばかりに` 因为不好的原因。
> **记忆点**：くらいなら = 与其A不如B；接续：V辞書形 + くらいなら；后常接〜ほうがいい

22. 予約をキャンセルする（　）、前日までに連絡してください。
    1. 際は 2. 以上 3. 反面 4. 一方で

> **答案：1** 際は
> **考点**：N2语法句型 — 際(は)
> **解析**：`際は` 是正式的"在……的时候"，如「予約(よやく)をキャンセルする際(さい)は、前日(ぜんじつ)までに連絡(れんらく)してください」（取消预约时，请在前一天之前联系）。`以上` 是既然，`反面` 是另一方面，`一方で` 是与此同时/另一方面。
> **记忆点**：際(さい)は = 〜とき(正式)；接续：V辞書形/した + 際；多用于书面/通知

23. 忙しい（　）、毎日少しでも復習している。
    1. ながらも 2. ために 3. ことから 4. によって

> **答案：1** ながらも
> **考点**：N2语法句型 — ながらも
> **解析**：`ながらも` 表示"虽然……但是……"，如「忙(いそが)しいながらも、毎日(まいにち)少(すこ)しでも復習(ふくしゅう)している」（虽然忙，但每天多少都在复习）。前接い形容词可用 `忙しいながらも`。
> **记忆点**：ながらも = 虽然…但是…；接续：い形/な形+ながらも；易混：ものの(虽然，更书面)

24. この資料は、初心者にもわかる（　）作られている。
    1. ように 2. ためで 3. うえは 4. だけに

> **答案：1** ように
> **考点**：N3语法句型 — ように
> **解析**：`ように` 表示目的或状态达成："为了能……"，如「初心者(しょしんしゃ)にもわかるように作(つく)られている」（被做成连初学者也能看懂的样子）。这里是"让初学者也能明白"。
> **记忆点**：ように = 为了能…；接续：V辞書形/ない形 + ように；区别：ために(意志动词目的)

25. 彼は遅刻した（　）、謝りもしなかった。
    1. にもかかわらず 2. にしたがって 3. に限って 4. に対して

> **答案：1** にもかかわらず
> **考点**：N2语法句型 — にもかかわらず
> **解析**：`にもかかわらず` 表示"尽管……却……"，后面常接反常结果，如「遅刻(ちこく)したにもかかわらず、謝(あやま)りもしなかった」（明明迟到了，却连道歉都没有）。
> **记忆点**：にもかかわらず = 尽管…却…；接续：V/い形/な形/N + にもかかわらず；语气：意外/不满

26. 海外で働く（　）、語学力だけでなく文化理解も必要だ。
    1. にあたって 2. につれて 3. ばかりか 4. どころか

> **答案：1** にあたって
> **考点**：N2语法句型 — にあたって
> **解析**：`にあたって` 表示做重要事情之前/之际的准备或注意，如「海外(かいがい)で働(はたら)くにあたって、語学力(ごがくりょく)だけでなく文化理解(ぶんかりかい)も必要(ひつよう)だ」（在海外工作之际，也需要文化理解）。
> **记忆点**：にあたって = 在…之际(正式)；接续：V辞書形/N + にあたって；场景：重要事情的准备

27. もう決めた（　）、最後までやるしかない。
    1. からには 2. ところで 3. わりに 4. うちに

> **答案：1** からには
> **考点**：N2语法句型 — からには
> **解析**：`からには` 表示"既然……就应该/必须……"，后面常接义务、决心，如「もう決(き)めたからには、最後(さいご)までやるしかない」（既然已经决定了，就只能做到最后）。
> **记忆点**：からには = 既然…就…；接续：V た形/辞書形 + からには；后常接：〜しかない／〜べきだ

28. この店は安い（　）、味もいい。
    1. うえに 2. ものを 3. くせに 4. かわりに

> **答案：1** うえに
> **考点**：N2语法句型 — うえに
> **解析**：`うえに` 表示追加优点或缺点："而且"，如「安(やす)いうえに、味(あじ)もいい」（不仅便宜，而且味道也好）。`くせに` 带责备，`かわりに` 是代替/作为交换。
> **记忆点**：うえに = 而且/不仅…还…；接续：い形/な形な/Vた + うえに；易混：くせに(带责备)

29. 彼の説明は専門用語が多く、理解し（　）。
    1. にくい 2. がたい 3. かねない 4. かねる

> **答案：1** にくい
> **考点**：N2语法辨析 — にくい/がたい
> **解析**：`〜にくい` 是"难以……"，多指物理/客观上的困难，如「専門用語(せんもんようご)が多(おお)く、理解(りかい)しにくい」（专业术语很多，难以理解）。`がたい` 多用于心理上难以接受，如 `信じがたい`。
> **记忆点**：にくい = 难以(客观困难)；がたい = 难以(心理上)；かねない = 有可能(不好的事)；かねる = 难以(委婉拒绝)

30. 雨が降っている（　）、試合は予定通り行われる。
    1. ものの 2. たびに 3. につき 4. とともに

> **答案：1** ものの
> **考点**：N2语法句型 — ものの
> **解析**：`ものの` 表示"虽然……但是……"，语气较书面，如「雨(あめ)が降(ふ)っているものの、試合(しあい)は予定通(よていどお)り行(おこな)われる」（虽然在下雨，比赛仍按计划举行）。
> **记忆点**：ものの = 虽然…但是…(书面)；接续：V/い形 + ものの；类：ながらも／にもかかわらず

### 問 文の組み立て（5题 × 4分 = 20分）

31. これは、今後の方針を ___ ___ ★ ___ 重要な資料だ。

> **答案：3** 参考に（★位置）
> **考点**：文の組み立て — うえで
> **解析**：正序是 `参考にするうえで`，星号位置是第 3 个词 `参考に`。`うえで` 表示"在……方面/时"。完整句为「今後(こんご)の方針(ほうしん)を参考(さんこう)にするうえで重要(じゅうよう)な資料(しりょう)だ」（这是今后制定方针时作为参考的重要资料）。
> **记忆点**：参考にする + うえで = 作为参考时；搭配：参考にする

32. 彼は ___ ___ ★ ___ 返事をくれた。

> **答案：3** すぐに（★位置）

> **记忆点**：にもかかわらず + すぐに → 强调意外性；搭配：丁寧な返事をくれた

33. 環境問題は、国だけでなく ___ ___ ★ ___ 取り組むべきだ。

> **答案：3** 真剣に（★位置）
> **考点**：文の組み立て — 一人一人が
> **解析**：正序是 `市民 / 一人一人が / 真剣に / 取り組むべきだ`，星号是 `真剣に`。完整句为「国(くに)だけでなく市民(しみん)一人一人(ひとりひとり)が真剣(しんけん)に取(と)り組(く)むべきだ」（不仅国家，市民每个人也应该认真应对）。
> **记忆点**：搭配：真剣に取り組む = 认真应对；一人一人が = 每个人

34. この薬は説明書を ___ ___ ★ ___ 飲んでください。
    1. 読んだ 2. うえで 3. よく 4. から

> **答案：2** うえで（★位置）
> **考点**：文の組み立て — た形+うえで
> **解析**：正序是 `よく / 読んだ / うえで / 飲んでください`，星号是 `うえで`。`た形+うえで` 表示"在……之后"。完整句为「説明書(せつめいしょ)をよく読(よ)んだうえで飲(の)んでください」（请仔细阅读说明书之后再服用）。
> **记忆点**：た形 + うえで = …之后再…；搭配：読んだうえで／確認したうえで

35. 新製品は発売前から ___ ___ ★ ___ 集めている。
    1. 多くの 2. 関心を 3. 消費者の 4. すでに

> **答案：3** 消費者の（★位置）
> **考点**：文の組み立て — 関心を集める

## 四、听解（60分）

44. 男の人はこれから何をしますか。
脚本：女「会議室の予約、もうしましたか。」男「すみません、まだです。資料の印刷は終わりました。」女「では、先に会議室をお願いします。」
1. 資料を印刷する 2. 会議室を予約する 3. 会議を中止する 4. 女の人に電話する

> **答案：2** 会議室を予約する
> **考点**：听解 — 下一步行动
> **解析**：女方最后说「先(さき)に会議室(かいぎしつ)をお願(ねが)いします」（请先去预约会议室），所以男方接下来要预约会议室。资料已经印完。
> **记忆点**：先に〜をお願いします → 次の行動を示す；聴解技巧：最後の指示に注目

45. 女の人はなぜ遅れますか。
脚本：男「もう駅に着いた？」女「ごめん、電車が止まっていて。別の線に乗り換えるから、十分くらい遅れる。」
1. 道に迷ったから 2. 電車が止まったから 3. 切符をなくしたから 4. 約束を忘れたから

> **答案：2** 電車が止まったから
> **考点**：听解 — 原因特定
> **解析**：迟到原因是「電車(でんしゃ)が止(と)まっていて」（电车停运了），后面的「別(べつ)の線(せん)に乗(の)り換(か)える」（换乘别的线路）是应对方法，不是根本原因。
> **记忆点**：聴解技巧：原因 vs 対策を区別 → 「〜ていて」が原因、「〜から」が対策

46. 男の人は何を選びますか。

> **记忆点**：〜にします = 决定选…；聴解技巧：消去法 → 売り切れ・排除理由を聞き取る

47. 女の人の意見はどれですか。
脚本：男「この案、すぐ始めてもいいと思う？」女「方向性はいいけど、費用の計算がまだ甘いと思う。そこを確認してからにしたほうがいい。」

> **记忆点**：〜けど → 転折後が本音；〜てからにしたほうがいい = 最好先…再…；甘い = 不充分

48. 男の人は何を忘れましたか。

> **记忆点**：まだ〜ていない = 还没做(未完了)；聴解技巧：「あ」は気づきのサイン → 忘れた事項

49. 女の人は何を頼みますか。

50. 男の人は週末どうしますか。
脚本：女「週末の山登り、行けそう？」男「天気がよければ行くつもりだったけど、風邪気味だから今回はやめておくよ。」
1. 山に行く 2. 家で休む 3. 仕事に行く 4. 友達を誘う
