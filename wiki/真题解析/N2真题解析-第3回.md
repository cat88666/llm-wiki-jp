## 一、文字・词汇（20分）

### 問1 汉字读音（5题 × 1分 = 5分）

1. 結果を**分析**する。
   1. ぶんせき 2. ふんせき 3. ぶんしゃく 4. ふんしゃく

> **答案：1** ぶんせき<br>**考点**：汉字读音 — 分析(ぶんせき)<br>**解析**：`分析` 读 `ぶんせき`。`析` 在这里读 `せき`。結果(けっか)を分析(ぶんせき)する＝分析结果。<br>**记忆点**：分析(ぶんせき) = 分析；搭配：結果を分析する/データを分析する

2. 事故の原因を**調査**する。
   1. ちょうさ 2. ちょうざ 3. じょうさ 4. しらさ

> **答案：1** ちょうさ<br>**考点**：汉字读音 — 調査(ちょうさ)<br>**解析**：`調査` 读 `ちょうさ`，表示调查事实。事故(じこ)の原因(げんいん)を調査(ちょうさ)する＝调查事故原因。<br>**记忆点**：調査(ちょうさ) = 调查；搭配：原因を調査する/実態を調査する

3. 彼は会社に大きく**貢献**した。
   1. こうけん 2. こうげん 3. くけん 4. こうこん

> **答案：1** こうけん<br>**考点**：汉字读音 — 貢献(こうけん)<br>**解析**：`貢献` 读 `こうけん`，常和 `〜に貢献する` 搭配。彼(かれ)は会社(かいしゃ)に大(おお)きく貢献(こうけん)した＝他为公司做出了很大贡献。<br>**记忆点**：貢献(こうけん) = 贡献；搭配：会社に貢献する/社会に貢献する

4. 新しい技術を**応用**する。
   1. おうよう 2. およう 3. こうよう 4. ようおう

> **答案：1** おうよう<br>**考点**：汉字读音 — 応用(おうよう)<br>**解析**：`応用` 读 `おうよう`，表示把知识/技术用到实际中。新(あたら)しい技術(ぎじゅつ)を応用(おうよう)する＝应用新技术。<br>**记忆点**：応用(おうよう) = 应用；搭配：技術を応用する；对比：応用 ↔ 基礎(きそ)

5. 予定を**変更**する。
   1. へんこう 2. へんごう 3. べんこう 4. へんかん

> **答案：1** へんこう<br>**考点**：汉字读音 — 変更(へんこう)<br>**解析**：`変更` 读 `へんこう`。予定(よてい)を変更(へんこう)する＝更改计划。`予定を変更する` 是固定搭配。<br>**记忆点**：変更(へんこう) = 变更/更改；搭配：予定を変更する/計画を変更する

### 問2 表记（5题 × 1分 = 5分）

6. 問題を（かいけつ）する。
   1. 解決 2. 解結 3. 改決 4. 回決

> **答案：1** 解決<br>**考点**：汉字表记 — 解決<br>**解析**：`解決` 是解决问题。`解結/改決/回決` 都不是正确表记。問題(もんだい)を解決(かいけつ)する＝解决问题。<br>**记忆点**：解決(かいけつ) = 解决；搭配：問題を解決する；注意「決」不写成「結」

7. 商品の（かち）を高める。
   1. 価値 2. 価地 3. 加値 4. 可知

> **答案：1** 価値<br>**考点**：汉字表记 — 価値<br>**解析**：`価値` 是价值。`価地/加値/可知` 不表示商品价值。商品(しょうひん)の価値(かち)を高(たか)める＝提高商品价值。<br>**记忆点**：価値(かち) = 价值；搭配：価値を高める/価値がある；注意「値」不写成「地」

8. 事故の（げんいん）を調べる。
   1. 原因 2. 原員 3. 現因 4. 元員

> **答案：1** 原因<br>**考点**：汉字表记 — 原因<br>**解析**：`原因` 表示原因，和 `調べる` 搭配自然。事故(じこ)の原因(げんいん)を調(しら)べる＝调查事故原因。<br>**记忆点**：原因(げんいん) = 原因；搭配：原因を調べる/原因を分析する；注意「原」不写成「現」「元」

9. 彼は強い（せきにん）感を持っている。
   1. 責任 2. 積任 3. 関任 4. 責人

> **答案：1** 責任<br>**考点**：汉字表记 — 責任<br>**解析**：`責任感` 是固定词。彼(かれ)は強(つよ)い責任感(せきにんかん)を持(も)っている＝他有很强的责任感。<br>**记忆点**：責任(せきにん) = 责任；搭配：責任感を持つ/責任を取る；注意「責」不写成「積」「関」

10. データを（ぶんせき）する。
    1. 分析 2. 分折 3. 文析 4. 分石

> **答案：1** 分析<br>**考点**：汉字表记 — 分析<br>**解析**：`データを分析する` 是固定搭配。データを分析(ぶんせき)する＝分析数据。<br>**记忆点**：分析(ぶんせき) = 分析；搭配：データを分析する；注意「析」不写成「折」「石」

### 問3 词汇・语境（5题 × 1分 = 5分）

11. 新しい制度にはまだ改善の（　）がある。
    1. 余地 2. 余計 3. 余分 4. 余裕

> **答案：1** 余地<br>**考点**：固定搭配 — 余地がある<br>**解析**：`余地がある` 表示还有空间/可能性。新(あたら)しい制度(せいど)にはまだ改善(かいぜん)の余地(よち)がある＝新制度还有改善空间。<br>**记忆点**：余地(よち) = 空间/可能性；搭配：改善の余地がある；区别：余裕(よゆう)=充裕、余計(よけい)=多余、余分(よぶん)=多出的

12. 彼の説明は短いが、要点を（　）いる。
    1. 押さえて 2. 押して 3. 押し込んで 4. 押し出して

> **答案：1** 押さえて<br>**考点**：固定搭配 — 要点を押さえる<br>**解析**：`要点を押さえる` 是固定搭配，表示抓住要点。彼(かれ)の説明(せつめい)は短(みじか)いが、要点(ようてん)を押(お)さえている＝他的说明虽短，但抓住了重点。<br>**记忆点**：要点(ようてん)を押(お)さえる = 抓住要点；区别：押す=推、押し込む=塞进、押し出す=推出

13. 売上が下がった原因を（　）する。
    1. 分析 2. 分解 3. 分担 4. 分類

> **答案：1** 分析<br>**考点**：近义词辨析 — 分析/分解/分担/分類<br>**解析**：原因要"分析"，不是分解/分担/分类。売上(うりあげ)が下(さ)がった原因(げんいん)を分析(ぶんせき)する＝分析销售额下降的原因。<br>**记忆点**：分析(ぶんせき)=分析、分解(ぶんかい)=拆解、分担(ぶんたん)=分担、分類(ぶんるい)=分类；搭配：原因を分析する

14. この商品は若い世代を（　）にしている。
    1. 対象 2. 対策 3. 対立 4. 対応

> **答案：1** 対象<br>**考点**：固定搭配 — 〜を対象にする<br>**解析**：`〜を対象にする` 表示以……为对象。この商品(しょうひん)は若(わか)い世代(せだい)を対象(たいしょう)にしている＝这个商品面向年轻一代。<br>**记忆点**：対象(たいしょう) = 对象；搭配：〜を対象にする；区別：対策=对策、対立=对立、対応=对应/应对

15. 予想より参加者が多く、会場は（　）だった。
    1. 満員 2. 満点 3. 満足 4. 満開

> **答案：1** 満員<br>**考点**：近义词辨析 — 満員/満点/満足/満開<br>**解析**：`会場が満員` 表示场地坐满/人满。予想(よそう)より参加者(さんかしゃ)が多(おお)く、会場(かいじょう)は満員(まんいん)だった＝参加者比预想多，会场满员了。<br>**记忆点**：満員(まんいん)=满员、満点(まんてん)=满分、満足(まんぞく)=满足、満開(まんかい)=盛开；搭配：会場は満員だ

### 問4 言い換え（5题 × 1分 = 5分）

16. 彼は**相変わらず**忙しい。
    1. 以前と同じように 2. 急に 3. たぶん 4. ほとんど

> **答案：1** 以前と同じように<br>**考点**：言い換え — 相変わらず<br>**解析**：`相変わらず` 表示状态和以前一样。彼(かれ)は相変(あいか)わらず忙(いそが)しい＝他还是和以前一样忙。<br>**记忆点**：相変(あいか)わらず = 以前と同じように；用于状态持续不变

17. この説明は**あいまい**だ。
    1. はっきりしない 2. 詳しい 3. 正しい 4. 新しい

> **答案：1** はっきりしない<br>**考点**：言い換え — あいまい<br>**解析**：`あいまい` 是不明确、不清楚。この説明(せつめい)はあいまいだ＝这个说明很含糊。<br>**记忆点**：あいまい = はっきりしない（不明确）；搭配：あいまいな表現/あいまいな返事

18. 予定を**延期する**。
    1. 後に延ばす 2. 早く始める 3. 中止する 4. 増やす

> **答案：1** 後に延ばす<br>**考点**：言い換え — 延期する<br>**解析**：`延期する` 是把时间往后推。予定(よてい)を延期(えんき)する＝延期计划。<br>**记忆点**：延期(えんき)する = 後(あと)に延(の)ばす；区別：中止(ちゅうし)する=取消

19. 彼は**わざと**間違えた。
    1. 故意に 2. 偶然 3. 急いで 4. 静かに

> **答案：1** 故意に<br>**考点**：言い換え — わざと<br>**解析**：`わざと` 表示故意，等于 `故意に`。彼(かれ)はわざと間違(まちが)えた＝他故意弄错了。<br>**记忆点**：わざと = 故意(こい)に；反义：偶然(ぐうぜん)/うっかり

20. この仕事は**手間がかかる**。
    1. 時間と労力が必要だ 2. とても簡単だ 3. 費用が不要だ 4. 危険が少ない

> **答案：1** 時間と労力が必要だ<br>**考点**：言い換え — 手間がかかる<br>**解析**：`手間がかかる` 表示需要时间和劳力。この仕事(しごと)は手間(てま)がかかる＝这项工作费时间费力。<br>**记忆点**：手間(てま)がかかる = 時間と労力が必要；搭配：手間を省(はぶ)く＝省事

## 二、语法（40分）

> 语法选择 Q21-30：每题 2 分（20分）/ 文の組み立て Q31-35：每题 4 分（20分）

21. 彼は新人（　）、仕事の覚えが早い。
    1. にしては 2. につれて 3. によって 4. に反して

> **答案：1** にしては<br>**考点**：N2语法句型 — にしては<br>**解析**：`にしては` 表示"以……标准来看却……"，常带出乎意料。彼(かれ)は新人(しんじん)にしては、仕事(しごと)の覚(おぼ)えが早(はや)い＝以新人来说，他工作学得很快。<br>**记忆点**：にしては = 以……来说却……；接续：N/V普通形 + にしては；后项多为出乎意料的评价

22. 健康のため、甘いものを控える（　）。
    1. ようにしている 2. ことになっている 3. ところだった 4. わけにはいかない

> **答案：1** ようにしている<br>**考点**：N2语法句型 — ようにしている<br>**解析**：`ようにしている` 表示有意识地保持某习惯。健康(けんこう)のため、甘(あま)いものを控(ひか)えるようにしている＝为了健康，我尽量少吃甜食。<br>**记忆点**：ようにしている = 有意识地保持某习惯；区別：ことにしている=已做出决定、ことになっている=规定如此

23. 一度引き受けた（　）、途中でやめるわけにはいかない。
    1. 以上 2. ほど 3. ばかり 4. ところ

> **答案：1** 以上<br>**考点**：N2语法句型 — 〜た以上<br>**解析**：`以上` 表示"既然……就……"，后面常接义务/责任。一度(いちど)引(ひ)き受(う)けた以上(いじょう)、途中(とちゅう)でやめるわけにはいかない＝既然已经接下，就不能中途放弃。<br>**记忆点**：〜た以上(いじょう) = 既然……就……；接续：V た形 + 以上；后项多为义务/决心

24. この本は難しい言葉が多く、初心者には読み（　）。
    1. にくい 2. がたい 3. っぽい 4. かけだ

> **答案：1** にくい<br>**考点**：N2语法句型 — 〜にくい<br>**解析**：`〜にくい` 表示客观上难以做某事。この本(ほん)は難(むずか)しい言葉(ことば)が多(おお)く、初心者(しょしんしゃ)には読(よ)みにくい＝这本书难词多，对初学者来说难读。<br>**记忆点**：〜にくい = 客观上难以……；区別：〜がたい=心理上难以接受（信じがたい）；接续：Vます形 + にくい

25. 彼は忙しいと言っているが、実は時間がない（　）やる気がないのだ。
    1. というより 2. にしても 3. だけあって 4. に応じて

> **答案：1** というより<br>**考点**：N2语法句型 — というより<br>**解析**：`というより` 表示修正说法："与其说A，不如说B"。彼(かれ)は忙(いそが)しいと言(い)っているが、実(じつ)は時間(じかん)がないというよりやる気(き)がないのだ＝他说忙，其实与其说没时间，不如说没干劲。<br>**记忆点**：というより = 与其说A不如说B；用于修正前项的说法，后项更接近真实

26. 説明を聞けば聞く（　）、ますますわからなくなった。
    1. ほど 2. だけ 3. しか 4. こそ

> **答案：1** ほど<br>**考点**：N2语法句型 — 〜ば〜ほど<br>**解析**：`〜ば〜ほど` 表示"越……越……"。説明(せつめい)を聞(き)けば聞(き)くほど、ますますわからなくなった＝说明越听越不懂。<br>**记忆点**：〜ば〜ほど = 越……越……；接续：Vば形 + V辞書形 + ほど；搭配：考えれば考えるほど

27. この問題は、経験のある人（　）簡単ではない。
    1. にとっても 2. によって 3. に沿って 4. にわたって

> **答案：1** にとっても<br>**考点**：N2语法句型 — にとっても<br>**解析**：`にとって` 表示从某人的立场来看，`も` 加强"也"。この問題(もんだい)は、経験(けいけん)のある人(ひと)にとっても簡単(かんたん)ではない＝这个问题即使对有经验的人来说也不简单。<br>**记忆点**：にとって = 对……来说；接续：N + にとって；にとっても = 即使对……来说也

28. 雨がやんだ（　）、急に気温が下がった。
    1. とたん 2. うえで 3. ものなら 4. だけに

> **答案：1** とたん<br>**考点**：N2语法句型 — とたん<br>**解析**：`たとたん` 表示"刚一……就……"，后面多是突发变化。雨(あめ)がやんだとたん、急(きゅう)に気温(きおん)が下(さ)がった＝雨刚停，气温就突然下降了。<br>**记忆点**：〜たとたん(に) = 刚一……就……；接续：V た形 + とたん；后项为意外/突发事件

29. 彼は約束を破った（　）、理由を説明しようともしない。
    1. ばかりか 2. につれて 3. に限り 4. として

> **答案：1** ばかりか<br>**考点**：N2语法句型 — ばかりか<br>**解析**：`ばかりか` 表示"不仅……而且还……"，后项更进一步。彼(かれ)は約束(やくそく)を破(やぶ)ったばかりか、理由(りゆう)を説明(せつめい)しようともしない＝他不仅违背约定，甚至连理由都不解释。<br>**记忆点**：ばかりか = 不仅……而且还……；接续：V普通形/N + ばかりか；后项程度更甚

30. 申し込みは明日までですので、参加希望の方は早めに（　）。
    1. お申し込みください 2. 申し込まれました 3. 申し込んでしまいました 4. 申し込むところでした

> **答案：1** お申し込みください<br>**考点**：敬语表达 — お〜ください<br>**解析**：`お申し込みください` 是礼貌请求表达，符合通知文体。申(もう)し込(こ)みは明日(あした)までですので、参加希望(さんかきぼう)の方(かた)は早(はや)めにお申(もう)し込(こ)みください＝报名截止到明天，希望参加者请尽早报名。<br>**记忆点**：お〜ください = 请……（尊敬语）；构造：お + Vます形 + ください；搭配：お申し込みください/お待ちください

### 問 文の組み立て（5题 × 4分 = 20分）

31. このサービスは ___ ___ ★ ___ 改善された。
    1. 利用者の声を 2. もとに 3. 何度も 4. して

> **答案：★ = 4** して<br>**考点**：语序组合 — 〜をもとにして<br>**解析**：正序 `利用者の声を / もとに / して / 何度も`，星号是 `して`。このサービスは利用者(りようしゃ)の声(こえ)をもとにして何度(なんど)も改善(かいぜん)された＝这项服务以用户声音为基础，多次改进。<br>**记忆点**：〜をもとにして = 以……为基础；接续：N + をもとに(して)；搭配：データをもとにする

32. 彼の話は ___ ___ ★ ___。
    1. あまりに 2. 現実 3. 離れしていて 4. 信じがたい
