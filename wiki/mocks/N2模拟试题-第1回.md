## 三、读解

### 短文1

最近、社内で「集中時間」を設ける会社が増えている。午前中の二時間は会議や電話を入れず、社員が自分の作業に集中できるようにする制度だ。導入した会社では、作業効率が上がっただけでなく、残業時間も減ったという。

36. この制度の目的は何か。
   1. 会議の回数を増やすこと
   2. 社員が作業に集中できる時間を作ること
   3. 電話対応を専門部署に任せること
   4. 残業代を増やすこと

37. 導入後の変化として合っているものはどれか。
   1. 午前中の仕事がなくなった
   2. 会議が長くなった
   3. 作業効率が上がった
   4. 社員数が増えた

### 短文2

本を選ぶとき、評判だけを頼りにすると失敗することがある。多くの人に評価されている本でも、自分の目的に合わなければ役に立たない。大切なのは、読む前に「何を知りたいのか」をはっきりさせることだ。

38. 筆者が言いたいことは何か。
   1. 評判の高い本は必ず読むべきだ
   2. 本は安いものから選ぶべきだ
   3. 目的を明確にして本を選ぶべきだ
   4. 多くの本を同時に読むべきだ

39. 評判だけで選ぶとどうなることがあるか。
   1. 目的に合わない本を選ぶ
   2. 読む速度が上がる
   3. 内容を忘れにくくなる
   4. 必ず役に立つ本に出会う

近年、地方の商店街では空き店舗を活用した取り組みが行われている。若者が短期間だけ店を開ける「チャレンジショップ」や、地域住民が自由に使える交流スペースなどである。これらは売り上げだけを目的にしているわけではない。人が集まる機会を増やし、地域への関心を取り戻すことも重要な狙いである。

40. 空き店舗の活用例として本文にあるものはどれか。
   1. 大型スーパー
   2. チャレンジショップ

## 四、听解

> 做题时先读问题，再让别人或 TTS 朗读脚本。自己做时可以遮住脚本。

44. 男の人はこれから何をしますか。  
脚本：女「会議室の予約、もうしましたか。」男「すみません、まだです。資料の印刷は終わりました。」女「では、先に会議室をお願いします。」  
1. 資料を印刷する 2. 会議室を予約する 3. 会議を中止する 4. 女の人に電話する

45. 女の人はなぜ遅れますか。  
脚本：男「もう駅に着いた？」女「ごめん、電車が止まっていて。別の線に乗り換えるから、十分钟くらい遅れる。」  
1. 道に迷ったから 2. 電車が止まったから 3. 切符をなくしたから 4. 約束を忘れたから

46. 男の人は何を選びますか。  
脚本：店員「黒は売り切れですが、青と白ならございます。」男「白は汚れが目立ちそうですね。じゃあ青にします。」  
1. 黒 2. 青 3. 白 4. 赤

47. 女の人の意見はどれですか。  
脚本：男「この案、すぐ始めてもいいと思う？」女「方向性はいいけど、費用の計算がまだ甘いと思う。そこを確認してからにしたほうがいい。」  
1. すぐ始めるべきだ 2. 案をやめるべきだ 3. 費用を確認してから始めるべきだ 4. 方向性が悪い

48. 男の人は何を忘れましたか。  
脚本：女「出張の準備できた？」男「ホテルの予約も新幹線の切符も大丈夫。あ、会社に出す申請書をまだ出していない。」  
1. ホテル予約 2. 新幹線の切符 3. 申請書 4. 名刺

49. 女の人は何を頼みますか。  
脚本：女「この表、数字は合っているんですが、見出しが少しわかりにくいので直してもらえますか。」男「はい、今日中に直します。」  
1. 数字を直す 2. 見出しを直す 3. 表を削除する 4. 明日提出する

50. 男の人は週末どうしますか。  
脚本：女「週末の山登り、行けそう？」男「天気がよければ行くつもりだったけど、風邪気味だから今回はやめておくよ。」  
1. 山に行く 2. 家で休む 3. 仕事に行く 4. 友達を誘う
