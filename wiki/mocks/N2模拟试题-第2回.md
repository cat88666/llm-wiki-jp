## 三、读解

### 短文1

オンライン会議は便利だが、すべての会議をオンラインにすればよいわけではない。短い確認や情報共有には向いているが、新しいアイデアを出し合う場面では、対面のほうが意見が出やすいこともある。

36. 筆者の考えに合うものはどれか。 1. 会議はすべて対面にすべきだ 2. 目的によって形式を選ぶべきだ 3. オンライン会議は役に立たない 4. 情報共有は対面がよい
37. オンライン会議に向いているものはどれか。 1. 短い確認 2. 新商品の開発合宿 3. 長期計画の議論 4. 交流会

### 短文2

語彙を覚えるとき、単語だけを何度も読むより、短い文の中で覚えたほうが使える知識になりやすい。単語は意味だけでなく、どんな言葉と一緒に使われるかが重要だからである。

38. 筆者は語彙学習で何が重要だと言っているか。 1. 単語を一つずつ書くこと 2. 文の中で使い方も覚えること 3. 辞書を毎日読むこと 4. 長い文章だけ読むこと
39. なぜ文の中で覚えるとよいのか。 1. 発音が不要になるから 2. 漢字を書かなくてよいから 3. 一緒に使う言葉がわかるから 4. 文法を勉強しなくてよいから

### 中文

地方の大学では、地域企業と協力して授業を行う例が増えている。学生は実際の課題に取り組むことで、教室だけでは得られない経験を積むことができる。一方、企業にとっても若い世代の視点を知る機会になる。ただし、教育と企業活動の目的は同じではないため、大学側が学習目的を明確にしておく必要がある。

40. 学生にとっての利点は何か。 1. 授業料が安くなる 2. 実際の課題に取り組める 3. 必ず就職できる 4. 試験がなくなる

## 四、听解

44. 女の人は何を買いますか。  
脚本：店員「こちらは軽いですが容量が小さいです。こちらは少し重いですが、たくさん入ります。」女「旅行用なので、重くても大きいほうにします。」  
1. 軽くて小さいかばん 2. 重くて大きいかばん 3. 軽くて大きいかばん 4. 小さい財布

45. 男の人はなぜ資料を直しますか。  
脚本：女「内容はいいんですが、グラフの数字が本文と合っていません。」男「わかりました。数字を確認して直します。」  
1. 内容が古いから 2. 数字が合わないから 3. グラフが多すぎるから 4. 本文が短いから

46. 会議はいつになりますか。  
脚本：男「火曜は部長が出張です。水曜の午前はどうですか。」女「午前は別件があります。午後なら大丈夫です。」男「では水曜午後にしましょう。」  
1. 火曜午前 2. 火曜午後 3. 水曜午前 4. 水曜午後

47. 男の人は何に困っていますか。  
脚本：男「新しいソフト、機能は多いんですが、どこに何があるかわかりにくくて。」女「最初は操作に慣れるまで時間がかかりますよね。」  
1. 値段が高いこと 2. 操作がわかりにくいこと 3. 機能が少ないこと 4. 動かないこと

48. 女の人は週末何をしますか。  
脚本：男「映画に行かない？」女「行きたいけど、日曜に試験があるから、今週は勉強する。」  
1. 映画を見る 2. 試験を受ける 3. 勉強する 4. 旅行する

49. 男の人は何を先にしますか。  
脚本：女「メールの返信と請求書の作成、どちらを先にしますか。」男「請求書は今日中で大丈夫ですよね。まず急ぎのメールを返します。」  
1. 請求書を作る 2. メールを返す 3. 会議に出る 4. 電話する

50. 女の人の考えはどれですか。  
脚本：男「広告をもっと増やせば売れると思うんだ。」女「それも大事だけど、まず商品の説明ページをわかりやすくしたほうがいいと思う。」  
1. 広告をやめるべきだ 2. 説明ページを先に改善すべきだ 3. 商品を減らすべきだ 4. 値段を上げるべきだ
