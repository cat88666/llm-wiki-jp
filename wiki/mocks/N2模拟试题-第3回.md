## 三、读解

### 短文1

メモを取る目的は、聞いたことをすべて書くことではない。大切なのは、後で見返したときに内容を思い出せるように、要点を整理して残すことである。

36. 筆者によると、メモで大切なことは何か。 1. すべて書くこと 2. 要点を整理して残すこと 3. きれいな字で書くこと 4. 長く書くこと
37. メモを見る目的は何か。 1. 内容を思い出すこと 2. 字の練習をすること 3. 他人に売ること 4. 時間を測ること

### 短文2

新しいことを始めるとき、最初から完璧を目指すと続かないことが多い。まずは小さく始め、続けながら少しずつ改善していくほうが、結果的に長く続きやすい。

38. 筆者がすすめている方法はどれか。 1. 最初から完璧にする 2. 小さく始めて改善する 3. 失敗したらすぐやめる 4. 計画だけを作る
39. 完璧を目指しすぎるとどうなることが多いか。 1. 続かない 2. 早く終わる 3. 費用が減る 4. 人が増える

### 中文

企業の研修では、知識を一方的に教えるだけでなく、参加者同士が話し合う時間を設けることが増えている。話し合いによって、自分とは異なる考え方に触れられるからだ。ただし、自由に話すだけでは学びが浅くなることもある。そのため、研修の目的に合ったテーマを用意し、最後に内容を整理することが重要である。

40. 話し合いの利点は何か。 1. 異なる考え方に触れられる 2. 研修時間を短くできる 3. 知識が不要になる 4. 講師が不要になる

## 四、听解

44. 男の人はなぜ店に戻りますか。  
脚本：男「さっき買ったシャツ、サイズを間違えたみたい。」女「レシートがあれば交換できるよ。」男「じゃあ、今から行ってくる。」  
1. 返金してもらうため 2. サイズを交換するため 3. 新しい店を探すため 4. レシートをもらうため

45. 女の人は何を心配していますか。  
脚本：男「発表の準備、順調？」女「内容はまとまったけど、時間内に終わるかどうかが心配。」  
1. 内容が足りないこと 2. 時間内に終わるかどうか 3. 資料がないこと 4. 会場が狭いこと

46. 男の人は何をすすめていますか。  
脚本：女「最近眠れなくて。」男「寝る前にスマホを見るのをやめてみたら？僕はそれで少しよくなったよ。」  
1. スマホを見ること 2. 寝る前に運動すること 3. 寝る前のスマホをやめること 4. 薬を飲むこと

47. 会議資料はどうなりますか。  
脚本：女「資料、全部印刷しますか。」男「人数が多いので、紙ではなくデータで配りましょう。」  
1. 全部印刷する 2. 一部だけ印刷する 3. データで配る 4. 配らない

48. 女の人はなぜ注文を変えますか。  
脚本：店員「本日のランチは魚料理ですが、少し時間がかかります。」女「急いでいるので、すぐ出るカレーにします。」  
1. 魚が嫌いだから 2. 急いでいるから 3. カレーが安いから 4. 注文を忘れたから

49. 男の人はどこで待ちますか。  
脚本：女「駅の改札は混んでいるから、本屋の前で待ち合わせにしない？」男「わかった。本屋の前ね。」  
1. 改札 2. 本屋の前 3. カフェ 4. バス停

50. 女の人の意見はどれですか。  
脚本：男「新しい机、安いほうでいいかな。」女「毎日使うものだから、少し高くても疲れにくいものを選んだほうがいいと思う。」  
1. 安い机が一番よい 2. 高くても使いやすい机がよい 3. 机は買わないほうがよい 4. 中古を探すべきだ
