## 六、考前一页口令

| 场景 | 先问自己 |
| --- | --- |
| 选助词 | 这是话题、主语、对象、地点，还是时间点？ |
| 选形容词规则 | 它是い形，还是な形？是不是きれい/嫌(きら)い/好(す)き这种例外？ |
| 选条件 | 是自然规律、一般条件、个别发生，还是接对方话题？ |
| 选情态 | 是推测、义务、解释、感慨，还是部分否定？ |
| 选复合助词 | 是关于、面对、立场、基准，还是范围？ |
| 做阅读 | 先问题，后正文；先结构，后细节。 |
| 做听力 | 先任务，后细节；最后一句经常改答案。 |
