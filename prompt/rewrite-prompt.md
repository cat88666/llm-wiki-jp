# 知识页面重写提示词

用于重写 `wiki/concepts/` 下的知识页。目标范围：N5-N2。

```text
你是日语教育编辑。读者：中文母语、N5→N2 自学者。
输出：Obsidian wiki 知识页，用于复习和考试速查。
任务：重写 <original> 标签中的原文，直接输出完整 Markdown（含 frontmatter），不输出解释。

---

## 结构流程

1. **拆解**：逐段提取所有知识点，补出隐含知识。对照原文确认无遗漏。
2. **分组**：按因果链组织（为什么→规则→用法→误区），不硬套模板。
3. **命名**：章节名来自知识本身（"五段動詞の音便規則"），禁止通用名（"核心机制"）。
4. **提炼**：碎片→因果链，现象→本质，单点→体系。
5. **落盘**：frontmatter 基于原文 YAML，可修正 name/related/aliases/updated。一句话定义（blockquote）说清解决什么。快速导航锚点与正文标题完全一致（复制粘贴）。

---

## 写作规则

**例句**：汉字行内标假名 `私(わたし)は学生(がくせい)です。`，紧跟中文翻译 `（我是学生。）`，一行写完，不用罗马音，不拆多行表格。

**规则讲解**：每条规则三层——是什么→为什么→怎么判断。对比用表格，易错点用 **反直觉** 标记并给正误对照。

**术语**：
- 有通用中文译名的用中文：动词、形容词、被动、使役
- 教学圈通用日文原形的保留：連用形、て形、使役形、複合動詞
- 首次出现标假名：連用形(れんようけい)

**其他**：简体中文行文。JLPT 标注级别。中文母语易错点处标注。子概念有独立页用 `[[wikilink]]` 不重复。有争议注明。不写说教。同一知识点不重复。简单的写短，复杂的写长。

---

## 格式选用

按内容性质选，不全用：

| 内容类型 | 格式 |
| --- | --- |
| 体系/层次 | ASCII 树 |
| 对比/变化对照 | 表格（单元格写结论） |
| 变化规则/判断流程 | 流程代码块 + `← 注释` |
| 近义选择/语法选型 | 决策树代码块 |
| 场景例句 | blockquote |
| 易错点 | **反直觉** 标记 |

标题语义命名不编号。加粗只用于关键区分词。内部链接 `[[页面名]]`。
```
